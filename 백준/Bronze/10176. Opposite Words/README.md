### 문제 설명

<p>One of the listeners of the NPR Sunday Puzzle show described the following puzzle: The word &quot;wizard&quot; has the peculiar property that its letters can be grouped in pairs,</p>

<ul>
	<li>A and Z</li>
	<li>B and Y</li>
	<li>C and X</li>
	<li>&hellip;</li>
	<li>M and N</li>
</ul>

<p>that are opposite each other in the alphabet. That is, A and Z are at opposite ends of the alphabet, C and X are three letters in from their respective ends, and M and N are 13 letters in from their respective ends. Can you name a well-known brand name in six letters that has this same property? Answer: La-Z-Boy (Notice the alphabetic opposites don&#39;t have to appear in opposite positions in the word)</p>

<p>Given a list of words you must determine which of the words are &ldquo;opposite&rdquo; words. Note that capitalization is not important when determining is a word is an &ldquo;opposite&rdquo; word.</p>

### 입력

<p>The first line of input will be a positive integer, n, indicating the number of problem sets that follow. Following the positive integer will be n text strings, one per line.</p>

### 출력

<p>For each problem set print &ldquo;Yes&rdquo; if the string is an &ldquo;opposite&rdquo; word and &ldquo;No&rdquo; if it is not an &ldquo;opposite&rdquo; word.</p>

### 제한

<ul>
	<li>1 &le; the length of text string &le; 15</li>
</ul>
