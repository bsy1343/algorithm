### 문제 설명

<p>There are many special irrational numbers in nature. One is the &ldquo;golden ratio&rdquo; represented by the Greek letter phi, &phi;, which has a value of</p>

<p style="text-align: center;">&phi; = (1 + &radic;5) / 2 = 1.61803399.....</p>

<p>Given 2 real numbers, determine whether their ratio is equal (or close to) the golden ratio. For this program, use &plusmn;1% as the tolerance.</p>

### 입력

<p>The first line will contain the number of data sets. Each data set consists of 2 real numbers (with or without decimals).</p>

### 출력

<p>Either print out &ldquo;golden&rdquo; or &ldquo;not&rdquo; for each data set.</p>
