### 문제 설명

<p>Gru has been trying to teach the minions English (they are born knowing only Minionese, which Lucy can&rsquo;t stand). He is currently stuck trying to teach them the difference between vowels and consonants, which the minions are always confused about, especially &ldquo;y&rdquo;, which they argue about endlessly (it is not a vowel). To help them practice, Gru starts a contest where the minions have to count the number of vowels in a word, i.e., given a word, they have to tell Gru what is the number of vowels in it. The winner gets ice cream, which the minions are crazy about. Kevin the Minion wants your help to cheat and eat a lot of ice cream.</p>

### 입력

<p>The first line in the test data file contains the number of test cases (&lt; 100). After that, each line contains one test case, a word, w, (provided as a String). You can assume that the words are provided in all lowercase letters (i.e., &ldquo;gru&rdquo;, not &ldquo;Gru&rdquo;).</p>

### 출력

<p>For each test case, you are to output the number of vowels in that word. The vowels are: &ldquo;a&rdquo;, &ldquo;e&rdquo;, &ldquo;i&rdquo;, &ldquo;o&rdquo;, and &ldquo;u&rdquo;. The exact format is shown below in the examples.</p>
