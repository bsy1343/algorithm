### 문제 설명

<p>두 수&nbsp;A와 B를 합치는 것은 A의 뒤에 B를 붙이는 것을 의미한다. 즉, 20과 30을 붙이면 2030이 된다.</p>

### 입력

<p>첫째 줄에 네 자연수 A, B, C, D가 주어진다. (1 &le; A, B, C, D &le; 1,000,000)</p>

### 출력

<p>A와 B를 붙인 수와 C와 D를 붙인 수의 합을 출력한다.</p>
