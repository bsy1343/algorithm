### 문제 설명

<p>Output the maximum area of a rectangle that can be inscribed into a circle of radius R. Rectangle&#39;s shorter side should not be longer than B.</p>

### 입력

<p>First line of the input contains an integer T (1 &lt;= T &lt;= 1000) - the number of test cases. Next T lines each contain two integers R and B (1 &lt;= R,B &lt;= 10000).</p>

### 출력

<p>For each test case print on a separate line the maximum area of a rectangle with a maximum shorter side B that can be inscribed into a circle of radius R. Your answer should be rounded to three digits after the decimal point (see sample output).</p>
