### 문제 설명

<p>A&times;B를 계산하다 지겨워진 형택이는 A&times;B를 새로운 방법으로 정의하려고 한다.</p>

<p>A에서 한 자리를 뽑고 &times; B에서 임의로 한 자리를 뽑아 곱한다.</p>

<p>의 가능한 모든 조합 (A가 n자리, B가 m자리 수라면 총 가능한 조합은 n&times;m개)을 더한 수로 정의하려고 한다.</p>

<p>예를 들어 121&times;34는</p>

<p>1&times;3 + 1&times;4 + 2&times;3 + 2&times;4 + 1&times;3 + 1&times;4 = 28</p>

<p>이 된다. 이러한 형택이의 곱셈 결과를 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 A와 B가 주어진다. 주어지는 두 수는 모두 10,000자리를 넘지 않는 음이 아닌 정수이다. 수가 0인 경우에는 0만 주어지며, 그 외의 경우 수는 0으로 시작하지 않는다.</p>

### 출력

<p>첫째 줄에 형택이의 곱셈 결과를 출력한다.</p>
