### 문제 설명

<p style="text-align:justify">Imagine que voc&ecirc; tenha uma m&aacute;quina do tempo que pode ser usada no m&aacute;ximo tr&ecirc;s vezes, e a cada uso da m&aacute;quina voc&ecirc; pode escolher voltar para o passado ou ir para o futuro. A m&aacute;quina possui tr&ecirc;s cr&eacute;ditos fixos; cada cr&eacute;dito representa uma certa quantidade de anos, e pode ser usado para ir essa quantidade de anos para o passado ou para o futuro. Voc&ecirc; pode fazer uma, duas ou tr&ecirc;s viagens, e cada um desses tr&ecirc;s cr&eacute;ditos pode ser usado uma vez apenas. Por exemplo, se os cr&eacute;ditos forem 5, 12 e 9, voc&ecirc; poderia decidir fazer duas viagens: ir 5 anos para o futuro e, depois, voltar 9 anos para o passado. Dessa forma, voc&ecirc; terminaria quatro anos no passado, em 2012. Tamb&eacute;m poderia fazer tr&ecirc;s viagens, todas indo para o futuro, usando os cr&eacute;ditos em qualquer ordem, terminando em 2042.</p>

<p style="text-align:justify">Neste problema, dados os valores dos tr&ecirc;s cr&eacute;ditos da m&aacute;quina, seu programa deve dizer se &eacute; ou n&atilde;o poss&iacute;vel viajar no tempo e voltar para o presente, fazendo pelo menos uma viagem e, no m&aacute;ximo, tr&ecirc;s viagens; sempre usando cada um dos tr&ecirc;s cr&eacute;ditos no m&aacute;ximo uma vez.</p>

### 입력

<p>A entrada consiste de uma linha contendo os valores dos tr&ecirc;s cr&eacute;ditos A, B e C.</p>

<p>Restri&ccedil;&otilde;es</p>

<ul>
	<li>1 &nbsp;&le; A, B, C&nbsp;&le; 1000</li>
</ul>

### 출력

<p>Seu programa deve imprimir uma linha contendo o caracter &ldquo;S&rdquo; se &eacute; poss&iacute;vel viajar e voltar para o presente, ou &ldquo;N&rdquo; caso contr&aacute;rio.</p>
