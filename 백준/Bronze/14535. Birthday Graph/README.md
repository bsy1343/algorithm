<p>Birthday graph can be display in many forms. It can a creative drawing consists of cupcakes, balloons, candles with names, or it can be in the form of simple bar chart to indicate the birthday frequency for the month.</p>

<p>Birthday graph apps will come handy to tabulate birthdates by month especially for a large group. Your task is to write a program that reads a list of birthdates and display the birthday graph as shown in the sample output below.</p>

### 입력

<p>The input consists of a few test cases. For each test case, the first line of input is a positive integer <em>N</em> (<em>N</em> &le; 100) which indicates the number of data in the test case. Each of the following <em>N</em> lines contains a valid date representing birthdays formatted as dd mm yyyy. Input is terminated by a test case where <em>N</em> is 0.</p>

### 출력

<p>For each test case, output a line in the format &quot;Case #x:&quot; where x is the case number (starting from 1), follow by the monthly birthday graph as shown in the sample output.</p>
