<p>Farmer John was performing his nightly bedtime reading duties for Bessie. &quot;Oh, this gives me a headache,&quot; moaned Bessie. &quot;But it&#39;s just simple number theory,&quot; replied FJ. &quot;Let&#39;s go over it again. The sigma function of a number is just the sum of the divisors of the number. So, for a number like 12, the divisors are 1, 2, 3, 4, 6, and 12. Summing them we get 28.&quot; &quot;That&#39;s all there is to it?&quot; asked Bessie. &quot;Yep.&quot; replied FJ. &quot;Perhaps someone will write a program to calculate the sigma function of an integer I (1 &lt;= I &lt;= 1,000,000).&quot;</p>

### 출력
