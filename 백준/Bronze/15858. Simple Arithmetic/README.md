### 문제 설명

<p>You are given three integers a, b, c (1 &le; a, b, c &le; 10<sup>9</sup>). Compute a&sdot;b/c, with an absolute precision of 10<sup>&minus;6</sup>.</p>

### 입력

<p>The first and only line of input contains the three integers a, b, c separated by a single space.</p>

### 출력

<p>Output a single floating point number. It must differ from a&sdot;b/c by at most 10<sup>&minus;6</sup> in absolute value, i.e., it should obey |x&minus;a&sdot;b/c| &le; 10<sup>&minus;6</sup>.</p>
