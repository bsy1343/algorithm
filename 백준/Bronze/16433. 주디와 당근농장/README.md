### 문제 설명

<p>주디는 오랜만에 부모님이 살고 계시는&nbsp;농장을 방문하여 일손을 돕기로 하였습니다.</p>

<p>주디가 할 일은 <em>N</em> &times; <em>N</em>&nbsp;격자 모양의 밭에 당근을 심는 일입니다. 각 칸에는 최대 하나의 당근을 심을 수 있는데 어떤 칸에 당근이 심겨&nbsp;있으면 그 칸의 변을 공유하는 칸들에는 당근을 심을 수 없습니다.</p>

<p>주디는 이미 위치가 (<em>R</em>, <em>C</em>) 인 칸에&nbsp;당근을 하나 심었고 나머지 칸에도 당근을 심으려고 합니다. 주디가 최대한 많은 당근을 심었을 때 밭이 어떤 모양일지 알려주세요. 주디는 심을 당근을 무한히 가지고 있다고 가정합니다.</p>

### 입력

<p>첫 번째 줄에 <em>N</em>, <em>R</em>, <em>C</em>&nbsp;(2 &le;&nbsp;<em>N</em> &le; 99, 1 &le; <em>R</em>, <em>C</em> &le; <em>N</em>) 가&nbsp;주어집니다.</p>

### 출력

<p>첫 번째 줄부터 <em>N</em>번째 줄까지 각 줄에 길이가 <em>N</em>인 문자열을 출력합니다.</p>

<p><em>i</em>행 <em>j</em>열에 당근이 심어졌다면 <em>i</em>번째 줄 <em>j</em>번째 문자를 &#39;<code>v</code>&#39; 로 표시하며 심어지지 않았다면 &#39;<code>.</code>&#39; 로 표시합니다.</p>
