### 문제 설명

<p>좌표평면상의 점의 배열&nbsp;P = P<sub>1</sub>,&nbsp;P<sub>2</sub>, ⋯, P<sub>N</sub>와&nbsp;Q = Q<sub>1</sub>,&nbsp;Q<sub>2</sub>, ⋯, Q<sub>M</sub>이 있다. Q 배열 상의 한 점을 중심으로, P 배열 상의 모든 점을 포함하는 최소 넓이의 원의 반지름 중 최댓값을 구하시오.</p>

### 입력

<p>첫 줄에 N과 M이 주어진다. (1&nbsp;&le; N, M&nbsp;&le; 1000)</p>

<p>N개의 줄에 걸쳐 x와 y가 주어지며, 이는 P<sub>i</sub>&nbsp;= (x,y)라는 뜻이다. (-10<sup>6</sup>&nbsp;&le; x,y&nbsp;&le; 10<sup>6</sup>)</p>

<p>M개의 줄에 걸쳐 x와 y가 주어지며, 이는 Q<sub>i</sub>&nbsp;= (x,y)라는 뜻이다. (-10<sup>6</sup>&nbsp;&le; x,y&nbsp;&le; 10<sup>6</sup>)</p>

### 출력

<p>Q&nbsp;배열 상의 한 점을 중심으로,&nbsp;P&nbsp;배열 상 모든 점을 포함하는 최소 넓이의 원의 반지름&nbsp;중 최댓값의 제곱을 출력한다.</p>
