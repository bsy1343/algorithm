### 문제 설명

<p>전북대학교 컴퓨터공학부 신입생인 시현이는 공대 필수 교양인 기초물리를 수강중이다.</p>

<p>공부를 열심히 하는 시현이는 물리 문제집를 풀다가 다음과 같은 문제를 만났다.</p>

<p style="text-align: center;"><img alt="" src="%EB%B0%B1%EC%A4%80/Bronze/17248.%E2%80%85%EB%AC%BC%EB%A6%AC%E2%80%85%EA%B3%B5%EB%B6%80/852c9c06.png" data-original-src="https://upload.acmicpc.net/1d022f14-a0cc-45a6-94fc-e486a263d7bf/-/preview/" onerror="this.onerror=null;this.src=this.dataset.originalSrc" /></p>

<p>평소 물리를 좋아하던 시현이는 ㄱ, ㄴ번은 단숨에 알았지만, ㄷ번을 풀 수 없어 절망에 빠져 있다.</p>

<p>절망에 빠져있는 시현이를 도와주도록 하자.</p>

### 입력

<p>첫째 줄에 테스트케이스&nbsp;T를 입력한다.&nbsp;(1 &le; T &le; 100)</p>

<p>다음 줄부터 각 테스트케이스마다 자동차 A와 자동차 B의 속력&nbsp;X,&nbsp;Y,&nbsp;그리고&nbsp;자동차 A의 가속도&nbsp;Z가 주어진다. (각각의 입력은 띄어쓰기로 구분한다.)</p>

<p>단,&nbsp;0 &le; X &lt; Y &le; 10,000이고,&nbsp;0 &lt;&nbsp;Z &le; 10,000이다.&nbsp;</p>

### 출력

<p>각 테스트케이스마다 자동차 A가&nbsp;자동차 B와&nbsp;속력이 같거나 커지는 시간(초)을 출력한다.</p>
