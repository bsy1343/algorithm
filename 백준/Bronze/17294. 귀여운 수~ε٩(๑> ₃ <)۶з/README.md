### 문제 설명

<p>욱제는 귀여운 수~&epsilon;٩(๑&gt; ₃ &lt;)۶з를 좋아한다. 귀여운 수~&epsilon;٩(๑&gt; ₃ &lt;)۶з는 수를 이루는 각 자릿수가 등차수열[*]을 이루는 수이다. 당신은 욱제한테 귀여운 수~&epsilon;٩(๑&gt; ₃ &lt;)۶з 하나를 선물해주고 싶다. 수 하나가 주어졌을 때 이 수가 귀여운지 판단하는 프로그램을 짜 보자.</p>

<p>[*] 등차수열을 이루는 숫자들은, 왼쪽에서 오른쪽으로 가면서&nbsp;일정한 크기(0일 수도 있음)만큼&nbsp;커지거나 일정한 크기만큼 작아진다. 뭔지 모르겠으면 예제를 보고 알아보자. ㅋㅋ!</p>

### 입력

<p>정수 k(1 &le; k &le; 10<sup>18</sup>)가&nbsp;주어진다. 입력은 0으로 시작하지 않는다.</p>

### 출력

<p>수가 귀여우면 &quot;<span class="emoji">◝(⑅&bull;ᴗ&bull;⑅)◜..&deg;♡</span> 뀌요미!!&quot;를, 귀엽지 않으면 &quot;흥칫뿡!! &lt;(￣ ﹌ ￣)&gt;&quot;를 따옴표 없이 출력한다.</p>
