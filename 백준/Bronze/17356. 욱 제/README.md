### 문제 설명

<p>오늘은 운명의 라이벌 권욱과 권제가 숙명의 대결을 펼치는 날이다. 대결 종목은 바로 2019년 천하제일 코딩대회 본선 문제 중 하나인 &quot;<a href="https://www.acmicpc.net/contest/problem/438/8">욱 제</a>&quot;를 더 빨리 푸는 것이다.</p>

<p>욱과 제는 &quot;욱 제&quot;를 빨리 푸는 능력, 일명 &quot;욱제력&quot;을 가지고 있다. 이 욱제력은 하나의 정수로 수치화할 수 있다. 욱의 욱제력이 A, 제의 욱제력이 B일 때, M을 $\frac{B-A}{400}$이라고 하자. 이때 욱이 제를 이길 확률은 $\frac{1}{1+10^M}$이다. 이 값은 얼마일까?</p>

### 입력

<p>욱의 욱제력과 제의 욱제력이 한 줄에 주어진다. 욱제력은 0 이상 5,000 이하의 정수이다.</p>

### 출력

<p>욱이 제를 이길 확률을 출력한다. 정답과의 절대오차 또는 상대오차가 10<sup>-4</sup> 이하이면 정답으로 인정한다.</p>

### 힌트

<p>소수의 거듭제곱을 계산하려면</p>

<ul>
	<li>C, C++의 경우 pow 함수를 사용하면 된다.</li>
	<li>Java의 경우 Math.pow 함수를 사용하면 된다.</li>
	<li>Python의 경우 ** 연산자를 사용하면 된다.</li>
</ul>
