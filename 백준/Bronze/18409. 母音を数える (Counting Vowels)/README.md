<var>S</var></pre>

### 출력

<p><var>S</var>&nbsp;のうち母音字の個数，つまり&nbsp;<code>a</code>，<code>i</code>，<code>u</code>，<code>e</code>，<code>o</code>&nbsp;の個数の総和を出力せよ．</p>

### 제한

<ul>
	<li><var>1 ≦ N ≦ 50</var>.</li>
