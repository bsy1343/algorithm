### 문제 설명

<p>JOI 君は情報の試験を&nbsp;<var>3</var>&nbsp;回受けた．試験の点数はすべて&nbsp;<var>0</var>&nbsp;以上&nbsp;<var>100</var>&nbsp;以下の整数である．</p>

<p>JOI 君の成績は&nbsp;<var>3</var>&nbsp;回の試験の点数のうち高い方から&nbsp;<var>2</var>&nbsp;つを足し合わせた合計によって決まる．</p>

<p><var>3</var>&nbsp;回の試験の点数&nbsp;<var>A, B, C</var>&nbsp;が与えられたとき，<var>3</var>&nbsp;回の試験の点数のうち高い方から&nbsp;<var>2</var>&nbsp;つを足し合わせた合計を出力するプログラムを作成せよ．</p>

### 입력

<p>入力は以下の形式で標準入力から与えられる．</p>

<pre>
<var>A</var> <var>B</var> <var>C</var></pre>

### 출력

<p><var>3</var>&nbsp;回の試験の点数のうち高い方から&nbsp;<var>2</var>&nbsp;つを足し合わせた合計を&nbsp;<var>1</var>&nbsp;行で出力せよ．</p>

### 제한

<ul>
	<li><var>0 ≦ A ≦ 100</var>．</li>
	<li><var>0 ≦ B ≦ 100</var>．</li>
	<li><var>0 ≦ C ≦ 100</var>．</li>
	<li>入力はすべて整数である．</li>
</ul>
