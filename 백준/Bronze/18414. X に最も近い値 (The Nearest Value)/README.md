### 문제 설명

<p>整数&nbsp;<var>X, L, R</var>&nbsp;が与えられる.&nbsp;<var>L</var>&nbsp;以上&nbsp;<var>R</var>&nbsp;以下の整数のうち，<var>X</var>&nbsp;との差の絶対値が最も小さいものを出力せよ．そのような整数はちょうど&nbsp;<var>1</var>&nbsp;つだけ存在することが証明できる．</p>

### 입력

<p>入力は以下の形式で標準入力から与えられる．</p>

<pre>
<var>X</var> <var>L</var> <var>R</var></pre>

### 출력

<p><var>L</var>&nbsp;以上&nbsp;<var>R</var>&nbsp;以下の整数のうち，<var>X</var>&nbsp;との差の絶対値が最も小さいものを出力せよ．</p>

### 제한

<ul>
	<li><var>1 ≦ X ≦ 100 000</var>．</li>
	<li><var>1 ≦ L ≦ R ≦ 100 000</var>．</li>
</ul>
