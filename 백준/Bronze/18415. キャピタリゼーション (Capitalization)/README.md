### 문제 설명

<p>JOI 君は長さ&nbsp;<var>N</var>&nbsp;の文字列&nbsp;<var>S</var>&nbsp;を見つけた．<var>S</var>&nbsp;に含まれる文字はすべて英小文字である．</p>

<p>JOI 君はこの文字列から自分の名前である&nbsp;<code>joi</code>&nbsp;が連続している部分を先頭から順にすべて探しだすことにした．そして&nbsp;<code>joi</code>&nbsp;を見つけるたびに，強調のためにそれを大文字の&nbsp;<code>JOI</code>&nbsp;に置き換えることにした．</p>

<p>文字列&nbsp;<var>S</var>&nbsp;が与えられたとき，<var>S</var>&nbsp;に含まれる&nbsp;<code>joi</code>&nbsp;をすべて&nbsp;<code>JOI</code>&nbsp;に置き換えた文字列を出力するプログラムを作成せよ．</p>

### 입력

<p>入力は以下の形式で標準入力から与えられる．</p>

<pre>
<var>N</var>
<var>S</var></pre>

### 출력

<p><var>S</var>&nbsp;に含まれる&nbsp;<code>joi</code>&nbsp;をすべて&nbsp;<code>JOI</code>&nbsp;に置き換えた文字列を&nbsp;<var>1</var>&nbsp;行で出力せよ．</p>

### 제한

<ul>
	<li><var>3 ≦ N ≦ 100</var>．</li>
	<li><var>S</var>&nbsp;は長さ&nbsp;<var>N</var>&nbsp;の文字列である．</li>
	<li><var>S</var>&nbsp;は英小文字からなる．</li>
</ul>
