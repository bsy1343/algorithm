### 문제 설명

<p>양의 정수 $N$이 주어질 때, $\displaystyle\frac{1}{2^N}$을 계산하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 정수 $N$이 주어진다.</p>

### 출력

<p>첫째 줄에 $\displaystyle\frac{1}{2^N}$의 정확한 값을 소수 형태로 출력한다. 단, 뒤에 불필요한 $0$을 덧붙여서는 안 된다.</p>

### 제한

<ul>
	<li>$1 &le; N &le; 250$</li>
</ul>
