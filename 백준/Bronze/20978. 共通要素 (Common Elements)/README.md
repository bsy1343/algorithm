### 문제 설명

<p>長さ&nbsp;<var>N</var>&nbsp;の整数列&nbsp;<var>A = (A<sub>1</sub>, A<sub>2</sub>, ..., A<sub>N</sub>)</var>&nbsp;と長さ&nbsp;<var>M</var>&nbsp;の整数列&nbsp;<var>B = (B<sub>1</sub>, B<sub>2</sub>, ..., B<sub>M</sub>)</var>&nbsp;が与えられる．<var>A</var>&nbsp;と&nbsp;<var>B</var>&nbsp;の両方に出現する整数をすべて，<var>1</var>&nbsp;個ずつ昇順で出力せよ．</p>

### 입력

<p>入力は以下の形式で標準入力から与えられる．</p>

<p><var>N</var>&nbsp;<var>M</var><br />
<var>A<sub>1</sub></var>&nbsp;<var>A<sub>2</sub></var>&nbsp;<var>...</var>&nbsp;<var>A<sub>N</sub></var><br />
<var>B<sub>1</sub></var>&nbsp;<var>B<sub>2</sub></var>&nbsp;<var>...</var>&nbsp;<var>B<sub>M</sub></var></p>

### 출력

<p><var>A</var>&nbsp;と&nbsp;<var>B</var>&nbsp;の両方に出現する整数をすべて，<var>1</var>&nbsp;個ずつ昇順で出力せよ．各整数は改行で区切ること．</p>

### 제한

<ul>
	<li><var>1 ≦ N ≦ 100</var>．</li>
	<li><var>1 ≦ M ≦ 100</var>．</li>
	<li><var>1 ≦ A<sub>i</sub>&nbsp;≦ 100</var>&nbsp;(<var>1 ≦ i ≦ N</var>)．</li>
	<li><var>1 ≦ B<sub>j</sub>&nbsp;≦ 100</var>&nbsp;(<var>1 ≦ j ≦ M</var>)．</li>
</ul>
