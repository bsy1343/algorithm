### 문제 설명

<p>Your friend has secretly picked $N$ consecutive positive integers between $1$ and $100$, and wants you to guess if their sum is even or odd.</p>

<p>If the sum must be even, output &#39;<code>Even</code>&#39;. &nbsp;If the sum must be odd, output &#39;<code>Odd</code>&#39;. &nbsp;If the sum could be even or could be odd, output &#39;<code>Either</code>&#39;.</p>

### 입력

<p>The input is a single integer $N$ with $1 \le N \le 10$.</p>

### 출력

<p>Output a single word. The word should be &#39;<code>Even</code>&#39;, &#39;<code>Odd</code>&#39;, or &#39;<code>Either</code>&#39;, according to the rules given earlier.</p>
