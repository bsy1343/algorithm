Васю заинтересовал вопрос: клеток какого цвета получилось больше &mdash; черного или белого.&nbsp;</p>

<p>Для того, чтобы выяснить это, он спросил у Ани, в какой цвет она раскрасила j-ю клетку в i-м ряду доски. По этой информации Вася попытался &nbsp;определить, клеток какого цвета на доске больше.&nbsp;&nbsp; &nbsp;&nbsp;</p>

<p>Требуется написать программу, которая по размерам доски и цвету j-й клетки в i-м ряду определит, клеток какого цвета на доске больше &mdash; черного или белого.</p>

### 입력

<p>Входной файл содержит пять целых чисел: m, n, i, j и c (1 &le; m, n &le; 10<sup>9</sup>, 1 &le; i &le; m, 1 &le; j &le; n, с = 0 или с = 1). Значение c = 0 означает, что j-я клетка в i-м ряду доски раскрашена в черный цвет, а значение c = 1 &mdash; в белый цвет.</p>

### 출력

<p>Выходной файл должен содержать одно из трех слов:</p>

<ul>
	<li>black, если черных клеток на доске больше,</li>
	<li>white, если белых клеток на доске больше,</li>
	<li>equal, если черных и белых клеток на доске поровну.</li>
</ul>
