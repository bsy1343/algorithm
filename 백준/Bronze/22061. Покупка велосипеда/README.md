### 문제 설명

<p>Сегодня Петя убирался в комнате и нашел под кроватью очень много мелочи &mdash; рублевых и двухрублевых монет. Внимательно все пересчитав, Петя понял, что у него есть&nbsp;<i>a</i>&nbsp;монет номиналом в один рубль и&nbsp;<i>b</i>&nbsp;монет номиналом в два рубля.</p>

<p>Петя очень давно хотел купить себе велосипед, поэтому сразу же пошел в магазин и узнал, что его заветное средство передвижения стоит&nbsp;<i>с</i>&nbsp;рублей. К сожалению, в магазине велосипедов проблемы с мелкими деньгами, а без сдачи Петя уходить не хочет. Поэтому теперь он задался вопросом: можно ли купить велосипед, заплатив при этом ровно&nbsp;<i>с</i>&nbsp;рублей?</p>

<p>Помогите Пете, скажите, можно ли, имея&nbsp;<i>a</i>&nbsp;рублевых монет и&nbsp;<i>b</i>&nbsp;двухрублевых, купить без сдачи велосипед стоимостью&nbsp;<i>с</i>&nbsp;рублей?</p>

### 입력

<p>Первая строка входных данных содержит одно число&nbsp;<i>t</i>&nbsp;(1&nbsp;&le;&nbsp;<i>t</i>&nbsp;&le;&nbsp;100000)&nbsp;&mdash; количество тестов. Следующие&nbsp;<i>t</i>&nbsp;строк содержат по тесту каждая. Каждый тест задается тремя целыми числами:&nbsp;<i>a</i>,&nbsp;<i>b</i>,&nbsp;<i>с</i>&nbsp;(0&nbsp;&le;&nbsp;<i>a</i>,&nbsp;<i>b</i>,&nbsp;<i>c</i>&nbsp;&le;&nbsp;10<sup>8</sup>)&nbsp;&mdash; количество рублевых, двухрублевых монет и стоимость велосипеда соответственно.</p>

### 출력

<p>Для каждого набора данных выведите единственную строку: &laquo;<code>YES</code>&raquo;, если можно купить велосипед без сдачи и &laquo;<code>NO</code>&raquo; в противном случае.</p>
