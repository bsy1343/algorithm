### 문제 설명

<p>В этой задаче упоминается популярная в последнее время игра 2048. Знание ее правил не требуется для решения задачи.</p>

<p>В свободное время Вася любит играть в популярную игру 2048. После того, как Вася догадался до правильной стратегии, ему наскучила эта игра и он решил сделать похожую игру с собственными правилами.</p>

<p>Игровое поле для Васиной игры имеет форму квадрата размером&nbsp;<i>n</i>&nbsp;&times;&nbsp;<i>n</i>, в некоторых клетках которого записаны числа. Будем называть игровое поле&nbsp;<em>хорошим</em>, если на поле есть свободная клетка, или есть две соседние по стороне клетки, в которых стоят одинаковые числа.</p>

<p>По заданному игровому полю определите, является ли оно хорошим.</p>

### 입력

<p>Входные данные содержат несколько тестовых примеров. Первая строка содержит число&nbsp;<i>T</i>&nbsp;&mdash; число тестовых примеров (1&nbsp;&le;&nbsp;<i>T</i>&nbsp;&le;&nbsp;100). Далее идет описание самих примеров в следующем формате.</p>

<p>Первая строка описания содержит одно натуральное число&nbsp;<i>n</i>&nbsp;(1&nbsp;&le;&nbsp;<i>n</i>&nbsp;&le;&nbsp;10) &mdash; размер игрового поля. Следующие&nbsp;<i>n</i>&nbsp;строк содержат по&nbsp;<i>n</i>&nbsp;неотрицательных целых чисел &mdash; описание игрового поля. Пустая клетка задается числом 0, заполненная клетка задается записанным в ней целым положительным числом от 1 до 1000.</p>

### 출력

<p>Выведите&nbsp;<i>T</i>&nbsp;строк. Для каждого тестового набора выведите &laquo;<code>YES</code>&raquo;, если поле является хорошим, либо &laquo;<code>NO</code>&raquo; в противном случае.</p>
