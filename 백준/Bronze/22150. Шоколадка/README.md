### 문제 설명

<p>У Игната праздник! И по этому поводу ему подарили шоколадку размером&nbsp;<i>n</i>&times;<i>n</i>&nbsp;долек. В разгаре веселья он положил эту шоколадку на стол и на время забыл о ней. Вернувшись за ней, он понял, что случилось ужасное! Его шоколадку кто-то сломал! И, мало того, что сломал, так ещё и мог съесть несколько долек. Помогите Игнату выяснить всю правду: ел ли кто-то его шоколадку?</p>

<p>Сломанная шоколадка представляет собой две половинки &mdash; левую и правую. Левая половинка выровнена по левому краю, правая &mdash; по правому. Каждая половинка состоит из&nbsp;<i>n</i>&nbsp;рядов. В&nbsp;<i>i</i>-м ряду левой половинки&nbsp;<i>l<sub>i</sub></i>&nbsp;долек, правой &mdash;&nbsp;<i>r<sub>i</sub></i>&nbsp;долек. Игнат считает, что шоколадку никто не ел, если из левой и правой половинок можно сложить целую шоколадку&nbsp;<i>n</i>&times;<i>n</i>, не поднимая их со стола и не переворачивая.</p>

<p>На картинке приведены шоколадки из первых двух запросов в примере.</p>

### 입력

<p>На вход подается несколько запросов. В первой строке задано число&nbsp;<i>t</i>&nbsp;(1&nbsp;&le;&nbsp;<i>t</i>&nbsp;&le;&nbsp;100) &mdash; количество запросов, на которые необходимо ответить.</p>

<p>Далее, каждый в своей строке, заданы сами запросы в формате: сначала&nbsp;<i>n</i>&nbsp;(2&nbsp;&le;&nbsp;<i>n</i>&nbsp;&le;&nbsp;100) &mdash; размер шоколадки, далее&nbsp;<i>n</i>&nbsp;пар чисел:&nbsp;<i>l</i><sub>1</sub>&nbsp;<i>r</i><sub>1</sub>,&nbsp;<i>l</i><sub>2</sub>&nbsp;<i>r</i><sub>2</sub>, ...,&nbsp;<i>l<sub>n</sub></i>&nbsp;<i>r<sub>n</sub></i>&nbsp;(1&nbsp;&le;&nbsp;<i>l<sub>i</sub></i>,&nbsp;<i>r<sub>i</sub></i>&nbsp;&lt; n,&nbsp;<i>l<sub>i</sub></i>&nbsp;+&nbsp;<i>r<sub>i</sub></i>&nbsp;&le;&nbsp;n), все числа целые.</p>

### 출력

<p>Для каждого запроса в отдельной строке выведите &laquo;<code>yes</code>&raquo;, если шоколадку кто-то ел, или &mdash; &laquo;<code>no</code>&raquo; в противном случае.</p>
