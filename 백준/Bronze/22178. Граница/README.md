### 문제 설명

<p>В международной политике важным понятием является граница между государствами. Нечеткое понимание сторонами того, где проходит граница, может привести к международным конфликтам и даже войнам.</p>

<p>В этой задаче ситуация обстоит несколько проще, так как у двух рассматриваемых в задаче государств есть четкое понимание, какая территория принадлежит какому из них.</p>

<p>Территория, занимаемая этими двумя государствами, представляет собой прямоугольник размером&nbsp;<i>h</i>&nbsp;на&nbsp;<i>w</i>&nbsp;километров, разбитый на квадраты со стороной в один километр. Каждый из этих квадратов полностью принадлежит либо первому государству, либо второму.</p>

<p>Необходимо определить длину границы между двумя государствами. Сторона единичного квадрата считается принадлежащей границе, если по одну сторону от нее лежит квадрат, принадлежащий первому государству, а по другую &mdash; принадлежащий второму.</p>

### 입력

<p>Первая строка содержит два целых числа:&nbsp;<i>w</i>&nbsp;и&nbsp;<i>h</i>&nbsp;&mdash; размеры прямоугольника в километрах (1 &le;&nbsp;<i>w</i>,&nbsp;<i>h</i>&nbsp;&le; 100). Далее следуют&nbsp;<i>h</i>&nbsp;строк, описывающих территорию. Каждая из них содержит&nbsp;<i>w</i>&nbsp;символов. Если символ равен&nbsp;<code>A</code>, то соответствующий единичный квадрат принадлежит первому государству, а если он равен&nbsp;<code>B</code>, то второму. Гарантируется, что каждому государству принадлежит хотя бы один квадрат.</p>

<p>Территории каждого из государств представляют собой связные области.</p>

### 출력

<p>Выведите одно целое число &mdash; длину границы между государствами в километрах.</p>
