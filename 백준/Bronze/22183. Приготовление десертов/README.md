### 문제 설명

<p>Приготовление десертов можно считать ремеслом, можно считать искусством, а можно считать и наукой. В этой задаче мы будем использовать третий из указанных подходов.</p>

<p>Одна из кофеен предлагает своим посетителям большой выбор десертов, каждый из которых состоит из кусочка пирога с начинкой, который сверху украшен шариком мороженого. При этом тесто пирога может быть приготовлено одним из&nbsp;<i>n</i>&nbsp;способов, существует&nbsp;<i>m</i>&nbsp;вариантов начинки и&nbsp;<i>k</i>&nbsp;сортов мороженого. К сожалению, не все типы теста сочетаются со всем начинками, не все начинки &mdash; со всеми сортами мороженого, и не все сорта мороженого &mdash; со всеми типами теста.</p>

<p>Ваша задача состоит в определении числа типов десертов, которые могут быть приготовлены так, чтобы все три ингредиента сочетались друг с другом.</p>

### 입력

<p>Первая строка содержит три целых числа:&nbsp;<i>n</i>,&nbsp;<i>m</i>&nbsp;и&nbsp;<i>k</i>&nbsp;(1&nbsp;&le;&nbsp;<i>n</i>,&nbsp;<i>m</i>,&nbsp;<i>k</i>&nbsp;&le;&nbsp;50).</p>

<p>Вторая строка содержит одно целое число&nbsp;<i>p</i>&nbsp;(0&nbsp;&le;&nbsp;<i>p</i>&nbsp;&le;&nbsp;200) &mdash; количество пар типов теста и начинок, не сочетающихся друг с другом. Каждая из последующих&nbsp;<i>p</i>&nbsp;строк содержит по два целых числа&nbsp;<i>a</i>&nbsp;и&nbsp;<i>b</i>&nbsp;(1&nbsp;&le;&nbsp;<i>a</i>&nbsp;&le;&nbsp;<i>n</i>, 1&nbsp;&le;&nbsp;<i>b</i>&nbsp;&le;&nbsp;<i>m</i>) &mdash; номера типа теста и начинки, которые не сочетаются друг с другом. Никакая пара несочетающихся типов теста и начинки не указана более одного раза.</p>

<p>Следующая строка содержит одно целое число&nbsp;<i>q</i>&nbsp;(0&nbsp;&le;&nbsp;<i>q</i>&nbsp;&le;&nbsp;200) &mdash; количество пар начинок и сортов мороженого, не сочетающихся друг с другом. Каждая из последующих&nbsp;<i>q</i>&nbsp;строк содержит по два целых числа&nbsp;<i>a</i>&nbsp;и&nbsp;<i>b</i>&nbsp;(1&nbsp;&le;&nbsp;<i>a</i>&nbsp;&le;&nbsp;<i>m</i>, 1&nbsp;&le;&nbsp;<i>b</i>&nbsp;&le;&nbsp;<i>k</i>) &mdash; номера начинки и сорта мороженого, которые не сочетаются друг с другом. Никакая пара несочетающихся начинки и сорта мороженого не указана более одного раза.</p>

<p>Следующая строка содержит одно целое число&nbsp;<i>r</i>&nbsp;(0&nbsp;&le;&nbsp;<i>r</i>&nbsp;&le;&nbsp;200) &mdash; количество пар типов теста и сортов мороженого, не сочетающихся друг с другом. Каждая из последующих&nbsp;<i>r</i>&nbsp;строк содержит по два целых числа&nbsp;<i>a</i>&nbsp;и&nbsp;<i>b</i>&nbsp;(1&nbsp;&le;&nbsp;<i>a</i>&nbsp;&le;&nbsp;<i>n</i>, 1&nbsp;&le;&nbsp;<i>b</i>&nbsp;&le;&nbsp;<i>k</i>) &mdash; номера типа теста и сорта мороженого, которые не сочетаются друг с другом. Никакая пара несочетающихся типа теста и сорта мороженого не указана более одного раза.</p>

### 출력

<p>Выведите одно число &mdash; число типов десертов, которые могут быть приготовлены так, чтобы все три ингредиента сочетались друг с другом.</p>
