### 문제 설명

<p>드립이는 &quot;driip&quot;으로 끝나는 문자열은 모두 귀엽다고 생각한다.</p>

<p>다음 문자열들은 드립이가 귀엽다고 생각하는 문자열들의 예시이다. &quot;dogdriip&quot;,&nbsp;&quot;catdriip&quot;, &quot;driip&quot;</p>

<p>다음 문자열들은 드립이가 귀엽다고 생각하지 않는 문자열들의 예시이다. &quot;dogdrip&quot;,&nbsp;&quot;driipcat&quot;,&nbsp;&quot;driiiiip&quot;</p>

<p>문자열이 주어지면, 드립이가 주어진 문자열을 귀엽다고 생각하는지 판단하는 프로그램을 작성하자.</p>

### 입력

<p>첫째 줄에 문자열 S(1 &le;&nbsp;|S| &le;&nbsp;1,000)가 주어진다. 주어지는 문자열은 영문 소문자로만 이루어져 있다.</p>

### 출력

<p>드립이가 생각하기에 주어진 문자열이 귀여우면 &quot;<code>cute</code>&quot;, 그렇지 않으면 &quot;<code>not cute</code>&quot;를 출력한다. (따옴표 제외)</p>
