### 문제 설명

<p>A 地点から B 地点に移動するのに&nbsp;<var>X</var>&nbsp;時間，B 地点から C 地点に移動するのに&nbsp;<var>Y</var>&nbsp;時間かかる．</p>

<p>A 地点から B 地点を経由して C 地点に移動するとき，<var>Z</var>&nbsp;時間&nbsp;<var>30</var>&nbsp;分以内に移動することができるか判定せよ．</p>

### 입력

<p>入力は以下の形式で標準入力から与えられる．</p>

<pre>
<var>X</var>
<var>Y</var>
<var>Z</var></pre>

### 출력

<p><var>Z</var>&nbsp;時間&nbsp;<var>30</var>&nbsp;分以内に移動することができるならば&nbsp;<var>1</var>&nbsp;を，そうでない場合は&nbsp;<var>0</var>&nbsp;を出力せよ．</p>

### 제한

<ul>
	<li><var>1 ≦ X ≦ 100</var>．</li>
	<li><var>1 ≦ Y ≦ 100</var>．</li>
	<li><var>1 ≦ Z ≦ 100</var>．</li>
	<li>入力される値はすべて整数である．</li>
</ul>
