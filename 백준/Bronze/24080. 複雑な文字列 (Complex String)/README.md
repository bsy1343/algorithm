### 문제 설명

<p>長さ&nbsp;<var>N</var>&nbsp;の文字列&nbsp;<var>S</var>&nbsp;が与えられる．<var>S</var>&nbsp;の各文字は&nbsp;<code>A</code>，<code>B</code>，<code>C</code>，<code>D</code>，<code>E</code>&nbsp;のいずれかである．</p>

<p><var>S</var>&nbsp;に&nbsp;<var>3</var>&nbsp;種類以上の文字が出現する場合は&nbsp;<code>Yes</code>&nbsp;を，そうでない場合は&nbsp;<code>No</code>&nbsp;を出力せよ．</p>

### 입력

<p>入力は以下の形式で標準入力から与えられる．</p>

<pre>
<var>N</var>
<var>S</var></pre>

### 출력

<p><var>S</var>&nbsp;に&nbsp;<var>3</var>&nbsp;種類以上の文字が出現する場合は&nbsp;<code>Yes</code>&nbsp;を，そうでない場合は&nbsp;<code>No</code>&nbsp;を出力せよ．</p>

### 제한

<ul>
	<li><var>1 ≦ N ≦ 100</var>．</li>
	<li><var>S</var>&nbsp;は長さ&nbsp;<var>N</var>&nbsp;の文字列である．</li>
	<li><var>S</var>&nbsp;の各文字は&nbsp;<code>A</code>，<code>B</code>，<code>C</code>，<code>D</code>，<code>E</code>&nbsp;のいずれかである．</li>
	<li><var>N</var>&nbsp;は整数である．</li>
</ul>
