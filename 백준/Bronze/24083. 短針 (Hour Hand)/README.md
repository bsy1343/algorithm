### 문제 설명

<p>JOI 高校の生徒である葵はアナログ時計を手に入れた．この時計には外周に沿って&nbsp;<var>12</var>&nbsp;個の目盛りがあり，時計回りに&nbsp;<var>1</var>&nbsp;から&nbsp;<var>12</var>&nbsp;までの番号が順に付けられている．</p>

<p>この時計の短針は時計回りに回っており，<var>1</var>&nbsp;時間で目盛り&nbsp;<var>1</var>&nbsp;つ分進む．</p>

<p>たった今，短針がある目盛りを指した．この目盛りの番号は&nbsp;<var>A</var>&nbsp;である．</p>

<p>この状態からちょうど&nbsp;<var>B</var>&nbsp;時間が経過したときに短針が指す目盛りの番号を出力せよ．</p>

### 입력

<p>入力は以下の形式で標準入力から与えられる．</p>

<pre>
<var>A</var>
<var>B</var></pre>

### 출력

<p>短針が&nbsp;<var>A</var>&nbsp;を指してからちょうど&nbsp;<var>B</var>&nbsp;時間が経過したときに短針が指す目盛りの番号を出力せよ．</p>

### 제한

<ul>
	<li><var>1 ≦ A ≦ 12</var>．</li>
	<li><var>1 ≦ B ≦ 100</var>．</li>
	<li>入力される値はすべて整数である．</li>
</ul>
