### 문제 설명

<p>長さ&nbsp;<var>N</var>&nbsp;の整数列&nbsp;<var>A = (A<sub>1</sub>, A<sub>2</sub>, &hellip;, A<sub>N</sub>)</var>&nbsp;が与えられる．</p>

<p><var>A</var>&nbsp;に出現する整数のうち，出現回数が最小である整数を出力せよ．ただし，そのような整数が複数考えられる場合は，考えられる整数のうち最も小さい整数を出力せよ．</p>

### 입력

<p>入力は以下の形式で標準入力から与えられる．</p>

<pre>
<var>N</var>
<var>A<sub>1</sub></var> <var>A<sub>2</sub></var> <var>&hellip;</var> <var>A<sub>N</sub></var></pre>

### 출력

<p><var>A</var>&nbsp;に出現する整数のうち，出現回数が最小である整数を出力せよ．ただし，そのような整数が複数考えられる場合は，考えられる整数のうち最も小さい整数を出力せよ．</p>

### 제한

<ul>
	<li><var>1 ≦ N ≦ 100</var>．</li>
	<li><var>1 ≦ A<sub>i</sub>&nbsp;≦ 2000</var>&nbsp;(<var>1 ≦ i ≦ N</var>)．</li>
	<li>入力される値はすべて整数である．</li>
</ul>
