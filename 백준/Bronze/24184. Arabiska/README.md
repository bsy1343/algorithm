### 문제 설명

<p>Det &auml;r ganska k&auml;nt att n&auml;r man skriver text p&aring; arabiska skriver man fr&aring;n h&ouml;ger till v&auml;nster. Det &auml;r d&auml;remot mindre v&auml;lk&auml;nt att man aldrig skriver ut korta vokaler.</p>

<p>Med vokaler menar vi en av bokst&auml;verna: <em>a,e,i,o,u,y</em>. En kort vokal defineras i detta problem som en vokal som f&ouml;ljs av tv&aring; eller fler konsonanter.</p>

<p>Skriv ett program som visar hur en mening skulle se ut om den skrevs p&aring; arabiska, d.v.s fr&aring;n h&ouml;ger till v&auml;nster och utel&auml;mnande alla korta vokaler fr&aring;n den ursprungliga meningen.</p>

### 입력

<p>Den f&ouml;rsta raden ineh&aring;ller ett heltal $N$ ($1 \le N \le 5$), antalet ord i meningen.</p>

<p>Den andra raden inneh&aring;ller $N$ ord, d&auml;r varje ord best&aring;r av maximalt 10 gemena bokst&auml;ver fr&aring;n det latinska alfabetet (<code>a</code> till <code>z</code>).</p>

### 출력

<p>Skriv ut $N$ ord -- hur meningen skulle skrivas p&aring; arabiska.</p>
