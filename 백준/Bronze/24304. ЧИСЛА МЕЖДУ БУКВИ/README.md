### 문제 설명

<p>Напишете програма nums_between_chars със следния вход и изход:</p>

### 입력

<p>На първия ред на стандартния вход е записано едно цяло число N, а на втория ред &ndash; поредица от точно четири знака, без разделящи ги интервали. Знаците в поредицата са букви от латинската азбука и/или цифри.</p>

### 출력

<p>На отделни редове на стандартния изход програмата трябва да изведе произведението на числото N с всяко открито в поредицата число.</p>

<p>Ако поредицата съдържа само букви, програмата да отпечата &ldquo;N/A&rdquo; (без кавичките!)</p>

### 제한

<ul>
	<li>0 &le; N &lt; 100</li>
</ul>
