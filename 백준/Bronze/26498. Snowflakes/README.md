### 문제 설명

<p>Snowflakes can be symmetrical both vertically and horizontally. Snowflakes that aren&rsquo;t symmetrical are useless. A snowflake is essentially a two dimensional array of features. A feature is a single digit between 0 and 9. Horizontal symmetry occurs if you flip the snowflake horizontally. It is still the same snowflake. Likewise, vertical symmetry is defined in the same way. A snowflake is &ldquo;Beautiful&rdquo; if its horizontally symmetrical, &ldquo;Graceful&rdquo; if its vertically symmetrical, and &ldquo;Magnificent&rdquo; if it is both. Otherwise it is &ldquo;Useless&rdquo;. For each snowflake, print what kind of snowflake it is.</p>

### 입력

<p>The first line of data contains n, the number of inputs. Each input starts with a single integer k, k &le; 20, denoting the dimension of the square snowflake. Then k lines follow with each line containing a string of length k denoting the digits in the snowflake.</p>

### 출력

<p>For each input. print the type of snowflake.</p>
