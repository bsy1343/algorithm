### 문제 설명

<p>Informatycy to też ludzie, więc jak wszyscy potrzebują trochę miłości. Bajtek właśnie postanowił wyznać miłość pięknej Bajtolinie i w tym celu zamierza narysować jej N serduszek w ASCII art złożonych ze znak&oacute;w małpy (<code>@</code>).</p>

<p>Jedno serduszko wygląda następująco:</p>

<pre>
 @@@   @@@ 
@   @ @   @
@    @    @
@         @
 @       @ 
  @     @  
   @   @   
    @ @    
     @     
</pre>

<p>Bajtek chciałby, żeby serduszka były wypisane jedno pod drugim. Pom&oacute;ż mu!</p>

<p>Napisz program, kt&oacute;ry: wczyta liczbę N &ndash; liczbę serduszek, kt&oacute;re Bajtek chce narysować Bajtolinie i wypisze na standardowe wyjście N serduszek w formacie opisanym powyżej.</p>

### 입력

<p>W pierwszym (jedynym) wierszu wejścia znajduje się jedna liczba całkowita dodatnia N, określająca liczbę serduszek do wypisania. Liczba ta będzie r&oacute;wna co najmniej 1 i co najwyżej 100 000.</p>

### 출력

<p>Tw&oacute;j program powinien wypisać na wyjście dokładnie N serduszek, jedno pod drugim.</p>
