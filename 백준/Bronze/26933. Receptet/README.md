### 문제 설명

<p>Du har best&auml;mt dig f&ouml;r att laga mat. F&ouml;r att laga maten beh&ouml;ver du $N$ ingredienser. F&ouml;r varje ingrediens vet du hur mycket du redan har hemma, hur mycket du beh&ouml;ver totalt samt kostnaden f&ouml;r ingrediensen om du m&aring;ste k&ouml;pa den. Du skall allts&aring; k&ouml;pa den m&auml;ngd av varje ingrediens som du saknar. Uppgiften &auml;r att ber&auml;kna kostnaden f&ouml;r att laga maten.</p>

### 입력

<p>Den f&ouml;rsta raden best&aring;r av ett heltal $N$ ($1 \le N \le 5$), antalet ingredienser.</p>

<p>De n&auml;sta $N$ raderna beskriver en ingrediens med tre heltal $H, B, K$ ($0 \le H, B \le 1\,000$, $1 \le K \le 1\,000$) som st&aring;r f&ouml;r hur mycket du har, hur mycket som beh&ouml;vs och hur mycket den kostar.</p>

### 출력
