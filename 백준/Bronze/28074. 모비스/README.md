<p>현대 모비스의 <strong>MOBIS</strong>는 어떤 뜻을 가지고 있을까?</p>

<p><strong>MOBIS</strong>는 기존에는 Mobile + System의 합성어에서 시작되었지만, 현재는 &quot;Mobility Beyond Integrated Solution&quot; 라는 의미로 재정의 되었다.</p>

<p>이는 사용자의 경험을 혁신하고, 고객의 요구에 최적화된 통합 솔루션, 그 이상의 가치를 전달하는 모빌리티 플랫폼 프로바이더로 도약하겠다는 뜻을 가지고 있다.</p>

<p style="text-align: center;"><img alt="" src="%EB%B0%B1%EC%A4%80/Bronze/28074.%E2%80%85%EB%AA%A8%EB%B9%84%EC%8A%A4/591ba2d4.png" data-original-src="%EB%B0%B1%EC%A4%80/Bronze/28074.%E2%80%85%EB%AA%A8%EB%B9%84%EC%8A%A4/591ba2d4.png" data-original-src="https://upload.acmicpc.net/f063b3ab-292d-414a-8eac-138120522923/-/preview/" onerror="this.onerror=null;this.src=this.dataset.originalSrc" /></p>

<p>이 뜻에 매료된 진익이는 스티커 용지에 인쇄되어 있는 문자들 중<strong> &#39;M&#39;, &#39;O&#39;, &#39;B&#39;, &#39;I&#39;, &#39;S&#39; </strong>만을 오리고 적절히 배치하여 노트북에 <strong>MOBIS</strong>를 붙여놓고자 한다.</p>

<p>스티커 용지에 인쇄되어 있는 문자열이 주어진다. 이 문자들을 이용해 MOBIS를 만들 수 있을까?</p>

### 입력

<p>첫째 줄에 문자열이 주어진다.</p>

### 출력

<p>주어진 문자열에 포함된 알파벳 대문자들을 이용해 <strong>MOBIS</strong>를 만들 수 있으면 &quot;<code>YES</code>&quot;, 그렇지 않으면 &quot;<code>NO</code>&quot;를 출력한다.</p>

### 제한

<ul>
	<li>1 &le; 문자열의 길이 &le; 100</li>
	<li>문자열은 알파벳 대문자로만 이루어져 있다.</li>
</ul>
