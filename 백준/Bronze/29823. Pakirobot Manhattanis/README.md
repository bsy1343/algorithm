### 문제 설명

<p>New Yorgis Manhattani linnaosas tegutseb isejuhtiv pakirobot. See robot on &uuml;sna suur ja suudab korraga kohale toimetada mitu pakki eri sihtkohtadesse. P&auml;rast k&otilde;ikide pakkide kohaletoimetamist peab robot minema tagasi lattu, et uued pakid v&otilde;tta.</p>

<p>Manhattani t&auml;navad moodustavad ruudustiku: k&otilde;ik t&auml;navad on t&auml;pselt p&otilde;hja-l&otilde;una v&otilde;i ida-l&auml;&auml;ne suunalised. Lisaks on k&otilde;ik k&otilde;rvutiste paralleelsete t&auml;navate vahekaugused v&otilde;rdsed.</p>

<p>Nimetame roboti <em>sammuks</em> &uuml;hest ristmikust m&ouml;&ouml;da mingit t&auml;navat j&auml;rgmise ristmikuni liikumist.</p>

<p>Leia, kui mitme sammuga on robotil v&otilde;imalik tagasi lattu j&otilde;uda.</p>

### 입력

<p>Sisendi esimesel real on roboti praeguseks tehtud sammude arv $N$ ($1 \le N \le 1\,000$). Teisel real on $N$ t&auml;hem&auml;rki, mis n&auml;itavad seda, mis suunas iga roboti samm oli: <code>N</code> t&auml;hendab sammu p&otilde;hja, <code>E</code> ida, <code>S</code> l&otilde;una ja <code>W</code> l&auml;&auml;ne suunas.</p>

### 출력

<p>V&auml;ljundi ainsale reale v&auml;ljastada minimaalne sammude arv, millega robot lattu tagasi saab.</p>
