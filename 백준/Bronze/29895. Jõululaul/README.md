### 문제 설명

<p>T&auml;nase teise advendi puhul meenutame paarisaja aasta vanust inglise j&otilde;ululaulu:</p>

<p>J&otilde;ulu <strong>esimesel</strong> p&auml;eval mu kallim saatis mul&#39; &uuml;he faasani pirnipuu sees.</p>

<p>J&otilde;ulu <strong>teisel</strong> p&auml;eval mu kallim saatis mul&#39; kaks turteltuvi ja faasani pirnipuu sees.</p>

<p>J&otilde;ulu <strong>kolmandal</strong> p&auml;eval mu kallim saatis mul&#39; kolm kirjut kana, kaks turteltuvi ja faasani pirnipuu sees.</p>

<p>J&otilde;ulu <strong>neljandal</strong> p&auml;eval mu kallim saatis mul&#39; neli laulvat lindu, kolm kirjut kana, kaks turteltuvi ja faasani pirnipuu sees.</p>

<p>Programmiharjutuste v&otilde;lumaal kestavad j&otilde;ulud $N$ p&auml;eva ja on olemas $N$ erinevat kingitust; t&auml;histame nad $A_1$, $A_2$, $\ldots$, $A_N$. J&otilde;ulu $k$-ndal p&auml;eval saadab kallim mulle &uuml;he $A_1$, kaks $A_2$, kolm $A_3$, $\dots$ ja $k$ $A_k$-te.</p>

<p>Kirjutada programm, mis leiab, mitu igat liiki kingitust kallim mulle kokku saadab.</p>

### 입력

<p>Sisendi ainsal real on t&auml;isarv $N$, v&otilde;lumaa j&otilde;ulude kestus p&auml;evades ($1 \le N \le 100\,000$).</p>

### 출력

<p>V&auml;ljastada t&auml;pselt $N$ rida, igale reale &uuml;ks t&auml;isarv; $k$-ndale reale v&auml;ljastada p&uuml;hade jooksul kingiks saadud $A_k$ koguarv.</p>
