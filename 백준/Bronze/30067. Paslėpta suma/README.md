### 문제 설명

<p>Simas ir Jurgis žaidžia tokį žaidimą. Simas sugalvoja devynis sveikus skaičius, teigiamus ir neigiamus, pavyzdžiui: $5, 4, -5, -3, 9, 1, 2, 5, -11$. Tuomet jis suskaičiuoja jų sumą $5 + 4 + (-5) + (-3) + 9 + 1 + 2 + 5 + (-11) = 7$, ir paslėpia ją kažkur tarp &scaron;ių skaičių. Taip gaunama i&scaron; viso de&scaron;imt skaičių:</p>

<p>$5, 4, -5, -3, 9, 1, 7, 2, 5, -11$</p>

<p>Tuomet Simas &scaron;ią seką parodo Jurgiui. Jurgio tikslas yra kuo greičiau surasti, kuris i&scaron; de&scaron;imties skaičių yra paslėptoji likusių skaičių suma.</p>

<p>Para&scaron;ykite programą, kuri padėtų Jurgiui surasti paslėptąją sumą.</p>

### 입력

<p>Pradinius duomenis sudarys de&scaron;imt eilučių, kuriose bus įra&scaron;yta po vieną skaičių.</p>

### 출력

<p>Jūsų programa turi i&scaron;vesti skaičių, kuris yra paslėptoji suma. Atsakymas visuomet bus vienareik&scaron;mis.</p>
