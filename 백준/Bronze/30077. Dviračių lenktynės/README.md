### 문제 설명

<p>Bitų karalystėje vyksta dviračių treko lenktynės. Visi dviratininkai startuoja vienu metu ir važiuoja $N$ ratų. Jei kurį lenktynininką ratu aplenkia kitas, aplenktasis toliau varžybų nebetęsia. Jei sportininkas, kirsdamas fini&scaron;o liniją, aplenkia varžovą ratu, aplenktasis taip pat i&scaron;krenta i&scaron; varžybų.</p>

<p>Žinodami, per kiek laiko kiekvienas dviratininkas įveikia vieną treko ratą bei kiek ratų trunka lenktynės, raskite, kiek dalyvių nebus aplenkti ratu ir fini&scaron;uos varžybose.</p>

### 입력

<p>Pirmojoje eilutėje pateikti trys natūralieji skaičiai atskirti tarpu:</p>

<ul>
	<li>$N$ &ndash; lenktynininkų skaičius;</li>
	<li>$M$ &ndash; varžybų trukmė ratais;</li>
	<li>$L$ &ndash; treko rato ilgis metrais.</li>
</ul>

<p>Kitose $N$ eilučių pateikta, per kiek sekundžių ($T_i$) kiekvienas i&scaron; sportininkų įveikia vieną treko ratą. $T_i$ - natūralusis.</p>

### 출력

<p>I&scaron;veskite sportininkų, kurie nebuvo aplenkti ratu ir baigė varžybas, skaičių.</p>

### 제한

<ul>
	<li>$2 &le; N, M, L, T_i &le; 10\,000$</li>
</ul>
