### 문제 설명

<p>Dr. O has two granddaughters (named Alyssa and Konari) and they keep Dr. O young!</p>

<p>When people ask Dr. O how old he is, rather than giving one positive integer (his age), Dr. O provides two positive integers a and k; Dr. O&rsquo;s age can then be computed using the expression (a &times; Alyssa&rsquo;s age) + (k &times; Konari&rsquo;s age).</p>

<p>Given three positive integers (Dr. O&rsquo;s age, Alyssa&rsquo;s age, and Konari&rsquo;s age), determine if the positive integers a and k exist.</p>

### 입력

<p>There is only one input line; it provides (respectively) Dr. O&rsquo;s age, Alyssa&rsquo;s age, and Konari&rsquo;s age. Assume that 1 &le; Konari&rsquo;s age &lt; Alyssa&rsquo;s age &lt; Dr. O&rsquo;s age &le; 150.</p>

### 출력

<p>Print 1 (one) if a and k exist, 0 (zero) otherwise. Note that a and k must both be greater than zero.</p>
