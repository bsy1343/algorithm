### 문제 설명

<p>You are given a string $s$ consisting of lowercase English letters. A &quot;<code>kick</code>&quot; is defined as a substring of $s$ that starts with the letter &#39;<code>k</code>&#39; followed by the letter &#39;<code>i</code>&#39; followed by the letter &#39;<code>c</code>&#39; followed by the letter &#39;<code>k</code>&#39;.</p>

<p>Your task is to count the number of distinct &quot;<code>kicks</code>&quot; in the string $s$. Note that the kicks can overlap.</p>

### 입력

<p>The input contains exactly a string $s$ consisting of lowercase English letters. The length of the string $s$ is no more than $5 \times 10^6$.</p>

### 출력

<p>Print the number of &quot;<code>kicks</code>&quot; on a line.</p>
