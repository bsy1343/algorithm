### 문제 설명

<p>Vilius steigia picų i&scaron;vežiojimo į namus verslą. Liko sugalvoti ir įsigyti lengvai įsimenamą telefono numerį. Vilius turi galimų įsigyti telefono numerių sąra&scaron;ą.</p>

<p>Jis mano, kad telefono numeris yra įsimintinas tik jei visi skaitmenys jame yra vienodi (pavyzdžiui, 2222), arba jeigu skaitmenys griežtai didėja (pavyzdžiui, 23679).</p>

<p>Duotame galimų telefono numerių sąra&scaron;e suraskite įsimintiną numerį.</p>

### 입력

<p>Pirmoje eilutėje pateiktas telefono numerių skaičius N. Kitose N eilučių yra po vieną skirtingą numerį &ndash; teigiamą skaičių K.</p>

### 출력

<p>I&scaron;veskite vieną skaičių &ndash; įsimintiną numerį. Jeigu sąra&scaron;e buvo keli įsimintini numeriai, i&scaron;veskite mažiausią i&scaron; jų. Jeigu sąra&scaron;e įsimintinų numerių nebuvo, rezultatas turi būti <code>NERASTA</code>.</p>

### 제한

<ul>
	<li>1 &le; N &le; 1 000</li>
	<li>1 &le; K &le; 100 000</li>
</ul>
