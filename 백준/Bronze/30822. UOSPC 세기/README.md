### 문제 설명

<p>배봉산에 살고 있는 여우는 문자열을 가지고 노는 것을 좋아한다. 여우는 특히 그중에서도 &#39;<code>uospc</code>&#39;라는 모양을 가장 좋아한다!</p>

<p>여우는 자신이 가진 문자열의 순서를 재배열하여 &#39;<code>uospc</code>&#39; 모양을 최대한 많이 만들려고 한다. 예를 들어 &#39;suspicion&#39;이라는 문자열을 재배열하면 &#39;<code><u><strong>uospc</strong></u>siin</code>&#39;라는 문자열을 얻을 수 있고. &#39;<code>processyourpurchase</code>&#39;라는 문자열을 재배열하면 &#39;<code><strong><u>uospcuospc</u></strong>reyrrhase</code>&#39;라는 문자열을 얻을 수 있다.</p>

<p>알파벳 소문자로만 이루어진 임의의 문자열 $S$가 주어졌을 때. 똑똑한 여우가 이 문자열을 적절히 재배열하여 얻을 수 있는 &#39;<code>uospc</code>&#39;의 최대 개수를 구하여라.</p>

### 입력

<p>첫 번째 줄에 문자열의 길이인 정수 $n$이 주어진다. ($1\leq n \leq 1\ 000$)</p>

<p>두 번째 줄에 영어 알파벳 소문자로만 구성되고 길이가 $n$인 문자열 $S$가 주어진다.</p>

### 출력

<p>문자열 $S$의 원소들을 적절히 재배열하여 만들 수 있는 &#39;<code>uospc</code>&#39;의 최대 개수를 출력한다.</p>
