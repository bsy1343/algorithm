### 문제 설명

<p>Your friend Bob is really bad at adding numbers, and he&rsquo;d like some help to make sure he&rsquo;s doing it correctly! Can you help Bob make sure he is adding correctly? Given 3 integers $A$, $B$, $C$, make sure that $A + B = C$, and that Bob indeed added $A$ and $B$ correctly.</p>

### 입력

<p>The input consists of a single line with 3 integers $A, B, C$ where $-10^9 \le A, B, C \le 10^9$.</p>

### 출력

<p>Output either <code>correct!</code> if $A + B = C$, or <code>wrong!</code> if $A + B \ne C$.</p>
