### 문제 설명

<p><var>3</var> つの整数 <var>A, B, C</var> が与えられる．</p>

<p><var>3</var> つの値の合計が <var>21</var> 以下なら <var>1</var> を，そうでないなら <var>0</var> を出力せよ．</p>

### 입력

<p>入力は以下の形式で与えられる．</p>

<pre><var>A</var>
<var>B</var>
<var>C</var></pre>

### 출력

<p><var>3</var> つの値の合計が <var>21</var> 以下なら <var>1</var> を，そうでないなら <var>0</var> を出力せよ．</p>

<p>答え以外は何も出力しないこと．(入力を促す文章なども出力しないこと．)</p>

### 제한

<ul>
	<li><var>1 ≦ A ≦ 13</var>．</li>
	<li><var>1 ≦ B ≦ 13</var>．</li>
	<li><var>1 ≦ C ≦ 13</var>．</li>
	<li>入力される値はすべて整数である．</li>
</ul>
