### 문제 설명

<p>제1회 피갤컵이 2024년 8월에 열렸다.  </p>

<p>제2회 피갤컵은 2025년 3월에 열린다.  </p>

<p>이를 통해 피갤컵은 7개월 주기로 열린다는 사실을 알 수 있다.  </p>

<p>그렇다면 $N$번째 피갤컵은 언제 열릴까?</p>

### 입력

<p>첫째 줄에 정수 $N$이 주어진다. $(1 \le N \le 5)$</p>

### 출력

<p>첫째 줄에 $N$번째 피갤컵이 열리는 연도와 월을 공백을 두고 출력한다.</p>
