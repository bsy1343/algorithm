### 문제 설명

<p>오늘은 시립대 컴퓨터과학부의 MT날이다. 학생회 임원들은 필요한 물건을 박스에 정리해 버스에 실으려 한다. 박스에 담을 물건들의 개수가 박스의 크기를 초과하지 않을 때만 박스에 안전하게 물건을 담을 수 있다. 이때, 필요한 박스의 최소 개수를 맞혀보자.</p>

### 입력

<p>물건의 개수 $N$과 박스의 크기를 나타내는 정수 $M$이 공백으로 구분되어 주어진다. $(0 \le N \le 100; 1 \le M \le 100)$</p>

### 출력

<p>필요한 박스의 최소 개수를 출력한다.</p>
