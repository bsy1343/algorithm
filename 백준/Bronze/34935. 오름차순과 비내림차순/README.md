### 문제 설명

<p>서울사이버대학교 빅데이터·정보보호학과, 인공지능학과 등에서 파이썬 수업을 하던 노교수는 학생들이 오름차순과 비내림차순을 잘 구분하지 못하는 것을 보고 문제를 만들어야 겠다고 결심했다. 입력으로 비내림차순 수열 $A$가 주어질 때, 오름차순인지 아닌지를 판단하자.</p>

### 입력

<p>첫 번째 줄에 데이터의 수 $N$이 주어진다. $(1 \le N \le 10^6)$</p>

<p>두 번째 줄에 $N$개의 정수 데이터 $A_i$가 순서대로 공백으로 구분되어 주어진다. $A$는 비내림차순이다. ($-10^{18} \le A_i \le 10^{18}$)</p>

### 출력

<p>오름차순이면 $1$, 그렇지 않으면 $0$을 출력하자.</p>

### 힌트

<p>길이가 $N$인 수열 $A_1, A_2, \cdots, A_N$에 대하여, 아래의 조건을 만족하는 경우 오름차순이라고 한다. 즉, 뒤로 갈수록 값이 커져야 하며, 작거나 같아서는 아니된다.</p>

<ul>
<li>$A_i &lt; A_{i+1}, \forall i \in \{1, \cdots, N-1\}$</li>
</ul>

<p>길이가 $N$인 수열 $A_1, A_2, \cdots, A_N$에 대하여, 아래의 조건을 만족하는 경우 비내림차순이라고 한다. 즉, 뒤로 갈수록 값이 커지거나 같아야 하며, 작아져서는 아니된다.</p>

<ul>
<li>$A_i \le A_{i+1}, \forall i \in \{1, \cdots, N-1\}$</li>
</ul>
