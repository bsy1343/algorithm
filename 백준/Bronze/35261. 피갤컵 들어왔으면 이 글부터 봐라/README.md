<p>알파벳 소문자로 이뤄진 길이가 $N$인 문자열 $S$가 주어진다. $S$에서 최소한의 글자를 다른 글자로 바꿔 이글이 보이게 하려 한다. 이글이 보인다는 것은 $S$가 <code><span style="color:#e74c3c;">eagle</span></code>을 연속 부분 문자열로 가진다는 것이다. 최소 몇 개의 글자를 바꿔야 하는지 구해보자.</p>

### 입력

<p>첫째 줄에 정수 $N$이 주어진다. $(5\leq N \leq 100)$</p>

<p>둘째 줄에 알파벳 소문자로 이루어진 길이 $N$의 문자열 $S$가 주어진다.⁩⁩</p>

### 출력

<p>첫째 줄에 답을 출력한다.</p>
