<p style="text-align: center;"><small>두바이 쫀득 쿠키를 먹는 수아의 모습</small></p>

<p>유림이와 하람이는 두바이 쫀득 쿠키를 좋아하는 수아를 위해 두바이 쫀득 쿠키를 만들어 선물하려고 한다.</p>

<p>두바이 쫀득 쿠키에는 다음 재료들이 들어가며, 두바이 쫀득 쿠키 $1$개를 만들기 위한 재료별 가격은 다음과 같다.</p>

<ul>
<li>구운 카다이프 $900$원</li>
<li>버터 $60$원</li>
<li>피스타치오 스프레드 $600$원</li>
<li>화이트초콜릿 $170$원</li>
<li>마시멜로 $160$원</li>
<li>코코아파우더 $110$원</li>
</ul>

<p>현재 유림이와 하람이는 $N$원을 가지고 있다. 유림이와 하람이가 두바이 쫀득 쿠키를 최대 몇 개까지 만들 수 있는지 구해보자.</p>

### 입력

<p>첫 번째 줄에 정수 $N$이 주어진다. ($0 \leq N \leq 10^9$)</p>

### 출력

<p>첫 번째 줄에 유림이와 하람이가 만들 수 있는 두바이 쫀득 쿠키의 최대 개수를 출력한다.</p>
