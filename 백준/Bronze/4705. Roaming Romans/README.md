### 문제 설명

<p>The English word &ldquo;mile&rdquo; derives from the Latin &ldquo;mille passus&rdquo;, meaning &ldquo;a thousand paces&rdquo;. A Roman mile was the distance a soldier would walk in 1000 paces (a pace being two steps, one with each foot).</p>

<p>Over time, the actual distance referred to as a &ldquo;mile&rdquo; has changed. The modern English mile is 5280 (modern) feet. The Roman mile is believed to have been about 4,854 (modern) feet.</p>

<p>Write a program to convert distances in English miles into Roman miles and paces.</p>

### 입력

<p>Input will consist of one or more lines, each containing a single positive floating point number denoting a distance in English miles. End of input is indicated by a value of 0.0.</p>

### 출력

<p>For each input value, print a line of output of the form</p>

<p>X English miles equals Y Roman miles or K paces.</p>

<p>where X is the value from the input, Y is the equivalent distance in Roman miles. Each of these should be printed to 2 decimal digits precision. K is an integer denoting the number of paces, and should be rounded to the closest integer.</p>
