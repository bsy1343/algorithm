<p>싱기한 네자리 숫자란, [1000,9999]인 10진수 숫자중에서,&nbsp; 다음의 조건을 만족하는 숫자를 말한다.</p>

</ul>

<p>여러분은&nbsp;싱기한 네자리 숫자를 모두 출력해야 한다.</p>
