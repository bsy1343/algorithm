### 문제 설명

<p>Write a program that repeatedly reads a positive integer, determines if the integer is&nbsp;deficient,&nbsp;perfect, or&nbsp;abundant, and outputs the number along with its classification.</p>

<p>A positive integer,&nbsp;<var>n</var>, is said to be&nbsp;perfect&nbsp;if the sum of its proper divisors equals the number itself. (Proper divisors include 1 but not the number itself.) If this sum is less that&nbsp;<var>n</var>, the number is&nbsp;deficient, and if the sum is greater than&nbsp;<var>n</var>, the number is&nbsp;abundant.</p>

### 입력

<p>The input starts with the number of integers that follow. For each of the following integers, your program should output the classification, as given below. You may assume that the input integers are greater than 1 and less than 32500.</p>
