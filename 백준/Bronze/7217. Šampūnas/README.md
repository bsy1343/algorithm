### 문제 설명

<p>Bitlandijoje visi gyventojai plauna galvą tomis pačiomis dienomis, pagal tokias taisykles:</p>

<ol>
	<li>Jeigu &scaron;iandien &scaron;ventinė diena &ndash; plauti galvą.</li>
	<li>Jeigu vakar galva neplauta &ndash; plauti galvą.</li>
	<li>Prie&scaron;ingu atveju, galvos neplauti.</li>
</ol>

<p>Bitlandijos vyriausybė nori suplanuoti, kiek &scaron;ampūno reikia pagaminti ateinančiam periodui, tad jie norėtų žinoti, kiek kartų per ateinančias N dienų Bitlandijos gyventojai plaus galvą.</p>

<p>Žinomas &scaron;ventinių dienų kalendorius. Suskaičiuokite, kiek kartų Bitlandijos gyventojai plaus galvą per ateinančias N dienų. Laikykite, kad pirmą laikotarpio dieną gyventojai plauna galvą.</p>

### 입력

<p>Pirmoje eilutėje įra&scaron;yti du sveikieji skaičiai: bendras dienų skaičius N ir &scaron;ventinių dienų skaičius K.</p>

<p>Kitose K eilučių įra&scaron;yta po vieną sveikąjį skaičių d<sub>i</sub> &ndash; tai &scaron;ventinių dienų numeriai (1 &le; d<sub>i</sub> &le; N). &Scaron;ie skaičiai pateikti didėjimo tvarka.</p>

### 출력

<p>I&scaron;veskite vieną sveikąjį skaičių: kiek kartų Bitlandijos gyventojai plaus galvą per nurodytą laikotarpį.</p>

### 제한

<ul>
	<li>0 &le; K &lt; N &le; 100 000</li>
</ul>
