### 문제 설명

<p>Jurgita tyrinėja skaitmenis ir skaičius. Ji pasirinko tris skirtingus skaitmenis nuo 1 iki 9, i&scaron; jų sudarė visus įmanomus triženklius skaičius ir juos i&scaron;rikiavo didėjimo tvarka.</p>

<p>Duotas i&scaron; pasirinktų skaitmenų sudarytas triženklis skaičius x. Nustatykite, kelintas, didėjimo tvarka i&scaron;rikiuotoje skaičių sekoje, yra skaičius x.</p>

### 입력

<p>Pirmoje eilutėje pateikti Jurgitos pasirinkti skaitmenys a, b, c. Antroje eilutėje pateiktas i&scaron; pasirinktų skaitmenų sudarytas skaičius x.</p>

### 출력

<p>I&scaron;veskite, kelintas sekoje yra duotasis skaičius x.</p>

### 제한

<ul>
	<li>1 &le; a, b, c &le; 9; a &ne; b; a &ne; c; b &ne; c;</li>
</ul>
