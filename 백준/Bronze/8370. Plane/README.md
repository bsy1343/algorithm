### 문제 설명

<p>Byteland Airlines recently extended their aircraft fleet with a new model of a plane. The new acquisition has <em>n</em><sub>1</sub>&nbsp;rows of seats in the business class and <em>n</em><sub>2</sub>&nbsp;rows in the economic class. In the business class each row contains <em>k</em><sub>1</sub>&nbsp;seats, while each row in the economic class has <em>k</em><sub>2</sub>&nbsp;seats.</p>

<p>Write a program which:</p>

<ul>
	<li>reads information about available seats in the plane,</li>
	<li>calculates the sum of all seats available in that plane,</li>
	<li>writes the result.</li>
</ul>

### 입력

<p>In the first and only line of the standard input there are four integers <em>n</em><sub>1</sub>,&nbsp;<em>k</em><sub>1</sub>,&nbsp;<em>n</em><sub>2</sub>&nbsp;and&nbsp;<em>k</em><sub>2</sub>&nbsp;(1 &le; <em>n</em><sub>1</sub>,&nbsp;<em>k</em><sub>1</sub>,&nbsp;<em>n</em><sub>2</sub>,&nbsp;<em>k</em><sub>2</sub> &le; 1 000), separated by single spaces.</p>

### 출력

<p>The first and only line of the standard output should contain one integer - the total number of seats available in the plane.</p>
