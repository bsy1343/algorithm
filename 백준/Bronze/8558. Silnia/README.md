### 문제 설명

<p>Niech <em>n</em>&nbsp;będzie nieujemną liczbą całkowitą. Liczbę <em>n</em>!&nbsp;(czytaj&nbsp;<em>n</em>-silnia) definiuje się następująco. Jeśli <em>n</em> &le; 1, to <em>n</em>! = 1. Dla <em>n</em> &gt; 1, <em>n</em>!&nbsp;jest r&oacute;wne iloczynowi wszystkich liczb od 1 do&nbsp;<em>n</em>, czyli <em>n</em>! = 1 * 2 * ... * <em>n</em>. Na przykład 4! = 1 * 2 * 3 * 4 = 24.</p>

<p>Napisz program, kt&oacute;ry</p>

<ul>
	<li>wczyta ze standardowego wejścia nieujemną liczbę całkowitą <em>n</em>,</li>
	<li>policzy cyfrę jedności w zapisie dziesiętnym liczby <em>n</em>!,</li>
	<li>wypisze wynik na standardowe wyjście.</li>
</ul>

### 입력

<p>Pierwszy i jedyny wiersz standardowego wejścia zawiera dokładnie jedną nieujemną liczbę całkowitą <em>n</em>, 0 &le; <em>n</em> &le; 30 000.</p>

### 출력

<p>W pierszym i jedynym wierszu standardowego wyjścia Tw&oacute;j program powinien zapisać dokładnie jedną cyfrę r&oacute;wną cyfrze jedności w zapisie dziesiętnym liczby <em>n</em>!.</p>
