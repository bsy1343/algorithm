### 문제 설명

<p>Dla danej nieujemnej liczby całkowitej <em>n</em>&nbsp;należy obliczyć cyfrę jedności (ostatnią cyfrę w zapisie dziesiętnym) liczby 2<sup><em>n</em></sup>.</p>

<p>Napisz program, kt&oacute;ry</p>

<ul>
	<li>wczytuje ze standardowego wejścia nieujemną liczbę całkowitą&nbsp;<em>n</em>,</li>
	<li>oblicza cyfrę jedności w zapisie dziesiętnym liczby 2<sup><em>n</em></sup>,</li>
	<li>wypisuje wynik na standardowe wyjście.</li>
</ul>

### 입력

<p>Pierwszy i jedyny wiersz wejścia zawiera jedną liczbę całkowitą <em>n</em>, 0 &le; <em>n</em> &le; 10<sup>40</sup>.</p>

### 출력
