### 문제 설명

<p>Mamy dane <em>n</em>&nbsp;liczb całkowitych. Zastanawiamy się, ile jest uporządkowanych par liczb <em>a</em>&nbsp;i <em>b</em>, w kt&oacute;rych zachodzi warunek: <em>a</em> - <em>b</em> &gt; <em>a</em> + <em>b</em>.</p>

### 입력

<p>W pierwszym wierszu wejścia znajduje się jedna liczba całkowita <em>n</em>&nbsp;(1 &le; <em>n</em> &le; 10<sup>6</sup>). W kolejnym wierszu znajduje się <em>n</em>&nbsp;liczb całkowitych <em>a</em><sub>1</sub>, <em>a</em><sub>2</sub>, ..., <em>a<sub>n</sub></em>&nbsp;(-10<sup>9</sup> &le; <em>a<sub>i</sub></em> &le; 10<sup>9</sup>), gdzie&nbsp;<em>a<sub>i</sub></em>&nbsp;oznacza wartość <em>i</em>-tej liczby.</p>

### 출력

<p>W pierwszym i jedynym wierszu wyjścia powinna się znaleźć jedna liczba całkowita, r&oacute;wna liczbie uporządkowanych par, kt&oacute;rych r&oacute;żnica jest większa od sumy.</p>

### 힌트

<p>Wszystkie pary to: (-1, 2), (-1, 3), (2, -1), (2, 3), (3, -1), (3, 2). Dwie pary spełniają zadany warunek: (2, -1), (3, -1).</p>
