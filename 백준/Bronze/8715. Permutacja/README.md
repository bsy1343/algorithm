### 문제 설명

<p>Bajtek napisał na kartce <em>n</em>&nbsp;liczb. Zastanawia się teraz, czy są one permutacją liczb od 1&nbsp;do&nbsp;<em>n</em>, czyli czy każda z liczb 1, 2, 3, ..., <em>n</em> - 1, <em>n</em>, występuje dokładnie jeden raz w tym ciągu.</p>

### 입력

<p>Pierwszy wiersz standardowego wejścia zawiera jedną liczbę całkowitą <em>n</em>&nbsp;(1 &le; <em>n</em> &le; 10<sup>6</sup>), oznaczającą ilość liczb jakie wypisał Bajtek. Kolejny wiersz zawiera ciąg&nbsp;<em>n</em>&nbsp;liczb całkowitych <em>a</em><sub>1</sub>, <em>a</em><sub>2</sub>, ..., <em>a<sub>n</sub></em>&nbsp;(0 &le; <em>a<sub>i</sub></em> &le; 10<sup>9</sup>), gdzie&nbsp;<em>a<sub>i</sub></em>&nbsp;oznacza <em>i</em>-tą liczbę w ciągu Bajtka.</p>

### 출력

<p>Pierwszy i jedyny wiersz wyjścia powinien zawierać słowo &#39;TAK&#39;, jeśli ciąg Bajtka jest permutacją liczb od 1 do <em>n</em>, lub słowo &#39;NIE&#39;, jeśli ciąg Bajtka nie jest permutacją liczb od 1 do&nbsp;<em>n</em>.</p>
