### 문제 설명

<p>Jasio wypisał ciąg <em>n</em>&nbsp;liczb całkowitych. Ciągiem&nbsp;<i>dobrym</i>&nbsp;nazwiemy ciąg kolejnych liczb naturalnych 1, 2, 3, ...&nbsp;itd. Jasio chciałby wykreślić jak najmniej liczb w taki spos&oacute;b, aby pozostałe liczby utworzyły&nbsp;<i>dobry</i>&nbsp;ciąg. Jeśli Jasio nie może utworzyć żadnego z&nbsp;<i>dobrych</i>&nbsp;ciąg&oacute;w, to powinien wykreślić wszystkie liczby.</p>

### 입력

<p>Pierwszy wiersz wejścia zawiera jedną liczbę całkowitą <em>n</em>&nbsp;(1 &le; <em>n</em> &le; 10<sup>6</sup>), oznaczająca ilość liczb w ciągu Jasia. Kolejny wiersz zawiera ciąg&nbsp;<em>n</em>&nbsp;liczb całkowitych <em>a</em><sub>1</sub>, <em>a</em><sub>2</sub>, ..., <em>a<sub>n</sub></em>&nbsp;(1 &le; <em>a<sub>i</sub></em> &le; 10<sup>9</sup>), oznaczających kolejne wartości liczb w ciągu Jasia.</p>

### 출력

<p>Pierwszy i jedyny wiersz wyjścia powinien zawierać jedną liczbę całkowitą r&oacute;wną minimalnej ilości liczb, kt&oacute;re powinien wykreślić Jaś.</p>
