### 문제 설명

<p>Bajtuś znalazł w lesie 3 patyki. Teraz chciałby wiedzieć, czy może z nich zbudować tr&oacute;jkąt prostokątny lub r&oacute;wnoboczny. Bajtuś nie może łamać patyk&oacute;w, może je wykorzystać tylko w całości.</p>

### 입력

<p>Pierwszy wiersz wejścia zawiera trzy liczby całkowite <em>a</em>, <em>b</em>, <em>c</em>&nbsp;(1 &le; <em>a</em>, <em>b</em>, <em>c</em> &le; 1000), oznaczające odpowiednio długości pierwszego, drugiego i trzeciego patyka.</p>

### 출력

<p>Pierwszy i jedyny wiersz wyjścia powinien zawierać jedną liczbę: 0 - jeśli Bajtuś nie może zbudować, ani tr&oacute;jkąta prostokątnego ani r&oacute;wnobocznego, 1 - jeśli Bajtuś może zbudować tylko tr&oacute;jkąt prostokątny, 2 - jeśli Bajtuś może zbudować tylko tr&oacute;jkąt r&oacute;wnoboczny,</p>
