### 문제 설명

<p>A letter bank is a word such that all the letters of one letter bank can be used as many times as desired (at least once each) to make a new word. For example, &lsquo;IMPS&rsquo; is a bank of &lsquo;MISSISSIPPI.&rsquo;</p>

<p>Given two words &alpha; and &beta;, you are to write a program for deciding whether &alpha; is a bank of &beta;. Assume that the first word &alpha; has no repeated letters. Every word consists only of capital Roman letters: A, B, &hellip;, Z, and no blank.&nbsp;</p>

### 입력

<p>Your program is to read from standard input. The input consists of T (1 &le; T &le; 20) test cases. The number T of test case is given in the first line of the input. For each test case, two words are given in one line. There is a single space between the first word and the second word. And each word has length at most 80.&nbsp;</p>

### 출력

<p>Your program is to write a standard output. Print exactly one line for each test case. For each test case, print &ldquo;YES&rdquo; if the first word is a bank of the second word. Otherwise, print &ldquo;NO&rdquo;.&nbsp;</p>
