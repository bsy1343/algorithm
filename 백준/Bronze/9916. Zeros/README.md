### 문제 설명

<p>The factorial of a positive integer n, written as n!, is the product of the first n positive integers. &nbsp;That is,</p>

<p>n! = 1 &times; 2 &times; ... &times; n</p>

<p>Given a positive integer n, find the number of zeros in the decimal representation of n!. &nbsp;Of course, leading zeros should not be counted. &nbsp;(Note that decimal representation means base ten representation.)</p>

### 입력

<p>The input contains a single positive integer n &lt; 100.</p>

### 출력

<p>The output contains a single integer giving the number of zeros in the decimal representation of n!.</p>
