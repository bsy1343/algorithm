### 문제 설명

<p>You are given a set of N integer sequences Z = {Z<sub>1</sub>, Z<sub>2</sub>, &middot;&middot;&middot; , Z<sub>N</sub>}, where |Z<sub>i</sub>| = K for i = 1, 2, &middot;&middot;&middot; , N.</p>

<p>The N integer sequences in the set were generated by the following generation process:</p>

<ol>
	<li>For i-th generation, select a binary integer sequence X &isin; B arbitrary, where B is a set of binary sequences of length K. Let&rsquo;s denote X = {x<sub>1</sub>, x<sub>2</sub>, &middot;&middot;&middot; , x<sub>K</sub>} where x<sub>j</sub> &isin; {0, 1} for j = 1, 2, &middot;&middot;&middot; , K.</li>
	<li>Select a binary integer sequence Y &isin; B arbitrary so that dist(X, Y) &le; 2. Here dist(X, Y) denotes the hamming distance between X and Y , which is the distance measure between two sequences of equal length is the number of positions at which the corresponding digits are different. For example, dist({1, 0, 1, 1}, {1, 1, 1, 1}) is 1 and dist({1, 0, 1, 1, 1, 0, 1}, {1, 0, 0, 1, 0, 0, 1}) is 2.</li>
	<li>As the final result, we can generate an integer sequence Z<sub>i</sub>, by given X and Y.&nbsp;Z<sub>i</sub> is the i-th generated integer sequence defined by Z<sub>i</sub> = {x<sub>1</sub> + y<sub>1</sub>, x<sub>2</sub> + y<sub>2</sub>, &middot; &middot; &middot; , x<sub>K</sub> + y<sub>K</sub>}.</li>
</ol>

<p>For example, an integer sequence Z<sub>i</sub> = {1, 0, 1, 2, 2} can be generated by two binary sequences X = {1, 0, 0, 1, 1} and Y = {0, 0, 1, 1, 1}.</p>

<p>Given N integer sequences generated, write a program to find how many elements in the set B. If there are several possible solutions, find the one with minimum set size.</p>

### 입력

<p>The first line of the input contains two integers K and N (1 &le; K &le; 20, 1 &le; N &le; 24). A whitespace character separates those two numbers. Each of the following N lines contains elements of Z. The jth number in the i-th line represents Z<sub>i,j</sub>, the value of the j-th element of Z<sub>i</sub>. There are no delimiter characters between adjacent numbers in these N lines.</p>

### 출력

<p>Print the minimum number of elements in the set B.</p>

### 힌트

<p>In the sample, we can generate each element in the set Z by letting X = {1, 0, 0, 1, 1}, Y = {0, 0, 1, 1, 1} and X = {1, 0, 0, 1, 1}, Y = {1, 0, 0, 1, 1}, respectively. Therefore, Z can be constructed by the set B = {{1, 0, 0, 1, 1} and {0, 0, 1, 1, 1}}.</p>
