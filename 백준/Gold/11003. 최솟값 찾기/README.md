### 문제 설명

<p>N개의 수 A<sub>1</sub>, A<sub>2</sub>, ..., A<sub>N</sub>과 L이 주어진다.</p>

<p>D<sub>i</sub> = A<sub>i-L+1</sub> ~ A<sub>i</sub> 중의 최솟값이라고 할 때, D에 저장된 수를 출력하는 프로그램을 작성하시오. 이때, i &le;&nbsp;0 인 A<sub>i</sub>는 무시하고 D를 구해야 한다.</p>

### 입력

<p>첫째 줄에 N과 L이 주어진다. (1 &le; L &le; N &le; 5,000,000)</p>

<p>둘째 줄에는 N개의 수 A<sub>i</sub>가 주어진다. (-10<sup>9</sup> &le; A<sub>i</sub> &le; 10<sup>9</sup>)</p>

### 출력

<p>첫째 줄에 D<sub>i</sub>를 공백으로 구분하여 순서대로 출력한다.</p>
