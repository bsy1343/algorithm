### 문제 설명

<p>Given a positive integer U, find the largest integer L such that L &le; U and L does not contain any digit more than twice.</p>

### 입력

<p>The input consists of a single line that contains an integer U (1 &le; U &le; 10<sup>18</sup>).</p>

### 출력

<p>Output a line with an integer representing the largest number less than or equal to U that does not contain any digit more than twice.</p>
