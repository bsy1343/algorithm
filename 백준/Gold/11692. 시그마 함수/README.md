### 문제 설명

<p>시그마 함수&nbsp;&sigma;(n)은 정수 n의 약수의 합을 구하는 함수이다.</p>

<p>예를 들어,&nbsp;&sigma;(2) = 1+2 = 3이고,&nbsp;&sigma;(6) = 1+2+3+6 = 12,&nbsp;&sigma;(12) = 1+2+3+4+6+12 = 28 이다.</p>

<p>m이 주어졌을 때, 1 &le; n &le; m인 모든 n의&nbsp;&sigma;(n) 중에서 값이 짝수인 것이 몇 개 있는지 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 m (1 &le; m &le; 10<sup>12</sup>)이 주어진다.&nbsp;</p>

### 출력

<p>첫째 줄에&nbsp;1 &le; n &le; m인 모든 n의&nbsp;&sigma;(n) 중에서 값이 짝수인 것의 개수를 출력한다.</p>
