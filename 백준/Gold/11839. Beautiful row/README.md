### 문제 설명

<p>Ali-Amir wrote N numbers in a row. A row is called beautiful if any two of the neighbour numbers in the row have got the same amount of ones in binary or ternary notations.</p>

<p>Ali-Amir wants to count the number of ways the all given numbers can be written in a beautiful row.</p>

### 입력

<p>The first line of input file contains integer N (2 &le; N &le; 20). The next line contains N non-negative integers not exceeding 10<sup>9</sup> each.</p>

### 출력

<p>Output the number of ways the all given numbers can be placed in a beautiful row.</p>

### 힌트

<p>In the sample 5 = 12<sub>3</sub> and 1 = 1<sub>3</sub>, 5 = 101<sub>2</sub> and 6 = 110<sub>2</sub>, thus rows 1 5 6 and 6 5 1 are beautiful.</p>
