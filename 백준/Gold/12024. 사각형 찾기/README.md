### 문제 설명

<p>주어진 그래프가 있다. 문제는 간단하다 사각형의 개수를 찾는 것이다. 즉, 서로 다른 정점들로 이루어진 길이가 4인 Cycle의 개수를 세는 것이다. 이때 정점 방문 순서가 다르면 다른 경우로 간주한다. (1 -&gt; 2 -&gt; 3 -&gt; 4 -&gt; 1 과 2 -&gt; 3 -&gt; 4 -&gt; 1 -&gt; 2은 다른 경우이다.)</p>

### 입력

<p>정점의 개수 N이 주어진다. (1 &le; N &le; 250)</p>

<p>그 후 N줄에는 그래프가 양방향 그래프인 인접행렬 A로 주어진다. A<sub>ii</sub>가 1인 경우는 없다.</p>

### 출력

<p>서로 다른 길이 4짜리의 Cycle의 개수를 출력한다.</p>
