### 문제 설명

<p>Mirko and Slavko are bored on their skiing trip, so they came up with an interesting game they could play. First, Mirko specifies a number N. Then Slavko writes N letters he will use to create his word. Then Mirko writes a word consisting of N letters. Slavko&rsquo;s goal is to create a word using the letters he chose, but so that not a single letter in his word matches the letter at the same position in Mirko&rsquo;s word. In order to make the game even more intense, Slavko must find the lexicographically smallest such word. This word will surely exist​. Since Mirko and Slavko are still young, they know only 3 letters: a, b, and c, which greatly affects their programming skills.</p>

### 입력

<p>The first line of input contains the positive integer N (1 &le; N &le; 5000).<br />
The following line contains a string of N lowercase letters &lsquo;a&rsquo;, &lsquo;b&rsquo;, or &lsquo;c&rsquo;, the letters Slavko chose.<br />
The third line contains a string of N lowercase letters &lsquo;a&rsquo;, &lsquo;b&rsquo;, or &lsquo;c&rsquo;, the word Mirko wrote.<br />
&nbsp;</p>

<p>&lt;Scoring&gt;</p>

<p>In test cases worth 40 points in total, it will hold 1 &le; N &le; 20.</p>

### 출력

<p>The first and only line of output must contain the word Slavko found.</p>
