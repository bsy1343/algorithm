### 문제 설명

<p>Alice와 Bob은 두 명이 할 수 있는 게임을 하나 만들었다. 규칙은 다음과 같다.</p>

<ol>
	<li>n개의 서로 다른 자연수로 이루어진 집합을 하나 갖고 시작한다.&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;</li>
	<li>두 명은 번갈아서 차례를 가지며, 각자의 차례에 다음과 같은 동작을 수행한다: 집합 내에서 서로 다른 두 자연수 x와 y를 고른다. 단, |x-y|가 집합 내에 존재해서는 안 된다. 이렇게 x와 y를 고른 이후, 그것을 고른 사람이 |x-y|를 집합 내에 넣는 것으로 그 사람의 차례가 종료된다.</li>
	<li>더 이상 고를 수 있는 (x,y)가 없는 사람이 패배한다.</li>
</ol>

<p>두 람이 모두 최적의 전략으로 플레이할 때, 처음에 주어진 n개의 자연수에 대해 누가 승리하게 될지 출력하시오. 단, 처음 시작하는 사람은 항상 Alice이다.</p>

### 입력

<p>첫 번째 줄에는 처음 집합의 크기 n이 주어진다. (2 &le; n &le; 100)</p>

<p>다음 줄에는 n개의 자연수 c<sub>1</sub>~c<sub>n</sub>이 빈 칸을 사이에 두고 주어진다. (1 &le; c<sub>i </sub>&le; 10<sup>9</sup>)</p>

<p>모든 c<sub>i</sub>는 서로 다름이 보장된다.</p>

### 출력

<p>게임에서 Alice가 승리한다면 &ldquo;Alice&rdquo;, Bob이 승리한다면 &ldquo;Bob&rdquo;을 출력한다.</p>
