### 문제 설명

<p>N명의 학생이 있습니다. 각 학생의 힘은 작은 크기의 음이 아닌 정수로 표현됩니다. 여러분은 이 학생들을 트라이슬 시합을 위해 세 개의 팀으로 나누려고 합니다. 각 학생은 단 하나의 팀에만 속해야 하며, 각 팀에 아무도 없을 수는 없습니다. 팀이 다 정해졌다면, 각 팀의 힘은 팀원들의 힘을 XOR 비트 연산한 값이 됩니다. 홍준이는 세 팀의 힘의 합을 &lsquo;굉장함&rsquo;이라고 생각합니다. 그리고 이 값이 최대가 될 때 그 값이 얼마인지 궁금해졌습니다. 가능한 굉장함의 최댓값을 구하는 프로그램을 작성하세요.</p>

### 입력

<p>첫째 줄에는 학생의 수를 나타내는 정수 N(1 &le; N &le; 100)이 주어집니다.</p>

<p>둘째 줄에는 각 학생의 힘을 나타내는 N개의 정수들 X(i)가 공백을 사이에 두고 주어집니다. (0 &le; X(i) &le; 255)</p>

### 출력

<p>가능한 굉장함의 최댓값을 출력하세요.</p>

### 힌트

<p>힘이 7인 첫 번째 학생이 한 팀, 힘이 3인 두 번째 학생이 한 팀, 힘이 5인 3번째 학생과 힘이 2인 4번째 학생이 한 팀을 하면 굉장함이 17이 됩니다.</p>
