### 문제 설명

<p>이 문제는 특이한 수를 찾는 문제이다. 특이한 수 N의 성질은 다음과 같다.</p>

<ul>
	<li>a자리의 양의 정수 (leading zero 없음)</li>
	<li>마지막 자리의 숫자를 맨 앞으로 옮겨서 만든 새로운 수를 N&#39; 이라고 하면, N&#39; = b &times; N 을 만족함</li>
</ul>

<p>a, b 가 주어졌을 때, 특이한 수들 중 가장 작은 것을 구해보자.</p>

### 입력

<p>첫째 줄에 a, b (1 &le; a &le; 10<sup>6</sup>, 1 &le; b &le; 9)가 주어진다.</p>

### 출력

<p>첫째 줄에 문제의 조건을 만족하는 특이한 수를 출력한다. 만약 조건을 만족하는 특이한 수가 없다면 &quot;Impossible&quot; 을 출력한다.</p>
