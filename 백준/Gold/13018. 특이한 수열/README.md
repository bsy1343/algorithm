### 문제 설명

<p>이 문제는 특이한 수열 A를 찾는 문제이다. 특이한 수열의 성질은 다음과 같다.</p>

<ul>
	<li>수열 A의 길이는 n</li>
	<li>1이상 n이하의 정수가 빠짐없이 모두 등장해야 하며, 각 수는 한번만 등장해야함</li>
	<li>1 &le; i &le; n 인 i에 대해 gcd(i, A[i]) &gt; 1 을 만족하는 i가 정확히 k개여야함</li>
</ul>

<p>n, k 가 주어졌을 때, 특이한 수열을 아무거나 하나 구해보자.</p>

### 입력

<p>첫째 줄에 n, k (1 &le; n &le; 10<sup>5</sup>, 0 &le; k &le; n)가 주어진다.</p>

### 출력

<p>첫째 줄에 문제의 조건을 만족하는 특이한 수열 A를 출력한다. 답이 여러 가지가 있다면 그 중 아무거나 출력해도 된다. 만약 조건을 만족하는 특이한 수열이 없다면 &quot;Impossible&quot; 을 출력한다.</p>
