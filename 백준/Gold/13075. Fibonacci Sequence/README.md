### 문제 설명

<p>Your job is to take an integer x as the input and compute f(x) mod 10<sup>9</sup>, where f(x) is the x-th value in the well-known Fibonacci sequence.&nbsp;</p>

<p>Note that Fibonacci sequence is defined as follows:</p>

<p>f(1) = f(2) = 1<br />
f(k) = f(k-1) + f(k-2) for any k &gt; 2</p>

### 입력

<p>The first line of the input includes the number of test cases, 1 &le; t &le; 1000. Each test case is a line containing an integer 1 &le; x &le; 2<sup>48</sup>.</p>

### 출력

<p>For each test case, print one line containing f(x) mod 10<sup>9</sup></p>
