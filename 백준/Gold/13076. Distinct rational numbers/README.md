### 문제 설명

<p>You are to write a program to compute the number of distinct rational numbers a/b for the given positive integer N where 0 &le; a &le; b &le; N.</p>

### 입력

<p>The first line of the input includes the number of test cases, 1 &le; t &le; 10000. Each test case comes in one line containing 2 &le; N &le; 10000.</p>

### 출력

<p>For each test case, print the number of distinct rational numbers in one line.</p>
