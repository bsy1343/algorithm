### 문제 설명

<p>길이가 N인 수열 A<sub>1</sub>, A<sub>2</sub>, ..., A<sub>N</sub>이&nbsp;주어진다. 이때, 1 &le; i &lt; j &lt; k &le; N 이면서, A<sub>j</sub> - A<sub>i</sub> = A<sub>k</sub> - A<sub>j</sub>를 만족하는 (i, j, k) 쌍의 개수를 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 수열의 크기 N (3&nbsp;&le; N &le; 100,000)가 주어진다.</p>

<p>둘째 줄에는 A<sub>1</sub>, A<sub>2</sub>, ..., A<sub>N</sub>이 주어진다. (1 &le; A<sub>i</sub>&nbsp;&le; 30,000)</p>

### 출력

<p>1 &le; i &lt; j &lt; k &le; N 이면서, A<sub>j</sub>&nbsp;- A<sub>i</sub>&nbsp;= A<sub>k</sub>&nbsp;- A<sub>j</sub>를 만족하는 (i, j, k) 쌍의 개수를 출력한다.</p>
