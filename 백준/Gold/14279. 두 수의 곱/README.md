### 문제 설명

<p>a, b, c가 주어졌을 때, A * B = C 이면서 |A - a| + |B - b| + |C - c|의 최솟값을 구하는 프로그램을 작성하시오. 이때, A, B, C는 양의 정수이어야 한다.</p>

<p>|x|는 x의 절댓값이다.</p>

### 입력

<p>첫째 줄에 a, b, c (1 &le; a, b, c &le; 1,000,000)가 주어진다.</p>

### 출력

<p>첫째 줄에 |A - a| + |B - b| + |C - c|의 최솟값을 출력한다. 이때, A*B는 C와 같아야 한다.</p>
