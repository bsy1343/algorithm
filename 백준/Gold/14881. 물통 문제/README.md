### 문제 설명

<p>용량이 a, b 리터인 두 물통이 있다. 이때, 물을 적절히 부어서 정확하게 c리터를 만들 수 있는지 아닌지 구하는 프로그램을 작성하시오. 물은 무한히 많다.</p>

### 입력

<p>첫째 줄에 테스트 케이스의 개수 T(1 &le;&nbsp;T &le; 10,000)가 주어진다.</p>

<p>둘째 줄부터 T개의 줄에는 테스트 케이스가 한 줄에 하나씩 주어진다. 각 테스트 케이스는 3개의 정수 a, b, c로 이루어져 있다. (1 &le; a, b, c &le;&nbsp;10<sup>9</sup>)</p>

### 출력

<p>각각의 테스트 케이스에 대해서, 정확하게 c리터를 만들 수 있으면 &quot;YES&quot;, 없으면 &quot;NO&quot;를 출력한다.</p>

### 힌트

<p>첫 번째 테스트 케이스의 경우에 다음과 같은 방법으로 4리터를 만들 수 있다.</p>

<ul>
	<li>a를 채운다. (a = 5, b = 0)</li>
	<li>a의 물을 b로 붓는다. (a = 2, b = 3)</li>
	<li>b에 들어있는 물을 버리고, a에 들어있는 물을 b로 붓는다. (a = 0, b = 2)</li>
	<li>a를 채운다. (a = 5, b = 2)</li>
	<li>a의 물을 b로 붓는다. (a = 4, b = 3)</li>
</ul>
