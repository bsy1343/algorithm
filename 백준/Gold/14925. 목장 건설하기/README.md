### 문제 설명

<p>랜드 씨는 퇴직금으로 땅을 사서 목장을 지으려 한다.&nbsp; 그가 사려고 소개받은 땅은 직사각형이고 대부분 들판이지만, 여기저기에 베기 어려운 나무와 치울 수 없는 바위가 있다.</p>

<p>그는 목장을 하나의 정사각형으로 최대한 크게 지으려 하는데, 그 안에 나무나 바위는 없어야 한다.&nbsp;</p>

<p>땅의 세로 길이가 M미터, 가로 길이가 N미터일 때, 1미터 간격의 격자로 된 땅의 지도를 M x N행렬로 표현하자.&nbsp;</p>

<p>이때, 행렬의 원소 0은 들판, 1은 나무 그리고 2는 돌을 의미한다.&nbsp; 랜드씨의 땅에서 지을 수 있는 가장 큰 정사각형 목장의 한 변의 크기 L을 출력하시오.</p>

### 입력

<pre>
M N
M x N 행렬</pre>

<ul>
	<li>1 &lt;= M &lt;= 1000</li>
	<li>1 &lt;= N &lt;= 1000</li>
</ul>

### 출력

<pre>
L</pre>
