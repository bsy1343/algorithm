### 문제 설명

<p>여러 가지 블록들을 이용하여 직사각형 모양을 만들려고 한다. 우리에게는&nbsp;1&thinsp;&times;&thinsp;<em>N</em>&nbsp;블록,&nbsp;2&thinsp;&times;&thinsp;<em>N</em>&nbsp;블록, ...,&nbsp;<em>N</em>&thinsp;&times;&thinsp;<em>N</em>&nbsp;블록이 무한하게 있다. 이 블록들을 사용하여&nbsp;<em>N</em>&thinsp;&times;&thinsp;<em>M</em>&nbsp;모양을 만들고 싶다. 만들 수 있는 총 방법의 수를&nbsp;1999로 나눈 나머지를 구하여라.</p>

### 입력

### 출력

<p>총 가능한 경우의 수를 1999로 나눈 나머지를 출력한다.</p>
