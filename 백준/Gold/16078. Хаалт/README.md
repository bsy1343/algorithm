### 문제 설명

<p>Тодорхой байрлал дахь хаалтууд өгөгдсөн. Нээх &ldquo;(&ldquo;, хаах &ldquo;)&rdquo; хаалтыг &ldquo;.&rdquo;-ийн оронд нэмэх замаар зөв хаалтуудын илэрхийлэл үүсгэнэ. Бүх зөв илэрхийллийн тоог ол. &nbsp;</p>

### 입력

### 출력
