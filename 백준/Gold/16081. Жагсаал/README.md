### 문제 설명

<p>Цэргийн ангид n тооны цэрэг байгаа. Цэргүүд жагсах үед цэрэг бүр нэг бол баруун, үгүй бол зүүн гар тал руугаа толгойгоо эргүүлж зогсдог. Мөн цэрэг бүр өөрийн жагсаалд эзлэх байрлалын дугаар болох 1 &ndash; ээс n хүртлэх тоогоор дугаарлагдана. i &ndash; р цэргийн өндөр нь hi. Дараах нөхцлүүд биелж байх үед i &ndash; р цэрэг j &ndash; р цэргийг харж чадна:</p>

<ul>
	<li>i &ndash; р цэрэг j &ndash; р цэргийн зүгт харж байх</li>
	<li>тэр хоёрын дунд байгаа бүх цэргүүд j &ndash; р цэргээс өндөр биш байх.</li>
</ul>

<p>Жишээ нь h<sub>1</sub> = 177, h<sub>2</sub> = 179, h<sub>3</sub> = 171, h<sub>4</sub> = 189 гэсэн өндрүүдтэй дөрвөн цэрэг жагссан байгаа бөгөөд бүгд зүүн гар тийшээгээ харсан байгаа тохиолдолд 2 &ndash; р цэрэг зөвхөн 1 &ndash; р цэргийг харж чадах ба 3 &ndash; р цэрэг зөвхөн 2 &ndash; р цэргийг (учир нь 1 &ndash; р цэргийг илүү өндөр 2 &ndash; р цэрэг хааж байгаа), 4 &ndash; р цэрэг 2 &ndash; р болон 3 &ndash; р цэргийг харж чадна.&nbsp;</p>

<p>Цэрэг бүр хэдэн өөр цэргийг харж чадаж байгааг офицер Бат тогтоох шаардлагатай болсон тул түүнд туслаарай.</p>

### 입력

<p>Эхний мөрөнд жагсаал дахь цэргүүдийн тоо болох n тоо өгөгдөнө (1 &le; n &le; 10<sup>5</sup>).&nbsp;</p>

<p>Хоёр дахь мөрөнд жагсаал дахь цэргүүдийн өндрүүд болох h<sub>1</sub>, h<sub>2</sub>, &hellip;, h<sub>n</sub> гэсэн n ширхэг тоо өгөгдөнө (1 &le; h<sub>i</sub> &le; 10<sup>9</sup>).&nbsp;</p>

<p>Гурав дахь мөрөнд цэрэг бүрийн харах чиглэлийг заах n ширхэг тэмдэгт байрлана: i &ndash; р тэмдэгт &ldquo;L&rdquo; бол i &ndash; р цэрэг зүүн гар тийш харсан байх ба өөрөөр хэлбэл түүний харах боломжтой цэргүүд нь 1, 2, &hellip;, i &ndash; 1 &ndash; р цэргүүд байна. &nbsp;i &ndash; р тэмдэгт &ldquo;R&rdquo; бол i &ndash; р цэрэг баруун гар тийш харсан байх ба өөрөөр хэлбэл түүний харах боломжтой цэргүүд нь i + 1, i + 2, &hellip;, n &ndash; р цэргүүд байна.</p>

### 출력

<p>i &ndash; р тоо нь i &ndash; р цэргийн жагсаалд зогсч байхдаа харж байгаа цэргүүдийн тоо байхаар n ширхэг тоог гаргана.&nbsp;</p>
