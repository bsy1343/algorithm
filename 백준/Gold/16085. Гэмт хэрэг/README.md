### 문제 설명

<p>Тамир бол хотынхоо шилдэг мөрдөгч. Өнөөдөр тэдний хотод нэгэн гэмт хэрэг гарчээ. Гэмт хэргийн сэжигтэн хамгийн сүүлд хотын зам дээр харагдсан гэх мэдээлэл ирэв. Одоо мэдээлэл ирснээс хойш T секундын хугацаа өнгөрсөн байгаа ба Тамир ажиллагаагаа эхлэхийн тулд эхлээд сэжигтний байж болох бүсийн талбайг олох шаардлагатай ажээ. Тамирын амьдардаг хотын машинууд зөвхөн зам дээр л явах боломжтой бөгөөд тэдний хот цорын ганц замтай тэр нь төгсгөлгүй урт шулуун хэлбэртэй. Мөн сэжигтэн хамгийн ихдээ явган W м/с хурдтай, машинтай C м/с хурдтай явж чадах ба зам дээр хаанаас ч машин олдох боломжтой, машинаас бууж суухдаа хугацаа зарцуулдаггүй.&nbsp;</p>

<p>Таны даалгавар бол Тамирт сэжигтний байж болох газрын талбайг олж өгөх явдал юм.</p>

### 입력

<p>Эхний мөрөнд тестийн тоог илэрхийлэх N &le; 100 бүхэл тоо өгөгдөнө.</p>

<p>Дараагийн N мөрөнд T, W, C бүхэл тоонууд өгөгдөнө. 1 &le; T, W, C &le; 100&nbsp;</p>

### 출력

<p>Тест бүрт харгалзах хариу болох талбайг м2 -аар илэрхийлсэн тоог 3 орны нарийвчлалтай хэвлэ.</p>
