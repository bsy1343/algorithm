### 문제 설명

<p>Write program sequences to count the number of all non-decreasing sequences of length n containing integers from 1 to m, where every element can occur at most k times.</p>

### 입력

<p>On the standard input, there are given integers n, m and k, separated by spaces.</p>

### 출력

<p>On the standard output, your program has to print the count of the described sequences.&nbsp;</p>

### 제한

<ul>
	<li>0 &lt; n &lt; 31</li>
	<li>0 &lt; m &lt; 31</li>
	<li>0 &lt; k &lt; 31.</li>
</ul>

### 힌트

<p>The sequences are: (1,1,2), (1,1,3), (1,1,4), (1,2,2), (1,2,3), (1,2,4), (1,3,3), (1,3,4), (1,4,4), (2,2,3), (2,2,4), (2,3,3), (2,3,4), (2,4,4), (3,3,4), (3,4,4).</p>
