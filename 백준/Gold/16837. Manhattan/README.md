### 문제 설명

<p>In Manhattan, there are streets x = i and y = i for each integer i. It is known that both Snuke&rsquo;s house and Smeke&rsquo;s house are on streets, and the Euclidean distance between them is exactly d. Compute the maximal possible distance between their houses when they travel along streets.</p>

### 입력

<p>The input contains one number d.</p>

<ul>
	<li>0 &lt; d &le; 10</li>
	<li>d contains exactly three digits after the decimal point.</li>
</ul>

### 출력

<p>Print the answer. The answer is considered to be correct if its absolute or relative error is at most 10<sup>&minus;9</sup>.</p>
