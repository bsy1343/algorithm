### 문제 설명

<p>성씨가 &#39;씨&#39;인 이 혈통은 전 세계에 총&nbsp;N명이 존재한다. 2019년 5월 11일, &#39;씨&#39;씨 혈통의 모든 사람들이 한 자리에 모인 역사적인 날이다.</p>

<p>하지만 누군가에게는 고통스러운 날이다. 성씨가 씨이고 이름이 씨인 &#39;씨씨&#39;는 모임이 끝나면 &#39;씨&#39;씨 혈통의 족보를 완성해야하기 때문이다.&nbsp;<br />
&#39;씨&#39;씨 혈통의 모든 사람들은 시크릿하기 때문에 자신을 직접 소개하기를 무척 꺼려한다. 족보를 완성해야만 하는 &#39;씨씨&#39;는 총&nbsp;M개의 대화를 엿들어 두 사람이 부르는 호칭을 듣고 그 촌수를 알아낼 수 있었다.&nbsp;</p>

<p>내일은 큰 어르신께서 족보를 인쇄하기 전에 검사하러 오실 예정인데, 어떤 사람&nbsp;a와&nbsp;b가 몇 촌인지를 빠르게 맞춰야 검사를 통과할 수 있다. 편의상 사람은 번호로 나타내며, 오늘 모임에 참석한 순서대로 1번부터&nbsp;N번까지 있다.</p>

<p>만약 검사를 하나라도 통과하지 못한다면 가문에서 쫓겨날 위기에 처해있다. 아직 모든 연결 관계를 외우지 못한 &#39;씨씨&#39;는 이대로라면 위험하다. &#39;씨씨&#39;가&nbsp;Q번의 검사를 모두 통과할 수 있도록 도와주자.</p>

### 입력

<p>첫째 줄에 이 성씨의 인구&nbsp;N과 엿들은 대화의 수&nbsp;M이 주어진다.&nbsp;(2&nbsp;&le;&nbsp;N &le; 200,&nbsp;1 &le; M &le; 20,000)</p>

<p>다음&nbsp;MM개의 줄에 걸쳐 두 사람&nbsp;a와&nbsp;b, 그리고 두 사람의 촌 수&nbsp;k가 주어진다. (1 &le; a, b &le; N,&nbsp;1 &le; k &le; 10)</p>

<p>같은 대화를 여러 번 들을 수도 있으며, 자기 자신과 대화하는 혼잣말을 엿듣는 경우는 없다. (대화 중에 두 사람의 촌 수가 변하는 경우는 없습니다.)</p>

<p>검사 횟수&nbsp;Q가 주어지고, 다음&nbsp;Q(1 &le; Q &le; 100,000)개의 줄에 촌 수를 맞춰야 하는 두 사람&nbsp;x,&nbsp;y (x&ne;y)가 주어진다</p>

### 출력

<p>Q개의 줄에 걸쳐, 각 검사의 정답을 출력한다. 촌 수를 알 수 없는 경우에는 -1을 출력한다.</p>
