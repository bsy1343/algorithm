# [Gold I] 홀수 부분열 - 17279

[문제 링크](https://www.acmicpc.net/problem/17279)

### 성능 요약

시간 제한: 3 초, 메모리 제한: 512 MB

### 통계

제출: 177, 정답: 74, 맞힌 사람: 54, 정답 비율: 42.520%

### 분류

다이나믹 프로그래밍, 배낭 문제

### 문제 설명

<p>길이 n인 음이 아닌 정수로 이루어진 배열 A에 대해,&nbsp;0개 이상의 원소를 지워서 얻을 수 있는 배열을 &quot;부분열&quot;라고 한다.</p>

<p>배열 A의 부분열 S에 대해, S가 다음 조건을 만족하면 S를 A의 &quot;홀수 부분열&quot;이라고 정의한다:</p>

<ul>
	<li>S의 원소의 합의 자릿수 중 홀수인 것들이 홀수이다.</li>
</ul>

<p>음이 아닌 정수 배열 A를 입력받아 A의 서로 다른 홀수 부분열의 개수를 출력하시오. 단, 부분열의 원소들을 정렬했을 시 그 결과가 같다면 같은 부분열로 취급한다.</p>

<p>A = [3, 3, 6, 8, 6] 경우 정답은 8이다.</p>

<ul>
	<li>길이가 0인 홀수 부분열: X</li>
	<li>길이가 1인 홀수 부분열: [3]
	<ul>
		<li>합이 3, 자릿수 중 홀수가 3으로 하나</li>
	</ul>
	</li>
	<li>길이가 2인 홀수 부분열: [3, 6], [6, 6], [6, 8]
	<ul>
		<li>합이 각각 9, 12, 14로 자릿수 중 홀수가 9, 1, 1로 하나씩</li>
	</ul>
	</li>
	<li>길이가 3인 홀수 부분열: [3, 3, 6], [3, 3, 8]</li>
	<li>길이가 4인 홀수 부분열: [3, 3, 6, 6], [3, 6, 6, 8]</li>
	<li>길이가 5인 홀수 부분열: X</li>
</ul>

### 입력

<p>첫 줄에 테스트 케이스의 수 T가 주어진다 (1 &le;&nbsp;T &le; 10).</p>

<p>각 테스트 케이스에 대해 첫 줄에 A의 길이 n 이 주어진다. 둘째 줄에 n개의 0이상의 정수가 공백으로 구분되어 주어진다. A의 각 원소는 0 이상 2,000 이하이다.</p>

### 출력

<p>각 테스트 케이스에 대해 홀수 부분열의 수를 한 줄에 출력한다.</p>

### 힌트

<ul>
	<li>테스트 케이스 1: {3}, {3,6}, {3, 3, 6} 세 종류의 홀수-부분열이 있다.</li>
	<li>테스트 케이스 2: {1}, {3}, {0,1}, {0,3}, {1,2}, {2,3}, {0,1,2}, {0,2,3} 여덟 종류의 홀수-부분열이 있다.</li>
	<li>테스트 케이스 3: 문제에서 서술된대로 여덟 종류의 홀수-부분열이 있다.</li>
</ul>
