### 문제 설명

<p>Dla liczby naturalnej n, przez f(n) oznaczymy sumę kwadrat&oacute;w jej cyfr w zapisie dziesiętnym. Dane są trzy liczby naturalne k, a, b. Twoim zadaniem jest wyznaczenie liczby takich naturalnych n, że a &le; n &le; b oraz n jest rozwiązaniem r&oacute;wnania</p>

<p style="text-align: center;">k &middot; f(n) = n.</p>

### 입력

<p>Pierwszy i jedyny wiersz wejścia zawiera trzy liczby całkowite z treści zadania: k, a, b, (1 &le; k, a, b &le; 10<sup>18</sup>, a &le; b).</p>

### 출력

<p>Tw&oacute;j program powinien wypisać jedną liczbę całkowitą będącą liczbą rozwiązań r&oacute;wnania z treści w przedziale [a, b].</p>

### 힌트

<p>Jedynymi liczbami naturalnymi n z przedziału [5000, 10000] spełniającymi r&oacute;wnanie dla k = 51 są 7293, 7854 i 7905.</p>
