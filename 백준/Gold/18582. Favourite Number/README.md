### 문제 설명

<p>Volodymyr&rsquo;s favourite number is A and it has an odd number of positive divisors. When you add K to this number, the resulting sum also has an odd number of positive divisors. Given K, find all possible values of A, Volodymyr&rsquo;s favourite number.</p>

### 입력

<p>The only line of input contains K, a positive integer not exceeding 10<sup>9</sup>.</p>

### 출력

<p>On the first line, print the number of possible values of A. On the second line, print all possible values of A in ascending order, separated by a single space.</p>

### 힌트

<p>Note that since zero has infinitely many divisors, it cannot be classified as a number with an odd number of divisors.</p>
