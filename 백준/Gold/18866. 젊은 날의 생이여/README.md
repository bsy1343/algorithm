### 문제 설명

<p>욱제는 입대를 앞두고 &lt;이등병의 편지&gt;를 부르고 있다.</p>

<p style="text-align: center;"><em>&hellip; 이제 다시 시작이다 젊은 날의 생이여 &hellip;&nbsp;</em></p>

<p>그런데 과연 욱제에게 젊은 날이 있었을까?&nbsp;욱제에게 젊은 날이 있었는지 알아보자.</p>

<p>먼저, N년간 욱제의 행복도와 피로도가 주어진다.&nbsp;행복도와 피로도는 양의 실수 값을 가진다.&nbsp;어떤 1 &le;&nbsp;K &lt; N에 대해 1, 2, &hellip;, K년은 욱제의 젊은 날이고, K + 1, K + 2, &hellip;, N년은 욱제의 늙은 날이다.</p>

<p>젊은 날과 늙은 날은 다음의 조건을 만족한다:</p>

<ul>
	<li>임의의 젊은 날의 행복도는 임의의 늙은 날의 행복도보다 높다.</li>
	<li>임의의 젊은 날의 피로도는 임의의 늙은 날의 피로도보다 낮다.</li>
</ul>

<p>욱제는 자신의 행복도와 피로도를 이용하여 자신의 젊은 날을 알아보려 한다. 하지만, 일부 값이 누락되었다. 욱제를 도와주자.</p>

### 입력

<p dir="ltr">첫째 줄에 N이 주어진다.</p>

<p dir="ltr">둘째 줄부터 N개의 줄에는 두 <strong>정수</strong> u<sub>i</sub>, v<sub>i</sub>&nbsp;(1 &le; i &le; N)가&nbsp;주어진다. u<sub>i</sub>, v<sub>i</sub>가 1 이상일 경우 각각 i년의 행복도와 피로도를 나타내며, 0인 경우에는 값이 누락되었음을 의미한다.</p>

### 출력

<p>욱제의 젊은 날이 될 수 있는&nbsp;최대 기간, 즉 문제의 조건을 만족할 수 있는 최대의 1&nbsp;&le; K&nbsp;&lt;&nbsp;N을 출력한다. 단, 이러한 K가 없을 경우, -1을 출력한다.</p>

### 제한

<ul>
	<li>2&nbsp;&le;&nbsp;N &le;&nbsp;1,000,000</li>
	<li>0 &le; u<sub>i</sub>, v<sub>i</sub> &le; 10<sup>9</sup></li>
	<li>주어지는 N개의 행복도 중 0이 아닌 값은&nbsp;모두 다르다.</li>
	<li>주어지는 N개의 피로도 중 0이 아닌 값은&nbsp;모두 다르다.</li>
</ul>
