### 문제 설명

<p>Greedy Smurf is opening a new shop in Smurf Village. &nbsp;Smurfs use coins with four denominations: 1, 5, 10 and 25 SmurfCoins. Write a program that will compute for Greedy the number of ways that he can give change of $n$ SmurfCoins.</p>

<p>Output the number of different ways of giving change modulo $10^9 + 7$. &nbsp;Two ways of giving change are considered different if they differ in the amount of used coins of some denomination.</p>

### 입력

<p>First and only input line contains $n$ ($1 \leq n \leq 10^{18}$) -- the amount of change.</p>

### 출력

<p>Output the number of different ways of giving change modulo $10^9 + 7$. &nbsp;Two ways of giving change are considered different if they differ in the amount of used coins of some denomination.</p>
