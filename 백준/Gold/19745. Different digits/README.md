### 문제 설명

<p>Senya likes integers not containing two consecutive equal digits. You are given an integer $n$. Help Senya to find the smallest integer he likes, which is strictly greater than $n$.</p>

### 입력

<p>Input contains integer $n$ ($1 \le n \le 10^{18}$).</p>

### 출력

<p>Print the smallest integer strictly greater than $n$, which doesn&#39;t contain any two consecutive equal digits.</p>
