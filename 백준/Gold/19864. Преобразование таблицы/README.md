<p>После выполнения всех операций Ваня вычисляет контрольную сумму для таблицы: сумма по всем ячейкам $[i, j]$ значений $(v[i][j] \times 17^i \times 19^j) \bmod (10^9 + 7)$. Здесь $v[i][j]$ означает значение в ячейке $[i, j]$, а операция &lt;&lt;$\bmod$&gt;&gt; означает операцию взятия остатка. Например, контрольная сумма для таблицы из примера вычисляется следующим образом: $(6 \times 17^0 \times 19^0 + 2 \times 17^0 \times 19^1 + \ldots + 14 \times 17^2 \times 19^4) \bmod (10^9 + 7) = 564\,830\,737$.</p>

<p>Помогите Ване выполнить все операции и вычислить контрольную сумму таблицы, которая получится в итоге.</p>

<p>Поскольку входные данные для этой задачи слишком велики, чтобы задавать их непосредственно, для ввода вам потребуется процедура <em>расширения массива</em>. Эта процедура не имеет специфических особенностей, которые надо использовать в решении задачи, предполагаемое жюри решение этой задачи в явном виде генерирует все массивы и далее работает с ними так же, как если бы оно считало их из входных данных.</p>

<p>Опишем процедуру генерации массива длины $n$ по массиву длины $2 \le k \le n$. Пусть задан массив целых неотрицательных чисел $A = (a[1], a[2], \ldots, a[k])$. Массив целых чисел $Ax = (ax[1], ax[2], \ldots, ax[n])$ будем называть расширением массива $A$ по модулю $r$ до размера $n$, если его элементы вычисляются по следующим формулам.</p>

<ul>
	<li>Если $1 \le i \le k$, то $ax[i] = a[i]$.</li>
	<li>Если $k + 1 \le i \le n$, то $ax[i] = (10007 \times ax[i - 2] + 10009 \times ax[i - 1] + 87277) \bmod r$.</li>
</ul>

<p>Здесь как &lt;&lt;mod&gt;&gt; также обозначена операция взятия остатка по модулю $r$.</p>

<p>Например, выполним расширение массива $A=(1, 4, 3)$ до 5 элементов по модулю 13.</p>

<ul>
	<li>$ax[1] = a[1] = 1$.</li>
	<li>$ax[2] = a[2] = 4$.</li>
	<li>$ax[3] = a[3] = 3$.</li>
	<li>$ax[4] = (10007 \times ax[2] + 10009 \times ax[3] + 87277) \bmod 13 = 157332 \bmod 13 = 6$.</li>
	<li>$ax[5] = (10007 \times ax[3] + 10009 \times ax[4] + 87277) \bmod 13 = 177352 \bmod 13 = 6$.</li>
</ul>

<p>Таким образом, $Ax = (1, 4, 3, 6, 6)$.</p>

### 입력

<p>Первая строка ввода содержит числа $h$, $w$, $n$ --- размеры таблицы и число преобразований, которые необходимо выполнить ($1 \le h, w \le 5\,000$, $2 \le n \le 10^6$).</p>

<p>Вторая строка содержит строку $s$ длины $n$ --- последовательность типов преобразований, $s[i]$ = &lt;&lt;<code>c</code>&gt;&gt; задает операцию обмена столбцов, $s[i]$ = &lt;&lt;<code>r</code>&gt;&gt; --- операцию обмена строк, $s[i]$ = &lt;&lt;<code>f</code>&gt;&gt; --- операцию обмена ячеек.</p>

<p>Следующие четыре строки задают массивы $A$, $B$, $C$, $D$, соответственно. Каждый массив задается числом $k$, $2 \le k \le n$, $k \le 1000$, после чего следует $k$ чисел --- элементы массива. Элементы массивов $A$ и $C$ удовлетворяют ограничению $0 \le a[i], c[i] \le h - 1$, элементы массивов $B$ и $D$ удовлетворяют ограничению $0 \le b[i], d[i] \le w - 1$.</p>

<p>Пусть $Ax$ является расширением массива $A$ по модулю $h$ до размера $n$, массив $Bx$ является расширением массива $B$ по модулю $w$ до размера $n$, массив $Cx$ является расширением массива $C$ по модулю $h$ до размера $n$ и массив $Dx$ является расширением массива $D$ по модулю $w$ до размера $n$.</p>

<p>Операции, которые требуется выполнить с таблицей, определяются следующим образом: тип $i$-й операции задается символом $s[i]$, а параметры получаются из массивов $Ax$, $Bx$, $Cx$, $Dx$.</p>

<ul>
	<li>Если $s[i]$ = &lt;&lt;<code>c</code>&gt;&gt;, то $i$-я операция &lt;&lt;<code>c</code> $Bx[i]$ $Dx[i]$&gt;&gt;;</li>
	<li>Если $s[i]$ = &lt;&lt;<code>r</code>&gt;&gt;, то $i$-я операция &lt;&lt;<code>r</code> $Ax[i]$ $Cx[i]$&gt;&gt;;</li>
	<li>Если $s[i]$ = &lt;&lt;<code>f</code>&gt;&gt;, то $i$-я операция &lt;&lt;<code>f</code> $Ax[i]$ $Bx[i]$ $Cx[i]$ $Dx[i]$&gt;&gt;;</li>
</ul>

### 출력

<p>Выведите одно число --- контрольную сумму таблицы после выполнения всех преобразований.</p>
