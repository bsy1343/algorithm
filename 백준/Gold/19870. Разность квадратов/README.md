### 문제 설명

<p>Вы участвуете в разработке программного модуля для системы символьных вычислений. Модуль будет использоваться для решения специального вида диофантовых уравнений.</p>

<p>По заданному целому неотрицательному целому числу n разрабатываемый модуль должен найти два натуральных числа x и y, для которых выполнено равенство x<sup>2</sup> &minus; y<sup>2</sup> = n. Найденные числа не должны превышать 2<sup>62</sup> &minus; 1.</p>

<p>Требуется написать программу, которая по заданному целому неотрицательному числу n находит натуральные числа x и y, не превышающие 2<sup>62</sup> &minus; 1, разность квадратов которых равна n.</p>

### 입력

<p>В единственной строке дано одно целое число n (0 &le; n &le; 2<sup>60</sup>).</p>

<p>Обратите внимание, что заданное во вводе число не помещается в 32-битный тип данных, необходимо использовать 64-битный тип данных (например, &laquo;long long&raquo; в C++, &laquo;int64&raquo; в паскале, &laquo;long&raquo; в Java).</p>

### 출력

<p>Если искомые x и y существуют, то необходимо вывести две строки: в первой строке выведите слово &laquo;Yes&raquo;, а во второй &mdash; искомые x и y.</p>

<p>Если подходящих пар x и y несколько, можно вывести любую из них, но должно выполняться условие 1 &le; x, y &le; 2<sup>62</sup> &minus; 1.</p>

<p>Если решения нет, в единственной строке необходимо вывести слово &laquo;No&raquo;.</p>
