### 문제 설명

<p>Ваня, Сережа и Дима любят физические нагрузки. Больше всего им нравится поднимать тяжести. Ребята очень давно начали заниматься гиревым спортом и уже собрали набор гирь для тренировок. Этот набор состоит из $n$ гирь с различными целыми весами от 1 до $n$.</p>

<p>Совсем недавно ребята нашли новый спортивный зал и решили перенести туда все свои гири. Так как им очень нравится их поднимать, то каждому хочется нести как можно больший вес. Но ребята очень честные, и весь вес решили распределить поровну.</p>

<p>Помогите спортсменам разделить набор из $n$ гирь с весами $1, 2, \ldots, n$ на три равные по весу части.</p>

### 입력

<p>Входной файл содержит единственное целое число $n$ ($1 \leq n \leq 100\,000$) --- количество гирь.</p>

### 출력

<p>Выведите для каждого спортсмена набор гирь, который ему нужно перенести в новый спортивный зал. Наборы выводятся следующим образом. В первой строке выведите количество гирь в наборе. Далее, во второй строке через пробел выведите веса гирь.</p>

<p>Если разбить все гири на три равных по весу множества нельзя, выведите &lt;&lt;<code>Impossible</code>&gt;&gt;.</p>

<p>Если существует несколько решений, можно вывести любое.</p>
