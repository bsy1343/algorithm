### 문제 설명

<p>ㅋㅋ루ㅋㅋ 문자열은 다음과 같이 정의한다.</p>

<ol>
	<li><code>R</code>로만 이루어진 문자열은 ㅋㅋ루ㅋㅋ 문자열이다. 단, 빈 문자열은 ㅋㅋ루ㅋㅋ 문자열이 아니다.</li>

</ol>

### 입력

<p>첫째 줄에 <code>K</code>와<code><font face="sans-serif, Arial, Verdana, Trebuchet MS, Apple Color Emoji, Segoe UI Emoji, Segoe UI Symbol">&nbsp;</font>R</code>로만 이루어진 문자열이 주어진다. 문자열의 길이는 최대 3,000,000이다.</p>

### 출력

### 힌트

<p>어떤 문자열에서 몇 개의 문자를 지워서&nbsp;부분 수열을 만들 수 있다. 예를 들어, ABC의 부분 수열은 ABC, AB, BC, AC, A, B, C와 빈 문자열이다.</p>
