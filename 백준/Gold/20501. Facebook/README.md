### 문제 설명

<p>&quot;마, 내가 누군지 아나? 저커버그가 내 후임이다.&quot;</p>

<p>준원이는 저커버그와 함께 페이스북을 만들던 리즈시절을 회상하곤 한다...&nbsp;저커버그가 페이스북을 다 만들기 직전,&nbsp;바로 그 순간 저커버그에게 급똥이 찾아왔다. 그는 마지막 남은 기능 하나를 준원이에게 맡겼고, 그 기능이 바로 &#39;함께 아는 친구&#39; 기능이었다.</p>

<p>페이스북의 사용자는 총 N명이고,&nbsp;각 사용자는 1번에서 N번까지로 번호 붙어 있었다. 준원이는, 사용자의 친구관계에 대한 정보가 모두 주어졌을 때,</p>

<p style="text-align: center;">&quot;a번 사용자, b번 사용자와 공통적으로 친구 관계인 사용자의 수&quot;</p>

<p>를 묻는 Q개의 질문에 차례로 답해야 했다.</p>

<p>준원이는 천하코일제딩대회 참가자 여러분에게 이 기능을 외주 맡겼다. 여러분이 이 기능을 대신 구현하면, 준원이는 여러분이 천하코일대딩제회에서 맞은 문제를 몰래 하나 늘여주겠다고 한다. 어떤가?</p>

### 입력

<p>첫째 줄에, 사용자의 수를 나타내는 정수 N이 주어진다.</p>

<p>이후 N개의 줄에 걸쳐, 각 사람의 친구관계를 나타내는 길이 N의 문자열이 N개 주어진다.&nbsp;i번째 줄의 j번째 문자는, i번 사용자가 j번 사용자와 친구일 때에 <code>1</code>, 아닐 때에 <code>0</code>이다.</p>

<p>이후, 질문의 개수를 나타내는 정수 Q가 주어진다.</p>

<p>이후 Q개의 줄에 걸쳐, 질문을 나타내는 두 정수 a와 b가 공백을 사이에 두고 주어진다.</p>

### 출력

<p>Q개의 줄에 걸쳐, 각 질문에 대한 답을 나타내는 정수를 하나씩 차례대로 출력한다.</p>

### 제한

<ul>
	<li>2 &le;&nbsp;N &le;&nbsp;2,000</li>
	<li>친구 관계는 대칭적이다. 즉, i번 사용자가 j번 사용자와 친구이면, j번 사용자는 i번 사용자와 친구이다.</li>
	<li>자기 자신과 친구인 사용자는 없다.</li>
	<li>1 &le;&nbsp;Q &le; 500,000</li>
	<li>1 &le;&nbsp;a, b &le;&nbsp;N</li>
	<li>a &ne;&nbsp;b</li>
</ul>

### 힌트

<p>입출력 양이 많다. 느린 입출력 함수를 쓰면 입력 받는 데에 걸리는 시간만으로 시간 초과가 발생할 수 있으므로 유의하라.</p>

<ul>
	<li>C++에서는, <code>cin</code>, <code>cout</code>&nbsp;함수를 가급적 쓰지 말고, <code>scanf</code>, <code>printf</code>&nbsp;함수로 입출력을 진행하는 편이 좋다.</li>
	<li>Java에서는, <code>Scanner</code>&nbsp;클래스 대신, <code>BufferedReader</code>&nbsp;클래스를 사용하여 입력받는 편이 좋다.</li>
	<li>Python에서는, <code>input</code> 함수를 쓰는 대신, <code>import sys</code>를 코드의 앞 부분에서 선언하면 같은 역할을 하는 더 빠른 함수인 <code>sys.stdin.readline</code>을 쓸 수 있다.</li>
</ul>
