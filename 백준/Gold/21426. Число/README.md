### 문제 설명

<p>По заданным числам $a$, $b$, $c$ и $d$, найдите наименьшее натуральное число $n$, большее $ac$, которое нельзя представить в виде произведения двух натуральных чисел $u$ и $v$, таких что $a \le u \le b$ и $c \le v \le d$.</p>

### 입력

<p>Входной файл содержит одну строку, состоящую из натуральных чисел $a$, $b$, $c$, $d$ ($1 \le a \le b \le 10^6$, $1 \le c \le d \le 10^6$).</p>

### 출력

<p>Выведите в выходной файл искомое число $n$.</p>
