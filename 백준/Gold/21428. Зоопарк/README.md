### 문제 설명

<p>В городском зоопарке содержатся животные $n$ разных видов. Для участия в международной выставке &lt;&lt;Три твари&gt;&gt; зоопарк должен представить трех животных различных видов. Теперь служителей зоопарка интересует, сколькими способами можно выбрать трех животных для участия в выставке.&nbsp;</p>

<p>Например, если в зоопарке два медведя, тигр, лев и пингвин, то есть семь способов выбрать трех животных:</p>

<ol>
	<li>первый медведь, тигр и лев;</li>
	<li>первый медведь, тигр и пингвин;</li>
	<li>первый медведь, лев и пингвин;</li>
	<li>второй медведь, тигр и лев;</li>
	<li>второй медведь, тигр и пингвин;</li>
	<li>второй медведь, лев и пингвин;</li>
	<li>тигр, лев и пингвин.</li>
</ol>

### 입력

<p>В первой строке входного файла содержится натуральное число $n$ --- количество видов животных в городском зоопарке ($1 \le n \le 10^5$).</p>

<p>В каждой из следующих $n$ строк содержится одно натуральное число --- количество животных соответствующего вида. Общее число животных в зоопарке не превышает $10^5$.</p>

### 출력

<p>В выходной файл выведите количество способов выбрать трех животных для международной выставки.</p>
