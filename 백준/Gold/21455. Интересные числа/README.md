### 문제 설명

<p>Роман коллекционирует числа, кажущиеся ему интересными. Например, сейчас он считает <em>интересным</em> положительные числа, запись которых в системе счисления с основанием $k$ заканчивается нечетным числом нулей. Например, при $k=2$ такими числами являются $2_{10}=10_2$, $24_{10}=11000_{2}$.</p>

<p>Для того, чтобы пополнить свою коллекцию, Роман хочет найти $n$-ое в порядке возрастания такое число. Поскольку $n$ он взял достаточно большим, то вручную у него это сделать не получается.&nbsp;</p>

<p>Помогите Роману --- напишите программу, которая найдет число, которое нужно ему для пополнения коллекции.&nbsp;</p>

### 입력

<p>Первая строка входного файла содержит два целых числа ($1 \le n \le 10^{15}$, $2 \le k \le 10$).</p>

### 출력

<p>В выходной файл выведите $n$-ое в порядке возрастания число, запись которого в системе счисления с основанием $k$ заканчивается на нечетное число нулей. Это число необходимо вывести в десятичной системе счисления.</p>
