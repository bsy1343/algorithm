### 문제 설명

<p>На далекой планете Тау Кита есть непонятные нам обычаи. Например, таукитяне очень необычно для землян выбирают имена своим детям. Родители так выбирают имя ребенку, чтобы оно могло быть получено как удалением некоторого набора букв из имени отца, так и удалением некоторого набора букв из имени матери. Например, если отца зовут &laquo;abacaba&raquo;, а мать &mdash; &laquo;bbccaa&raquo;, то их ребенок может носить имена &laquo;a&raquo;, &laquo;bba&raquo;, &laquo;bcaa&raquo;, но не может носить имена &laquo;aaa&raquo;, &laquo;ab&raquo; или &laquo;bbc&raquo;. Возможно, что имя ребенка совпадает с именем отца и/или матери, если оно может быть получено из имени другого родителя удалением нескольких (возможно, ни одной) букв.</p>

<p>Пусть отец по имени X и мать по имени Y выбирают имя своему новорожденному ребенку. Так как в таукитянских школах учеников часто вызывают к доске в лексикографическом порядке имен учеников, то есть в порядке следования имен в словаре, то они хотят выбрать своему ребенку такое имя, чтобы оно лексикографически следовало как можно позже.</p>

<p>Формально, строка S лексикографически больше строки T, если выполняется одно из двух условий:</p>

<ul>
	<li>строка T получается из S удалением одной или более букв с конца строки S;</li>
	<li>первые (i - 1) символов строк T и S не различаются, а буква в i-й позиции строки T следует в алфавите раньше буквы в i-й позиции строки S.</li>
</ul>

<p>Требуется написать программу, которая по именам отца и матери находит лексикографически наибольшее имя для их ребенка.&nbsp;</p>

### 입력

<p>Первая строка входного файла содержит X &mdash; имя отца. Вторая строка входного файла содержит Y &mdash; имя матери. Каждое имя состоит из строчных букв латинского алфавита, включает хотя бы одну букву и имеет длину не более 10<sup>5</sup> букв.&nbsp;</p>

### 출력

<p>Выходной файл должен содержать искомое лексикографически наибольшее из возможных имен ребенка. В случае, если подходящего имени для ребенка не существует, выходной файл должен быть пустым.&nbsp;</p>

### 힌트

<p>В первом примере имя ребенка не может начинаться с буквы большей с, так как имя отца не содержит таких букв. Буква с содержится в обоих именах, следовательно, имя ребенка может начинаться с этой буквы. Единственная буква, которая может идти следом за буквой с в имени ребенка &mdash; это буква a.&nbsp;</p>
