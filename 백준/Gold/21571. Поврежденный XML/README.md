### 문제 설명

<p>Формат XML является распространенным способом обмена данными между различными программами. Недавно программист Иванов написал небольшую программу, которая сохраняет некоторую важную информацию в виде XML-строки.</p>

<p>XML-строка состоит из открывающих и закрывающих тегов.</p>

<p>Открывающий тег начинается с открывающей угловой скобки (<code>&lt;</code>), за ней следует имя тега &mdash; непустая строка из строчных букв латинского алфавита, а затем закрывающая угловая скобка (<code>&gt;</code>). Примеры открывающих тегов: <code>&lt;a&gt;</code>, <code>&lt;dog&gt;</code>.</p>

<p>Закрывающий тег начинается с открывающей угловой скобки, за ней следует прямой слеш (<code>/</code>), затем имя тега &mdash; непустая строка из строчных букв латинского алфавита, а затем закрывающая угловая скобка. Примеры закрывающихся тегов: <code>&lt;/a&gt;</code>, <code>&lt;/dog&gt;</code>.</p>

<p>XML-строка называется корректной, если она может быть получена по следующим правилам:</p>

<ul>
	<li>Пустая строка является корректной XML-строкой.</li>
	<li>Если A и B &mdash; корректные XML-строки, то строка AB, получающаяся приписыванием строки B в конец строки A, также является корректной XML-строкой.</li>
	<li>Если A &mdash; корректная XML-строка, то строка &lt;X&gt;A&lt;/X&gt;, получающаяся приписыванием в начало A открывающегося тега, а в конец &mdash; закрывающегося с таким же именем, также является корректной XML-строкой. Здесь X &mdash; любая непустая строка из строчных букв латинского алфавита.</li>
</ul>

<p>Например, представленные ниже строки:&nbsp;</p>

<pre style="text-align: center;">
&lt;a&gt;&lt;/a&gt;
&lt;a&gt;&lt;ab&gt;&lt;/ab&gt;&lt;c&gt;&lt;/c&gt;&lt;/a&gt;
&lt;a&gt;&lt;/a&gt;&lt;a&gt;&lt;/a&gt;&lt;a&gt;&lt;/a&gt;</pre>

<p>являются корректными XML-строками, а такие строки как:</p>

<pre style="text-align: center;">
&lt;a&gt;&lt;/b&gt;
&lt;a&gt;&lt;b&gt;
&lt;a&gt;&lt;b&gt;&lt;/a&gt;&lt;/b&gt;</pre>

<p>не являются корректными XML-строками.</p>

<p>Иванов отправил файл с сохраненной XML-строкой по электронной почте своему коллеге Петрову. Однако, к сожалению, файл повредился в процессе пересылки: ровно один символ в строке заменился на некоторый другой символ.</p>

<p>Требуется написать программу, которая по строке, которую получил Петров, восстановит исходную XML-строку, которую отправлял Иванов.</p>

### 입력

<p>Входной файл содержит одну строку, которая заменой ровно одного символа может быть превращена в корректную XML-строку. Длина строки лежит в пределах от 7 до 1000, включительно. Строка содержит только строчные буквы латинского алфавита и символы &laquo;<code>&lt;</code>&raquo; (ASCII код 60), &laquo;<code>&gt;</code>&raquo;(ASCII код 62) и &laquo;<code>/</code>&raquo;(ASCII код 47).&nbsp;</p>

<p>Строка во входном файле заканчивается переводом строки.</p>

### 출력

<p>Выходной файл должен содержать корректную XML-строку, которая может быть получена из строки во входном файле заменой ровно одного символа на другой. Если вариантов ответа несколько, можно вывести любой.&nbsp;</p>
