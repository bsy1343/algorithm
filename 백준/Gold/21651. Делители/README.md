### 문제 설명

<p>Натуральное число a называется делителем натурального числа b, если b = ac для некоторого натурального числа c. Например, делителями числа 6 являются числа 1, 2, 3 и 6. Два числа называются взаимно простыми, если у них нет общих делителей кроме 1. Например, 16 и 27 взаимно просты, а 18 и 24 &mdash; нет.</p>

<p>Будем называть нормальным набор из k чисел (a<sub>1</sub>, a<sub>2</sub>, &hellip;, a<sub>k</sub>), если выполнены следующие условия:</p>

<ol>
	<li>каждое из чисел ai является делителем числа n;&nbsp;</li>
	<li>выполняется неравенство a<sub>1</sub> &lt; a<sub>2</sub> &lt; &hellip; &lt; a<sub>k</sub>;</li>
	<li>числа a<sub>i</sub> и a<sub>i+1</sub> для всех i от 1 до k &ndash; 1 являются взаимно простыми;&nbsp;</li>
	<li>произведение a<sub>1</sub>a<sub>2</sub>&hellip;a<sub>k</sub> не превышает n.&nbsp;</li>
</ol>

<p>Например, набор (2, 9, 10) является нормальным набором из 3 делителей числа 360.</p>

<p>Требуется написать программу, которая по заданным значениям n и k определяет количество нормальных наборов из k делителей числа n.</p>

### 입력

<p>Первая строка входного файла содержит два целых числа: n и k (2 &le; n &le; 10<sup>8</sup>, 2 &le; k &le; 10).</p>

### 출력

<p>Первая строка входного файла содержит два целых числа: n и k (2 &le; n &le; 108, 2 &le; k &le; 10).</p>
