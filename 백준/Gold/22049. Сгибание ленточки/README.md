### 문제 설명

<p>Из клетчатой бумаги вырезали полоску размером 1&times;2<sup><i>n</i></sup>. Затем ее положили горизонтально на стол и&nbsp;<i>n</i>&nbsp;раз согнули ровно пополам, каждый раз либо левую половину клали поверх правой (назовем это &laquo;левый поворот&raquo;), либо правую поверх левой (назовем это &laquo;правый поворот&raquo;). В итоге получилась стопка квадратиков 1&times;1. После этого полоску развернули обратно и положили на стол той же стороной, которой она лежала изначально.</p>

<p>Ваша задача теперь состоит в том, чтобы быстро отвечать на запросы, в какую сторону ориентирован&nbsp;<i>i</i>-й сгиб полоски, находящийся между&nbsp;<i>i</i>-м и&nbsp;<i>i</i>+1-м квадратиками 1&times;1, считая слева.</p>

### 입력

<p>Первая строка входных данных содержит одно число&nbsp;<i>t</i>&nbsp;(1&nbsp;&le;&nbsp;<i>t</i>&nbsp;&le;&nbsp;100)&nbsp;&mdash; количество тестов. Далее следуют описания тестов. Каждый тест задается следующим образом: первая строка содержит число&nbsp;<i>n</i>&nbsp;(1&nbsp;&le;&nbsp;<i>n</i>&nbsp;&le;&nbsp;60)&nbsp;&mdash; количество сгибов полоски. Следующая строка теста содержит строку из&nbsp;<i>n</i>&nbsp;символов, каждый из которых описывает левый или правый повороты соответственно (символ &laquo;L&raquo; описывает левый поворот, а &laquo;R&raquo;&nbsp;&mdash; правый).</p>

<p>В следующей строке теста содержится число&nbsp;<i>q</i>&nbsp;(1&nbsp;&le;&nbsp;<i>q</i>&nbsp;&le;&nbsp;10<sup>5</sup>)&nbsp;&mdash; количество запросов. Затем через пробел идут сами запросы, каждый из которых представляется одним числом&nbsp;<i>i</i>&nbsp;(1&nbsp;&le;&nbsp;<i>i</i>&nbsp;&le;&nbsp;2<sup><i>n</i>&nbsp;</sup>&minus;1)&nbsp;&mdash; номер сгиба, ориентацию которого требуется узнать.</p>

<p>Гарантируется, что суммарное количество запросов не превосходит 10<sup>5</sup>.</p>

### 출력

<p>На каждый запрос выведите &laquo;<tt>U</tt>&raquo;, если сгиб ориентирован вверх, и &laquo;<tt>D</tt>&raquo;, если сгиб ориентирован вниз.</p>
