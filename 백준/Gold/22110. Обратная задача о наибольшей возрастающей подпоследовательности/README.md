### 문제 설명

<p>Обратные задачи часто встречаются в исследованиях по теоретической информатике. Обычно обратная задача формулируется так: по заданному решению требуется построить набор входных данных, на котором это решение достигается. В этой задаче вам требуется решить обратную задачу о наибольшей возрастающей подпоследовательности.</p>

<p>Напомним, что&nbsp;<em>возрастающей подпоследовательностью</em>&nbsp;последовательности&nbsp;<i>a</i>[1..<i>n</i>] длины&nbsp;<i>n</i>&nbsp;называют последовательность&nbsp;<i>a</i>[<i>i</i><sub>1</sub>] &lt;&nbsp;<i>a</i>[<i>i</i><sub>2</sub>] &lt; ...&lt;&nbsp;<i>a</i>[<i>i<sub>k</sub></i>], где 1 &le;&nbsp;<i>i</i><sub>1</sub>&nbsp;&lt;&nbsp;<i>i</i><sub>2</sub>&nbsp;&lt; ... &lt;&nbsp;<i>i<sub>k</sub></i>&nbsp;&le;&nbsp;<i>n</i>. Задача о наибольшей возрастающей подпоследовательности (НВП) заключается в том, чтобы найти возрастающую подпоследовательность заданной последовательности, имеющую максимальное число элементов.</p>

<p>При решение задачи о НВП, строится массив&nbsp;<i>d</i>[1..<i>n</i>], где&nbsp;<i>d</i>[<i>i</i>] &mdash; длина наибольшей возрастающей последовательности последовательности&nbsp;<i>a</i>[1..<i>i</i>], заканчивающейся на&nbsp;<i>a</i>[<i>i</i>].</p>

<p>Например, для последовательности&nbsp;<i>a</i>&nbsp;= [3, 2, 4, 1, 5, 6] построенный массив&nbsp;<i>d</i>&nbsp;имеет вид&nbsp;<i>d</i>&nbsp;= [1, 1, 2, 1, 3, 4].</p>

<p>Требуется по заданному массиву&nbsp;<i>d</i>&nbsp;найти такую последовательность&nbsp;<i>a</i>, такую что при решении задачи о НВП построенный массив&nbsp;<i>d</i>&nbsp;совпадает с заданным. Все числа в последовательности&nbsp;<i>a</i>&nbsp;должны быть различными целыми положительными и не превосходить 10<sup>15</sup>.</p>

### 입력

<p>Первая строка содержит целое положительное число&nbsp;<i>t</i>&nbsp;&mdash; число тестовых примеров во входных данных. Далее следуют описания тестовых примеров.</p>

<p>Каждый тестовый пример описывается двумя строками. Первая строка содержит целое положительное число&nbsp;<i>n</i>&nbsp;&mdash; длину последовательности (1&nbsp;&le;&nbsp;<i>n</i>&nbsp;&le;&nbsp;300<sub>&nbsp;</sub>000). Вторая строка содержит&nbsp;<i>n</i>&nbsp;целых положительных чисел &mdash; массив&nbsp;<i>d</i>.</p>

<p>Гарантируется, что сумма значений&nbsp;<i>n</i>&nbsp;по всем тестовым примерам не превышает 300<sub>&nbsp;</sub>000.</p>

### 출력

<p>Для каждого тестового примера выведите&nbsp;<i>n</i>&nbsp;различных целых положительных чисел, не превосходящих 10<sup>15</sup>&nbsp;&mdash; искомую последовательность&nbsp;<i>a</i>. Гарантируется, что входные данные таковы, что искомая последовательность существует. Если возможных решений несколько, можно вывести любое.</p>
