### 문제 설명

<p>Рассмотрим прямоугольную таблицу, составленную из&nbsp;<i>n</i>&times;<i>m</i>&nbsp;клеток. Каждую клетку разрешается покрасить в черный либо в белый цвет. Раскраска называется&nbsp;<em>правильной</em>, если не существует четырех клеток одинакового цвета, центры которых расположены в углах невырожденного прямоугольника со сторонами, параллельными сторонам таблицы.</p>

<p>Требуется посчитать число различных правильных раскрасок прямоугольника. Так как это число может быть большим, необходимо вывести его по заданному модулю&nbsp;<i>r</i>.</p>

<p>Например, для прямоугольника 2&times;2 число таких раскрасок равно 14: подходят все раскраски, кроме тех, у которых все четыре клетки покрашены в один цвет.</p>

### 입력

<p>В первой строке входного файла содержится натуральное число&nbsp;<i>t</i>&nbsp;&mdash; количество тестов (1&nbsp;&le;&nbsp;<i>t</i>&nbsp;&le;&nbsp;150).</p>

<p>В каждой из следующих&nbsp;<i>t</i>&nbsp;строк содержится 3 целых числа:&nbsp;<i>n</i>,&nbsp;<i>m</i>&nbsp;и&nbsp;<i>r</i>&nbsp;&mdash; размеры таблицы и модуль, по которому требуется вывести ответ. (1&nbsp;&le;&nbsp;<i>n</i>,&nbsp;<i>m</i>,&nbsp;<i>r</i>&nbsp;&le;&nbsp;10<sup>18</sup>)</p>

### 출력

<p>Для каждого тестового запроса выведите ответ в отдельной строке.</p>
