### 문제 설명

<p>Открытый в 2345 году химический элемент млуран очень радиоактивен. Неправильное обращение с млураном может привести к непредсказуемым последствиям, поэтому при его использовании, хранении и транспортировке необходимо соблюдать особую осторожность.</p>

<p>Существует много различных изотопов млурана, каждый из которых характеризуется одним натуральным числом&nbsp;&mdash; своей атомной массой. При длительном хранении вместе любой пары изотопов, сумма масс которых равна одному из&nbsp;<i>k</i>&nbsp;&laquo;критических чисел&raquo;, эта пара изотопов вступает в реакцию и происходит взрыв. Известно, что все критические числа являются неотрицательными степенями двойки.</p>

<p>В ядерной лаборатории Флатландии образцы млурана хранятся на двух специальных складах. Ученым удалось получить&nbsp;<i>n</i>&nbsp;различных изотопов млурана, массы которых являются числами от 1 до&nbsp;<i>n</i>. Теперь необходимо распределить изотопы по складам для хранения. Безопасными называются такие способы хранения, при которых каждый изотоп хранится ровно на одном складе, а любые два различных изотопа, сумма масс которых равна одному из &laquo;критических чисел&raquo;, хранятся на разных складах.</p>

<p>Помогите ученым выяснить, сколько существует различных безопасных способов хранения млурана.</p>

### 입력

<p>Входные данные к задаче содержат несколько тестовых наборов. Описание каждого набора состоит из двух строк.</p>

<p>Первая строка описания содержит два целых числа&nbsp;<i>n</i>&nbsp;и&nbsp;<i>k</i>&nbsp;(1&nbsp;&le;&nbsp;<i>n</i>&nbsp;&le;&nbsp;10<sup>18</sup>, 1&nbsp;&le;&nbsp;<i>k</i>&nbsp;&le;&nbsp;61)&nbsp;&mdash; количество изотопов, которые необходимо хранить, и количество &laquo;критических чисел&raquo;. Следующая строка содержит&nbsp;<i>k</i>&nbsp;различных натуральных чисел, каждое из которых является степенью двойки и не превышает 2<i>n</i>&nbsp;&mdash; критические числа. Критические числа отделены друг от друга пробелами.</p>

<p>Количество наборов данных в каждом тесте не превышает 10000. Последняя строка теста содержит два нуля.</p>

### 출력

<p>Для каждого тестового набора на отдельной строке выведите количество безопасных способов хранения всех изотопов на двух складах, взятое по модулю 10<sup>9</sup>&nbsp;+ 7.</p>
