### 문제 설명

<p>За последние несколько лет электронные очереди прочно вошли в повседневную жизнь. Во многих государственных учреждениях можно встретить терминал, печатающий бумажку с номером, и, не задавая привычный вопрос &laquo;Кто последний?&raquo;, посетители с помощью электронного табло узнают, сколько еще им ждать и когда наступит их очередь.</p>

<p>Однако, пока такие системы далеки от совершенства. Например, вызывает вопросы стандартный принцип любой очереди: &laquo;Первым обслуживается тот, кто первым пришел&raquo;. При разработке инновационной ГАС &laquo;Очередь&raquo; решено было сделать ее такой, что выполнение этого принципа не требуется. Вместо этого, новая система призвана минимизировать количество негатива, приходящегося на чиновника, на прием к которому стоят люди в очереди.</p>

<p>Известно, что у каждого человека есть такой критерий, как раздражительность. Если этот параметр равен&nbsp;<i>w</i>, то через&nbsp;<i>t</i>&nbsp;часов ожидания в очереди этот человек обрушит на голову чиновника ровно&nbsp;<i>wt</i>&nbsp;единиц злобы и ругани. Так, если посетителя начнут обслуживать сразу же после его прихода, то чиновник не пострадает, а если посетитель пришел в начале третьего часа, а его обслуживание началось только в начале пятого &mdash; количество гнева будет равно 2<i>w</i>.</p>

<p>Также известно, что на обслуживание каждого посетителя уходит ровно час, а каждый посетитель приходит в начале какого-либо часа. Ваша задача заключается в том, чтобы по данным вам показателям раздражительности и временам прихода посетителей определить, сколько негатива достанется чиновнику при оптимальном порядке обслуживания клиентов.</p>

### 입력

<p>В первой строке задано одно целое число&nbsp;<i>t</i>&nbsp;&mdash; количество случаев, которые вам предстоит обработать. Далее следуют&nbsp;<i>t</i>&nbsp;описаний самих случаев.</p>

<p>Описание каждого случая состоит из: числа&nbsp;<i>n</i>&nbsp;в первой строке &mdash; количества посетителей, и&nbsp;<i>n</i>&nbsp;описаний самих посетителей. Для каждого посетителя в отдельной строке указаны два целых числа&nbsp;<i>r<sub>i</sub></i>&nbsp;и&nbsp;<i>w<sub>i</sub></i>&nbsp;(1&nbsp;&le;&nbsp;<i>r<sub>i</sub></i>,&nbsp;<i>w<sub>i</sub></i>&nbsp;&le;&nbsp;10<sup>6</sup>) &mdash; номер часа, в начале которого посетитель пришел, и его коэффициент раздражительности, соответственно.</p>

<p>Суммарное количество посетителей во всех случаях одного теста не превосходит 10<sup>5</sup>.</p>

### 출력

<p>Для каждого случая в отдельной строке выведите ответ &mdash; минимальное суммарное количество негатива, которое получит чиновник.</p>
