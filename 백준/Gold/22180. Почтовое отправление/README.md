### 문제 설명

<p>Для подготовки заключительного этапа Russian Code Cup организаторам потребовалось отправить на место проведения&nbsp;<i>n</i>&nbsp;предметов. Для каждого предмета известна его масса в граммах&nbsp;<i>m<sub>i</sub></i>.</p>

<p>Для отправки решено было воспользоваться почтовой службой &laquo;Суперэкспресс&raquo;. Служба принимает к пересылке бандероли, в каждой из которых может пересылаться один или несколько предметов. При этом масса бандероли равна сумме масс пересылаемых в ней предметов.</p>

<p>Пересылка бандероли стоит 1 рубль за грамм, за исключением бандеролей, которые попадают под действие специального предложения. А именно, если бандероль весит ровно один килограмм, то стоимость ее пересылки составляет&nbsp;<i>P</i>&nbsp;рублей.</p>

<p>Организаторы Russian Code Cup хотят переслать все предметы, затратив минимальную возможную сумму денег. Помогите им распределить предметы по бандеролям, чтобы добиться этого.</p>

### 입력

<p>Первая строка содержит два целых числа:&nbsp;<i>n</i>&nbsp;и&nbsp;<i>P</i>&nbsp;(1 &le;&nbsp;<i>n</i>&nbsp;&le; 14; 1 &le;&nbsp;<i>P</i>&nbsp;&le; 1000) &mdash; количество предметов и стоимость пересылки бандероли в рамках специального предложения. Вторая строка содержит&nbsp;<i>n</i>&nbsp;целых чисел:&nbsp;<i>m</i><sub>1</sub>,&nbsp;<i>m</i><sub>2</sub>, ...,&nbsp;<i>m<sub>n</sub></i>&nbsp;(1&nbsp;&le;&nbsp;<i>m<sub>i</sub></i>&nbsp;&le;&nbsp;1000 для всех&nbsp;<i>i</i>&nbsp;от 1 до&nbsp;<i>n</i>).</p>

### 출력

<p>Выведите одно число &mdash; минимальную суммарную стоимость пересылки всех предметов в рублях.</p>
