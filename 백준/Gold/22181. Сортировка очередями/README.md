### 문제 설명

<p>Очередь &mdash; структура данных, основанная на принципе &laquo;первый пришел &mdash; первый вышел&raquo; (FIFO, First In &mdash; First Out). Добавление элемента возможно только в конец очереди, извлечение &mdash; только из начала очереди. Таким образом, элементы извлекаются из очереди в том же порядке, в котором они были в нее добавлены.</p>

<p>Задача сортировки состоит в упорядочивании заданного массива чисел (или других объектов) по возрастанию или убыванию. У этой задачи существует достаточно многих вариантов, для многих из которых существуют весьма эффективные алгоритмы.</p>

<p>Далее в задаче рассматривается специальное устройство, содержащее входной поток, выходной поток и&nbsp;<i>k</i>&nbsp;очередей, пронумерованных числами от 1 до&nbsp;<i>k</i>.</p>

<p>Ваша задача состоит в том, чтобы по заданной последовательности различных чисел, которую требуется отсортировать, составить программу для этого устройства, которая будет выводить в выходной поток те же числа, что поступили во входной поток, но упорядоченные по возрастанию.</p>

<p>Программа должна содержать ровно 2<i>n</i>&nbsp;операций, каждая из которых либо читает число из входного потока и добавляет его в одну из очередей, либо извлекает число из одной из очередей и выводит его в выходной поток.</p>

### 입력

<p>Первая строка содержит целое число&nbsp;<i>n</i>&nbsp;(1 &le;&nbsp;<i>n</i>&nbsp;&le; 300). Вторая строка содержит&nbsp;<i>n</i>&nbsp;различных целых чисел&nbsp;<i>a</i><sub>1</sub>, ...,&nbsp;<i>a<sub>n</sub></i>&nbsp;в том порядке, в котором они поступают из входного потока (1 &le;&nbsp;<i>a<sub>i</sub></i>&nbsp;&le; 10<sup>9</sup>&nbsp;для всех&nbsp;<i>i</i>&nbsp;от 1 до&nbsp;<i>n</i>). Третья строка содержит целое число&nbsp;<i>k</i>&nbsp;(1 &le;&nbsp;<i>k</i>&nbsp;&le;&nbsp;<i>n</i>).</p>

### 출력

<p>Если сортировку выполнить невозможно, выведите слово&nbsp;<code>NO</code>.</p>

<p>Иначе выведите слово&nbsp;<code>YES</code>&nbsp;и 2<i>n</i>&nbsp;строк, задающих программу сортировки. Каждая из этих строк должна описывать одну операцию и иметь следующий формат:</p>

<ul>
	<li><code>I(j)</code>&nbsp;&mdash; считать элемент из входного потока и добавить его в очередь номер&nbsp;<i>j</i>&nbsp;(1 &le;&nbsp;<i>j</i>&nbsp;&le;&nbsp;<i>k</i>);</li>
	<li><code>R(j)</code>&nbsp;&mdash; извлечь элемент из очереди номер&nbsp;<i>j</i>&nbsp;(1 &le;&nbsp;<i>j</i>&nbsp;&le;&nbsp;<i>k</i>) и вывести его в выходной поток.</li>
</ul>
