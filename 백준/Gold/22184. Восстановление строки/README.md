### 문제 설명

<p>Во многих прикладных задачах, таких как поиск в сети или расшифровка генома, требуется совершать некоторые действия со строками. Например, часто требуется по некоторым данным о строке восстановить ее саму.</p>

<p>Вам даны две строки&nbsp;<i>S</i><sub>1</sub>&nbsp;и&nbsp;<i>S</i><sub>2</sub>. Известно, что одна из них была суффиксом искомой строки&nbsp;<i>S</i>, а другая &mdash; ее префиксом. Также известна длина искомой строки&nbsp;<i>L</i>, а также то, что строка&nbsp;<i>S</i>&nbsp;состояла только из строчных букв латинского алфавита.</p>

<p>Необходимо определить число строк, удовлетворяющих этим ограничениям. Так как это число может быть достаточно большим, то необходимо вывести его по модулю&nbsp;<i>m</i>.</p>

### 입력

<p>Первая строка содержит одно целое число&nbsp;<i>t</i>&nbsp;(1&nbsp;&le;&nbsp;<i>t</i>&nbsp;&le;&nbsp;100) &mdash; количество наборов входных данных, которые необходимо обработать.</p>

<p>Описание каждого из наборов входных данных содержит три строки. Первая из них содержит два целых числа:&nbsp;<i>L</i>&nbsp;и&nbsp;<i>m</i>&nbsp;(1&nbsp;&le;&nbsp;<i>L</i>&nbsp;&le;&nbsp;10<sup>9</sup>, 1&nbsp;&le;&nbsp;<i>m</i>&nbsp;&le;&nbsp;10<sup>4</sup>). Вторая и третья строки содержат соответственно строки&nbsp;<i>S</i><sub>1</sub>&nbsp;и&nbsp;<i>S</i><sub>2</sub>. Они непусты, состоят из строчных букв латинского алфавита, их длины не превышают 200 символов.</p>

### 출력

<p>Для каждого набора входных данных выведите в отдельной строке остаток от деления числа строк, удовлетворяющих требованиям условия, на&nbsp;<i>m</i>.</p>
