### 문제 설명

<p>Недавно в одной популярной социальной сети появилось приложение &quot;Угадай число!&quot;. Его пользователям предлагается игра, на каждом из уровней которой необходимо определить загаданное число по некоторой информации о нем.</p>

<p>В частности, на одном из самых сложных уровней необходимо угадать рациональное число&nbsp;<i>x</i>&nbsp;(0 &lt;&nbsp;<i>x</i>&nbsp;&lt; 1), о котором известно, что в результате умножения на натуральное число&nbsp;<i>k</i>&nbsp;в его десятичной записи произошло ровно одно изменение &mdash; в ней поменялись местами&nbsp;<i>i</i>-ая и&nbsp;<i>j</i>-ая цифры после десятичной точки (цифры нумеруются с единицы в направлении слева направо). При этом цифра до десятичной точки не изменилась, то есть выполнено неравенство 0 &lt;&nbsp;<i>kx</i>&nbsp;&lt; 1. Отметим, что исходно в десятичной записи&nbsp;<i>x</i>&nbsp;может быть бесконечно много знаков после десятичной точки.</p>

<p>Ваша задача состоит в том, чтобы написать программу, которая будет определять значение&nbsp;<i>x</i>&nbsp;по числам&nbsp;<i>i</i>,&nbsp;<i>j</i>,&nbsp;<i>k</i>.</p>

### 입력

<p>Первая строка содержит три целых числа&nbsp;<i>i</i>,&nbsp;<i>j</i>,&nbsp;<i>k</i>&nbsp;(1&nbsp;&le;&nbsp;<i>i</i>&nbsp;&lt;&nbsp;<i>j</i>&nbsp;&le; 1000; 2&nbsp;&le;&nbsp;<i>k</i>&nbsp;&le;&nbsp;10<sup>9</sup>).</p>

### 출력

<p>Если искомое число существует, то выведите два целых числа &mdash; числитель&nbsp;<i>a</i>&nbsp;и знаменатель&nbsp;<i>b</i>&nbsp;несократимой дроби, задающей искомое число (<i>a</i>,&nbsp;<i>b</i>&nbsp;&gt; 0). В противном случае выведите фразу NO SOLUTION.</p>
