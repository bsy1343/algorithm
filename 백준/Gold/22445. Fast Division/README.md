### 문제 설명

<p>イクタ君は速いプログラムが大好きである。最近は、除算のプログラムを高速にしようとしている。しかしなかなか速くならないので、「常識的に考えて典型的」な入力に対してのみ高速にすればよいと考えた。イクタ君が解こうとしている問題は次のようなものである。</p>

<p>与えられた非負整数$n$に対し、10進法で$p(n) - 1$桁の正整数$11...1$を$p(n)$で割ったあまりを求めよ。ただし$p(n)$は$2^{2^{^{.^{.^{.^{2}}}}}}$（2が$n$個）より大きい最小の素数を表すとする。$p(0) = 2$とする。</p>

<p>あなたの仕事は、イクタ君より速くプログラムを完成させることである。</p>

### 입력

<p>入力は以下の形式で与えられる。</p>

<blockquote>
<p>$n$</p>
</blockquote>

<p>問題の入力の非負整数$n$があたえられる。</p>

### 출력

<p>問題の解を1行に出力せよ。</p>

### 제한

<p>入力中の各変数は以下の制約を満たす。</p>

<ul>
	<li>
	<p>$0 \leq n &lt; 1000$</p>
	</li>
</ul>
