# [Gold IV] Usaneko Matrix - 22567

[문제 링크](https://www.acmicpc.net/problem/22567)

### 성능 요약

시간 제한: 1 초, 메모리 제한: 512 MB

### 통계

### 문제 설명

<p>うさぎとねこが勝負をしている. ルールは以下の通りである.</p>

<p>まず2 匹はそれぞれ<i>n</i>&nbsp;行<i>n</i>&nbsp;列の正方形状に<i>n</i><sup>2</sup>&nbsp;個の整数を紙に書き, トランプを1 枚ずつ引く. 次に, 1 から 1 000 000 までの数が1 つずつ書かれた1 000 000 枚のカードを2 匹でシャッフルし, これを1 枚ずつ交互に引いていく. 2 匹はカードが引かれるたび, カードと同じ数が自分の紙に書かれていたらそれに印をつける. 「印がついた<i>n</i>&nbsp;個の数の組であって, 一直線上に並んでいるもの」の個数が, はじめに引いたトランプの数以上になることを勝利条件とする.</p>

<p>与えられた<i>m</i>&nbsp;枚目のカードまでで, うさぎとねこのどちらが勝つか答えよ. ただし勝敗は, あるカードが引かれて印をつけ終わった段階で2 匹のうち片方のみが勝利条件をみたしたときに決まるものとし, それ以外の場合は引き分けとする. いずれかが勝利条件をみたした後でもカードが引かれることはあるが, これは勝敗に影響しない.</p>

### 입력

<p>1 行目：&ldquo;<i>n u v m</i>&rdquo; (正方形のサイズ, うさぎのトランプの数, ねこのトランプの数, 引かれるカードの枚数) 2-(<i>N</i>&nbsp;+ 1) 行目：うさぎが紙に書く<i>n</i><sup>2</sup>&nbsp;個の数(<i>N</i>&nbsp;+ 2)-(2<i>N</i>&nbsp;+ 1) 行目：ねこが紙に書く<i>n</i><sup>2</sup>&nbsp;個の数(2<i>N</i>&nbsp;+ 2)-(2<i>N</i>&nbsp;+<i>M</i>&nbsp;+ 1) 行目：引かれるm 枚のカード</p>

<ul>
	<li>1 &le;&nbsp;<i>n</i>&nbsp;&le; 500</li>
	<li>1 &le;&nbsp;<i>u</i>,&nbsp;<i>v</i>&nbsp;&le; 13</li>
	<li>1 &le;&nbsp;<i>m</i>&nbsp;&le; 100 000</li>
	<li>1 &le; (書かれる数) &le; 1 000 000</li>
</ul>

<p>うさぎが紙に書く<i>n</i><sup>2</sup>&nbsp;個の数, ねこが紙に書く<i>n</i><sup>2</sup>&nbsp;個の数, 引かれる<i>m</i>&nbsp;枚のカードに書かれた数はそれぞれの中で異なる.</p>

### 출력

<p>うさぎが勝つ場合には&rdquo;USAGI&rdquo;を, ねこが勝つ場合には&rdquo;NEKO&rdquo;を, 引き分けならば&rdquo;DRAW&rdquo;, それぞれ 一行に出力せよ.</p>
