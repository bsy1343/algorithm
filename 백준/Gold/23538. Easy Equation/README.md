<p>Find the number of solutions of the equation $$x^5 + y^4 + z^3 + w^2 + t = n,$$, where $x$, $y$, $z$, $w$, and $t$ are positive integer variables, and $n$ is a known positive integer.</p>

### 입력

<p>A single line containing integer $n$ ($1 \le n \le 10^9$).</p>

### 출력

<p>Print a single integer -- the number of solutions.</p>
