### 문제 설명

<p>플레이어 $N$명이 $1$개 이상의&nbsp;팀으로 나누어 게임을 진행하려 한다.&nbsp;플레이어는 각각 정확히 한 팀에 속해야 한다. $i$번째 플레이어는&nbsp;같은 팀에 속한 인원 수와 $a_i$를 곱한 것만큼의 점수를 얻는다.</p>

<p>팀을 적절히 나누었을 때, 모든 플레이어의 점수의&nbsp;합의 최댓값을 구해보자.</p>

### 입력

<p>첫째 줄에 $N$ ($1 \leq N \leq 10^5$)이 주어진다.</p>

<p>둘째 줄에&nbsp;$N$개의 정수가&nbsp;주어진다.&nbsp;$i$번째 수는 $a_i$이다.&nbsp;($ -10^5&nbsp;\leq a_i&nbsp;\leq 10^5$)</p>

### 출력

<p>첫째 줄에 팀을 적절히 나누었을 때 모든 플레이어들의 점수의 합의 최댓값을 출력한다.</p>
