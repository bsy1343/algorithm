### 문제 설명

<p>태영이는 SASA의 축제라고 불리는 &quot;인문예술탐사주간&quot;을 보내게 되었다. &quot;인문예술탐사주간&quot;을 맞이하여 세종호수공원에 가게 된 태영이는 아름다운 경치에 놀라움을 금치 못했다.</p>

<p>세종호수공원은 일직선으로 뻗어있는 모습이다. 이 공원에는 나무가 총 $N$ 그루 있으며, $i$ 번째 나무의 위치는 $P_i$이다.</p>

<p>태영이는 카메라를 들고&nbsp;파노라마 사진을 $Q$ 번&nbsp;찍어, 이 아름다운 풍경을 담으려고 한다. 태영이는 이 사진에 점수를 매기려고 하는데, 사진의 점수는 사진을 찍은 위치로부터 각 나무까지의 거리 합이다. 이 때, 두 위치의 거리는 두 위치의 차의 절댓값으로 정의된다.</p>

<p>태영이가 찍은 사진에 대해서, 각 사진의 점수를 매겨주자.</p>

### 입력

<p>첫째&nbsp;줄에는 나무의 개수&nbsp;$N$과 태영이가 찍은 사진의 개수 $Q$가 공백으로 구분되어 주어진다.</p>

<p>둘째&nbsp;줄에는 나무의&nbsp;위치 $P_1, P_2, \cdots, P_N$이 주어진다.</p>

<p>셋째 줄부터 $Q$개의 줄의 $i$ 번째 줄에는 태영이가 $i$ 번째로 사진을 찍은 위치 $X_i$가 주어진다.</p>

### 출력

<p>$i$ 번째 줄에는 태영이가 $i$ 번째 찍은 사진의 점수를 한 줄에 하나씩 차례대로 출력한다.</p>

### 제한

<ul>
	<li>$1 \leq N,Q \leq 10^5$</li>
	<li>$1 \leq P_i&nbsp;\leq 10^7$ ($1 \le i \le N$)</li>
	<li>$1 \leq X_i&nbsp;\leq 10^7$ ($1 \le i \le Q$)</li>
	<li>입력으로 주어지는 모든 수는 정수이다.</li>
</ul>
