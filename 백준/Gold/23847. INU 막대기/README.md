### 문제 설명

<p><em>N</em>개의 INU 막대기가 있다.</p>

<p>INU 막대기의&nbsp;양 끝에는 각각 <code>I, N, U</code>&nbsp;중 하나가 적혀있다.</p>

<p>영현이는 몇 개의 막대기를 연결하여 가장 긴 막대기를 만들려 한다.</p>

<p>막대기는 접합면에 적힌 글자가 동일한 경우만 연결할 수 있고, 뒤집어서 사용하는 것이 가능하다.</p>

<p>영현이가 만들 수 있는 가장 긴 막대기의 길이는 얼마인가?</p>

### 입력

<p>첫 줄에 막대기의 개수 <em>N</em>이 주어진다. (1 &le; <em>N</em> &le; 10<sup>5</sup>)</p>

<p>다음<em> </em><em>N</em>줄에 각 막대기의 각 양쪽 끝에 적힌 두 알파벳과 길이 정수&nbsp;<em>t</em><sub><em>i</em>&nbsp;</sub>가 주어진다. (1 &le; <em>t<sub>i</sub></em>&nbsp;&le; 10<sup>3</sup>)</p>

### 출력

<p>영현이가 만들 수 있는 가장 긴 막대기의 길이를 출력한다.</p>
