### 문제 설명

<p>숫자로 이루어진 문자열 <em>S</em>가<em> </em>있다. 문자열<em> S</em>를&nbsp;<em>K</em>-1번 잘라&nbsp;<em>K</em>개의 비어있지 않은 연속 부분 문자열로 나눌&nbsp;때, 어떠한 연속 부분 문자열에도 leading zero가 없도록 나누는 경우의 수를 구해보자.</p>

### 입력

<p>첫째 줄에 문자열의 길이 <em>N</em>과 연속 부분 문자열의 개수 <em>K</em>가 주어진다. (1 &le; <em>K</em> &le; <em>N</em> &le; 2,500)</p>

<p>둘째 줄에 문자열&nbsp;<em>S</em>가 주어진다.</p>

### 출력

<p>첫째 줄에 경우의 수를 1,000,000,007로 나눈 나머지를 출력한다.</p>

### 힌트

<p>leading zero는 숫자로 이루어진 문자열에서 0을 제외한&nbsp;첫 번째 숫자&nbsp;앞에 오는 모든 숫자 0을&nbsp;의미한다.</p>

<p>단, &quot;0&quot;은 leading zero에 해당하지 않는다.</p>
