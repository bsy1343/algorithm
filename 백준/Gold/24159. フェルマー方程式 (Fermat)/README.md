### 문제 설명

<p>素数 p と自然数 n &ge; 1 が与えられたとき，</p>

<p style="text-align: center;">x<sup>n</sup> + y<sup>n</sup> &equiv; z<sup>n</sup> (mod p)</p>

<p>をみたす整数 x, y, z (0 &le; x, y, z &le; p &minus; 1) の組 (x, y, z) の個数 m を求めるプログラムを作れ．こ こで，a &equiv; b (mod p) とは，a &minus; b が p で割り切れることを意味する．</p>

### 입력

<p>入力の１行目には素数 p (p &lt; 10000) が書かれている．2 行目に は自然数 n (1 &le; n &le; 10000) が書かれている．</p>

### 출력

<p>標準出力に 1 つの整数 m のみからなる 1 行を出力せよ．</p>

### 힌트

<p><strong>注意</strong> 採点に用いる入力データに対しては，m の値は 2<sup>31</sup> よりも小さい．</p>
