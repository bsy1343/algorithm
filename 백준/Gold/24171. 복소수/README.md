### 문제 설명

<p>$\displaystyle{{b_0&nbsp;+ c_0&nbsp;\sqrt{d_0} \over a_0} + {b_1&nbsp;+ c_1&nbsp;\sqrt{d_1} \over a_1} i}$</p>

<ul>
	<li>$i = \sqrt{-1}$</li>
	<li>$j=0, \, 1$에 대해
	<ul>
		<li>$a_j,&nbsp;\, b_j, \, c_j, \, d_j$는 정수</li>
		<li>$a_j &gt; 0$;&nbsp;$0 \le d_j \ne 1$</li>
		<li>$\gcd (a_j, \, b_j, \, c_j) = 1$</li>
		<li>&#39;$c_j = 0$&#39;과 &#39;$d_j = 0$&#39;는 필요충분조건</li>
		<li>$d_j &gt; 0$일 때 $d_j$의 약수 중 $1$보다 큰 제곱수가 없음</li>
	</ul>
	</li>
	<li>$d_0=d_1$</li>
</ul>

<p>위 형식으로 표현되는 복소수&nbsp;$A$와 $B$가 주어질 때, $A+B$, $A-B$, $A \times B$, $A \div B$의 값을 출력하는 프로그램을 작성하세요.</p>

### 입력

<p>첫 번째 줄에 $A$의 $a_0, \, b_0, \, c_0, \, d_0, \, a_1, \, b_1, \, c_1, \, d_1$ 값이 하나씩 주어집니다.</p>

<p>두 번째 줄에는 마찬가지로 $B$의 $a_0, \, b_0, \, c_0, \, d_0, \, a_1, \, b_1, \, c_1, \, d_1$ 값이 하나씩 주어집니다.</p>

### 출력

<p>첫 번째 줄부터 네 번째 줄까지 각각&nbsp;$A+B$, $A-B$, $A \times B$, $A \div B$의 값을&nbsp;문제에서 설명한 형식으로 표현했을 때의 $a_0, \, b_0, \, c_0, \, d_0, \, a_1, \, b_1, \, c_1, \, d_1$&nbsp;값을 출력합니다.</p>

### 제한

<ul>
	<li>$B \ne 0$</li>
	<li>$A$와 $B$의 $d_0$ 값은 동일</li>
	<li>입력은&nbsp;문제에서 설명한 형식의 조건을 만족</li>
	<li>입력으로 주어지는 수는 모두 절댓값이 $10$ 이하</li>
</ul>
