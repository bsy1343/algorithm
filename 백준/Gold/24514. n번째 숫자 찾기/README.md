### 문제 설명

<p>수학과인 영진이는 연속하는 수를 좋아한다. 영진이는 아래와 같이 두 가지 수를 정의하였다.</p>

<p>$X_K(N)$&nbsp;: $1$부터 $N$까지의 연속하는&nbsp;$K$진법 수를&nbsp;이어 만든 수</p>

<p>(ex. $X_3(2)&nbsp;= 12,&nbsp;X_3(4)&nbsp;= 121011, X_7(2) = 12, X_7(4) = 1234$)</p>

<p>$YJ_K$ : $X_K(1)$부터 $X_K(10^{100})$까지의 수를&nbsp;이어 만든 수</p>

<p>(ex. $YJ_3&nbsp;= X_3(1)X_3(2)X_3(3)X_3(4)\cdots&nbsp;= 1121210121011\cdots$, $YJ_7 = X_7(1)X_7(2)X_7(3)X_7(4)\cdots&nbsp;= 1121231234\cdots$)</p>

<p>영진이는 수학과 신입생 환영회에 참가해 후배들에게 $YJ_K$의 $N$번째 자리에 오는 숫자를 맞히지 못하면 술을 마셔야 한다고 하였다.</p>

<p>신입생 세원이는 살아서 집에 돌아가고 싶다. 세원이를 위해 $YJ_K$의 $N$번째 숫자를 찾아서 알려주자.</p>

### 입력

<p>첫 줄에 영진이의 질문 횟수&nbsp;$T$와 $K$가 공백을 사이에 두고 주어진다.&nbsp;($1&nbsp;\le&nbsp;T &lt;&nbsp;100, 2&nbsp;\le&nbsp;K &lt; 10$)</p>

<p>다음 줄부터&nbsp;$T$개의 줄에 $N$이 주어진다. ($1&nbsp;\le&nbsp;N&nbsp;&lt; 2\times10^{9}$)</p>

### 출력

<p>각 질문에서 $YJ_K$의 $N$번째 숫자를 출력한다.</p>
