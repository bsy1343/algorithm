### 문제 설명

<p>포함된 K의 개수가 S의 개수의 정확히 $2$배이면서, S와 K가 적어도 한 번은 등장하는 문자열을 SKK 문자열이라고 한다.</p>

<p>SKK 문자열은 S, K 말고도 다른 알파벳 또한 포함할 수 있다.</p>

<p>알파벳 대문자로만 이루어진 문자열 $S$가 주어질 때, $S$의 부분 문자열 중 길이가 가장 긴 SKK 문자열을 찾는 프로그램을 작성하라.</p>

### 입력

<p>첫째 줄에 길이가 $1$이상 $100,000$ 이하인 알파벳 대문자로만 이루어진 문자열 $S$가 주어진다.&nbsp;</p>

### 출력

<p>$S$의 부분 문자열 중 길이가 가장 긴 SKK 문자열의 길이를 출력한다. 만약 그러한 문자열이 없으면 -1을 출력한다.</p>

### 힌트

<p>문자열 $S$에서 문자를 연속하여 선택할 때 만들어지는 새로운 문자열을&nbsp;$S$의 부분 문자열이라 한다.</p>

<p>예를 들어, &quot;appl&quot;, &quot;ap&quot;, &quot;ple&quot; 등은 &quot;apple&quot;의 부분 문자열이고, &quot;ppe&quot;, &quot;apl&quot; 등은 &quot;apple&quot;의 부분 문자열이 아니다.</p>
