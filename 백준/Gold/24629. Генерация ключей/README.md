### 문제 설명

<p>В криптографической системе &lt;&lt;EasyCrypt&gt;&gt; ключом может быть любое неотрицательное целое число, двоичная запись которого содержит ровно $K$ единиц, и не превосходящее заданного целого положительного числа $N$.</p>

<p>Вычислите, сколько различных ключей существует для заданных $N$ и $K$. Так как ответ может быть очень большим, выведите остаток от его деления на простое число $998\,244\,353$.</p>

### 입력

<p>Первая строка входных данных содержит одно целое число $N$, записанное в <em>шестнадцатеричной</em> системе счисления без ведущих &nbsp;нулей ($1 \le N &lt; 16^{250}$). Цифры, большие 9, обозначаются заглавными латинскими буквами от &#39;<code>A</code>&#39; до &#39;<code>F</code>&#39;.</p>

<p>Вторая строка содержит одно целое число $K$ --- число бит в ключе, равных единице ($0 \le K \le 1\,000$).</p>

### 출력

<p>Выведите одно число --- остаток от деления количества различных ключей, существующих для заданных $N$ и $K$, на простое число $998\,244\,353$.</p>
