### 문제 설명

<p>청정수열은 길이가 $2N$이고 $1$부터 $N$까지의 정수들이&nbsp;정확히 두 번씩 등장하는 수열이다.</p>

<p>청정수열의 점수는&nbsp;$1$이상&nbsp;$N$이하인 모든 정수&nbsp;$i$에 대해 다음 값의 합이다.</p>

<ul>
	<li>(두 개의 $i$ 사이에 있는 수의&nbsp;합) &times; $i$</li>
</ul>

<p>이때, &quot;사이&quot;는 양 끝의 $i$를 포함한다.</p>

<p>길이가 $2N$인 청정수열의 최대 점수와 점수가 최대인 청정수열의&nbsp;개수를 구해보자. 답이 매우 커질 수 있으니 $10^9+7$로 나눈 나머지를 출력한다.</p>

### 입력

<p>첫째&nbsp;줄에 정수 $N$이 주어진다.&nbsp;($1 \le N&nbsp;\le 10^5$)</p>

### 출력

<p>첫째 줄에 길이가 $2N$인 청정수열의 점수의 최댓값을 $10^9+7$로 나눈 나머지와 점수가 최대인&nbsp;청정수열의 개수를 $10^9+7$로 나눈 나머지를 공백을 사이에 두고 출력하라.</p>

### 힌트

<p>예시로 $[3, 1, 2, 1, 3, 2]$는 $N$이 $3$인 청정수열이고 이 청정수열의 점수는 다음과 같이 계산되어 $50$점이 된다.</p>

<p>$1$과 $1$의&nbsp;사이의 수들은&nbsp;$[1,2,1]$ 이다. 따라서 $(1+2+1)\ &times;\ 1$을 점수에 더한다.</p>

<p>$2$와&nbsp;$2$의 사이의 수들은 $[2,1,3,2]$ 이다. 따라서 $(2+1+3+2)\ &times;\ 2$를&nbsp;점수에 더한다.</p>

<p>$3$과 $3$의 사이의 수들은&nbsp;$[3,1,2,1,3]$ 이다. 따라서 $(3+1+2+1+3)\ &times;\ 3$을&nbsp;점수에 더한다.</p>

<p>따라서 이 청정수열의 점수는 $4+16+30$으로 $50$점이다.</p>
