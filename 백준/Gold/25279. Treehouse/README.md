### 문제 설명

<p>Pusheen wants to build a treehouse in the <em>Treehouse forest</em> in Brunnsh&ouml;g in the north of Lund. The treehouse should to be built on a square platform in the treetops, with a tree in each of the four corners. When Pusheen has picked a spot to built the treehouse, trees that are located between the corners will be cut down. Pusheen has a 2D map with all the <em>(x, y)</em>-coordinates of the trees in the forest. In how many places can they build a square treehouse?</p>

### 입력

<p>The first line has an integer $4 \leq N \leq 2\, 000 $, the number of trees in the forest. Each of the following $N$ lines has two integers $-10,000\leq x_i, y_i \leq 10,000$, the x- and y-coordinates of tree $i$. All points are distinct.</p>

### 출력

<p>An integer, the number of suitable spots for a tree house.</p>
