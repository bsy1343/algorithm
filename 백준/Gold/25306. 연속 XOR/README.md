### 문제 설명

<p>준원이는 다음과 같이 $A$에서 $B$까지의 자연수들을 나열했다.</p>

<p>$$A, A+1, A+2, \dots, B-2, B-1, B$$</p>

<p>이 수들에 모두 <strong>비트&nbsp;XOR</strong>을 취한 값을 구하라.</p>

### 입력

<p>두 자연수 $A$, $B$가 공백을 사이에 두고 주어진다.</p>

### 출력

<p>$A$ 이상 $B$ 이하인 모든 자연수들을 XOR한 값을 구하여라.</p>

### 제한

<ul>
	<li>$1 &le; A &le; B &le; 1\,000\,000\,000\,000\,000\,000 = 10^{18}$</li>
</ul>

### 힌트

<p><strong>비트 XOR</strong>&nbsp;(Bitwise XOR) 에&nbsp;대해 잘 모르는 사람은 이&nbsp;<a href="https://en.wikipedia.org/wiki/Bitwise_operation#XOR">위키백과 링크</a>를 참조하라.</p>
