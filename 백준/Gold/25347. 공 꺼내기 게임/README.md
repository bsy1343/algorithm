### 문제 설명

<p>Azber와 Benedict는 빨간색 주머니와 파란색 주머니를 가지고 게임을 하려고 한다. 주머니에는 공이 여러 개 들어 있는데, 각 공에는 $1$부터 $n$까지의 자연수 중 하나가 적혀 있다. $1$ 이상 $n$ 이하의 자연수 $i$에 대해, $i$가 적힌 공은 빨간색 주머니에 $r_i$개, 파란색 주머니에 $b_i$개가 있다.</p>

<p>먼저 Azber는 Benedict가 모르게 두 주머니 중에 하나를 고른다. 고른 주머니에서 Azber가 무작위로 공을 하나 골라 그 공에 적혀 있는 수를 Benedict에게 알려준다. Benedict는 그 수만 보고 Azber가 어떤 색의 주머니에서 공을 꺼냈을지 알아맞혀야 한다.</p>

<p>Benedict는 미리 전략을 정해서 Azber가 뽑은 수 $i$에 대해 $p_i$의 확률로 빨간색이라고 답을 하고, $1-p_i$의 확률로 파란색이라고 답을 할 것이다.</p>

<p>Benedict는 이상한 집착이 있어서 답이 파란색일 때 오답을 말하는 경우를 피하고 싶어한다. 즉, 답이 파란색일 때 오답을 말할 확률이 $q$ 이하가 되어야만 한다.</p>

<p>Benedict는 이를 만족하는 전략들 중에서 답이 빨간색일 때 오답을 말할 확률이 최소가 되는 전략을 만들 것이다.</p>

<p>Benedict를 도와 전략을 짜 보자. 구체적으로, $1$ 이상 $n$ 이하의 자연수 $i$에 대해 $p_i$를 정하면 된다.</p>

### 입력

<p>입력의 첫째 줄에 자연수 $n$, $x$, $y$가 공백으로 구분되어 주어진다. $q = \cfrac{x}{y}$임을 의미한다.</p>

<p>입력의 둘째 줄에 $n$개의 자연수 $r_i$가 공백으로 구분되어 주어진다. ($1 \leq i \leq n$)</p>

<p>입력의 셋째 줄에 $n$개의 자연수 $b_i$가 공백으로 구분되어 주어진다. ($1 \leq i \leq n$)</p>

### 출력

<p>$n$개의 줄에 걸쳐 $p_1, \dots, p_n$을 출력한다. $0 \leq p_i \leq 1$ 이어야 하고 전략이 여러 가지일 경우 그 중 하나만 출력한다.</p>

<p>답이 파란색일 때 오답을 말할 확률이 $q + 10^{-6}$ 이하이며, 정답과 <b data-stringify-type="bold">답이 빨간색일 때 오답을 말할 확률의 </b>절대/상대 오차가 $10^{-6}$ 이하일 경우 정답으로 처리된다.</p>

### 제한

<ul>
	<li>$1 \leq n \leq 10^6$</li>
	<li>$1 \leq x \leq y \leq 10^9$</li>
	<li>$1 \leq r_{i}, b_{i} \leq 10^9$</li>
</ul>
