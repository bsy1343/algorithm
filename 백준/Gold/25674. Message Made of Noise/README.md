### 문제 설명

<p>Alisa wants to send a message to Eva using a <em>number wire</em>. The message is one English word.</p>

<p>Unfortunately, right now the number wire transmits just some noise: random integers from $0$ to $10^{9} - 1$ inclusive. Alisa knows the sequence of the next $10\,000$ integers that will be transmitted.</p>

<p>Fortunately, Alisa has a superpower: she can erase any number of elements from any positions in the sequence. The relative order of the remaining elements does not change.</p>

<p>Unfortunately, after that, around half of the integers will be lost in transmission: each transmitted integer will disappear with a probability of $1 / 2$. The relative order of the remaining elements once again does not change.</p>

<p>How should Alisa and Eva act to transmit a given word?</p>
