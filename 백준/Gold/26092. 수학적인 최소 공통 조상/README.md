### 문제 설명

<p>$10^{12}$ 개의 정점으로 이루어진 트리가 주어진다. 트리의 각 정점은 $1$번부터 $10^{12}$번까지 번호가 매겨져 있고 $1$번 정점은 루트이다.</p>

<p>이 트리는 다음과 같은 특수한 성질을 가지고 있다.</p>

<ul>
	<li>$2$ 이상 $N$ 이하의 정수 $x$에 대해 $x$의 가장 작은 소인수를 $p$라고 하자.</li>
	<li>$x$번 정점의 부모 정점은 $\frac{x}{p}$번 정점이다.</li>
</ul>

<p>두 정점의 가장 가까운 공통 조상은, 두 정점을 모두 자손으로 가지면서 깊이가 가장 깊은 정점으로 정의한다.</p>

<p>두 정수 $a$와 $b$가 주어졌을 때, $a$번 정점과 $b$번 정점의 가장 가까운 공통 조상의 번호를 구해보자.</p>

### 입력

<p>첫째 줄에 정수 $a$와 $b$가 공백으로 구분되어 주어진다. $(1\leq a,b\leq 10^{12})$</p>

### 출력

<p>첫째 줄에 $a$번 정점과 $b$번 정점의 가장 가까운 공통 조상의 번호를 출력한다.</p>

### 힌트

<p>어떤 정점의 자손은 자기 자신을 포함한다.</p>
