# [Gold III] お小遣いの最大化 - 26474

[문제 링크](https://www.acmicpc.net/problem/26474)

### 성능 요약

시간 제한: 8 초 (추가 시간 없음), 메모리 제한: 1024 MB

### 통계

제출: 22, 정답: 20, 맞힌 사람: 13, 정답 비율: 86.667%

### 분류

다이나믹 프로그래밍

### 문제 설명

<p>鐘星くんの今月のお小遣いの金額は，次のようなゲームで決まることになった．</p>

<ol>
	<li>はじめに，<i>N</i> 枚のカードが横一列に並んでいる．それぞれのカードには，1 から 9 までのいずれかの数字が 1 つ書かれている．また，整数 <i>K</i> が指定される．ここで <i>N</i> は <i>K</i> の倍数である．</li>
	<li>鐘星くんは，これら <i>N</i> 枚のカードの中から，何枚かのカードを選んで破り捨てることができる．ここで，破り捨てるカードの枚数は，<i>K</i> の倍数でなければならない．なお，1 枚も破り捨てなくてもよいし，<i>N</i> 枚全部破り捨ててもよい．</li>
	<li>残ったカードの枚数は <i>K</i> の倍数である．これらのカードを，順番を変えることなく，左から順に <i>K</i> 枚ずつのグループに分ける．</li>
	<li>それぞれのグループのカードに書かれた数字を，<i>K</i> 桁の十進数とみなす．これら <i>K</i> 桁の数をすべて足し合わせた値が，鐘星くんの今月のお小遣いの金額となる．</li>
</ol>

<p>たとえば，<i>N = 9</i>, <i>K = 3</i> で，カードに書かれた数字が左から順に 2, 9, 9, 7, 9, 2, 4, 5, 8 であったとする．これらのカードを 1 枚も破り捨てない場合，鐘星くんの今月のお小遣いは 299 + 792 + 458 = 1549 円である．一方，1 枚目，6 枚目，7 枚目の計 3 枚のカードを破り捨てる場合，残った 6 枚のカードに書かれた数字は左から順に 9, 9, 7, 9, 5, 8 となり，お小遣いは 997 + 958 = 1955 円となる．なお，すべてのカードを破り捨てた場合，お小遣いは 0 円である．</p>

<p>鐘星くんが破り捨てるカードを適切に選んだとき，今月のお小遣いは最大でいくらになるだろうか．</p>

### 입력

<p>入力は <i>50</i> 個以下のデータセットからなる．各データセットは次の形式で表される．</p>

<blockquote>
<p><i>K</i></p>

<p><i>s<sub>1</sub>s<sub>2</sub>...s<sub>N</sub></i></p>
</blockquote>

<p>1 行目には整数 <i>K</i> が与えられる．<i>K</i> は <i>2</i> 以上 <i>14</i> 以下であることが保証される．</p>

<p>2 行目には文字列が与えられ，その長さが <i>N</i> である．<i>N</i> は <i>K</i> 以上 <i>200,000</i> 以下の整数であり，かつ <i>K</i> の倍数であることが保証される．また，この文字列の <i>i</i> 番目の文字 <i>s<sub>i</sub></i> は，はじめに左から <i>i</i> 番目のカードに書かれている数字であり，1 以上 9 以下であることが保証される．</p>

<p>入力の終わりは 1 つのゼロからなる行で表される．</p>

### 출력

<p>各データセットに対し，鐘星くんの今月のお小遣いの最大値を表す整数を，1 行に出力せよ．</p>

<p>なお，この問題の制約から，答えが <i>2<sup>63</sup></i> 未満であることは証明できる．ただし，32 ビット整数型では表現できない場合があることに注意せよ．</p>
