### 문제 설명

<p>文字数が多い英単語はしばしば，前後の1文字ずつだけを残し，それ以外の中間部分をその文字数で置き換える形で省略されることがある． たとえば，&quot;internationalization&quot; は &quot;i18n&quot;，&quot;localization&quot; は &quot;l10n&quot;，&quot;accessibility&quot; は &quot;a11y&quot; などと略記される． ここではこの略記を一般化し，前後の文字を <i>1</i> 文字以上であれば好きなだけ残してよいものとし，この表記を「字数略記」と呼ぶこととする． 例えば，&quot;i18n&quot;，&quot;int13tion&quot;，&quot;i1ternationalization&quot; はいずれも &quot;internationalization&quot; の字数略記であるが，&quot;i123n&quot;，&quot;i4national3tion&quot;，&quot;19n&quot; はいずれも &quot;internationalization&quot; の字数略記ではない．ただし，どの区間も字数に置き換えないそのままの &quot;internationalization&quot; も，&quot;internationalization&quot; の字数略記として有効であるものとする．</p>

<p>あなたは，英小文字のみからなる単語が <i>N</i> 個含まれた辞書を持っており，この辞書内の単語をそれぞれ字数略記に変換した辞書を作ろうと思っている． ここで，もしもある字数略記が変換前の辞書に含まれる複数の単語の字数略記として有効であると，その字数略記だけを見て元の単語を一意に復元することができなくなってしまう．よって，そのような字数略記を変換後の辞書に含めることはできない． 例えば，&quot;love&quot; と &quot;live&quot; がともに辞書に含まれる場合，いずれかでも &quot;l2e&quot; という字数略記に変換してしまうと，&quot;l2e&quot; を見たときに元が &quot;love&quot; だったのか &quot;live&quot; だったのかわからず，復元できなくなってしまう．この場合，例えば &quot;love&quot; を &quot;lo1e&quot;，&quot;live&quot; を &quot;li1e&quot; と字数略記すれば，いずれも元の単語を一意に特定することができる．</p>

<p>与えられた辞書に対し，字数略記変換後の辞書として考えられるもののうち，変換に使用した整数の和として考えられるものの最大値を求めよ． ただし，単語そのままを字数略記とする場合に使用した整数は，0 であるとみなす．</p>

### 입력

<p>入力は 50 個以下のデータセットからなる．各データセットは次の形式で表される．</p>

<blockquote>
<p><i>N</i></p>

<p><i>s<sub>1</sub></i></p>

<p><i>s<sub>2</sub></i></p>

<p><i>...</i></p>

<p><i>s<sub>N</sub></i></p>
</blockquote>

<p>最初の行は辞書に含まれる単語の個数 <i>N</i> からなる (<i>1 &le; N &le;1,000</i>)．続く <i>N</i> 行には，変換前の辞書に含まれる単語が，各行ごとに <i>1</i> つ与えられる．各単語の長さは <i>3</i> 以上 <i>30</i> 以下であり，英小文字アルファベットのみからなる．辞書に含まれる単語はすべて相異なることが保証される．</p>

<p>入力の終わりは，<i>1</i> つのゼロからなる行で表される．</p>

### 출력

<p>各データセットについて，各単語を一意に復元できるよう字数略記に変換するとき，使用した整数の和の最大値を <i>1</i> 行に出力せよ．</p>
