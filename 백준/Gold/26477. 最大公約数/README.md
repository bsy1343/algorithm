# [Gold I] 最大公約数 - 26477

[문제 링크](https://www.acmicpc.net/problem/26477)

### 성능 요약

시간 제한: 8 초 (추가 시간 없음), 메모리 제한: 1024 MB

### 통계

제출: 17, 정답: 12, 맞힌 사람: 9, 정답 비율: 64.286%

### 분류

수학, 정수론, 유클리드 호제법, 포함 배제의 원리, 소인수분해, 오일러 피 함수

### 문제 설명

<p>正の整数 <i>M</i>, <i>A</i> が与えられる．整数 <i>x</i> が以下の条件をすべて満たすとき，<i>x</i> は良い整数である．良い整数の個数を求めよ．</p>

<ul>
	<li><i>1 &le; x &lt; M</i>．</li>
	<li><i>M</i> と <i>x</i> は互いに素．</li>
	<li><i>M</i> と <i>A</i> の最大公約数を <i>g</i> とするとき，<i>Ax &equiv; g (mod M)</i>．</li>
</ul>

### 입력

<p>入力は <i>500</i> 個以下のデータセットからなる．各データセットは次の形式で表される．</p>

<blockquote>
<p><i>M</i> <i>A</i></p>
</blockquote>

<p>各データセットは整数 <i>M</i> と <i>A</i> を含む 1 行からなる．<i>2 &le; M &le; 10<sup>12</sup></i> および <i>1 &le; A &lt; M</i> を満たすことが保証される．</p>

<p>入力の終わりは 2 つのゼロからなる行で表される．</p>

### 출력

<p>各データセットに対し，良い整数の個数を 1 行に出力せよ．</p>
