### 문제 설명

<p>Bajtek zapomniał hasła do swojego telefonu. Pamięta on, że składało się ono z trzech r&oacute;żnych dodatnich liczb całkowitych a &lt; b &lt; c oddzielonych haszami. Dodatkowo suma tych liczb wynosiła n oraz dla każdej pary liczb (spośr&oacute;d (a, b), (a, c) oraz (b, c)) jedna z liczb była wielokrotnością drugiej.</p>

<p>Pom&oacute;ż mu policzyć ile możliwych tr&oacute;jek musi sprawdzić, aby m&oacute;gł zdecydować, czy warto marnować na to czas, czy lepiej kupić nowy telefon.</p>

### 입력

<p>W pierwszym i jedynym wierszu wejścia dana jest liczba n (1 &le; n &le; 10<sup>9</sup>).</p>

### 출력

<p>Należy wypisać pojedynczą liczbę &ndash; liczbę szukanych tr&oacute;jek liczb (a, b, c).</p>

### 힌트

<p>Wyjaśnienie do przykładu: Te dwie tr&oacute;jki to (1, 2, 32) oraz (5, 10, 20).</p>
