### 문제 설명

<p>수학토끼는 역학을 비롯한 친구들과 2학기 기말고사 성적으로 내기를 했다. 내기에 참여한 인원은 총 $N$명이다. 내기의 내용은 다음과 같다:</p>

<ul>
	<li>임의의 두 학생 A, B에 대해 A, B 중 평균 점수가 더 작은 학생이 $y$원을 내고, A와 B의 점수 차이 $x$점 당 추가로 $y$원을 낸다. (즉, 두 명의 점수가 다르고, 그 차이가 $(k-1)x$점 이상 $kx$점 미만이면 점수가 더 낮은 쪽이 $ky$원을 낸다. 단, $k$는 자연수이다.)</li>
	<li>예를 들어, 두 학생의 A, B의 평균 점수가 57점, 63점이고 $x=4, y=1000$이면 A는 $2000$원을 내고, B는 아무것도 내지 않는다.</li>
	<li>단, 두 학생의 점수가 같다면 아무 일도 일어나지 않는다.</li>
</ul>

<p>위 과정을 모든 학생 쌍에 대해 반복한다. 즉 위 과정은 총 $\frac{N(N-1)}{2}$번 일어난다. 이렇게 모인 금액은 나중에 좋은 곳에 사용될 예정이다.</p>

<p>수학토끼는 이 내기로 학생들이 낸 금액의 총합과 자신이 낸 금액이 궁금해졌다. 수학토끼를 포함한 $N$명의 시험 평균 점수 $X_i~(1 \le i \le N)$와 $x, y$가 주어질 때, 이 내기로 모인 총 금액과 수학토끼가 낸 금액을 출력하라.</p>

### 입력

<p>첫째 줄에 세 정수 $N$과 $x, y$가 띄어쓰기를 사이에 두고 주어진다.</p>

<p>둘째 줄에는 학생 $N$명의 시험 평균 점수 $X_1$, $X_2$, $\cdots$, $X_{N}$이 띄어쓰기를 사이에 두고 주어진다. 이 중 $X_1$이 수학토끼의 시험 평균 점수이다.</p>

### 출력

<p>학생들이 낸 금액의 총합과 수학토끼가 낸 금액을 각각 $998\ 244\ 353$으로 나눈 나머지를 띄어쓰기를 사이에 두고 출력한다.</p>

### 제한

<ul>
	<li>$2 \le N \le 2 \times 10^5$</li>
	<li>$1 \le x \le 2 \times 10^5$</li>
	<li>$1 \le y \le 10^9$</li>
	<li>$1 \le X_i \le 2 \times 10^5 ~ (1 \le i \le N)$</li>
	<li>입력으로 주어지는 수는 모두 정수이다.</li>
</ul>
