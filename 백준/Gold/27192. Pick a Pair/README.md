### 문제 설명

<p>Friends play an interesting game with words. The goal of this game is arranging words in pairs.</p>

<p>Friends have $n$ words of the same length. They are choosing the largest number $k$ such that it is possible to divide the words into pairs so that words in each pair have common prefix of length at least $k$.</p>

<p>Find the maximum possible value of $k$.</p>

### 입력

<p>The first line of input contains an even integer $n$ --- number of words  ($1 \leq n \leq 2\cdot 10^5$).</p>

<p>The following $n$ lines contain words that the friends have. All words have same length. Total words length is less or equal to $2 \cdot 10^6$.</p>

### 출력

<p>Output maximal possible value of $k$.</p>
