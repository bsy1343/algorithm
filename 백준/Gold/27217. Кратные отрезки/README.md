### 문제 설명

<p>Задан массив натуральных чисел $A = [a_1, a_2, \ldots, a_n]$. Отрезком массива $A$ с $l$ по $r$ будем называть массив $[a_l, a_{l+1}, \ldots, a_r]$.</p>

<p>Для заданного массива $A$ и числа $k$ требуется найти количество пар $(l, r)$, таких что $l \le r$ и сумма чисел на отрезке массива $A$ с $l$ по $r$ делится на $k$ без остатка.</p>

### 입력

<p>На первой строке ввода заданы целые числа $n$  --- число элементов массива $A$ и $k$ ($1 \le n \le 200\,000$, $2 \le k \le 10^9$).</p>

<p>На второй строке заданы целые числа $a_1, a_2, \ldots, a_n$ --- элементы массива $A$ ($1 \le a_i \le 10^9$).</p>

### 출력

<p>Выведите одно число: количество пар $(l, r)$, таких что $l \le r$, и сумма чисел на отрезке массива $A$ с $l$ по $r$ делится на $k$ без остатка.</p>

### 힌트

<p>В примере подходят следующие отрезки:</p>

<ul>
	<li>$l = 1$, $r = 3$, отрезок $[1, 2, 3]$</li>
	<li>$l = 1$, $r = 4$, отрезок $[1, 2, 3, 4]$</li>
	<li>$l = 2$, $r = 2$, отрезок $[2]$</li>
	<li>$l = 2$, $r = 5$, отрезок $[2, 3, 4, 5]$</li>
	<li>$l = 3$, $r = 5$, отрезок $[3, 4, 5]$</li>
	<li>$l = 4$, $r = 4$, отрезок $[4]$</li>
</ul>
