### 문제 설명

<p>Множество $A = \{a_1, a_2, \ldots, a_k\}$ различных натуральных чисел с суммой $a_1+a_2+\ldots+a_k=n$ называется <em>генератором квадратов</em>, если сумма любых $k-1$ элементов этого множества является полным квадратом целого числа.</p>

<p>Например, множество $\{1, 22, 41, 58\}$ является генератором квадратов, так как $1 + 22 + 41 = 64 = 8^2$, $1 + 22 + 58 = 81 = 9^2$, $1 + 41 + 58 = 100 = 10^2$, $22 + 41 + 58 = 121 = 11^2$.</p>

<p>По заданным $n$ и $k$ постройте множество из $k$ различных натуральных чисел с суммой $n$, которое является генератором квадратов, либо выясните, что такого нет.</p>

### 입력

<p>На ввод подаются два целых числа $n$ и $k$ ($2 \le n \le 200\,000$, $2 \le k \le 30$).</p>

### 출력

<p>Если искомый генератор квадратов существует, выведите &lt;&lt;<code>YES</code>&gt;&gt; на первой строке, а на второй строке выведите $k$ натуральных чисел --- искомое множество.</p>

<p>Если генератора квадратов с заданными параметрами не существует, выведите &lt;&lt;<code>NO</code>&gt;&gt;.</p>
