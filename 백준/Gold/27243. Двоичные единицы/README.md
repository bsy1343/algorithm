### 문제 설명

<p>Рассмотрим натуральное число $x$. Требуется прибавить к нему минимальное возможное целое неотрицательное число $y$, чтобы двоичная запись получившегося числа $x+y$ имела ровно $k$ единиц.</p>

### 입력

<p>Первая строка ввода содержит натуральное число $x$ ($1 \le x \le 10^{18}$).</p>

<p>Вторая строка ввода содержит натуральное число $k$ ($1 \le k \le 60$).</p>

### 출력

<p>Выведите минимальное возможное целое неотрицательное число $y$, такое что двоичная запись числа $x+y$ имеет ровно $k$ единиц.</p>
