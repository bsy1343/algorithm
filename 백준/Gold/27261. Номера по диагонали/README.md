### 문제 설명

<p>Пронумеруем клетки прямоугольной таблицы с $r$ строками и $c$ столбцами, начиная с левого верхнего угла. Нумерацию будем вести по диагоналям, идущим справа-сверху налево-вниз, клетки одной диагонали будем нумеровать сверху вниз. </p>

<p>Например, для таблицы $3 \times 5$ клетки будут пронумерованы следующим образом:</p>

<table class="table table-bordered table-center-30 td-center">
	<tbody>
		<tr>
			<td>1</td>
			<td>2</td>
			<td>4</td>
			<td>7</td>
			<td>10</td>
		</tr>
		<tr>
			<td>3</td>
			<td>5</td>
			<td>8</td>
			<td>11</td>
			<td>13</td>
		</tr>
		<tr>
			<td>6</td>
			<td>9</td>
			<td>12</td>
			<td>14</td>
			<td>15</td>
		</tr>
	</tbody>
</table>

<p>Задано $q$ номеров клеток. Для каждого номера найдите, в какой клетке он находится.</p>

### 입력

<p>Первая строка ввода содержит три целых числа: $r$, $c$ и $q$ ($1 \le r, c \le 10^9$, $1 \le q \le 100$).</p>

<p>Вторая строка содержит $q$ целых чисел $1 \le n_1 &lt; n_2 &lt; \ldots &lt; n_q \le r\cdot c$.</p>

### 출력

<p>Выведите $q$ строк. Для каждого числа $n_i$ выведите два числа: номер строки и номер столбца, где находится соответствующая клетка. Строки нумеруются с 1 сверху вниз. Столбцы нумеруются с 1 слева направо.</p>
