### 문제 설명

<p>Задано целое положительное число $n$. Требуется найти число способов представить его в виде суммы нечетных слагаемых. При этом разбиения, отличающиеся только порядком слагаемых, считаются одинаковыми.</p>

<p>Например, число 6 можно представить следующими способами: $1+1+1+1+1+1$, $1+1+1+3$, $3+3$, $1+5$.</p>

### 입력

<p>На вход подается число $n$ ($1 \le n \le 1000$).</p>

### 출력

<p>Выведите число способов представить $n$ в виде суммы нечетных слагаемых.</p>
