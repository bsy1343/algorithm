### 문제 설명

<p>어떤 트리에서 차수가 $1$ 이하인 모든 정점을 동시에 제거하는 작업을 반복적으로 수행할 수 있다. 정점을 제거하면 그 정점에 연결된 간선들 역시 같이 제거된다.  </p>

<p>$i$번째 작업에서 제거한 정점의 개수를 $A_i$라 하고, 이 $A_i$를 순서대로 이어 붙여 만든 수열을 <strong>리프 수열</strong>이라고 하자. 이 정의에 따르면 모든 트리는 그에 대응되는 리프 수열을 유일하게 가진다.</p>

<p>길이 $N$의 수열 $A$가 주어진다. 주어진 수열을 리프 수열으로 가지는 트리를 <strong>아무거나 하나</strong> 구해보자. 만약 그러한 트리가 존재할 수 없다면 <span style="color:#e74c3c;"><code>-1</code></span>을 출력한다.</p>

### 입력

<p>첫째 줄에 수열의 길이 $N$이 주어진다. $(1 \le N \le 100\ 000)$</p>

<p>둘째 줄에 $A_1, A_2, \dots, A_N$이 공백을 두고 주어진다. $(1 \le A_i \le 100\ 000;$ $\sum_{i=1}^{N}A_i \le 100\ 000)$</p>

### 출력

<p>첫째 줄에 주어진 수열을 리프 수열으로 가지는 트리의 정점 개수 $M$을 출력한다.</p>

<p>다음 $M-1$개의 줄에는 트리의 간선이 연결하는 두 정점의 번호를 공백을 두고 출력한다. 정점의 번호는 $1$ 이상 $M$ 이하의 정수여야 한다.</p>

<p>만약 주어진 수열을 리프 수열으로 가지는 트리가 존재하지 않는다면 <span style="color:#e74c3c;"><code>-1</code></span>을 출력한다.</p>

### 힌트

<p>그래프에서 <strong>차수(Degree)</strong>란 어떤 정점에 연결된 간선의 개수를 말한다.</p>
