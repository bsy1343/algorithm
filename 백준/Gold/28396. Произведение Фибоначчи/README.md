### 문제 설명

<p>Напомним, что последовательность чисел Фибоначчи определяется следующим образом: $F_0 = 1$, $F_1 = 1$, $F_n = F_{n-2} + F_{n-1}$. Последовательность чисел Фибоначчи начинается так: $1, 1, 2, 3, 5, 8, 13, 21, 34, \dots$.</p>

<p>Дано натуральное число $n$. Требуется посчитать количество способов представить его как произведение чисел Фибоначчи, каждое из которых больше $1$.</p>

### 입력

<p>Первая строка ввода содержит целое число $t$ &mdash; количество тестов ($1 \le t \le 50$)</p>

<p>Следующие $t$ строк содержат тесты, каждая строка содержит одно целое число $n$ ($2 \le n \le 10^{18}$).</p>

### 출력

<p>Для каждого теста вывести одно число &mdash; искомое количество способов.</p>

### 힌트

<p>В примере:</p>

<ul>
	<li>число $2$ можно представить в виде произведения чисел Фибоначчи единственным способом $2 = 2$;</li>
	<li>число $7$ нельзя представить в виде произведения чисел Фибоначчи;</li>
	<li>число $8$ можно представить двумя способами: $8 = 2 \cdot 2 \cdot 2$ и $8 = 8$;</li>
	<li>число $40$ можно представить двумя способами: $40 = 2 \cdot 2 \cdot 2 &middot; 5$ и $40 = 5 \cdot 8$.</li>
</ul>
