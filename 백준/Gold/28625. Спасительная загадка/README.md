### 문제 설명

<p>Загадочник похищает окружного прокурора Джила Колсона. </p>

<p>Чтобы спастись, Колсон должен отгадать три загадки. Бэтмен помогает ему отгадать первые две, но третья загадка оказалась слишком сложной. Сможете ли вы помочь ему с отгадкой?</p>

<p>Загадка устроена следующим образом:</p>

<ol>
	<li>Загадочник записал и спрятал массив целых чисел $a$ длины $n$.</li>
	<li>Затем он циклически сдвинул его на какую-то величину $x$ влево, после чего полученный сдвиг поэлементно вычел из исходного массива.</li>
	<li>Полученный в результате массив $$b = [a_1 - a_{x+1}, a_2 - a_{x+2}, \ldots, a_{n} - a_{x}]$$ (иными словами, массив, в котором $b_i = a_i - a_{(i + x) \bmod n}$ для всех $i$), Загадочник сообщил Колсону.</li>
</ol>

<p>Требуется по данному массиву $b$ восстановить <strong>все возможные</strong> величины сдвига $x$, которые могли привести к получению такого массива $b$ из <strong>какого-то</strong> массива $a$.</p>

### 입력

<p>В первой строке ввода дано единственное целое число $n$ --- длина исходного массива ($1 \leqslant n \leqslant 10^6$).</p>

<p>Во второй строке через пробел перечислены $n$ целых чисел $b_1$, $b_2$, \ldots, $b_n$ --- элементы полученного Загадочником массива $b$ ($|b_i| \leqslant 10^9$).</p>

### 출력

<p>Выведите через пробел $n - 1$ целое число, $i$-е из которых равно $1$, если $x = i$ могло иметь место, и $0$ иначе.</p>

### 힌트

<p>В первом примере такой массив $b$ мог быть получен, например, из массива $a = [2, 4, 4]$ сдвигом на $1$ влево или из массива $a = [2, 2, 4]$ сдвигом на $2$ влево.</p>

<p>В первом примере данный массив $b$ ни при каком $a$ не мог быть получен сдвигом на $3$, а для сдвигов $1$ или $4$, например, подошли бы массивы $a = [1, 2, 0, 3, 7, 3]$ и $a = [3, 7, 0, 3, 4, 5]$, соответственно.</p>

<p>Для третьего примера можно показать, что такой $b$ в принципе не мог быть получен описанным в условии образом.</p>
