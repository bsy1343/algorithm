### 문제 설명

<p>Очень голодный Пеннивайз вновь проснулся спустя 27 лет. Преследуя детей, он случайно отвлекся, и им удалось спрятаться в комнате с кодовым замком. На двери комнаты имеется табло с двумя строками $s$ и $t$. </p>

<p>Чтобы узнать код, клоун должен посчитать количество способов выбрать непустую подстроку $s$, которую можно собрать из букв строки $t$. Подстрокой строки называется отрезок подряд идущих символов. Две подстроки считаются различными, если различаются позиции их начала или конца.</p>

<p>Помогите Пеннивайзу получить код и позавтракать!</p>

### 입력

<p>В первой строке дана строка $s$ ($1 \le |s| \le 10^6$). Во второй строке дана строка $t$ ($1 \le |t| \le 10^6$).</p>

<p>Строки состоят из строчных латинских букв.</p>

### 출력

<p>Выведите одно число --- искомое количество способов выбрать подстроку $s$.</p>

### 힌트

<p>В первом тесте существуют следующие способы выбрать подстроку (выделена скобками): </p>

<ol>
	<li><code>[a]aa</code></li>
	<li><code>a[a]a</code></li>
	<li><code>aa[a]</code></li>
	<li><code>[aa]a</code></li>
	<li><code>a[aa]</code></li>
</ol>

<p>Во втором тесте существуют следующие способы выбрать подстроку:</p>

<ol>
	<li><code>[a]bacaba</code></li>
	<li><code>a[b]acaba</code></li>
	<li><code>ab[a]caba</code></li>
	<li><code>aba[c]aba</code></li>
	<li><code>abac[a]ba</code></li>
	<li><code>abaca[b]a</code></li>
	<li><code>abacab[a]</code></li>
	<li><code>[ab]acaba</code></li>
	<li><code>a[ba]caba</code></li>
	<li><code>ab[ac]aba</code></li>
	<li><code>aba[ca]ba</code></li>
	<li><code>abac[ab]a</code></li>
	<li><code>abaca[ba]</code></li>
	<li><code>a[bac]aba</code></li>
	<li><code>aba[cab]a</code></li>
</ol>
