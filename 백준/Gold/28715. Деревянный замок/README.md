### 문제 설명

<p>Чтобы попасть в заброшенный дом, в котором прячется Оно, ребятам нужно открыть дверь с хитроумным замком. Этот замок представляет собой дерево из $n$ вершин, каждая из которых покрашена в белый или черный цвет. Чтобы открыть замок, нужно уничтожить все вершины этого дерева. Для этого ребята могут выполнять две операции:</p>

<ol>
	<li>Перекрасить еще не уничтоженную вершину из белого в черный, или из черного в белый.</li>
	<li>Запустить цепную реакцию, уничтожающую группу связных вершин одного цвета. Формально, ребята могут выбрать любую еще не уничтоженную вершину цвета $c$, уничтожить ее и все вершины цвета $c$, достижимые из нее по еще не уничтоженным вершинам цвета $c$.</li>
</ol>

<p>Разумеется, ребятам хочется поскорее попасть в дом, поэтому им интересно узнать, какое минимальное количество операций им потребуется, чтобы открыть замок.</p>

### 입력

<p>В первый строке дано целое число $n$ --- количество вершин в графе ($1 \le n \le 200\,000$). В следующей строке дана строка $s$ длины $n$ из символов $0$ и $1$. Если $i$-й символ строки $s$ равен $0$, то $i$-я вершина покрашена в белый цвет, иначе --- в черный. В следующих $n - 1$ строках дано по два целых числа $a_i$ и $b_i$ --- ребра дерева ($1 \le a_i, b_i \le n$).</p>

<p>Гарантируется, что ребра образуют дерево.</p>

### 출력

<p>Выведите одно число --- минимальное количество операций, необходимое, чтобы открыть замок.</p>

### 힌트

<p>В первом тесте замок можно открыть за два действия следующим образом:</p>

<ol>
	<li>Перекрасить вершину $1$ в белый цвет.</li>
	<li>Запустить цепную реакцию из вершины $1$, она уничтожит все вершины.</li>
</ol>
