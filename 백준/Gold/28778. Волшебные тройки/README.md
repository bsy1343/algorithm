### 문제 설명

<p>Пока Иэн и Барли ехали по шоссе, чтобы Барли не скучал, Иэн предложил ему посчитать количество волшебных троек. Тройка натуральных чисел $a$, $b$ и $c$ ($1 \le a &lt; b &lt; c \le n$) называется волшебной, если $a \cdot b$, $a \cdot c$ и $b \cdot c$ --- квадраты натуральных чисел.</p>

<p>Помогите Барли решить задачку Иэна, найдите количество волшебных троек.</p>

### 입력

<p>В единственной строке дано одно целое число $n$ ($1 \le n \le 200\,000$).</p>

### 출력

<p>Выведите одно число --- количество волшебных троек.</p>

### 힌트

<p>В первом примере единственной волшебной тройкой является $a = 1$, $b = 4$, $c = 9$.</p>

<p>Во втором примере существуют следующие волшебные тройки:</p>

<ul>
	<li>$a = 1$, $b = 4$, $c = 9$</li>
	<li>$a = 1$, $b = 4$, $c = 16$</li>
	<li>$a = 1$, $b = 9$, $c = 16$</li>
	<li>$a = 4$, $b = 9$, $c = 16$</li>
	<li>$a = 2$, $b = 8$, $c = 18$</li>
</ul>
