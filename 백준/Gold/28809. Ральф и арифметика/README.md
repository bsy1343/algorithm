### 문제 설명

<p>Ральф --- второстепенный персонаж компьютерной игры, и ему надоело находиться в тени главного героя. Ральф заметил кое-что общее между его компьютерной игрой и арифметикой. </p>

<p>Ральф считает, что в арифметике некоторые цифры встречаются чаще других, делая все остальные цифры <code>второстепенными</code>. Чтобы проверить свою гипотезу, Ральф выписал все второстепенные цифры и теперь хочет узнать количество чисел от $1$ до $n$, которые не содержат второстепенных цифр в своей десятичной записи. Помогите ему это сделать.</p>

### 입력

<p>Первая строка входного файла содержит целое число $n$ ($1 \le n \le 10^{18}$).</p>

<p>Вторая строка содержит целое число $k$ --- количество цифр, которые Ральф считает второстепенными ($1 \le k \le 9$).</p>

<p>В третьей строке через пробел записаны сами второстепенные цифры $d_1, \ldots d_k$ ($0 \le d_1 &lt; d_2 &lt; \ldots &lt; d_k \le 9$).</p>

### 출력

<p>В единственной строке выходного файла выведите одно число --- количество чисел от $1$ до $n$, в десятичной записи которых не встречаются второстепенные цифры.</p>

### 힌트

<p>В первом тестовом примере подходят все числа от $1$ до $9$, кроме $3$ и $4$.</p>

<p>Во втором тестовом примере подходят только числа $1$, $11$ и $111$.</p>

<p>В третьем тестовом примере подходят все числа длиной от $1$ до $5$, состоящие только из $3$ и $4$.</p>
