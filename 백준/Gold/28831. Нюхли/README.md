### 문제 설명

<p>Ньют Саламандер в очередной раз наблюдает за детенышами нюхлей. Ему интересно, так ли хорошо они ищут золото, как и взрослые особи.</p>

<p>Для испытаний Ньют взял $n$ коробок и соединил их $n - 1$ двунаправленными тоннелями так, чтобы между каждыми двумя коробками был ровно один простой путь. Ньют называет  <em>тупиком</em> любую коробку, в которую можно попасть только по одному тоннелю.</p>

<p>Ньют хочет разместить нюхля в одном тупике, а в каком-то другом тупике разместить золотую монету. Однако так как нюхль еще маленький, Ньют хочет выбрать тупики так, чтобы детеныш прошел как можно меньше тоннелей при поиске монеты.</p>

<p>Ваша задача помочь Ньюту найти минимальное число тоннелей, которое придется пройти детенышу нюхля, чтобы найти монету при оптимальном выборе тупиков.</p>

### 입력

<p>В первой строке дано целое число $n$ --- число коробок ($2 \le n \le 10^5$).</p>

<p>В следующих $n - 1$ строках заданы по два числа $a_i$, $b_i$ --- номера коробок, которые соединены $i$-м тоннелем ($1 \le a_i, b_i \le n$).</p>

<p>Гарантируется, что между любыми двумя коробками, существует ровно один простой путь.</p>

### 출력

<p>Выведите одно число --- минимальное расстояние, которое нужно пройти нюхлю, чтобы найти монету.</p>
