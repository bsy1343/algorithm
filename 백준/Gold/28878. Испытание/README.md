### 문제 설명

<p>После схватки с Хелой Тор свалился на весьма странную планету Сакаар. Его сразу же захватили  и отправили в качестве гладиатора на арену принимать участие в битве чемпионов Грандмастера, правителя планеты. Первое же испытание оказалось не физическим, а умственным, и показалось Тору чрезвычайно тяжелым. </p>

<p>Первоначально герою дали массив $f$ из $n$ чисел и поставили перед ним непосильную задачу: применить к массиву определенное количество раз запрашиваемые операции и после каждой операции ответить верно на вопрос, озвученный ниже. Операции бывают двух видов:</p>

<ul>
	<li>OR x --- к каждому элементу массива применить операцию побитового &lt;&lt;ИЛИ&gt;&gt; с числом $x$ (то есть $f_i = f_i\,|\,x$ для всех $1 \leq i \leq n$).</li>
	<li>AND x --- к каждому элементу массива применить операцию побитового &lt;&lt;И&gt;&gt; с числом $x$ (то есть $f_i = f_i\&amp;x$ для всех $1 \leq i \leq n$).</li>
</ul>

<p>После каждой операции Тор должен назвать минимальное число непрерывных отрезков массива, на каждом из которых элементы расположены в неубывающем порядке. </p>

<p>Если вы желаете, чтобы Тор одержал победу в битве, помогите ему справиться с поставленной задачей. Только ваши усилия способны ему помочь!</p>

### 입력

<p>Первая строка входных данных содержит натуральное число $n$ --- размер массива $f$ ($1 \le n \le 10^5$).</p>

<p>Во второй строке находятся $n$ целых чисел $f_i$ --- исходные элементы массива ($0 \le f_i \le 10^9$).</p>

<p>Третья строка содержит число $m$ --- количество операций ($1 \le m \le 10^5$). Следующие $m$ строк содержат сами операции. Каждая строка содержит тип операции и целое число $x$, в формате, указанном в условии ($0 \le x \le 10^9$).</p>

### 출력

<p>Для каждой операции в отдельной строке выведите ее результат.</p>
