### 문제 설명

<p>После многочисленных приключений Т&#39;Чалле предстоит финальная битва против Киллмонгера! Так как герои уже порядком устали за время фильма, они решили выяснить отношения в более спокойном интеллектуальном соревновании.</p>

<p>Правила соревнования установили следующие: сначала Киллмонгер придумывает строку $s_1 s_2 \dots s_n$, состоящую из $n$ строчных латинских букв.</p>

<p>Назовем <em>грубостью</em> строки $t_1 t_2 \dots t_k$ количество пар целых чисел $(i, j)$, где $1 \le i &lt; j \le k$, при этом $t_i = $ &lt;&lt;<code>a</code>&gt;&gt;, а также $t_j = $ &lt;&lt;<code>b</code>&gt;&gt;. Иными словами, грубость строки --- это количество способов вычеркнуть все ее символы, кроме двух, так, чтобы осталась строка, состоящая из двух букв: латинских букв &lt;&lt;<code>a</code>&gt;&gt; и &lt;&lt;<code>b</code>&gt;&gt; (именно в этом порядке).</p>

<p>После того, как Киллмонгер придумал строку, Т&#39;Чалла должен выбрать некоторую непустую ее подстроку $s_l s_{l + 1} \dots s_r$. При этом грубость выбранной подстроки не должна превышать числа $c$, иначе за такую грубость Т&#39;Чалла получит техническое поражение в игре.</p>

<p>Т&#39;Чалла побеждает в игре, если среди всех возможных подстрок строки $s$, грубость которых не превышает $c$, он выберет максимальную по длине (любую из них, если искомых подстрок максимальной длины несколько). Т&#39;Чалла не просит вас помогать ему находить икомую подстроку, ведь он и сам может справиться с этой задачей, однако для того, чтобы проверить себя, он просит вас узнать, какова же максимальная возможная длина искомой подстроки.</p>

### 입력

<p>Первая строка входных данных содержит два целых числа $n$ и $c$ --- длина строки, загаданной Киллмонгером, и максимальная разрешенная грубость выбранной подстроки ($1 \le n \le 10^6$, $0 \le c \le 10^{18}$).</p>

<p>Вторая строка содержит строку $s$, задуманную Киллмонгером. Строка состоит из $n$ строчных латинских букв.</p>

### 출력

<p>Выведите единственное число --- максимальную длину подстроки загаданной строки, которая имеет грубость не более $c$.</p>
