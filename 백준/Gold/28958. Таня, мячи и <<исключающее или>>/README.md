### 문제 설명

<p>У Тани было $n$ мячей, она пронумеровала их от $1$ до $n$. Но, к сожалению, Таня уронила все мячи в реку и сильно расстроилась.</p>

<p>Чтобы утешить ее, старший брат Сережа предложил Тане забавное математическое развлечение: посчитать сумму попарных <em>исключающих или</em> от номеров ее мячей.</p>

<p>Исключающее или двух чисел обозначается как $\oplus$ и соответствует операции &lt;&lt;<code>xor</code>&gt;&gt; в паскале или &lt;&lt;<code>\char 94</code>&gt;&gt; в других языках. Чтобы вычислить $x \oplus y$ для двух целых чисел, необходимо сделать следующее: представить каждое из чисел в двоичной системе счисления и сделать $i$-й разряд результата единицей, если он равен единице ровно в одном из чисел $x$ и $y$. Например, $3 \oplus 2 = 11_2 \oplus 10_2 = 1_2 = 1$, $17 \oplus 5 = 10001_2 \oplus 101_2 = 10100_2 = 20$.</p>

<p>Помогите Тане! Посчитайте сумму по всем парам ее мячей исключающих или их номеров. Таня не любит большие числа, так что ответ необходимо вывести по модулю $10^9 + 7$.</p>

<p>Например, если у Тани было $3$ мяча, то искомое значение равно $(1 \oplus 2) + (1 \oplus 3) + (2 \oplus 3) = 3 + 2 + 1 = 6$.</p>

### 입력

<p>В первой строке задано число $n$ --- количество мячей у Тани ($1 \le n \le 10^9$).</p>

### 출력

<p>Выведите сумму по всем парам ее мячей исключающих или их номеров, взятую по модулю $10^9 + 7$.</p>
