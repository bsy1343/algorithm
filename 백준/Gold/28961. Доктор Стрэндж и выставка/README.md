### 문제 설명

<p>У доктора Стрэнджа есть сад, в котором в ряд выставлены $n$ горшков с цветами. На каждом горшке написано некоторое число. На позиции номер $i$ стоит горшок с числом $a_i$. Иначе говоря, горшки образуют массив $a$.</p>

<p>Грядет выставка цветов. Доктор Стрэндж выберет для нее ровно $k$ горшков. Он хочет, чтобы его коллекция была самая запоминающаяся. Также доктор Стрэндж любит закономерности, поэтому он верит, что если побитовый <code>AND</code> чисел, написанных на выбранных горшках, будет равняться нулю, то его цветы произведут на всех неизгладимое впечатление.</p>

<p>Помогите доктору Стрэнджу понять, можно ли выбрать $k$ горшков, которые удовлетворяют этому условию.</p>

<p>Побитовый <code>AND</code> --- это бинарная операция, действие которой эквивалентно применению логического <code>AND</code> к каждой паре битов, которые стоят на одинаковых позициях в двоичных представлениях операндов.</p>

### 입력

<p>В первой строке находятся два натуральных числа $n, k$ ($1 \le n \le 2 \cdot 10^4, 1 \le k \le n$).</p>

<p>В следующей строке находятся $n$ неотрицательных целых чисел $a_i$ ($0 \le a_i &lt; 2^{12}$).</p>

### 출력

<p>В первой строке выведите <code>YES</code>, если существует способ выбрать $k$ горшков, чтобы их побитовый <code>AND</code> был равен нулю.</p>

<p>Если ответа не существует --- выведите <code>NO</code>.</p>
