### 문제 설명

<p>Все знают, что Симка активный фиксик. Сегодня она решила устроить день физкультуры, поэтому она будет заниматься спортом, пока не устанет. Для начала она попрыгает в длину.</p>

<p>Симка может прыгнуть на любое целое количество метров из интервала $[A \dots B]$. Чтобы немного разнообразить свои занятия, она хочет выбрать $a$, $A \le a \le B$, и каждый раз прыгать ровно на $a$ метров. Она собирается сделать от $C$ до $D$ прыжков. В результате своих занятий она хочет пропрыгать от $X$ до $Y$ метров.</p>

<p>Формально говоря, вам даны интервалы $[A \dots B], [C \dots D], [X \dots Y]$, от вас требуется выбрать целые $a$ и $b$ ($A \le a \le B, C \le b \le D$), такие что $X \le a \times b \le Y$.</p>

### 입력

<p>В единственной строке входного файла находятся шесть натуральных чисел $A$, $B$, $C$, $D$, $X$, $Y$ ($A \le B \le 10^{12}, C \le D \le 10^{12}, X \le Y \le 10^{12}$).</p>

### 출력

<p>В первой строке выведите <code>YES</code>, если решение существует, и в следующей строке два натуральных числа $a$ и $b$ ($A \le a \le B, C \le b \le D$). В противном случае, выведите <code>NO</code>.</p>

<p>Если существует несколько ответов --- выведите любой из них.</p>
