### 문제 설명

<p>Однажды, убираясь в комнате, Паша нашел массив, с которым он очень любил играть в детстве. Однако, сейчас Паша понимает, что массивы, в которых на $i$-ом месте стоит число большее, чем $a_i$, являются очень некрасивыми. </p>

<p>Кроме массива, он нашел листок, на котором были написаны операции, с помощью которых найденный массив был получен из массива, заполненного нулями. Операции имели вид: &lt;&lt;<em>на отрезке от $l$ до $r$ всем элементам добавить $1$</em>&gt;&gt;. Теперь Паша хочет убрать некоторые операции так, чтобы массив стал красивым. Помогите ему сэкономить время --- найдите минимальное число операций, которые требуется убрать!</p>

### 입력

<p>В первой строке входного файла задано одно число $n$ ($1 \le n\le 10^5$) --- размер массива. Во второй строке задано $n$ чисел $a_i$ ($1 \le a_i \le 10^5$) --- число в $i$-ой ячейке массива. В третьей строке задано число $m$ ($1 \le m \le 10^5$) --- число операций. В следующих $m$ строках задано по два числа $l_i, r_i$($1 \le l \le r \le n$) - описание операций. </p>

### 출력

<p>Выведите одно число --- ответ на задачу.</p>

### 힌트

<p>В первом примере необходимо убрать, например, четвертый отрезок.</p>
