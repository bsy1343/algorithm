### 문제 설명

<p>В альтернативной геометрии, изучаемой последователями доктора Иллитида, одной из ключевых фигур является $n$-угольная суперзвезда. Она используется для того, чтобы произносить заклинания, призывать Ктулху и делать много других крайне полезных вещей. Описание процесса построения фигуры приведено практически в любом справочнике для юных последователей, и состоит всего из трех пунктов:</p>

<ul>
	<li>возьмите окружность радиусом $10$ метров и поставьте на ней $n$ точек так, что длины дуг между всеми парами соседних точек одинаковы</li>
	<li>пронумеруйте точки от $0$ до $n - 1$, обходя окружность по часовой стрелке</li>
	<li>соедините отрезками те и только те точки, для которых верно равенство $(a + 2) \mod n = b$, где $a$ и $b$ --- номера соединяемых точек</li>
</ul>

<p>После этого вы можете встать в центр окружности и призвать Ктулху, у которого будет ровно $n$ щупалец. Все, что вам для этого понадобится --- правильно назвать площадь нарисованной вами звезды. Напишите программу, решающую эту задачу за вас.</p>

### 입력

<p>Входной файл содержит одно целое число $n$($5 \le n \le 100$) --- количество щупалец у Ктулху, которого вы хотите призвать.</p>

### 출력

<p>В выходной файл выведите одно целое число --- ответ на задачу, выведенный с точностью не меньшей, чем 5 знаков после запятой.</p>
