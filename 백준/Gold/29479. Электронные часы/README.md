### 문제 설명

<p>Однажды юный часовщик Максим нашел поломанные электронные часы с дисплеем, который показывает количество часов, минут и секунд. Максиму не составило труда починить их, и он решил на часах выставить определенное время.</p>

<p>Однако, найденные часы обладали интересной особенностью --- у них была всего одна кнопка. Данная кнопка при срабатывании прибавляла к количеству минут на циферблате часов одну минуту, не изменяя количества часов и секунд. Поясним, что если количество минут при срабатывании кнопки на циферблате было равно &lt;&lt;59&gt;&gt;, то при срабатывании кнопки количество минут на циферблате станет равно &lt;&lt;00&gt;&gt;, а количество часов не изменится. Например, при срабатывании кнопки во время &lt;&lt;21:59:23&gt;&gt; время на циферблате станет равно &lt;&lt;21:00:23&gt;&gt;.</p>

<p>Несмотря на починку, срабатывала кнопка не мгновенно, а по истечению $l$ секунд после нажатия. Например, если $l$ равно пяти, а Максим нажал кнопку в &lt;&lt;21:59:23&gt;&gt;, то через пять секунд на циферблате часов будет время &lt;&lt;21:00:28&gt;&gt;. А если на часах было &lt;&lt;21:59:55&gt;&gt;, то через пять секунд будет время &lt;&lt;22:01:00&gt;&gt;. В течении данных $l$ секунд нажатие кнопки не влияет на состояние часов.</p>

<p>Максим желает за наименьшее время выставить на циферблате часов определенное время. Напишите программу, которая определит, за какое количество секунд юный часовщик сможет достичь желаемого.</p>

### 입력

<p>Первая строка входного файла содержит время на часах в начальный момент в формате &lt;&lt;ЧЧ:ММ:СС&gt;&gt;. Вторая строка содержит в том же формате желаемое Максимом время на часах. Третья строка содержит целое число $l$ ($1 \le l \le 10$).</p>

### 출력

<p>Выведите в выходной файл единственное целое число --- количество секунд, которое понадобится Максиму для того, чтобы перевести часы.</p>
