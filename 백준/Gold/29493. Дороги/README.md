### 문제 설명

<p>Дорожная сеть города Нью-Флетсити устроена довольно просто. Все дороги являются отрезками единичной длины с концами в точках с целыми координатами. Этот факт --- своего рода достопримечательность Нью-Флетсити.</p>

<p>Недавно пришедший к власти мэр считает, что он тратит слишком много времени на дорогу из дома в мэрию и обратно. Он решил построить несколько новых дорог так, чтобы этот путь был как можно короче. Естественно, новые дороги должны также являться единичными отрезками с концами в целых точках.</p>

<p>Вам, как главному инженеру Нью-Флетсити, поручено вычислить минимальное количество дорог, которое придется построить для осуществления плана мэра.</p>

### 입력

<p>Первая строка входного файла содержит целое число $n$ --- количество дорог в Нью-Флетсити ($0 \le n \le 100$). Далее следуют $n$ строк с четырьмя целыми числами, разделенными пробелами: $x_i, y_i, x_j, y_j$ --- координаты начала и конца соответствующей дороги ($0 \le x_i, y_i, x_j, y_j \le 100$). Последняя строка содержит два целых числа $m_x$ и $m_y$ --- координаты дома мэра ($0 \le m_x, m_y \le 100$). Мэрия расположена в точке $(0, 0)$.</p>

<p>Все дороги расположены либо по горизонтали, либо по вертикали, а длина каждой из этих дорог равна единице. Движение по дорогам возможно в обе стороны.</p>

### 출력

<p>В выходной файл на первой строке выведите число $M$ --- количество новых дорог, которые нужно построить в Нью-Флетсити.</p>
