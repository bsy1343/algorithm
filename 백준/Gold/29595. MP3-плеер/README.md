# [Gold II] MP3-плеер - 29595

[문제 링크](https://www.acmicpc.net/problem/29595)

시간 제한: 2 초, 메모리 제한: 1024 MB

제출: 13, 정답: 11, 맞힌 사람: 7, 정답 비율: 87.500%

수학, 다이나믹 프로그래밍, 조합론, 포함 배제의 원리

### 문제 설명

<p>Вася недавно купил себе новый MP3-плеер <em>doPi Shuffle</em>. Этот плеер отличается от всех других, которые использовал Вася, тем, что у него есть функция &lt;&lt;перемешивания&gt;&gt; списка проигрываемых песен (плейлиста). </p>

<p>Вася загрузил в плеер $n$ своих любимых песен. До этого он их постоянно слушал на компьютере и очень хорошо запомнил порядок, в котором они были в плейлисте. </p>

<p>Разумеется, плеер переставил песни в случайном порядке. Прослушав плейлист несколько раз на плеере, Вася с удивлением обнаружил, что никакие две, шедшие подряд друг за другом в старом плейлисте, не идут подряд в новом. Он задумался, насколько часто возможна такая ситуация. </p>

<p>Пронумеруем песни числами от 1 до $n$ в том порядке, в котором они идут в плейлисте на компьютере. Напишите программу, которая вычисляет число способов их переставить так, чтобы никакие две песни с последовательными номерами не шли непосредственно друг за другом. Так как это число может быть достаточно большим, выведите его по модулю $m$.</p>

### 입력

<p>Входной файл содержит два целых числа: $n$ и $m$ ($2 \le n \le 2000$, $2 \le m \le 10^9$).</p>

### 출력

<p>В выходной файл выведите остаток от деления числа способов переставить песни требуемым образом на $m$.</p>
