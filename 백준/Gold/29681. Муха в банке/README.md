# [Gold III] Муха в банке - 29681

[문제 링크](https://www.acmicpc.net/problem/29681)

### 문제 설명

<p>Как настоящий юный энтомолог, Вася содержит у себя дома личную домашнюю муху. Муха у него живет в тонком, почти плоском стеклянном сосуде, по форме напоминающем прямоугольник.</p>

<p>По наблюдениям Васи, муха летает внутри сосуда с постоянной скоростью, при столкновении со стенками &lt;&lt;отражаясь&gt;&gt; от них согласно законам отражения света.</p>

<p>Ему стало интересно, где муха будет находится через некоторый момент времени, если изначально она была в самом центре сосуда (то есть центре правильного прямоугольника) и задано ее начальное направление полета. Для этого он попросил Вас написать программу.</p>

<p>В процессе своего полета муха, возможно, несколько раз отражается от стенок сосуда, являющихся сторонами прямоугольника, но никогда не попадает в какой-либо угол.</p>

### 입력

<p>Первая строка входного файла содержит два целых числа $w$ и $h$ ($1 \le w, h \le 100$) --- соответственно ширина и высота прямоугольника. Вторая строка содержит два целых числа $vx$ и $vy$ ($-100 \le vx, vy \le 100$) --- точка, в направлении которой изначально движется муха из центра прямоугольника, если центр прямоугольника находится в начале координат (точке $(0, 0)$), его стороны параллельны осям координат (сторона длиной $w$ вдоль оси OX, сторона длиной $h$ вдоль оси OY). При этом $vx$ и $vy$ не могут быть одновременно равны 0. Третья и последняя строка содержит единственное целое число $t$ ($0 \le t \le 100$) --- время полета мухи.</p>

### 출력

<p>В выходной файл выведите два вещественных числа $x$ и $y$ --- координаты мухи по истечении времени $t$, с точность не меньшей, чем $10^{-3}$.</p>
