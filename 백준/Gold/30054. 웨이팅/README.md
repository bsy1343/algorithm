### 문제 설명

<p>테이블이 1개 있는 24시간 맛집이 있다. 이 식당은 예약제로 운영되며 1시간에 오직 1명의 예약을 받는다.</p>

<p>어느 날, 식당 사장님은 예약하고도 제때 오지 않는 손님들이 많아 다음과 같은 규칙을 만들었다.</p>

<ol>
	<li>식당 입장은 매 정각에 이루어지며, 식당에 입장한 손님은 1시간 뒤에 퇴장한다.</li>
	<li>자신의 예약 시간이 아닌 시간에 도착한 손님들은 대기 줄에 선다. 동시에 도착한 손님들은 예약 시간이 빠른 순으로 선다.</li>
	<li>자신의 예약 시간에 늦지 않게 도착한 손님은 예약 시간이 되면 먼저 입장한다.</li>
	<li>예약이 없거나 예약자가 도착하지 않았다면 대기 줄의 첫 번째 손님이 입장한다.</li>
</ol>

<p>대기 줄에 있는 손님들은 인내심이 뛰어나기 때문에 식당에 입장할 때까지 줄을 이탈하지 않는다.</p>

<p>각 손님이 예약한 시각 $t_1$과 도착한 시각 $t_2$가 주어졌을 때, 식당에 도착해 입장할 때까지 가장 오래 기다린 손님이 몇 시간을 기다렸는지 구하여라.</p>

### 입력

<p>첫 번째 줄에 예약한 손님의 수 $N$이 주어진다. $(1 \leq N \leq 100\,000)$</p>

<p>두 번째 줄부터 각 사람이 예약한 시각 $t_1$과 도착한 시각 $t_2$가 공백으로 구분되어 주어진다. $t_1$과 $t_2$는 규칙을 도입한 첫 정각으로부터 몇 단위 시간이 지났는지를 의미한다. $t_1$과 $t_2$는 모두 정수이며 $t_1$은 서로 다르다. $(1 \leq t_1, t_2 \leq 200\,000)$</p>

### 출력

<p>식당에 도착해 가장 오래 기다린 손님이 몇 시간을 기다렸는지 출력하라.</p>
