### 문제 설명

<p>Вчера Вадим нашёл на дороге бинарное дерево $a$ с корнем в 0 из $N$ вершин. Однако любимым у него является бинарное дерево $b$ с корнем в 0 из $N$ вершин. Поэтому он решил преобразовать дерево $a$ в дерево $b$, используя следующую операцию:</p>

<ul>
	<li>Выбирается произвольная вершина $v$, кроме корня. Её поддерево, включая саму вершину, переподвешивается за другую вершину $u$, которая не принадлежит выбранному поддереву. Результатом должно получиться бинарное дерево с корнем в 0.</li>
</ul>

<p>Вадим уверен, что с помощью подобной операции возможно привести найденное бинарное дерево в изоморфное его любимому, используя не более, чем $N$ преобразований. Помогите ему найти последовательность этих преобразований.</p>

<p>Напомним, что <em>бинарное дерево</em> --- это такое дерево, что каждая вершина является предком не более, чем 2 других вершин, у корня предка нет. Два корневых бинарных дерева называются <em>изоморфными</em>, если:</p>

<ol>
	<li>Эти два дерева состоят из одной вершины;</li>
	<li>Количество детей у корней этих деревьев одинаковое, поддерево каждого ребёнка первого изоморфно поддереву какого-то ребёнка второго и поддерево каждого ребёнка второго изоморфно поддереву какого-то ребёнка первого.</li>
</ol>

### 입력

<p>В первой строке дано целое число $N$ --- количество вершин в найденном и любимом деревьях $(2 \le N \le 10^3)$.</p>

<p>Во второй строке даны $N-1$ целых чисел $pa_i$ --- предки вершин найденного дерева с номерами от 1 до $N-1$ $(0 \le pa_i \le N - 1)$.</p>

<p>Во третьей строке даны $N-1$ целых чисел $pb_i$ --- предки вершин любимого дерева с номерами от 1 до $N-1$ $(0 \le pb_i \le N - 1)$.</p>

<p>Гарантируется, что данные деревья бинарные.</p>

### 출력

<p>В первой строке выведите целое число $M$ --- количество использованных операций $(0 \le M \le N)$.</p>

<p>В следующих $M$ строках выведите пары чисел $v$ и $u$ --- корень выбранного поддерева и вершина, за которую это поддерево подвешивается во время текущей операции $(1 \le v \le N - 1, 0 \le u \le N - 1)$. Вершина $u$ не может находиться в поддереве вершины $v$. Полученное после каждой операции дерево должно быть бинарным.</p>

<p>Гарантируется, что ответ существует. Если ответов несколько, выведите любой из них.</p>
