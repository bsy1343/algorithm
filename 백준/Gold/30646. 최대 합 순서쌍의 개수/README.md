### 문제 설명

<p>크기가 <em>N</em>인 배열 a가 주어진다. 배열 a의 임의의 위치를 나타내는 두 수 i, j를 골랐을 때, 아래 두 조건을 만족하면 같은 수 순서쌍 (i, j)를 만들 수 있다.</p>

<ul>
	<li>1 &le; i &le; j &le; <em>N</em></li>
	<li>a<sub>i</sub> = a<sub>j</sub></li>
</ul>

<p>만들어진 같은 수 순서쌍 (i, j)의 합은 a<sub>i</sub>부터 a<sub>j</sub>까지의 합 a<sub>i</sub> + a<sub>i+1</sub> + a<sub>i+2 </sub>+ &hellip; + a<sub>j-1</sub> + a<sub>j</sub>로 정의된다. 이때 주어진 배열에서 만들 수 있는 같은 수 순서쌍의 최대 합을 찾고, 최대 합을 가지는 같은 수 순서쌍의 개수를 출력하는 프로그램을 작성하시오.</p>

### 입력

<p>첫 번째 배열 a의 크기 <em>N</em>이 주어진다.</p>

<p>두 번째 줄에 배열 a의 원소 a<sub>1</sub>, a<sub>2</sub>, &hellip;, a<sub>N</sub>이 주어진다.</p>

### 출력

<p>주어진 배열에서 만들 수 있는 같은 수 순서쌍의 최대 합과 최대 합을 가진 같은 수 순서쌍의 개수를 출력한다.</p>

### 제한

<ul>
	<li>1 &le; <em>N</em> &le; 200,000</li>
	<li>1 &le; a<sub>i</sub> &le; 10<sup>9</sup></li>
</ul>
