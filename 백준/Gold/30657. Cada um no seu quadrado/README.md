### 문제 설명

<p>Ana e Bob gostam de brincar de geometria. Semana passada, enquanto ouviam uma can&ccedil;&atilde;o que falava de quadrados, Bob, que estava brincando com seus blocos de madeira, se perguntou quantos quadrados poderiam ser formados usando os blocos como v&eacute;rtices, mas sem mov&ecirc;-los. Ana, que aprendeu a programar recentemente, teve a ideia de desenvolver um programa para resolver esse problema. Entretanto, Ana teve dificuldade para resolver esse problema e decidiu pedir a sua ajuda. Como os blocos s&atilde;o pequenos quando comparados &agrave; dist&acirc;ncia entre eles, voc&ecirc; pode assumir que eles s&atilde;o pontos em um plano.</p>

### 입력

<p>A entrada consiste de multiplos casos de teste. Cada caso de teste come&ccedil;a com uma linha contendo um &uacute;nico inteiro 4 &le; n &le; 1.000. As n linhas seguintes cont&eacute;m 2 inteiros x e y cada, &minus;1.000.000 &le; x, y &le; 1.000.000, referentes &agrave;s coordenadas dos blocos. A entrada termina com uma linha contendo n = 0, que n&atilde;o deve ser processada.</p>

### 출력

<p>Para cada caso de teste, voc&ecirc; deve imprimir uma linha contendo um &uacute;nico inteiro, o n&uacute;mero de quadrados que podem ser obtidos a partir das posi&ccedil;&otilde;es dadas.</p>
