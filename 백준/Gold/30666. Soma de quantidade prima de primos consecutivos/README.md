### 문제 설명

<p>Um número k é primo se e somente se k &ge; 2 e k tem exatamente dois divisores, 1 e k. Dois primos k e l são consecutivos se e somente se k &lt; l e não existe um primo p tal que k &lt; p &lt; l.</p>

<p>Dado n, dizer se n pode ser obtido como a soma de q primos consecutivos, onde q é primo.</p>

### 입력

<p>A entrada consiste de vários casos de teste. Cada caso de teste consiste de uma única linha contendo um inteiro 2 &le; n &le; 1.000.000.</p>

<p>A última linha da entrada conterá um inteiro n = 0. Esse caso não deve ser processado.</p>

### 출력

<p>Para cada linha da entrada, uma linha da saída deve ser gerada, contendo &ldquo;SIM&rdquo; caso n possa ser escrito da forma desejada ou &ldquo;NAO&rdquo;, caso contrário.</p>
