### 문제 설명

<p>Вам дана строка $s_1 s_2 \ldots s_n$, состоящая из $n$ маленьких английских букв. Жюри загадало перестановку $p_1, p_2, \ldots, p_n$ и получило новую строку $t$, поставив $i$-й символ строки $s$ на $p_i$-ю позицию в строке $t$. Вам необходимо найти строку $t$.</p>

<p>Для этого вы можете <strong>один раз</strong> задать вопрос, состоящий из $k$ выбранных вами пар индексов. Для пары индексов $(a_i, b_i)$ ($1 \leq i \leq k$, $1 \leq a_i &lt; b_i \leq n$) вы узнаете, верно ли, что $p_{a_i} &lt; p_{b_i}$.</p>

<p>Вы хотите определить искомую строку, спросив про наименьшее количество пар, то есть минимизируя значение числа $k$, при условии, что проверяющая программа является адаптивной. Это означает, что ответ к каждому тесту может быть различен в зависимости от того, про какие пары индексов вы спрашиваете. Другими словами, ваше решение должно успешно восстанавливать строку $t$ для любой возможной перестановки $p_1, p_2, \ldots, p_n$.</p>

### 힌트

<p>В первом примере сделан запрос про пару индексов $(1, 2)$, в ответ получена строка &lt;&lt;<code>1</code>&gt;&gt;, что означает, что $p_1 &lt; p_2$, и, следовательно, загадана перестановка $p = $&lt;&lt;$1$ $2$&gt;&gt;, то есть искомая строка --- &lt;&lt;<code>ab</code>&gt;&gt;.</p>

<p>Во втором примере сделан запрос про пару индексов $(1, 2)$, в ответ получена строка &lt;&lt;<code>0</code>&gt;&gt;, это означает, что $p_1 \geq p_2$, и, следовательно, загадана перестановка $p = $&lt;&lt;$2$ $1$&gt;&gt;, то есть искомая строка --- &lt;&lt;<code>ba</code>&gt;&gt;.</p>

<p>В третьем примере не спрашивается ни про какие пары индексов, а сразу выводится ответ --- строка &lt;&lt;<code>qqq</code>&gt;&gt;.</p>

<p>В четвертом примере была загадана перестановка &lt;&lt;$2$ $3$ $1$&gt;&gt;.</p>
