### 문제 설명

<p>В качестве новогоднего подарка Андрей получил коробку с конфетами. Или не совсем коробку. На самом деле он быстро обнаружил, что внутри коробки находятся ещё несколько одинаковых коробок меньшего размера, внутри которых содержатся ещё меньшие коробки и так далее... Формально, скажем что конфета является коробкой уровня $0$, а коробка уровня $i$ содержит в себе $a_i$ коробок уровня $i - 1$. Коробка, подаренная Андрею, имеет уровень $n$.</p>

<p>Сегодня к Андрею в гости придут друзья и он хочет поделиться с ними некоторым количеством конфет, для чего ему придётся открыть некоторое количество коробок. Разумеется, Андрей не может открыть коробку, если она находится внутри ещё не открытой коробки. Ему хотелось бы знать, какое минимальное количество коробок ему потребуется открыть, чтобы достать $x$ конфет. Поскольку Андрей ещё не уверен, сколько друзей к нему сегодня придут, он просит вас решить задачу для нескольких значений $x$.</p>

### 입력

<p>В первой строке входных данных записаны числа $n$ и $m$ ($1 \leq n, m \leq 300\,000$) --- количество коробок и количество вопросов Андрея соответственно.</p>

<p>Во второй строке записаны $n$ целых чисел $a_1, a_2, \ldots, a_n$ ($1 \leq a_i \leq 10^9$).</p>

<p>В третьей строке записаны $m$ чисел $x_1, x_2 \ldots, x_m$ ($1 \leq x_i \leq 10^{12}$) --- значения количества конфет, для которых Андрей хочет знать ответ. Гарантируется, что каждое значение $x_i$ не превосходит общего число конфет в коробке уровня $n$.</p>

### 출력

<p>Выведите $m$ целых чисел, $i$-е из них должно равняться минимальному количеству коробок, которое потребуется открыть Андрею, чтобы получить хотя бы $x_i$ конфет.</p>

### 힌트

<p>В первом примере единственная конфета спрятана в пяти уровнях коробок.</p>

<p>Во втором примере, чтобы получить $13$ конфет, Андрей должен открыть самую большую коробку, затем две коробки уровня $2$, и, наконец, пять из шести имеющихся коробок уровня $1$.</p>
