### 문제 설명

<p>Вера очень много работала в этом году, подавая своим коллегам пример настоящего труженика. На восьмое марта за прекрасное исполнение служебных обязанностей Вера получила подарок &mdash; долгожданный отпуск в Теплой Стране! Тяжелые трудовые будни закончились, и Вера уже нежится на пляже на берегу Теплого Моря.</p>

<p>Любимое хобби Веры &mdash; пляжный волейбол, и как же Вера ждала момента, когда она сможет испытать невероятный азарт этой игры! Вера уже познакомилась с несколькими симпатичными волейболистами, но она пока не решила, какая же команда достойна иметь в своем составе такого замечательного игрока.</p>

<p>Каждый из N капитанов команд мечтает заполучить Веру в состав своей команды, поэтому они хотят максимально проявить себя. Так как поиграть хотят все, они решили действовать следующим образом: все N команд выстроились в очередь. Первый матч играется между двумя командами, которые стоят в очереди раньше остальных. Победитель игры остается на площадке, а проигравший отправляется в конец очереди. В каждом из следующих матчей победитель предыдущего играет с первой командой из очереди, а проигравший в очередной встрече опять становится в конец очереди. Каждая команда имеет некоторую силу, причем для простоты будем предполагать, что силы всех команд различны, а победителем в матче является команда, сила которой больше. Матчей может быть как угодно много.</p>

<p>Вера решила для себя, что она будет действовать по самому справедливому принципу &laquo;считалочки&raquo;: она будет играть с одной из двух команд, играющих матч с соответствующем считалке номером K. Но затем Вера поняла, что уже выбрала себе команду, в которой хотела бы играть, причем ориентируясь не только на ее силу. Ей известны Q считалок, соответствующих различным значениям K. Для каждого из этих чисел K<sub>i</sub> необходимо узнать, а кто же именно будет сражаться за столь ценный приз, то есть какие две команды будут играть в матче с номером K<sub>i</sub>.</p>

### 입력

<p>Первая строка входных данных содержит единственное целое число N &mdash; количество команд (2 6 N 6 100 000). Вторая строка содержит N различных чисел от 1 до N &mdash; силы команд: первое число &mdash; сила команды, стоящей в начале очереди, второе &mdash; сила следующей по очереди команды, ..., последнее &mdash; сила команды, стоящей в конце очереди.</p>

<p>Третья строка содержит единственное целое число Q (1 &le; Q &le; 100 000) &mdash; количество известных Вере считалок. Каждая из следующих Q строк содержит число K<sub>i</sub> (1 &le; K<sub>i</sub> &le; 10<sup>18</sup>) &mdash; номер очередного интересующего Веру матча. Обратите внимание, Ki может быть больше N.</p>

### 출력

<p>Выведите Q строк: для каждого интересующего Веру числа Ki два числа в любом порядке &mdash; силы команд, сыграющих на K<sub>i</sub>-м шаге. Первая строка должна содержать ответ на первый запрос, вторая &mdash; на второй и так далее.</p>
