### 문제 설명

<p>Пятиклассник Вася живет в доме, в котором всего N квартир. Квартиры нумеруются с 1 по N, и около каждой квартиры на табличке написан ее номер. Вася хочет заменить часть табличек так, чтобы сумма номеров квартир стала равна S.</p>

<p>В магазине Вася может купить таблички с любыми натуральными числами. Помогите Васе определить минимальное количество табличек, которые ему придется купить, чтобы осуществить свой коварный план.</p>

### 입력

<p>В первой и единственной строке вводятся два целых числа: N и S (1 ⩽ N ⩽ 10<sup>9</sup>, 1 ⩽ S ⩽ 10<sup>18</sup>).</p>

### 출력

<p>Выведите единственное число: минимальное количество табличек, которые должен купить Вася, чтобы получить сумму S. Гарантируется, что это можно сделать.</p>
