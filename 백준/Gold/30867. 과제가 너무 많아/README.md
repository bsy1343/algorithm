### 문제 설명

<p>과제가 너무 많은 로하는 과도한 두통에 시달리며 영어 단어에 있는 모든 <span style="color:#e74c3c;"><code>wh</code></span>를 <span style="color:#e74c3c;"><code>hw</code></span>로 보게 되었다! 두통이 한번 올 때마다, 문자열 $S$는 다음의 시행에 따라 변화한다. 문자열 $S$는 $1$번째 글자부터 $L$번째 글자까지 있는 길이 $L$의 문자열이다.</p>

<ul>
	<li>문자열 $S$의 $i=1$번째 글자부터 $L-1$번째 글자까지, 다음 과정을 반복한다.
	<ul>
		<li>만약 $i$번째 글자와 $i+1$번째 글자가 차례로 <span style="color:#e74c3c;"><code>w</code></span>와 <span style="color:#e74c3c;"><code>h</code></span>라면, $i$번째 글자와 $i+1$번째 글자를 각각 <span style="color:#e74c3c;"><code>h</code></span>와 <span style="color:#e74c3c;"><code>w</code></span>로 바꾼다.</li>
	</ul>
	</li>
</ul>

<p>로하가 $N$번의 두통을 겪고 나서 주어진 문자열을 어떤 문자열로 보게 될지 출력하여라.</p>

### 입력

<p>첫 번째 줄에 문자열의 길이 $L$과 두통의 횟수 $N$이 공백으로 구분되어 주어진다.$(1\leq L\leq 200\, 000;$ $1\leq N\leq 200\, 000)$</p>

<p>두 번째 줄에 문자열 $S$가 주어진다. 문자열은 영어 소문자로만 이루어져 있다.</p>

### 출력

<p>첫 번째 줄에 로하가 $N$번의 두통을 겪은 후 보게 될 문자열을 출력하여라.</p>
