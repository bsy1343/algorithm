### 문제 설명

<p>서윤이네 뒷마당에는 $M$개의 귤이 열려 있는 커다란 귤나무가 있다.</p>

<p>이웃집에 사는 $N$마리의 곰곰이들은 이 귤나무에 매일 귤을 따러 온다. 매일 $1$번 곰곰이부터 시작해서 $N$번 곰곰이까지 차례대로 귤을 따려고 시도하는데, $i$번 곰곰이는 $A_i$개의 귤을 따려고 시도하며 나무에 남은 귤이 $A_i$개 미만이라면 아무 행동도 하지 않는다.</p>

<p>$10^{100}$ 일이 지났을 때, 귤나무에 남아있는 귤의 개수는 몇 개일지 구해보자.</p>

### 입력

<p>첫째 줄에 곰곰이의 수와 귤의 개수 $N$, $M$이 공백으로 구분되어 주어진다. $(1 \leq N \leq 200\,000;$ $1 \leq M \leq 10^{18})$</p>

<p>둘째 줄에 각 곰곰이가 따갈 귤의 개수 $A_1$, $A_2$, $\cdots$, $A_N$이 공백으로 구분되어 주어진다. $(1 \leq A_i \leq 10^{12})$</p>

<p>입력으로 주어지는 모든 수는 정수이다.</p>

### 출력

<p>$10^{100}$ 일이 지났을 때 귤나무에 남아있는 귤의 개수를 출력한다.</p>
