### 문제 설명

<p>아래와 같이 물음표를 포함한 부등식이 있다.</p>

<p>\[\frac{A_1}{A_2} &lt; \frac{?}{B} &lt; \frac{?}{C} &lt; \frac{?}{D} &lt; \frac{E_1}{E_2}\]</p>

<p>이때, 부등식이 성립하게 물음표를 양의 정수로 바꾸는 경우의 수는 모두 몇 개가 있을까?</p>

### 입력

<p>첫째 줄에 B, C, D가 주어진다. (1 &le; B, C, D &le; 1000)</p>

<p>둘째 줄에 A<sub>1</sub>, A<sub>2</sub>가 주어진다. (1 &le; A<sub>1</sub>, A<sub>2</sub> &le; 1000)</p>

<p>셋째 줄에 E<sub>1</sub>, E<sub>2</sub>가 주어진다. (1 &le; E<sub>1,</sub> E<sub>2</sub> &le; 1000)</p>

### 출력

<p>첫째 부등식을 만족하게 물음표를 양의 정수로 바꾸는 방법의 수를 출력한다.</p>

### 힌트

<p>세 번째 예제의 경우 다음과 같이 3가지 방법이 있다.</p>

<p>\[\frac{14}{5} &lt; \frac{3}{1} &lt; \frac{28}{9} &lt; \frac{22}{7} &lt; \frac{10}{3}\]\[\frac{14}{5} &lt; \frac{3}{1} &lt; \frac{28}{9} &lt; \frac{23}{7} &lt; \frac{10}{3}\]\[\frac{14}{5} &lt; \frac{3}{1} &lt; \frac{29}{9} &lt; \frac{23}{7} &lt; \frac{10}{3}\]</p>
