### 문제 설명

<p><var>N</var> 個のライトが横一列に並んでおり，左から順に <var>1</var> から <var>N</var> までの番号が付けられている．それぞれのライトの色は赤，緑，青のいずれかである．ライトの色は文字列 <var>S</var> によって表され，ライト <var>i</var> (<var>1 ≦ i ≦ N</var>) の色は，<var>S</var> の <var>i</var> 文字目が <code>R</code> のとき赤，<code>G</code> のとき緑，<code>B</code> のとき青である．最初すべてのライトは点灯している．</p>

<p>JOI 君は点灯しているライトが <var>1</var> つ以上ある限り，以下の <var>3</var> 種類の操作を好きな順番で好きな回数行うことができる．操作を <var>1</var> 回も行わなくても構わない．</p>

<ul>
	<li><var>A</var> 円を払って点灯しているライトの中で最も左にあるものを消灯する．</li>
	<li><var>B</var> 円を払って点灯しているライトの中で最も右にあるものを消灯する．</li>
	<li><var>C</var> 円を払って点灯しているライトを <var>1</var> つ選び，好きな色へ点灯し直す．</li>
</ul>

<p>JOI 君は，遠くからこのライトの列を見たときにきれいな白色に見えるようにしたい．そのためには，点灯しているライトを左から見たときの色の並びが <code>RGBRGB...RGB</code> のように <code>RGB</code>（赤緑青）の繰り返しになっている必要がある．ただし，<var>1</var> つも点灯しているライトが存在しない場合も <code>RGB</code> の繰り返しであるとみなす． <code>GBRGBR</code> や <code>RGBRG</code> などの色の並びは条件を満たさないことに注意せよ．</p>

<p>ライトと操作に必要な金額の情報が与えられたとき，点灯しているライトの色の並びを <code>RGB</code> の繰り返しにするために必要な金額の最小値を求めるプログラムを作成せよ．</p>

### 입력

<p>入力は以下の形式で与えられる．</p>

<pre>
<var>N</var>
<var>S</var>
<var>A</var>   <var>B</var>   <var>C</var></pre>

### 출력

<p>点灯しているライトの色の並びを <code>RGB</code> の繰り返しにするために必要な金額の最小値を，単位 (円) を省いて <var>1</var> 行で出力せよ．</p>

### 제한

<ul>
	<li><var>1 ≦ N ≦ 200 000</var>．</li>
	<li><var>S</var> は長さ <var>N</var> の文字列である．</li>
	<li><var>S</var> の各文字は <code>R</code>，<code>G</code>，<code>B</code> のいずれかである．</li>
	<li><var>1 ≦ A ≦ 10<sup>9</sup></var>．</li>
	<li><var>1 ≦ B ≦ 10<sup>9</sup></var>．</li>
	<li><var>1 ≦ C ≦ 10<sup>9</sup></var>．</li>
	<li><var>N, A, B, C</var> は整数である．</li>
</ul>
