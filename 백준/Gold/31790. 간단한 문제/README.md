### 문제 설명

<p><code>PULSE</code>를 떠나서 대학원에 입학한 산지니는 휴일 태종대에서 월척을 낚기 위해 낚시를 하고 있다. 하지만 몇 시간이 지나도 아무 소식이 없다. 산지니는 낚싯대를 너무 오래 쥐고 있어 졸리던 차에 간단한 수열을 생각하게 되었다.</p>

<p>길이가 $N$인 순열 $a$에 대해 $S$와 수열 $b$를 아래와 같이 정의하자.</p>

<ul>
	<li>$S_{i,j}(1 \leq i \leq N)$는 순열 $a$의 연속 부분 수열 $a_1, a_2, ..., a_i$의 원소 중 $p$로 나눈 나머지가 $j(0 \leq j &lt; p)$인 원소의 개수이다.</li>
	<li>$b_i(1 \leq i \leq N)$는 $S_{i,0}, S_{i,1}, ... , S_{i,p-1}$ 중 최댓값이다.</li>
</ul>

<p>산지니는 조금 더 생각하다 어떤 숫자 $p$와 수열 $b$에 대해서도 순열 $a$가 존재하는지 궁금해졌다. 조건을 만족하는 순열 $a$가 존재하는지 알아보자. </p>

### 입력

<p>첫 번째 줄에 수열 $b$의 길이 $N$, 정수 $p$가 공백으로 구분되어 주어진다. $(1 \leq N, p \leq 5 \times 10^5)$</p>

<p>두 번째 줄에 수열 $b$의 원소 $b_1, b_2, &hellip;, b_N$이 공백으로 구분되어 차례대로 주어진다. $(1 \leq b_i \leq N)$</p>

### 출력

<p> 조건을 만족하는 순열 $a$가 존재하면<span style="color:#e74c3c;"> </span><span style="color:#e74c3c;"><code>YES</code></span>, 존재하지 않으면 <span style="color:#e74c3c;"><code>NO</code></span>를 출력한다.</p>

### 힌트

<p>길이가 $N$인 순열은 $1$부터 $N$까지 정수가 한 번씩만 사용되는 유한수열이다. 예를 들어 $\{1, 2, 4, 3\}$은 순열이지만 $\{1, 4, 2\}$, $\{1, 3, 3, 2\}$ 등은 순열이 아니다.</p>
