### 문제 설명

<p>익준이는 한 프로그램에서 함수 호출이 일어난 함수명이 담긴 함수 호출 리스트를 가지고 있다. 이 리스트에서 <strong>서로 다른 원소</strong>로 이루어진 함수 호출 구간이 연속하여 가장 많이 반복된 구간을 확인하고 싶다.</p>

<p>예를 들어, $[A, B, C, A, B, C]$와 같은 함수 호출 리스트가 있으면, $[A, B, C]$ 함수 호출 구간이 총 두 번 반복된 것을 알 수 있다.</p>

<p>그러나 $[A, B, C, B, A, B, C, B]$와 같은 함수 호출 리스트의 경우, 구간 $[A, B, C, B]$가 연속하여 반복되지만 함수명 $B$가 두 번 포함되어 있으므로 조건을 만족하지 않는다.</p>

### 입력

<p>첫 번째 줄에 정수 $N$이 주어진다. ($1 \leq N \leq 100\,000$)</p>

<p>두 번째 줄에 함수 호출 리스트에 속하는 영어 대문자로만 이루어진 문자열 $S_i$가 공백을 사이에 두고 $N$개 주어진다. ($1 \leq$ $| S_i |$ $\leq 10$)</p>

### 출력

<p>첫 번째 줄에 가장 많이 반복된 함수 호출 구간 중 제일 먼저 나오는 구간의 길이와 반복된 횟수를 출력한다.</p>

<p>두 번째 줄에 해당 함수 호출 구간을 공백을 사이에 두고 출력한다.</p>

<p>단, 여러 번 반복된 구간이 없다면 <span style="color:#e74c3c;"><code>-1</code></span>만을 출력한다.</p>

### 힌트

<p>$|S|$는 문자열 $S$의 길이를 의미한다.</p>
