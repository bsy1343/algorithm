# [Gold II] 시간을 달려서 (Rough) - 32285

[문제 링크](https://www.acmicpc.net/problem/32285)

### 성능 요약

시간 제한: 1 초, 메모리 제한: 1024 MB

### 통계

제출: 597, 정답: 85, 맞힌 사람: 54, 정답 비율: 15.698%

### 분류

그리디 알고리즘, 수학, 정수론

### 문제 설명

<blockquote>
<p>미처 말하지 못했어</p>

<p>다만 너를 좋아했어</p>

<p>어린 날의 꿈처럼 마치 기적처럼</p>

<p>시간을 달려서 어른이 될 수만 있다면</p>

<p>거친 세상 속에서 손을 잡아줄게</p>
</blockquote>

<p>나는 시간을 달려서 너를 만나고자 한다.</p>

<p>나와 네가 있는 세계에서 시간은 음이 아닌 정수로 나타낼 수 있다.</p>

<p>처음에 나는 시간 $0$에 있고, 너는 시간 $G$에 있다.</p>

<p>시간 $x$에 있는 내가 시간을 달려 이동할 수 있는 방법은 2가지가 있다.</p>

<ul>
	<li>$x+1$로 이동한다</li>
	<li>$2 \times x$로 이동한다</li>
</ul>

<p>내가 있는 시간은 이상한 구조로 되어 있기 때문에, 이동한 뒤 나의 시간이 $F$ 이상이라면 나는 $x \bmod F$ 시간으로 이동하게 된다. 여기서 $x \bmod F$란 $x$를 $F$로 나눈 나머지를 의미한다.</p>

<p>내가 시간 속에 갇혀 길을 헤매지 않도록, 너의 시간에 도착하기 위해 시간을 달려 이동하는 횟수의 최솟값을 구하여라.</p>

### 입력

<p>첫째 줄에 양의 정수 $G,F$가 주어진다.</p>

### 출력

<p>첫째 줄에 시간을 달려 이동하는 횟수의 최솟값을 출력하라.</p>

### 제한

<ul>
	<li>$0&lt;G&lt;F \leq 10^{18}$</li>
	<li>$G \leq 10^7$</li>
</ul>

### 힌트

<p></p>

<p>입력이 C/C++의 <span style="color:#e74c3c;"><code>int</code></span> 범위를 넘어갈 수 있으므로 <span style="color:#e74c3c;"><code>long long</code></span> 자료형을 사용하는 것을 추천한다.</p>
