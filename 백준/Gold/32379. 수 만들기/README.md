### 문제 설명

<p>두 양의 정수 $A,B$가 주어집니다.</p>

<p>$A$에 다음의 두가지 연산을 순서와 횟수에 상관 없이 원하는 만큼 반복할 수 있습니다.</p>

<ol>
	<li>$A$의 $10$진법 표현을 $d_{k-1} \cdots d_0$라고 합시다. 즉, $A = \sum\limits_{i = 0}^{k-1} d_i \cdot 10^i$, $d_i$는 $0 \le d_i \le 9$를 만족하는 정수이며, $d_{k-1} \ne 0$ 입니다. 	임의의 정수 $i (0 \le i &lt; k)$와 $x (0 \le x \le 9)$를 골라, $d_i$를 $x$로 교체합니다. 단, $i = k-1$이라면 $0&lt; x \le 9$을 만족해야 합니다. 	해당 연산의 비용은 $|x-d_i|$입니다.</li>
	<li>$y&gt;-A$인 임의의 정수 $y$를 골라, $A$에 더합니다. 	해당 연산의 비용은 $|y|$입니다.</li>
</ol>

<p>예를 들어, 수 $2024$에 $i=1, x=7$를 골라 $1$번 연산을 수행하면 $2074$가 되며, 해당 연산의 비용은 $|7-2| = 5$입니다.</p>

<p>그러나 $i=3, x=0$를 골라 $0024$를 만들거나, $i=4, x=1$를 골라 $12024$를 만드는 것은 조건을 만족하지 않으므로 불가능합니다.</p>

<p>또한, 수 $926$에 $y=-926$를 골라 $2$번 연산을 수행하여 $0$으로 만드는 것 역시 조건을 만족하지 않으므로 불가능합니다.</p>

<p>$A$를 $B$로 만드는 데 드는 비용의 합의 최솟값은 얼마일까요?</p>

### 입력

<p>첫 번째 줄에 두 양의 정수 $A, B (1 \leq A, B &lt; 5\,000\,000)$가 공백으로 구분되어 주어집니다.</p>

### 출력

<p>$A$를 $B$로 만드는 데 드는 비용의 합의 최솟값을 출력해주세요.</p>
