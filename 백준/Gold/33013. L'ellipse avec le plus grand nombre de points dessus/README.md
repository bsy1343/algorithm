### 문제 설명

<p>$0$이 아닌 어떤 실수 $a$, $b$와 실수 $c$에 대해 $\left(\frac{x}{a}\right)^2+\left(\frac{y}{b}\right)^2=c^2$를 만족시키는 점 $(x, y)$의 집합을 <strong>원점타원</strong>이라고 한다. 주어진 $N$개의 점들 중, 같은 원점타원 위에 있는 점들의 최대 개수를 구하시오.</p>

### 입력

<p>첫 번째 줄에 점의 개수 $N$이 주어진다. ($2 \le N \le 3000$)</p>

<p>다음 $N$개 줄에 점의 위치 $(x, y)$를 나타내는 정수 $x$, $y$가 공백으로 구분되어 한 줄에 하나씩 주어진다. 같은 위치에 여러 점이 있을 수 있다. ($-10^9 \le x, y \le 10^9$)</p>

### 출력

<p>같은 원점타원 위에 있는 점들의 최대 개수를 출력한다.</p>
