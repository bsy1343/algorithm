### 문제 설명

<p>bnb2011은 너무 똑똑하다. 그래서 매일 주변 사람들이 퍼즐을 던진다. 오늘 bnb2011이 받은 퍼즐의 내용은 다음과 같다.</p>

<p>$N$행 $N$열의 격자가 주어진다. 당신은 격자의 $N^2$개의 칸에 $0$ 이상, $10\,000$ 이하의 정수를 중복을 허용하여 자유롭게 채울 수 있다. 모든 칸에 정수를 채운 뒤, 격자의 $i$번째 행에 적힌 수들의 합을 $R_i$, $j$번째 열에 적힌 수들의 합을 $C_j$라고 하자. 이때, $\operatorname{mex}  (\{R_1, R_2, \cdots, R_N, C_1, C_2, \cdots, C_N\})$이 최대가 되도록 격자를 구성해 보자. $\operatorname{mex} (S)$는 집합 $S$에 포함되지 않은 가장 작은 음이 아닌 정수로 정의한다.</p>

<p>bnb2011은 적당한 휴식을 취하려고 했지만, 퍼즐이 계속 눈에 아른거려 쉬지 못하고 있다. bnb2011이 휴식을 취할 수 있도록 퍼즐을 해결해 보자.</p>

### 입력

<p>첫째 줄에 격자의 한 변의 길이를 나타내는 정수 $N$이 주어진다. $(1 \le N \le 2\,000)$</p>

### 출력

<p>$N$개의 줄에 걸쳐 각 줄에 $N$개의 정수를 출력한다. $i$번째 줄에서 $j$번째로 출력되는 정수는 격자의 $i$행 $j$열에 적힌 정수를 의미한다. 답으로 가능한 격자가 여러 가지 존재한다면, 그중 하나를 출력하면 된다.</p>
