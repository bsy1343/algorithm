### 문제 설명

<p>곧 있을 루미의 생일을 축하해 주기 위해 $N$명이나 되는 많은 사람들이 생일 축하 멘션을 보내왔다. 하지만 도배가 허용된 채팅방에 멘션을 허용해 준 나머지 다른 사람들의 멘션을 너무 많이 받아 답장하기 어려워졌다. 따라서 루미는 각 멘션마다 메시지를 보내주는 매크로를 만들기로 하였다.</p>

<p>그런데 루미는 한 사람이 보낸 서로 다른 멘션에 똑같은 메시지를 보내면 루미가 그 사람에게 매크로 의심을 받을 것이라고 생각했다. 그래서 루미는 매크로 의심을 피하기 위해 아래와 같이 매크로를 만들기로 했다.</p>

<ol>
	<li>서로 다른 준비된 $M$개의 메시지에 $1$번, $2$번, $3$번, ..., $M$번으로 번호를 매긴다.</li>
	<li>멘션을 $2$회 이상한 사람들이 멘션을 한 횟수의 평균을 구한다. 단, $2$회 이상 멘션을 보낸 사람이 없다면 평균은 $0$이다.</li>
	<li>멘션을 한 횟수가 평균 이하인 사람에게는 첫 번째 멘션부터 순서대로 각 멘션마다 $1$번 메시지부터 차례대로 보낸다. 만약 준비된 메시지가 멘션의 양보다 적어 $M$개의 메시지를 다 쓴 경우에는 $1$번 메시지로 돌아와 다시 메시지를 보내는 과정을 반복한다.</li>
	<li>멘션을 한 횟수가 평균을 초과한 사람에게는 그 사람의 첫 번째 멘션에만 $1$번 메시지를 $1$회 보내도록 한다.</li>
</ol>

<p>예를 들어</p>

<ol>
	<li>평균 멘션 횟수가 $5$회, 준비된 메시지 개수가 $6$개일 때
	<ul>
		<li>$4$회 멘션을 보낸 사람에게는 $1,2,3,4$번째 멘션에 $1,2,3,4$번 메시지를 각각 보내게 된다.</li>
		<li>$6$회 멘션을 보낸 사람에게는 $1$번 메시지만 $1$회 보낸다.</li>
	</ul>
	</li>
	<li>평균 멘션 횟수가 $5$회, 준비된 메시지 개수가 $3$개일 때
	<ul>
		<li>$4$회 멘션을 보낸 사람에게는 $1,2,3,4$번째 멘션에 $1,2,3,1$번 메시지를 각각 보내게 된다.</li>
		<li>$6$회 멘션을 보낸 사람에게는 $1$번 메시지만 $1$회 보낸다.</li>
	</ul>
	</li>
</ol>

<p>이와 같이 매크로를 만들고 $M$개의 메시지를 준비했을 때 $1$번부터 $M$번까지 각 메시지를 보낸 횟수를 구하라.</p>

### 입력

<p>첫 번째 줄에 $N, M$이 공백으로 구분되어 주어진다.</p>

<p>두 번째 줄에 $i(1\leq i\leq N)$번째 사람이 멘션한 횟수 $P_i$가 공백으로 구분되어 주어진다.</p>

### 출력

<p>한 사람이 보낸 서로 다른 멘션에 똑같은 메시지를 보내 매크로 의심을 받게 된다면 <span style="color:#e74c3c;"><code>-1</code></span>을, 아니라면 $1$번부터 $M$번까지 메시지를 보낸 횟수를 공백으로 구분해 출력하라.</p>

### 제한

<ul>
	<li>$1\leq N\leq 10^6$</li>
	<li>$1\leq M\leq 10^6$</li>
	<li>$1\leq P_i\leq 10^9$</li>
	<li>입력으로 주어지는 수는 모두 정수이다.</li>
</ul>
