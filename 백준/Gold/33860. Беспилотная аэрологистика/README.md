### 문제 설명

<p>На всероссийской олимпиаде по информатике 2224 года, которая проходит в Иннополисе, доставкой занимаются роботы нового поколения, которые способны создавать своих клонов. Доставку можно получить прямо через окно, не выходя из дома.</p>

<p>Изначально есть только один робот-доставщик. В любой момент верхний робот может создать одного или нескольких новых роботов прямо над собой. Так образуется колонна роботов. Высота каждого робота равна высоте одного этажа.</p>

<p style="text-align: center;"><img alt="" src="%EB%B0%B1%EC%A4%80/Gold/33860.%E2%80%85%D0%91%D0%B5%D1%81%D0%BF%D0%B8%D0%BB%D0%BE%D1%82%D0%BD%D0%B0%D1%8F%E2%80%85%D0%B0%D1%8D%D1%80%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0/bb8c0e5c.png" data-original-src="https://boja.mercury.kr/assets/problem/33860-1.png" onerror="this.onerror=null;this.src=this.dataset.originalSrc" style="width: 518px; height: 150px;"></p>

<p>В процессе доставки колонна одинаковых роботов-клонов перемещается вдоль корпусов общежития слева направо. В базе данных у роботов содержится список сделанных заказов, для каждого из которых известно окно, в которое его нужно доставить. Когда колонна роботов проходит мимо окна, соответствующего какому-то заказу, она может произвести доставку, если в колонне есть робот, расположенный на уровне окна.</p>

<p style="text-align: center;"><img alt="" src="%EB%B0%B1%EC%A4%80/Gold/33860.%E2%80%85%D0%91%D0%B5%D1%81%D0%BF%D0%B8%D0%BB%D0%BE%D1%82%D0%BD%D0%B0%D1%8F%E2%80%85%D0%B0%D1%8D%D1%80%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0/3c2d6272.jpg" data-original-src="https://boja.mercury.kr/assets/problem/33860-2.jpg" onerror="this.onerror=null;this.src=this.dataset.originalSrc" style="width: 527px; height: 150px;"></p>

<p>Во время перемещения конструкция из роботов может натолкнуться на препятствие. После препятствия движение продолжают только те экземпляры роботов, которые находились выше препятствия. Они оказываются на земле непосредственно за препятствием, по прежнему в виде вертикальной колонны, и могут продолжать движение, создавать новых клонов и доставлять заказы.</p>

<p style="text-align: center;"><img alt="" src="%EB%B0%B1%EC%A4%80/Gold/33860.%E2%80%85%D0%91%D0%B5%D1%81%D0%BF%D0%B8%D0%BB%D0%BE%D1%82%D0%BD%D0%B0%D1%8F%E2%80%85%D0%B0%D1%8D%D1%80%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0/8d91ae97.png" data-original-src="https://boja.mercury.kr/assets/problem/33860-3.png" onerror="this.onerror=null;this.src=this.dataset.originalSrc" style="width: 533px; height: 150px;"></p>

<p>Расстояние между препятствиями и окнами достаточно большое, поэтому во время переезда через препятствие роботы не будут проезжать мимо окна.</p>

<p>За доставку одного заказа компания-организатор доставки получает $p$ крипторублей. Стоимость создания одного нового робота равна $c$ крипторублей. Итоговая прибыль равна суммарному доходу от доставки заказов за вычетом суммарной стоимости создания всех роботов. Компания хочет максимизировать свою прибыль. При этом, она не обязана выполнить все заказы, а роботы могут в любой момент остановиться, и прекратить процесс доставки.</p>

<p>Определите максимальную прибыль, которую может получить компания.</p>

### 입력

<p>В первой строке входных данных находятся четыре целых числа $n$, $m$, $c$, $p$ ($0 \le n, m \le 100\,000$, $1 \le c, p \le 10^6$) --- количество препятствий, количество заказов в базе, стоимость создания клона робота и стоимость доставки одного заказа, соответственно.</p>

<p>В следующих $n+m$ строках идёт описание препятствий и окон, в которые нужно доставить заказы, в порядке следования колонны роботов вдоль общежитий слева направо. Каждая строка содержит два целых числа $t_i$ и $h_i$ ($1 \le t_i \le 2$, $1 \le h_i \le 10^6$) --- тип объекта $t_i$ ($1$ для препятствия и $2$ для окна) и $h_i$ --- высота препятствия в этажах или этаж, на котором находится окно.</p>

<p>Гарантируется, что ровно $n$ объектов имеют тип $1$, и оставшиеся $m$ объектов имеют тип $2$.</p>

### 출력

<p>Выведите одно число --- максимальную величину прибыли, которую можно получить.</p>
