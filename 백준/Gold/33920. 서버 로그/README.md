# [Gold III] 서버 로그 - 33920

[문제 링크](https://www.acmicpc.net/problem/33920)

시간 제한: 1 초, 메모리 제한: 1024 MB

### 통계

제출: 100, 정답: 60, 맞힌 사람: 27, 정답 비율: 47.368%

### 분류

누적 합, 수학, 이분 탐색, 정렬

### 문제 설명

<p>데이터 센터에는 $N$개의 서버가 운영 중이다. 각 서버에는 로그가 저장되며, 로그는 하루가 지날 때마다 $1$씩 누적된다. 처음에는 서버마다 $D_1, D_2, \cdots, D_N$ 만큼의 로그가 쌓여 있다.</p>

<p>지속적으로 로그가 쌓이면 저장 공간이 부족해질 수 있기 때문에, 시스템 관리자는 다음 $M$일 동안 정기적으로 로그를 정리하는 프로그램을 만들었다. $i$번째 날에는, 로그가 $C_i$​를 초과한 서버들의 로그를 정리하여, 각 서버의 로그 양을 정확히 $C_i$​로 맞춘다. 프로그램은 그 날의 로그가 누적되기 전의 시점에 로그를 정리한다.</p>

<p>프로그램이 의도대로 작동하는지 확인하기 위해, 하루가 끝날 때마다 정리되어야 하는 로그의 총 용량을 미리 구해서 날마다 비교하려고 한다. 정리 프로그램이 정상적으로 작동했을 때, 다음 $M$일동안 정리할 로그의 용량을 각각 구해보자.</p>

### 입력

<p>첫 번째 줄에 $N$과 $M$이 공백을 사이에 두고 주어진다.</p>

<p>두 번째 줄에 $N$개의 정수 $D_1, D_2, \cdots , D_N$이 공백을 사이에 두고 주어진다.</p>

<p>세 번째 줄에 $M$개의 정수 $C_1, C_2, \cdots , C_M$이 공백을 사이에 두고 주어진다.</p>

### 출력

<p>총 $M$줄에 걸쳐 출력한다. $i$번째 줄에는, $i$번째 날에 정리될 로그의 용량을 출력한다.</p>

### 제한

<ul>
<li>주어지는 모든 수는 정수이다.</li>
<li>$1 \le N, M \le 500\,000$</li>
<li>$1 \le i \le N$ 인 각 $i$ 에 대하여: $1 \le D_i \le 500\,000$</li>
<li>$1 \le j \le M$ 인 각 $j$ 에 대하여: $1 \le C_j \le 500\,000$</li>
</ul>
