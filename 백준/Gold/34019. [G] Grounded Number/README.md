### 문제 설명

<p>양의 정수 $N$이 주어질 때, 아래 연산을 원하는 만큼 반복하여 $N$을 $0$으로 만들 수 있는지 구해보자.</p>

<ul>
	<li>이 연산을 $k$번째로 실행할 때 $N$이 $k$의 배수라면 $N$에 $1$을 더하고 아니라면 $N$에서 $1$을 뺀다.</li>
</ul>

### 입력

<p>첫째 줄에는 양의 정수 $N$이 주어진다. $(1 \le N \le 10^{18})$</p>

### 출력

<p>첫째 줄에 $N$에 연산을 원하는 만큼 반복하여 $0$으로 만들 수 있다면 <span style="color:#e74c3c;"><code>Yes</code></span>를, 아니면 <span style="color:#e74c3c;"><code>No</code></span>를 출력한다.</p>
