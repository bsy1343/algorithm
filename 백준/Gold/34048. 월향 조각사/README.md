### 문제 설명

<p>조각의 신 아이보리는 본인의 능력을 널리 뽐내고자 $1 \times 1$ 대리석 블록들을 이용하여 피라미드 조각상을 만들려고 한다.</p>

<p style="text-align: center;"><img alt="" src="%EB%B0%B1%EC%A4%80/Gold/34048.%E2%80%85%EC%9B%94%ED%96%A5%E2%80%85%EC%A1%B0%EA%B0%81%EC%82%AC/09a251fb.png" data-original-src="https://boja.mercury.kr/assets/problem/34048-1.png" onerror="this.onerror=null;this.src=this.dataset.originalSrc"></p>

<p>크기가 $h (h \geq 1, h$는 정수$)$인 피라미드는 다음 조건을 만족해야 한다.</p>

<ul>
	<li>전체 너비가 $2h - 1$이다.</li>
	<li>피라미드를 이루는 위치는 연속해야 한다.</li>
	<li>피라미드를 이루는 위치의 높이가 순서대로 $1, 2, \cdots, h - 1, h, h - 1, \cdots, 2, 1$이다.</li>
	<li>피라미드를 이루지 않는 위치의 높이는 $0$이다.</li>
</ul>

<p>현재 아이보리의 작업실에는 $i$번 위치에 대리석 블록 $A_i$개가 세로로 쌓여있다. 일류 조각가인 아이보리는 본인이 조각할 수 있는 피라미드의 수가 궁금해졌다.</p>

<p>$f(h)$를 <strong>대리석 블록을 적절히 제거하여 크기 $h$인 피라미드를 조각하는 경우의 수</strong>라고 정의할 때, $\sum{f(h)}$를 구해보자! 단, 피라미드의 크기와 중심의 위치가 모두 같으면 같은 경우로 취급한다.</p>

### 입력

<p>첫 번째 줄에 대리석의 개수 $N$이 주어진다. $(1 \leq N \leq 200\,000)$</p>

<p>두 번째 줄에 $i$번 위치에 쌓여 있는 대리석 블록의 개수 $A_1, A_2, \cdots, A_N$이 공백으로 구분되어 주어진다. $(1 \le A_i \le 10^9)$ </p>

### 출력

<p>첫 번째 줄에 $\sum{f(h)}$를 출력한다.</p>
