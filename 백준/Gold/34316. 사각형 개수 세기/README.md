### 문제 설명

<p>$N \times M$ 격자의 $(i, j)$ 칸에는 $1$ 이상 $9$ 이하의 숫자 $A_{i, j}$가 쓰여있다. 다음을 만족하는 $(x_1, x_2, y_1, y_2)$ 순서쌍의 개수를 세자.</p>

<ul>
<li>$1 \leq x_1 &lt; x_2 \leq N$, $1 \leq y_1 &lt; y_2 \leq M$</li>
<li>$A_{x_1, y_1} + A_{x_1, y_2} + A_{x_2, y_1} + A_{x_2, y_2} = 20$</li>
</ul>

### 입력

<p>첫 줄에는 $N$과 $M$이 공백으로 구분되어 주어진다.</p>

<p>이후 $N$개의 줄에 걸쳐 $A_{i, 1}, A_{i, 2}, \cdots, A_{i, M}$이 공백으로 구분되어 주어진다.</p>

### 출력

<p>위 조건을 만족하는 $(x_1, x_2, y_1, y_2)$의 개수를 출력하라.</p>

### 제한

<ul>
<li>$2 \leq N, M \leq 10^5$</li>
<li>$NM \leq 10^5$</li>
<li>$1 \leq A_{i, j} \leq 9$</li>
</ul>
