### 문제 설명

<p>강가에 $N$개의 가게가 일렬로 늘어져 있다.</p>

<p>이 세계에서 하루는 $1$시부터 $K$시까지 총 $K$시간으로 이루어져 있다. $i$번째 가게는 $S_i$시 1분 전에 문을 열고, $E_i$시 1분 후에 문을 닫는다. 즉, $S_i \le t \le E_i$인 정수 $t$에 대해 $i$번째 가게는 $t$시 정각에 영업하고 있다.</p>

<p>당신은 연속한 몇 개의 가게가 모두 열려 있을 때에 이들을 한 묶음으로 세고 있다. 이를테면, 1, 2, 4, 6, 7, 8번째 가게가 열려 있고 나머지 가게들이 닫혀있다고 하자. 이들은 총 세 묶음으로 이루어져 있는데, 1, 2번째 가게들이 한 묶음, 4번째 가게가 한 묶음, 6, 7, 8번째 가게들이 한 묶음이다.</p>

<p>매 시 정각에, 강가에 연속으로 열려 있는 가게들의 묶음의 수를 계산하는 프로그램을 작성하여라.</p>

### 입력

<p>첫째 줄에 $N$, $K$가 공백을 사이에 두고 주어진다.</p>

<p>둘째 줄에 $N$개의 정수 $S_1, \cdots, S_N$이 공백을 사이에 두고 주어진다.</p>

<p>셋째 줄에 $N$개의 정수 $E_1, \cdots, E_N$이 공백을 사이에 두고 주어진다.</p>

### 출력

<p>첫째 줄에 총 $K$개의 수를 출력하라. $i$번째로 출력하는 수는 $i$시 정각에 강가에 연속으로 열려 있는 가게들의 묶음의 수여야 한다.</p>

### 제한

<ul>
<li>$1 \le N \le 200\,000$</li>
<li>$1 \le K \le 200\,000$</li>
<li>각 $1 \le i \le N$에 대해, $1 \le S[i] \le E[i] \le K$.</li>
</ul>
