<p>플랜 $1,2,3$의 정원은 각각 $P_1,P_2,P_3$명이다. 정원을 초과하여 배정할 수는 없지만 반드시 모두 채울 필요는 없으며, 남는 인원은 정원 제한이 없는 플랜 $0$에 배정된다. 각 부원은 정확히 하나의 플랜에 배정된다.</p>

<p>운영진은 사전 조사를 통해 각 부원이 네 가지 플랜에 대해 느끼는 만족도를 기록하였다. 만족도가 음수면 불만족, $0$이면 중립, 양수면 만족을 뜻한다.</p>

<p>오리엔테이션 당일, 부원들은 행사장에 도착하는 순서대로 줄을 섰다. 운영진은 이 줄을 본 뒤 전체 만족도 합을 최대화하도록 각 부원의 플랜을 배정한다.</p>

<p>또한, 행사 인솔을 위해 줄은 회의에서 정한 플랜의 나열 순서대로 연속된 블록을 이루어야 했다. 배정 인원이 $0$인 플랜은 길이 $0$의 블록으로 간주한다.</p>

<p>이에 운영진은 임의의 두 사람의 자리를 교환하는 방법을 사용하여 줄을 다시 세우려고 한다. 단, 같은 플랜에 속한 부원들의 원래 줄에서의 상대적 순서는 유지되어야 한다.</p>

<p>운영진은 전체 만족도 합이 최대인 배정이 여러 개라면, 그중 필요한 교환 횟수가 최소인 배정을 선택한다. 운영진이 선택한 배정에서 필요한 자리 교환 횟수를 구해보자.</p>

### 입력

<p>첫째 줄에 쉬었음 부원의 인원수 $N$이 주어진다. $(1 \leq N \leq 10)$</p>

<p>둘째 줄에는 현재 행사장에 줄 서 있는 순서대로 각 부원의 번호 $A_i$가 공백으로 구분되어 주어진다. $(1 \le A_i \le N)$</p>

<p>${A_1,A_2,\cdots,A_N}$ 의 값은 <strong>서로 다르다</strong>.</p>

<p>셋째 줄에는 플랜 $1$, $2$, $3$의 정원을 나타내는 $P_1$, $P_2$, $P_3$가 공백으로 구분되어 주어진다. $(0 \leq P_1, P_2, P_3 \leq N)$</p>

<p>넷째 줄부터 $N$개의 줄에 걸쳐, 각 부원이 플랜 $0$, $1$, $2$, $3$에 배정되었을 때의 만족도인 $V_{i0}, V_{i1}, V_{i2}, V_{i3}$가 공백으로 구분되어 주어진다. $(-10 \leq V_{i0}, V_{i1}, V_{i2}, V_{i3} \leq 10)$</p>

<p>마지막 줄에는 회의에서 정해놓은 플랜의 나열 순서 $X_1, X_2, X_3, X_4$가 공백으로 구분되어 주어진다. $(0 \le X_1,X_2,X_3,X_4 \le 3)$</p>

<p>${X_1,X_2,X_3,X_4}$ 의 값은 <strong>서로 다르다.</strong></p>

<p>입력되는 모든 수는 정수이다.</p>

### 출력

<p>운영진이 선택한 배정에서 필요한 자리 교환 횟수를 출력한다.</p>
