### 문제 설명

<p>근수와 승형이는 '근수의 카드게임3'을 즐기고 있다. 근수의 목표는 최종 점수 $S$를 최대화하는 것이고, 승형이의 목표는 $S$를 최소화하는 것이다. 초기값은 $S = 0$이다. 게임은 총 $N$턴 동안 진행되고 각 턴에는 다음 과정이 이루어진다.</p>

<ol>
<li>세 장의 카드가 주어진다. 각 턴마다 $1$, $2$, $3$이 적힌 카드가 각각 한 장씩 주어진다.</li>
<li>승형이가 세 장의 카드 중 한 장을 제거한다.</li>
<li>근수가 남은 두 장의 카드 중 한 장을 골라 그 카드의 값을 $S$에 더한다.</li>
</ol>

<p>모든 턴이 끝난 뒤 최종 점수 $S$가 $K$를 초과하면 최종 점수 $S$는 $S = -1$로 처리된다. 모든 정보는 모두에게 공개되어 있으며, 두 사람은 항상 최선의 전략으로 행동한다. 이때의 최종 점수 $S$를 구하여라.</p>

### 입력

<p>첫째 줄에 진행할 게임의 턴 수 $N$과 $K$가 주어진다. ($1 \le N \le 100\,000$; $1 \le K \le 3N$) </p>

### 출력

<p>근수와 승형이가 항상 최선의 전략으로 게임을 진행하였을 때, 최종 목표치 값을 구하여라.</p>
