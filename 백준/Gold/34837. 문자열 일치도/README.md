### 문제 설명

<p>길이가 같은 두 문자열의 <strong>일치도</strong>를 두 문자열이 같은 문자를 가지는 위치의 수라고 하자.</p>

<p>영어 알파벳 소문자로만 이루어진 두 문자열 $S$와 $T$가 주어진다. $S$와 $T$의 길이는 각각 $N$과 $M$이다. $N$은 $M$보다 크거나 같다.</p>

<p>$S$의 모든 길이가 $M$인 부분 문자열 $N-M+1$개에 대해 $T$와의 일치도의 합을 구하여라.</p>

<p>부분 문자열의 정의는 노트를 참고하라.</p>

### 입력

<p>첫째 줄에 문자열 $S$의 길이 $N$과 문자열 $T$의 길이 $M$이 공백으로 구분되어 주어진다. ($1 \leq M \leq N \leq 200\,000$)</p>

<p>둘째 줄에 영어 알파벳 소문자로만 이루어진 길이 $N$의 문자열 $S$가 주어진다. </p>

<p>셋째 줄에 영어 알파벳 소문자로만 이루어진 길이 $M$의 문자열 $T$가 주어진다. </p>

### 출력

<p>첫째 줄에 $S$의 모든 길이가 $M$인 부분 문자열 $N-M+1$개에 대해 $T$와의 일치도의 합을 출력한다.</p>

### 힌트

<p>부분 문자열이란, 문자열에서 연속한 부분에 해당하는 문자열이다. 예를 들어 <code>ab</code>, <code>bca</code>, <code>abca</code>는 <code>abca</code>의 부분 문자열이고, <code>ac</code>, <code>ba</code>는 <code>abca</code>의 부분 문자열이 아니다.</p>
