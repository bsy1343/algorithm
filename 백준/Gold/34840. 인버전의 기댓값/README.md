# [Gold V] 인버전의 기댓값 - 34840

[문제 링크](https://www.acmicpc.net/problem/34840)

### 성능 요약

### 통계

제출: 191, 정답: 155, 맞힌 사람: 124, 정답 비율: 79.487%

### 분류

기댓값의 선형성, 수학, 조합론, 확률론

### 문제 설명

<p>길이 $N$의 어떤 순열 $p$에 대해, 인버전의 개수란 $1 \leq i &lt; j \leq N$ 이고 $p_i &gt; p_j$인 순서쌍 $(i,j)$의 개수와 같다.</p>

<p>길이 $N$의 가능한 모든 $N!$개의 순열 중 균등한 확률로 하나를 뽑았을 때, 해당 순열의 인버전의 개수의 기댓값을 구하여라.</p>

### 입력

<p>첫째 줄에 순열의 길이 $N$이 주어진다. ($1 \leq N \leq 100$)</p>

### 출력

<p>첫째 줄에 인버전의 개수의 기댓값을 출력한다. 실제 정답과 출력값의 절대 오차 혹은 상대 오차가 $10^{-9}$ 이하라면 정답으로 인정한다.</p>

### 힌트

<p>길이 $N$의 순열은 $1$부터 $N$까지의 수가 정확히 한 번 등장하는 수열을 말한다.</p>

<p>예를 들어 $[3,5,1,2,4]$와 $[1,3,2]$는 순열이지만, $[2,3,2]$와 $[0]$은 순열이 아니다.</p>
