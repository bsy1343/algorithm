# [Gold IV] 자벌레 세기 - 34875

[문제 링크](https://www.acmicpc.net/problem/34875)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 2048 MB

### 통계

제출: 168, 정답: 101, 맞힌 사람: 80, 정답 비율: 60.150%

### 분류

수학, 조합론, 트리

### 문제 설명

<p>$N$개의 정점으로 구성된 트리가 있습니다. 트리는 간선에 방향이 없는 연결된 그래프로, 각 정점에서 다른 정점으로 가는 최단 경로가 유일합니다.</p>

<p>어떤 $6$개의 <strong>서로 다른</strong> 정점 $(u,v,w,x,y,z)$가 다음 조건을 모두 만족할 경우 그러한 $6$개의 정점으로 구성된 그래프를 <strong>자벌레</strong>라고 합니다.</p>

<ul>
<li>$u$와 $v$가 간선으로 직접 연결되어 있습니다.</li>
<li>$w,x,y,z$ 중 $2$개는 $u$와, 나머지 $2$개는 $v$와 간선으로 직접 연결되어 있습니다.</li>
</ul>

<p>이때, 두 자벌레 $A$와 $B$에 대해, $A$에 포함되어 있으면서 $B$에 포함되지 않은 정점이 존재한다면 $A$와 $B$는 <strong>서로 다른</strong> 자벌레로 취급합니다.</p>

<p>예를 들어, 다음 그림에 주어진 트리에 대해, 빨간색으로 색칠된 간선으로 연결된 $6$개의 정점이 자벌레의 예시입니다.</p>

<p style="text-align: center;"><img alt="" src="%EB%B0%B1%EC%A4%80/Gold/34875.%E2%80%85%EC%9E%90%EB%B2%8C%EB%A0%88%E2%80%85%EC%84%B8%EA%B8%B0/cc1f0b3d.png" data-original-src="https://boja.mercury.kr/assets/problem/34875-1.png" onerror="this.onerror=null;this.src=this.dataset.originalSrc" style="width: 512px; height: 256px;"></p>

<p>위의 트리에는 서로 다른 자벌레가 총 $6$개 존재합니다.</p>

<p>입력으로 주어진 트리에 대해서, 트리에 존재하는 서로 다른 자벌레의 수를 구하는 프로그램을 작성해 주세요.</p>

### 입력

<p>첫 번째 줄에 트리를 구성하는 정점의 개수 $N$이 주어집니다.</p>

<p>두 번째 줄부터 $N-1$개의 줄에 걸쳐 각각의 간선이 연결하는 두 정점의 번호 $u_i$, $v_i$가 공백으로 구분되어 주어집니다.</p>

### 출력

<p>입력으로 주어진 트리에 존재하는 서로 다른 자벌레의 수를 한 줄에 출력합니다.</p>

### 제한

<ul>
<li>$2 \le N \le 300\,000$</li>
<li>$1 \le u_i,v_i \le N$</li>
<li>$u_i \neq v_i$</li>
<li>주어진 그래프는 올바른 트리임이 보장됩니다.</li>
<li><strong>주어진 트리에 존재하는 서로 다른 자벌레의 수는 $10^{18}$을 초과하지 않음이 보장됩니다.</strong></li>
</ul>
