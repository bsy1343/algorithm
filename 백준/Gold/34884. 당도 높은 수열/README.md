### 문제 설명

<p><em>이 문제는 Output only 문제입니다.</em></p>

<p>길이 $N$의 순열의 당도는 다음과 같다:</p>

<ul>
<li>$\sum\limits_{1\le l\le r\le N} \max\{A_l, A_{l+1}, \dots, A_r\}$</li>
</ul>

<p>도훈이는 여자친구와 15일 기념으로 당도가 높은 순열을 만들어 여자친구에게 선물하려고 한다.</p>

<p>길이 $150\,000$의 순열 중 당도가 $1.5 \times 10^{15}$ 이상인 순열을 만들어주자.</p>

### 입력

<p>입력은 없다.</p>

### 출력

<p>첫째 줄에 당도가 $1.5 \times 10^{15}$ 이상인 순열 $P_1, P_2, \ldots, P_{150\,000}$을 공백으로 구분하여 출력한다.</p>

<p>만약 그러한 답이 여러 가지라면 아무 답이나 출력한다.</p>

### 힌트

<p>길이 $N$의 순열은 $1$부터 $N$까지의 수가 정확히 한 번 등장하는 수열을 말한다.</p>

<p>예를 들어, $[3,5,1,2,4]$와 $[1,3,2]$는 순열이지만, $[2,3,2]$ 또는 $[0]$은 순열이 아니다.</p>
