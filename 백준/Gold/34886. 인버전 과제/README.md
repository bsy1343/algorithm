### 문제 설명

<p><em>이 문제는 인터랙티브 문제입니다.</em></p>

<p>길이 $N$의 숨겨진 수열 $A_1, A_2, \ldots A_N$이 있다. $A_i$는 $0$ 또는 $1$이다. ($1 \le i \le N$)</p>

<p>수열 $A$의 인버전을 $(1 \leq i &lt; j \leq N)$를 만족하는 모든 정수 순서쌍 $(i,j)$에 대해 $A_i &gt; A_j$를 만족하는 순서쌍의 개수로 정의한다.</p>

<p>건우는 전공 과목 수업 과제로 수열 $A$의 인버전을 $0$으로 만들어 제출해야 한다. 이미 과제 제출 기간이 3일 지난 건우는 아직도 귀찮기 때문에 당신이 건우를 위해 과제를 대신 해 주자.</p>

<p>당신은 $A$를 $A_l, A_{l+1}, \ldots, A_r$의 각 원소를 $0$이라면 $1$로, $1$이라면 $0$으로 바꾸는 연산을 최대 $N$회 할 수 있다.</p>

<p>건우가 F를 받지 않도록 과제를 대신 해 도와주자.</p>

### 입력

<p>첫째 줄에 수열의 길이 $N$과 초기 수열 $A$의 인버전 $I$가 공백으로 구분되어 주어진다. ($2 \leq N \leq 100;I \neq 0$)</p>
