### 문제 설명

<p>민재와 재하는 수 맞히기 게임의 참가자이고, 유담이는 이 게임의 진행자이다.</p>

<p>유담이는 게임 시작 전 두 양의 정수 $N$, $M$을 정한다. 이때 두 수의 차 $|N-M|$을 $K$라 하자.</p>

<p>수 맞히기 게임의 진행 방식은 다음과 같다.</p>

<ol>
<li>유담이는 민재에게 $N$과 $K$를 알려주고 재하에게 $M$과 $K$를 알려준다.</li>
<li>민재는 $N$과 $K$의 값을 알게 된 후 재하에게 정수 $C$를 종이에 적어 전달한다. 이때 $C$는 $0$ 또는 $1$이어야 한다.</li>
<li>재하는 자신이 알고 있는 $M$과 $K$, 그리고 민재에게 전달받은 $C$를 이용하여 $N$의 값을 추측한다.</li>
</ol>

<p>게임 시작 전 민재와 재하는 전략을 논의할 수 있다. 그러나 게임이 시작된 후에 두 사람은 민재가 정수 $C$를 종이에 적어 재하에게 전달하는 것 이외에는 어떤 방법으로도 소통할 수 없다.</p>

<p>재하가 추측한 값이 실제 $N$과 같다면 민재와 재하는 게임에서 승리한다.</p>

<p>두 사람을 위해 게임에서 승리할 수 있는 전략을 만들어 주자.</p>
