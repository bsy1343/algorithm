### 문제 설명

<p>Pahom walks around as large an area as he wants, starting at daybreak and ending at sunset, using his spade to make markers as he goes. The polygon marked out by his markers as vertices represents the land Pahom has claimed.</p>

<p>Pahom walks at a speed of one metre per second, each marker cost $m$ seconds to make, the total available time between daybreak and sunset is $t$ seconds. Find the maximum area Pahom could claim.</p>

<p>Pahom must return to the starting point by the end.</p>

### 입력

<p>The first line of input contains the number of cases, $c$ ($1 \le c \le 1000$).</p>

<p>The next $c$ line of input each consist of two space-separated integers $m$ and $t$ ($1 \le m, t \le 10^8$) --- the time required to make one marker, and the total time available to Pahom.</p>

### 출력

<p>For each case, output a real number representing the maximum area.</p>

<p>Your answers should have an absolute or relative error of at most $10^{-6}$.</p>
