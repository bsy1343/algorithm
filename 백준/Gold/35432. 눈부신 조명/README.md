### 문제 설명

<p>일자로 된 지하철 선로에 $N$개의 조명이 설치되어 있다. 이 조명들은 모두 솔빛공업에서 만들었다.</p>

<p>솔빛공업의 조명은 고유한 <strong>등급</strong>과 <b>확산도</b>를 가진다. $i$번째 조명은 선로 위의 위치 $p_i$에 설치되어 있고, 등급은 $l_i$, 확산도는 $d_i$이다. $p_i, l_i, d_i$는 모두 정수이며, <strong>모든 $l_i$는 서로 다르다.</strong></p>

<p>조명이 설치되기 전 선로 위 모든 점의 초기 밝기는 $0$이고, 여기에 설치된 조명이 주변의 정수 지점을 비춰 밝기를 올린다. 선로 위의 정수 위치 $x$에서 $$L_{i,x}= l_i - \left \lfloor \frac{|p_i-x|}{d_i} \right \rfloor \geq 0$$을 만족하면, 위치 $x$의 밝기는 $i$번째 조명에 의해 $A^{L_{i,x}}$만큼 증가한다. 여기서 $A$는 솔빛공업의 모든 조명이 공유하는 밝기 상수로, $\boldsymbol{A \geq N}$<strong>이다.</strong></p>

<p>명준이는 지하철을 타면 얼마나 눈부실지 궁금해졌다. 명준이를 위해 지하철 선로에서 최대 밝기와 최대 밝기가 나타나는 위치를 구해주자.</p>

### 입력

<p>첫 번째 줄에 설치된 조명의 개수 $N$과 밝기 상수 $A$가 공백으로 구분되어 주어진다.</p>

<p>두 번째 줄부터 $N+1$번째 줄까지 $i + 1$번째 줄에 각 조명을 표현하는 세 정수 $p_i, l_i, d_i$가 공백으로 구분되어 주어진다.</p>

### 출력

<p>첫 번째 줄에 최대 밝기를 $10^9+7$로 나눈 나머지를 출력하라.</p>

<p>두 번째 줄에 최대 밝기를 가지는 정수 위치를 출력하라. 최대 밝기를 가지는 정수 위치가 여러 개 존재한다면, 그중 아무거나 출력하라.</p>

### 제한

<ul>
<li>$1 \le N \le 10^4$</li>
<li>$N \le A \le 10^9$</li>
<li>$-10^9 \le p_i \le 10^9$</li>
<li>$0 \le l_i \le 10^9$ $(1 \le i \le N)$</li>
<li>$l_i \neq l_j$ $(1 \le i &lt; j \le N)$</li>
<li><span style="color:#e74c3c;">$1 \le d_i \le 100$ $(1 \le i \le N)$</span></li>
</ul>
