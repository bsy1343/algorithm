### 문제 설명

<p>길이가 $N$인 수열 $A_1, A_2, \cdots, A_N$이 주어진다. 다음 쿼리를 수행하는 프로그램을 작성하자.</p>

<ul>
<li><code>1 i j v</code>: $A_i, A_{i+1}, \cdots, A_j$의 구간에 $v$씩 더한다.</li>
<li><code>2</code>: 수열이 단조증가수열이면 <code><span style="color:#e74c3c;">YES</span></code>를, 아니면 <span style="color:#e74c3c;"><code>NO</code></span>를 출력한다.</li>
</ul>

<p>단조증가수열이란, $1 \le i \le N-1$를 만족하는 모든 $i$에 대해 $A_i \le A_{i+1}$인 수열을 의미한다.</p>

### 입력

<p>첫 번째 줄에 수열의 길이 $N$과 쿼리의 개수 $Q$가 공백으로 구분되어 주어진다.</p>

<p>두 번째 줄에 수열 $A_1, A_2, \cdots, A_N$이 공백으로 구분되어 주어진다.</p>

<p>그다음 줄부터 $Q$개의 줄에 걸쳐 쿼리가 한 줄에 하나씩 주어진다.</p>

### 출력

<p>$2$번 쿼리에 대해서 정답을 한 줄에 하나씩 순서대로 출력한다.</p>

### 제한

<ul>
<li>입력으로 주어지는 모든 수는 정수이다.</li>
<li>$1 \le N,Q \le 500\,000$</li>
<li>$-10^9 \le A_i \le 10^9$</li>
<li>$-10^9 \le v \le 10^9$</li>
<li>$1$번 쿼리에 대해 $1 \leq i \leq j \leq N$</li>
<li>$2$번 쿼리는 하나 이상 주어진다.</li>
</ul>
