### 문제 설명

<p>R은 변의 길이가 정수인 직사각형이다. 직사각형은 단위 정사각형으로 나눌 수 있다.</p>

<p>f(R)은 직사각형의 한 대각선이 지나는 정사각형의 개수라고 정의한다. 변의 길이가 2와 4인 직사각형 R의 f(R) = 4이다.</p>

<p>N이 주어졌을 때, f(R) = N을 만족하는 직사각형의 개수를 구하는 프로그램을 작성하시오. 변의 길이가 a&times;b인 직사각형과 b&times;a인 직사각형은 같은 직사각형이다.&nbsp;</p>

### 입력

<p>첫째 줄에 자연수 N (0 &lt; N &lt; 10<sup>6</sup>)이 주어진다.&nbsp;</p>

### 출력

<p>첫째 줄에 f(R) = N인 R의 개수를 출력한다.</p>
