### 문제 설명

<p>
	은행수란 정수 m과 n의 쌍 (m,n)이다. 예를 들어, (1,1), (-2,1), (-3,-1)은 은행수이다.</p>

<p>
	은행수의 곱셈 (m,n) &middot; (x,y) = (mx - ny, my + nx) 이다. 예를 들어, (1,1)&middot;(-2,1) = (-3,-1)이다.</p>

<p>
	이때, (m,n) &middot; (x,y) = (p,q)를 만족하는 은행수 (x,y)가 존재한다면, (m,n)은 은행수 (p,q)의 약수라고 한다.</p>

<p>
	모든 (1,0), (0,1), (-1,0), (0,-1), (m,n), (-n,m), (-m,-n), (n,-m)은 임의의 은행수 (m,n)의 약수이다. 만약, m<sup>2</sup> + n<sup>2</sup> &gt; 1이라면, 앞의 8개 수는 모두 다를 것이다. 즉, m<sup>2</sup> + n<sup>2</sup> &gt; 1을 만족하는 은행수는 적어도 8개의 약수가 있다.</p>

<p>
	만약, m<sup>2</sup> + n<sup>2</sup> &gt; 1을 만족하는 은행수 (m,n)의 약수가 8개라면, 이 수를 소수라고 한다.</p>

<p>
	은행수가 주어졌을 때, 소수인지 아닌지 알아내는 프로그램을 작성하시오.</p>

### 입력

<p>
	첫째 줄에 테스트 케이스의 개수가 주어진다. 각 테스트 케이스는 한 줄로 이루어져 있으며, 은행수 (m,n)의 m과 n이 공백으로 구분되어 있다. (1 &lt; m<sup>2</sup> + n<sup>2</sup> &lt; 20000)</p>

### 출력

<p>
	각 테스트 케이스에 대해서, 입력으로 주어진 은행수가 소수라면 P를, 아니라면 C를 출력한다.</p>

### 힌트

<p>
	m<sup>2</sup> + n<sup>2</sup> &gt; 0인 경우에 m<sup>2</sup> + n<sup>2</sup>이 mp + nq와 mq - np의 공약수라면, (m,n)은 (p,q)의 약수이고, 그 역도 성립한다.</p>
<p>
	만약, (m,n) &middot; (x,y) = (p,q)라면, (m<sup>2</sup> + n<sup>2</sup>)(x<sup>2</sup> + y<sup>2</sup>) = p<sup>2</sup> + q<sup>2</sup>이다.</p>
