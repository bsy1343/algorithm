### 문제 설명

<p>학생회장을 하고 있는 상근이는 이번 학교 축제 행사로 학우들간의 친밀감을 돈독히 하고자 줄다리기를 하려 한다.</p>

<p>하지만 이 줄다리기에 형평성을 최대한 고려하기위해 두 팀간의 사람 수 차이를 1&nbsp;이하로 하려하며, 두 팀간의 몸무게의 차이가 최소화되도록 하고자 한다.</p>

<p>이때 상근이가 나누려 하는 두 팀의 몸무게를 각각 출력 하시오.</p>

### 입력

<p>가장 첫 번째 줄에 줄다리기를 하기 원하는 총 인원의 수(1&nbsp;&le;&nbsp;N&nbsp;&le;&nbsp;100)가 주어진다.</p>

<p>이후 N개의 줄에 줄다라기에 참여하기 원하는 사람의 몸무게(1&nbsp;&le; K&nbsp;&le; 450)가 주어진다.</p>

### 출력

<p>두팀의 몸무게를 작은 순서대로 순차적으로 출력한다.</p>
