### 문제 설명

<p>The factorial function, n! is defined thus for n a non-negative integer:</p>

<pre>
0! = 1
n! = n * (n-1)! &nbsp; (n &gt; 0)</pre>

<p>We say that a divides b if there exists an integer k such that</p>

<pre>
k*a = b</pre>

### 입력

<p>The input to your program consists of several lines, each containing two non-negative integers, n and m, both less than 2^31.</p>
