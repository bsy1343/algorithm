### 문제 설명

<p>길이 n+1의 비감소 수열 s<sub>1</sub>,...,s<sub>n+1</sub> (s<sub>i</sub> &le; s<sub>i+1</sub> for 1 &le; i &le; n) 을 생각해보자.</p>

<p>이러한 수열의 &quot;평균값 수열&quot;을 다음과 같이 정의한다 : 1 &le; i &le; n에서 m<sub>i</sub> = (s<sub>i</sub> + s<sub>i+1</sub>)/2.</p>

<p>예를 들어, S = {1,2,2,4} 일 때 M = {3/2,2,3}이다. 이렇듯 평균값 수열은 정수가 아닌 수가 나올 수도 있지만, 이 문제에서는 평균값 수열의 원소가 정수인 경우만 고려하기로 한다.</p>

<p>길이 n의 감소하지 않는 평균값 수열 m<sub>1</sub>,...,m<sub>n</sub>이 주어질때, 해당 수열을 평균값 수열로 가지는 수열 s<sub>1</sub>,...,s<sub>n+1</sub>의 개수를 세어라.</p>

### 입력

<p>첫 번째 줄에 평균값 수열의 길이 n이 주어진다. (2 &le; n &le; 5,000,000)</p>

<p>이후 n개의 줄에 평균값 수열 m<sub>i</sub>가 순서대로 주어진다. (0 &le; m<sub>i</sub> &le; 1,000,000,000)</p>

### 출력

<p>주어진 수열을 평균값 수열로 가지는 수열의 개수를 출력하라.</p>

### 힌트

<p>다음과 같은 4가지 경우만이 존재한다 : {2,2,8,10} / {1,3,7,11} / {0,4,6,12} / {-1,5,5,13}</p>
