### 문제 설명

<p>팰린드롬이란 대칭 문자열이다.&nbsp;즉, 왼쪽에서 오른쪽으로 읽었을때와 오른쪽에서 왼쪽으로 읽었을때 같다는 얘기다. 당신은 문자열이 주어졌을때, 최소 개수의 문자를&nbsp;삽입하여 팰린드롬이 되게 되는 문자의 개수를 구하는 프로그램을 작성하여라.</p>

<p>예제에서는, 2개의 문자를 삽입하여 팰린드롬이 된다. &quot;Ab3bd&quot;는 &quot;dAb3bAd&quot; 혹은 &quot;Adb3bdA&quot; 로 바뀔 수 있다. 하지만, 2개 미만의 문자를 삽입해서는 팰린드롬이 될 수 없다.</p>

### 입력

<p>첫 번째 줄에는 문자열의 길이 N (3 &le; N &le; 5000)이 주어진다. 두 번째 줄에는 길이가 N인 문자열이 주어진다. 문자열은 대문자 &#39;A&#39;-&#39;Z&#39;와 소문자 &#39;a&#39;-&#39;z&#39;, 숫자 &#39;0&#39;-&#39;9&#39;로 이루어진다. 대문자와 소문자는 구분되어야 한다.</p>

### 출력

<p>첫 번째 줄에 삽입해야할 최소 개수를 출력한다.</p>
