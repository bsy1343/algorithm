### 문제 설명

<p>Your task in this problem is to determine the number of divisors of $n \choose k$. Just for fun -- or do you need any special reason for such a useful computation?</p>

### 입력

<p>The input consists of several instances. Each instance consists of a single line containing two integers&nbsp;<em>n</em>&nbsp;and&nbsp;<em>k</em>&nbsp;(0 &le;&nbsp;<em>k</em>&nbsp;&le;&nbsp;<em>n</em>&nbsp;&le;&nbsp;431), separated by a single space.</p>

### 출력

<p>For each instance, output a line containing exactly one integer -- the number of distinct divisors of $n \choose k$. For the input instances, this number does not exceed 2<sup>63</sup>&nbsp;- 1.</p>
