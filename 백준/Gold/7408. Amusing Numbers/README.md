### 문제 설명

<p>Let us consider the set of integer numbers between 1 and N inclusive. Let us order them lexicographically (i. e. like in the vocabulary), for example, for N&nbsp;=&nbsp;11 the order would be: 1, 10, 11, 2, 3, 4, 5, 6, 7, 8, 9.</p>

<p>Let us denote the position of the number K in this ordering as Q<sub>N,K</sub>. For example, Q<sub>11,2</sub>&nbsp;=&nbsp;4. Given numbers K and M find the smallest N such that Q<sub>N,K</sub>&nbsp;=&nbsp;M.</p>

### 입력

<p>Input contains two integer numbers K and M (1&nbsp;&le;&nbsp;K,&nbsp;M&nbsp;&le;&nbsp;10<sup>9</sup>) separated by a space.</p>

### 출력

<p>If such N that Q<sub>N,K</sub>&nbsp;=&nbsp;M exists then write to the output file the smallest such N, otherwise write 0.</p>
