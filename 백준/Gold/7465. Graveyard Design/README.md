### 문제 설명

<p>King George has recently decided that he would like to have a new design for the royal graveyard. The graveyard must consist of several sections, each of which must be a square of graves. All sections must have different number of graves.</p>

<p>After a consultation with his astrologer, King George decided that the lengths of section sides must be a sequence of successive positive integer numbers. A section with side length s contains s<sup>2</sup> graves.</p>

<p>George has estimated the total number of graves that will be located on the graveyard and now wants to know all possible graveyard designs satisfying the condition. You were asked to find them.</p>

### 입력

<p>Input file contains n &mdash; the number of graves to be located in the graveyard (1 &le; n &le; 10<sup>14</sup>).</p>

### 출력

<p>On the first line of the output file print k &mdash; the number of possible graveyard designs. Next k lines must contain the descriptions of the graveyards. Each line must start with l &mdash; the number of sections in the corresponding graveyard, followed by l integers &mdash; the lengths of section sides (successive positive integer numbers).</p>
