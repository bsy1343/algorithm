### 문제 설명

<p>September 10th&nbsp;is forester day. On this occasion Magic forest dwellers decided to make a party. Kuzma beaver is assigned to take care about intellectual games. Kuzma has invented at his opinion quite interesting game.</p>

<p>Rules of the game are simple. Master announces positive integer&nbsp;<strong>N</strong>. Player needs to calculate product of digits for each positive integer up to N and report the greatest product.</p>

<p>All answers should be known beforehand to allow interactive communication during a game. This causes problems since&nbsp;<strong>N</strong>&nbsp;could be relatively big (<strong>1</strong>&nbsp;&le;&nbsp;<strong>N</strong>&nbsp;&le;&nbsp;<strong>2000000000</strong>). Kuzma is not good using computers; he asks you to write a program which having positive integer&nbsp;<strong>N</strong>&nbsp;will find correct answer.</p>

### 입력

<p>Each line at input contains integer&nbsp;<strong>N</strong>.</p>

### 출력

<p>Corresponding line at output should contain greatest product for this&nbsp;<strong>N</strong>.</p>
