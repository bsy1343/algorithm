### 문제 설명

<p>숫자 m개가 주어지며, 각 숫자를 n<sub>i</sub>라고 한다. (1 &le; i &le; m)</p>

<p>이때, 모든 i에 대해서, 연속하는 소수 n<sub>i</sub>개의 합으로 나타낼 수 있는 가장 작은 소수를 찾는 프로그램을 작성하시오.</p>

<p>예를 들어, m = 2, n<sub>1</sub> = 3, n<sub>2</sub> = 5인 경우에 정답은 83이다. (83 = 23 + 29 + 31 = 11 + 13 + 17 + 19 + 23)</p>

### 입력

<p>첫째 줄에 테스트 케이스의 개수가 주어진다. 각 테스트 케이스의 첫째 줄에는 1 &le; m &le; 10 이 주어진다. 둘째 줄에는 n<sub>i</sub>가 주어진다. (1 &le; n<sub>i</sub> &le; 10<sup>4</sup>)</p>

<p>항상 정답은 10<sup>7</sup>보다 작다.</p>

### 출력

<p>각 테스트 케이스마다 &quot;Scenario i:&quot;를 출력하고, 둘째 줄에 정답을 출력한다.</p>
