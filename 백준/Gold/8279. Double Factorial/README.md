### 문제 설명

<p>For a positive integer <em>n</em>, its&nbsp;<i>factorial</i>&nbsp;is defined as the product of all integers from 1&nbsp;to&nbsp;<em>n</em>, denoted as <em>n</em>!. Now&nbsp;<em>n</em>&nbsp;<i>double factorial</i>&nbsp;is the product of 1&nbsp;factorial, 2&nbsp;factorial, ..., up to&nbsp;<em>n</em>&nbsp;factorial: 1! &middot; 2! &middot; 3! &middot; ... &middot; <em>n</em>!. Given <em>n</em>, find the number of trailing zeros of a decimal representation of&nbsp;<em>n</em>&nbsp;double factorial.</p>

### 입력

<p>The first and only line of the standard input contains an integer <em>n</em>&nbsp;(1 &le; <em>n</em> &le; 10<sup>18</sup>).</p>

### 출력

<p>The first and only line of the standard output should contain the number of trailing zeros of <em>n</em>&nbsp;double factorial.</p>

### 힌트

<p>11 double factorial equals 265 790 267 296 391 946 810 949 632 000 000 000. This number has 9&nbsp;trailing zeros.</p>
