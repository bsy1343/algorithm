### 문제 설명

<p>Byteasar admires chess puzzles. He is a long-time subscriber of the Chess Player&#39;s Magazine. The most recent issue of this magazine contains the following puzzle:</p>

<p>A chessboard of size n&times;n is given. In how many ways one can place n rooks on the chessboard, so that no two rooks attack each other and the i-th rook is located neither in the i-th column nor in the i-th row? The rooks, the rows and the columns are numbered from 1 to n. The result should be given modulo m.</p>

<p>Puzzles of the type &quot;mate in 13 moves&quot; are a piece of cake for Byteasar, but this new type of puzzle seems very hard for him. Could you please help him?</p>

### 입력

<p>The only line of input contains two integers n and m (1 &le; n &le; 1018, 1 &le; m &le; 10<sup>6</sup>).</p>

### 출력

<p>Your program should output the number of possible placements of rooks.</p>
