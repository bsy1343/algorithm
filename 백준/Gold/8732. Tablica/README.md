### 문제 설명

<p>Kacper i Adi bardzo polubili system dw&oacute;jkowy. Każdy z nich napisał na tablicy ciąg zer i jedynek. Kacper chciałby teraz w każdym z tych ciąg&oacute;w skreślić niekt&oacute;re cyfry tak, żeby pozostałe ciągi były takie same oraz były uporządkowane, tj. po pierwszym wystąpieniu jedynki nie może już wystąpić żadne zero. Jaki najdłuższy ciąg może pozostać na tablicy?</p>

### 입력

<p>Pierwszy wiersz wejścia zawiera dwie liczby całkowite <em>n</em>, <em>m</em>&nbsp;(1 &le; <em>n</em>, <em>m</em> &le; 10<sup>6</sup>), oznaczające długość ciąg&oacute;w napisanych przez kolejno Kacpra i Adiego. W drugim wierszu znajduje się&nbsp;<em>n</em>&nbsp;cyfr 0 lub 1 - liczba napisana przez Kacpra. W trzecim wierszu znajduje się <em>m</em>&nbsp;cyfr 0 lub 1 - liczba na wyświetlaczu Adiego.</p>

### 출력

<p>Pierwszy i jedyny wiersz wyjścia powinien zawierać jedną liczbę - długość najdłuższego ciągu kt&oacute;ry może pozostać na tablicy.</p>
