### 문제 설명

<p>두 자연수 A, B가 주어졌을 때, A&nbsp;&le; x&nbsp;&le; B를 만족하는 모든 x에 대해 x를&nbsp;이진수로 표현했을 때 1의 개수의 합을 구하는 프로그램을 작성하시오.</p>

<p>즉, f(x) = x를 이진수로 표현 했을 때 1의 개수라고 정의하고, 아래 식의 결과를 구하자.</p>

<p>\[\sum_{x=A}^{B}{f(x)}\]</p>

### 입력

<p>첫 줄에 두 자연수 A, B가 주어진다. (1&nbsp;&le; A&nbsp;&le; B&nbsp;&le; 10<sup>16</sup>)</p>

### 출력

<p>1의 개수를 세어 출력한다.</p>
