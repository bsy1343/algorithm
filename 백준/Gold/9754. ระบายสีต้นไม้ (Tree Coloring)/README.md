### 문제 설명

<p>ตน้ ไมค้ือกราฟแบบไม่มีทิศทางที่เชื่อมต่อกนั (connected) และไม่มีวงรอบ (cycle) ให้ต้นไม้ที่มีN โหนด และเส้นเชื่อม N-1 เส้น เราต้องการจะ ระบายสีต้นไม้กล่าวคือเราตอ้งการกา หนดสีให้กบัโหนดต่าง ๆ โดยสีเป็ นจ านวนเต็มจากเซต {1,2,...,K} โดยที่รับประกนัวา่ โหนดที่มีเส้นเชื่อมติดกนั จะมีสีไม่ซ้า กน</p>

<p style="text-align: center;">ให้คุณเขียนโปรแกรมเพื่อคา นวณว่าสามารถทา ไดก้ี่แบบ เนื่องจากค าตอบสามารถมีได้จ านวนมาก ให้ตอบจ านวนวิธีmodulo 93563</p>

### 입력

<p>บรรทัดแรก ระบุจ านวนเต็ม T (1 &le; T &le; 10) แทนจ านวนข้อมูลชุดทดสอบ จากน้นัจะมีขอ้มูลชุดทดสอบ T ชุด ในรูปแบบดงัน้ี</p>

<ul>
	<li>บรรทัดแรกของข้อมูลชุดทดสอบระบุจ านวนเต็มสองจ านวน N และ K โดยที่ N แทนจ านวนโหนด (2 &le; N &le; 200) และ K แทน จ านวนสีที่สามารถใช้ได้(1 &le; K &le; 10) โหนดจะมีหมายเลขต้งัแต่ 1 ถึง N</li>
	<li>อีก N-1 บรรทัด ระบุข้อมูลของเส้นเชื่อมในต้นไม้แต่ละบรรทดัระบุจา นวนเตม็ สองจา นวน A และ B (1 &le; A &le; N; 1 &le; B &le; N; A &ne; B) ระบุวา่ มีเส้นเชื่อมระหวา่ งโหนดหมายเลข A และ B</li>
</ul>

### 출력

<p>มีท้งัสิ้น T บรรทัด สา หรับขอ้มูลนา เขา้แต่ละชุดให้พิมพจ์า นวนวิธีการระบายสีmodulo 93563 เรียงตามล าดับ บรรทดัและหน่ึงค่า</p>
