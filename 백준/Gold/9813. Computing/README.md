# [Gold I] Computing - 9813

[문제 링크](https://www.acmicpc.net/problem/9813)

### 성능 요약

시간 제한: 1 초, 메모리 제한: 128 MB

### 통계

제출: 574, 정답: 40, 맞힌 사람: 30, 정답 비율: 12.000%

### 분류

수학, 브루트포스 알고리즘, 사칙연산

### 문제 설명

<p>Input any five positive integral numbers n1, n2, n3, n4, n5, such that 0 &le; ni &le; 100, 1 &le; i &le; 5. To the first four positive integral numbers (n1, n2, n3, n4) the arithmetic operation, such as addition (+), subtraction (-), multiplication (*), or division (/) and brackets (&#39;(&#39;,&#39;)&#39;) may be freely applied, but in the arithmetic expression formed with these numbers and operations, every one of the four integral numbers should be used once and only once.&nbsp;</p>

<p>Write a program for finding an arithmetic expression that satisfies the above requirement and equals n5.</p>

### 입력

<p>The input file consists of a number of data sets.Each data set is a line of 5 numbers separated by blank.A line of a single -1 represents the end of input.</p>

### 출력

<p>For each data set output the original data set first.If the program finds out the expression for these four arbitrary input numbers, then it gives out the output &quot;OK!&quot;；On the contrary, if the program could not get the result of n5 by any arithmetic operations to the four input numbers, it gives output &quot;NO!&quot;.</p>
