### 문제 설명

<p>Given an integer interval [A,B] and an integer C, your job is to calculate the number of occurrences of C as string in the interval.</p>

<p>For example, 33 appears in 333 twice, and appears in 334 once. Thus the number of occurrences of 33 in [333,334] is 3.</p>

### 입력

<p>The test case is given by a line with the three integers, A, B, C (0 &le; A &le; B &le; 10<sup>10000</sup>, 0 &le; C &le; 10<sup>500</sup>).</p>

<p>&nbsp;</p>

### 출력

<p>Print the number of occurrences of C mod 1000000007.</p>
