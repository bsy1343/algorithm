### 문제 설명

<p>양의 정수 a, b에 대해서, a의 b&nbsp;초지수승은&nbsp;a &uarr;&uarr; b와 같이 나타낸다. 초지수승의 정의는 다음과 같다.</p>

<p>a &uarr;&uarr; 1 = a,</p>

<p>a &uarr;&uarr; (k + 1) = a ^&nbsp;(a &uarr;&uarr; k)</p>

<p>따라서 예를 들면,&nbsp;3&uarr;&uarr;2 = 3^3 = 27 이고, 3&uarr;&uarr;3 = 3^27 =&nbsp;7625597484987이다.</p>

<p>a &uarr;&uarr; b의 마지막 8자리를 출력하세요.</p>

### 입력

<p>양의 정수 a, b가 입력된다.&nbsp;(a,&nbsp;b &le; 20000)</p>

### 출력

<p>a &uarr;&uarr; b의 마지막 8자리를 출력하시오.</p>
