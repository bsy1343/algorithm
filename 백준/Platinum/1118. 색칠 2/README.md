### 문제 설명

<p>지민이는 종이에 색칠하기를 좋아한다. 지민이는 W&times;H 크기의 직사각형 종이를 가지고 있다. 지민이는 종이에 다음과 같이 색칠을 K번 하려고 한다. 0보다 크거나 같고, K-1보다 작거나 같은 i에 대해서, 다음과 같은 행동을 한 번씩 한다.</p>

<ol>
	<li>종이를 x = f[i]에 맞춰서 접는다. 이때, 왼쪽 종이가 오른쪽 종이 위에 올라오게 접는다.</li>
	<li>종이를 가로로 c[i] + 1개의 크기가 동일 한 구간으로 나눈다. 그 다음에 c[i]번 가장 위의 구간부터 차례대로 접는다.</li>
	<li>왼쪽 아래가 (x<sub>1</sub>[i], y<sub>1</sub>[i]) 이고, 오른쪽 위가 (x<sub>2</sub>[i], y<sub>2</sub>[i])인 직사각형을 찾는다. 이때, (0, 0)은 현재 접힌 상태에서 가장 왼쪽 아래 점이다. 그 직사각형을 칠한다. 이때, 페인트는 겹쳐있는 모든 곳에 스며든다.</li>
	<li>종이를 편다.</li>
</ol>

<p>예제는 1117번 문제를 참고한다.</p>

<p>W, H, K, 그리고 각각의 i(0 &le; i &lt; K)에 대해 f[i], c[i], x<sub>1</sub>[i], y<sub>1</sub>[i], x<sub>2</sub>[i], y<sub>2</sub>[i]가 주어질 때, 색칠되어 있지 않은 면적을 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 세 정수 W, H, K가 주어진다.</p>

<p>둘째 줄부터 N개의 줄에는 각각 접고 색칠하는 정보를 의미하는 6개의 정수 f[i], c[i], x<sub>1</sub>[i], y<sub>1</sub>[i], x<sub>2</sub>[i], y<sub>2</sub>[i]가 주어진다.</p>

### 출력

<p>첫째 줄에 색칠되지 않은 영역의 넓이를 출력한다.</p>

### 제한

<ul>
	<li>1 &le; W, H &le; 10<sup>9</sup></li>
	<li>1 &le; K &le; 50</li>
	<li>0 &le; f[i] &le; W</li>
	<li>0 &le; c[i]&nbsp;&le; 1,000</li>
	<li>c[i]+1은 H의 약수</li>
	<li>0 &le; x<sub>1</sub>[i] &lt; x<sub>2</sub>[i] &le; max(f[i], W-f[i])</li>
	<li>0 &le; y<sub>1</sub>[i] &lt; y<sub>2</sub>[i] &le; H/(c[i]+1)</li>
</ul>
