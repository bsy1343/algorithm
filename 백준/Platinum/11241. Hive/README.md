# [Platinum V] Hive - 11241

### 문제 설명

<p>กาลครั้งหนึ่งมีทุ่งดอกไม้สีชมพูแห่งหนึ่งขนาด N x M ตารางเมตร ในแต่ละตารางเมตรมีดอกไม้อยู่ a<sub>ij</sub> ดอก ทุกๆเช้าฝูงกระต่ายจะมาเก็บดอกไม้เหล่านี้กลับไปกินเพื่อให้ขนของพวกมันมีสีชมพู โดยกระต่ายแต่ละตัวจะเดินจากซ้ายบน ไปขวาล่าง และจะไม่เดินกลับไปทางด้านซ้ายหรือกลับขึ้นด้านบนเด็ดขาด ส าหรับทุกๆตารางเมตรที่เดินผ่านและยังมีดอกไม้อยู่กระต่ายจะเก็บดอกไม้ 1 ดอก ถามว่าต้องใช้กระต่ายอย่างน้อยกี่ตัวจึงจะเก็บดอกไม้ได้ทั้งหมด</p>

### 입력

<p>บรรทัดแรกเป็นจ านวนกรณีทดสอบ T ชุด (1 &le; T &le; 10) กรณีทดสอบแต่ชุดประกอบด้วยข้อมูลดังนี้</p>

<ul>
	<li>บรรทัดแรก เป็นจ านวนแถวและคอลัมน์ของทุ่งหญ้า N และ M ตามล าดับ (1 &lt; N, M &le; 1 600)</li>
	<li>บรรทัดที่ 2 ถึง N+1 ประกอบด้วยจ านวนเต็ม M จ านวน โดยแต่ละจ านวน a<sub>ij</sub> แสดงจ านวน ดอกไม้ในตารางเมตรนั้น (0 &le; a<sub>ij</sub> &le; 100)</li>
</ul>

### 출력

<p>ส าหรับแต่ละกรณีทดสอบ ให้แสดงจ านวนกระต่ายที่น้อยที่สุดที่ใช้ในการเก็บดอกไม้ทั้งหมด</p>
