### 문제 설명

<p>자연수 N이 주어졌을 때, N의 각 자리수를 K제곱 한 후에 그 합을 구하는 함수를 S<sub>K</sub>(N)이라고 하자. 예를 들어, S<sub>2</sub>(65) = 6<sup>2</sup> + 5<sup>2</sup> = 61이다.</p>

<p>이제 다음과 같은 수열을 하나 만들어보자. N, S<sub>K</sub>(N), S<sub>K</sub>(S<sub>K</sub>(N)), &hellip; 이때, A와 B와 K가 주어졌을 때, A보다 크거나 같고, B보다 작거나 같은 모든 N으로 각각 수열을 만들었을 때, 그 수열에서 가장 작은 수의 합을 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 세 정수 A, B, K가 주어진다.</p>

### 출력

<p>첫째 줄에 문제의 정답을 출력한다.</p>

### 제한

<ul>
	<li>1 &le; A &le; B &le; 1,000,000</li>
	<li>1 &le; K &le; 6</li>
</ul>
