### 문제 설명

<p>자연수 \(N\)과 정수 \(K\)가 주어졌을 때 이항 계수 \(\binom{N}{K}\)를 \(M\)으로 나눈 나머지를 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 \(N\),&nbsp;\(K\)와 \(M\)이&nbsp;주어진다. (1 &le; \(N\) &le; 10<sup>18</sup>, 0 &le; \(K\) &le; \(N\), 2 &le; \(M\) &le; 2,000, M은 소수)</p>

### 출력

<p>&nbsp;\(\binom{N}{K}\)를 \(M\)으로 나눈 나머지를&nbsp;출력한다.</p>
