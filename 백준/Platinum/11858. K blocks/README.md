### 문제 설명

<p>You are given a sequence A of N positive integers. Let&rsquo;s define &ldquo;value of a splitting&rdquo; the sequence to K blocks as a sum of maximums in each of K blocks. For given K find the minimal possible value of splittings.</p>

### 입력

<p>First line of the input file contains two integers N and K. Next line contains N integers A<sub>1</sub>, A<sub>2</sub>, . . . , A<sub>N</sub> (1 &le; A<sub>i</sub> &le; 10<sup>6</sup>) &mdash; the sequence elements.</p>

### 출력

<p>Output one number &mdash; minimal possible value of a splittings.</p>
