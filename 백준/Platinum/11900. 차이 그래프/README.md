### 문제 설명

<p>승현이는 정점의 개수가 N개인 방향성 그래프 G와 음이 아닌 정수들로 구성된 배열 A[1],A[2],⋯,A[N&minus;1]을 가지고 있습니다. 정점들에는 0 이상 N&minus;1 이하의 정수 번호가 붙어 있으며, 각 간선에는 가중치가 존재합니다. 초기에 이 방향성 그래프에는 간선이 하나도 없었기 때문에, 승현이는 허전함을 느꼈습니다. 이에 그는 아래와 같은 방식으로 간선들을 잇기로 했습니다.</p>

<ul>
	<li>임의의 두 정점 u와 v (0&le;u,v&lt;N,u&ne;v)에 대하여,
	<ul>
		<li>만약 u&gt;v이고 A[u&minus;v]의 값이 양수라면, u에서 v로 향하는 가중치가 A[u&minus;v]인 간선을 추가합니다.</li>
		<li>만약 u&lt;v이고 A[u&minus;v+N]의 값이 양수라면, u에서 v로 향하는 가중치가 A[u&minus;v+N]인 간선을 추가합니다.</li>
	</ul>
	</li>
</ul>

<p>승현이는 이렇게 간선을 추가한 뒤에 임의의 정점에서 출발하여 다른 정점에 도착하는 최단 경로의 길이를 구하고자 합니다.</p>

### 입력

<p>첫 번째 줄에 그래프 G의 정점의 수 N (5&le;N&le;2,000)이 주어집니다. 두 번째 줄에 A[1],A[2],⋯,A[N&minus;1] (0&le;A[1],A[2],⋯,A[N&minus;1]&le;10,000)이 공백을 사이로 두고 주어집니다. 세 번째 줄에 질의의 수 Q (1&le;Q&le;200,000)가 주어집니다. 이후 Q개 줄이 주어지는데, 이 중 i번째 줄 (1&le;i&le;Q)에는 두 개의 정수 a<sub>i</sub>와 b<sub>i</sub> (0&le;a<sub>i</sub>,b<sub>i</sub>&lt;N, a<sub>i</sub>&ne;b<sub>i</sub>)가 공백을 사이로 두고 주어집니다.</p>

### 출력

<p>여러분은 각 질의마다 a<sub>i</sub>번 정점에서 출발하여 b<sub>i</sub>번 정점에 도착하는 최단 경로의 길이를 한 줄에 하나씩 입력에서 주어진 순서대로 출력해야 합니다. 단 이러한 최단 경로가 존재하지 않을 경우 &minus;1을 출력합니다.</p>
