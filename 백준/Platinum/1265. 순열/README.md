### 문제 설명

<p>1부터 N까지의 수를 임의로 배열한 순열은 총 N! = N&times;(N-1)&times;&hellip;&times;2&times;1 가지가 있다.</p>

<p>모든 1 &le; i &le; N에 대해서 |P[i] - i| &le;&nbsp;K를 만족하는 순열 P의 개수를 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 N, K가 순서대로 주어진다. N은 100보다 작거나 같은 자연수이고, K는 6보다 작거나 같은 자연수이다.</p>

### 출력

<p>첫째 줄에 문제의 정답을 출력한다.</p>
