# [Platinum IV] 서브 트리의 크기 - 13024

[문제 링크](https://www.acmicpc.net/problem/13024)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 512 MB

### 통계

제출: 175, 정답: 68, 맞힌 사람: 44, 정답 비율: 32.353%

### 분류

수학, 다이나믹 프로그래밍, 트리, 정수론, 트리에서의 다이나믹 프로그래밍, 분할 정복을 이용한 거듭제곱, 모듈로 곱셈 역원, 트리에서의 전방향 다이나믹 프로그래밍

### 문제 설명

<p>양방향 트리 T가 주어진다. 트리의 정점은 0번부터 N-1번까지 번호가 매겨져 있다.</p>

<p>T의 서브 트리는 T의 연결된 서브 그래프를 의미한다. 서브 트리의 크기는 포함되어 있는 정점의 개수이다.</p>

<p>T의 모든 서브 트리의 크기의 합을 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 N(1 &le; N &le; 10<sup>5</sup>)이 주어진다. 둘째 줄부터 N-1개의 줄에 트리 T의 간선이 주어진다.</p>

### 출력

<p>첫째 줄에 트리 T의 모든 서브 트리의 크기의 합을 1,000,000,007로 나눈 나머지를&nbsp;출력한다.</p>
