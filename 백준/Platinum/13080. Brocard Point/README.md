### 문제 설명

<p>Label the vertices of a triangle in anticlockwise order by A, B and C. We know there is exactly one point P inside the triangle ABC such that &ang;PAB = &ang;PBC = &ang;PCA. This point is known as Brocard point.</p>

<p>You are to write a program that gets the coordinates of&nbsp;A, B and C and computes the coordinates of the Brocard point of the triangle ABC.</p>

### 입력

<p>The first line of the input includes the number of test cases, 1 &le;&nbsp;t &le; 10000. Each test case comes in one line containing six space separated real numbers x<sub>A</sub>, y<sub>A</sub>, x<sub>B</sub>, y<sub>B</sub>, x<sub>C</sub>, y<sub>C</sub>.</p>

### 출력

<p>For each test case, output in one line x-coordinate and y0coordinate of Brocard point rounded to five decimal places.&nbsp;</p>
