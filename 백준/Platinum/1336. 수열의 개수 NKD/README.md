### 문제 설명

<p>세 정수 $N$, $K$, $D$가 주어졌을 때, 다음 조건을 만족하는 크기가 $K$인 자연수 수열 $A_1, A_2, \dots, A_K$의 개수를 구해보자.</p>

<ul>
	<li>$A_1 + A_2 + \dots + A_K = N$</li>
	<li>$A_1 &lt; A_2 &lt; \dots &lt; A_K$</li>
	<li>$A_{i+1} - A_i&nbsp;\le D$ $(1 \le i &lt; K)$</li>
	<li>$A_1 \le D$</li>
</ul>

### 입력

<p>첫째 줄에 $N$, $K$, $D$가 주어진다.</p>

### 출력

<p>문제 조건을 만족하는 수열의 개수를 $10^9 + 7$로 나눈 나머지를 출력한다.</p>

### 제한

<ul>
	<li>$1 \le N, K, D \le 10^5$</li>
</ul>
