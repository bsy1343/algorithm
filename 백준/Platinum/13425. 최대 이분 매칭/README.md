# [Platinum II] 최대 이분 매칭 - 13425

[문제 링크](https://www.acmicpc.net/problem/13425)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 512 MB

### 통계

제출: 24, 정답: 12, 맞힌 사람: 9, 정답 비율: 60.000%

### 분류

수학, 그래프 이론, 이분 매칭

### 문제 설명

<p>무방향 그래프가 주어졌을 때 매칭을 구할 수 있다.</p>

<p>매칭은 간선의 부분 집합으로, 집합에 포함된 간선이 공통된 정점을 공유하면 안되다.</p>

<p>최대 매칭은 부분 집합의 크기가 가장 큰 매칭이다.</p>

<p>다음과 같은 성질을 만족하는 그래프가 존재하는지 확인하고, 존재한다면 그러한 그래프에 존재할 수 있는 간선의 최대 개수를 구하는 프로그램을 작성하시오.</p>

<ul>
	<li>그래프 G는 단순 무방향 그래프이다.</li>
	<li>G는 이분 그래프이고, 각 부분의 크기는 n1과 n2이다.</li>
	<li>G의 최대 매칭의 크기는 ans이다.</li>
	<li>모든 정점의 차수는 d 이상이다.</li>
</ul>

### 입력

<p>첫째 줄에 n1, n2, ans, d가 주어진다. (1 &le; n1, n2 &le; 10<sup>9</sup>, 1 &le; ans, d &le; min(n1, n2))</p>

### 출력

<p>입력으로 주어진 그래프가 존재하지 않으면 -1을 출력하고, 존재하면 그러한 그래프에 존재할 수 있는 간선의 최대 개수를 출력한다.</p>
