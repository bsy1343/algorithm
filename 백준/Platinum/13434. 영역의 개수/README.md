# [Platinum III] 영역의 개수 - 13434

[문제 링크](https://www.acmicpc.net/problem/13434)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 512 MB

### 통계

제출: 28, 정답: 8, 맞힌 사람: 5, 정답 비율: 83.333%

### 분류

다이나믹 프로그래밍, 기하학, 수학, 정수론

### 문제 설명

<p>두 정수 A와 B가 주어진다. 두 정수를 이용해서 평면 위의 A*B개의 직선을 만들 수 있다.</p>

<p>0 &le; a &lt; A, 0 &le; b &lt; B인 모든 (a, b) 쌍은 직선의 방정식 y = ax+b를 만들 수 있다.&nbsp;</p>

<p>A와 B가 주어졌을 때, 평면이 몇 개의 영역으로 나누어지는지 구하는 프로그램을 작성하시오.&nbsp;</p>

### 입력

<p>첫째 줄에 A와 B가 주어진다. (1 &le; A, B &le; 1,200)</p>

### 출력

<p>첫째 줄에 영역의 개수를 출력한다. 이 값은 int 범위를 넘어간다.</p>
