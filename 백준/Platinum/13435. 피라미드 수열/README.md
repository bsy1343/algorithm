# [Platinum III] 피라미드 수열 - 13435

[문제 링크](https://www.acmicpc.net/problem/13435)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 512 MB

### 통계

제출: 189, 정답: 109, 맞힌 사람: 92, 정답 비율: 73.600%

### 분류

수학, 정수론, 유클리드 호제법

### 문제 설명

<p>높이가 X &gt; 1인 피라미드 수열의 주기는 2X-2이다. 처음 2X-2개의 수는 1, 2, ..., X-1, X, X-1, ..., 2 이다.</p>

<p>두 피라미드 수열 A와 B의 높이 N과 M이 주어진다. 이때, 서로 다른 (A[i], B[i]) 쌍의 개수를 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 N, M이 주어진다. (2 &le; N, M &le; 1,000,000,000)</p>

### 출력

<p>첫째 줄에 서로 다른 (A[i], B[i]) 쌍의 개수를 출력한다.</p>

### 힌트

<p>예제 1의 경우에</p>

<ul>
	<li>A: 1, 2, 3, 2, 1, 2, 3, 2, 1, 2, 3, 2, 1, ...</li>
	<li>B: 1, 2, 3, 4, 3, 2, 1, 2, 3, 4, 3, 2, 1, ...</li>
</ul>

<p>(1, 1), (2, 2), (3, 3), (2, 4), (1, 3), (3, 1)이 있다.</p>
