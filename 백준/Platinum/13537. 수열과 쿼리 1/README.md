### 문제 설명

<p>길이가 N인 수열 A<sub>1</sub>, A<sub>2</sub>, ..., A<sub>N</sub>이 주어진다. 이때, 다음 쿼리를 수행하는 프로그램을 작성하시오.</p>

<ul>
	<li><code>i j k</code>: A<sub>i</sub>, A<sub>i+1</sub>, ..., A<sub>j</sub>로 이루어진 부분 수열 중에서 k보다 큰 원소의 개수를 출력한다.</li>
</ul>

### 입력

<p>첫째 줄에 수열의 크기 N (1 &le; N &le; 100,000)이 주어진다.</p>

<p>둘째 줄에는 A<sub>1</sub>, A<sub>2</sub>, ..., A<sub>N</sub>이 주어진다. (1 &le; A<sub>i</sub> &le; 10<sup>9</sup>)</p>

<p>셋째 줄에는 쿼리의 개수 M (1 &le; M &le; 100,000)이 주어진다.</p>

<p>넷째 줄부터 M개의 줄에는 쿼리 i, j, k가 한 줄에 하나씩 주어진다. (1 &le; i &le; j &le; N, 1 &le; k &le; 10<sup>9</sup>)</p>

### 출력

<p>각각의 쿼리마다 정답을 한 줄에 하나씩 출력한다.</p>
