# [Platinum III] K번째 부분 문자열 - 13541

[문제 링크](https://www.acmicpc.net/problem/13541)

### 문제 설명

<p>문자열 S의 모든 부분 문자열을 정렬했을 때, 사전 순으로 K번째 오는 문자열을 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 문자열 S, 둘째 줄에 K (1 &le; K &le; 100,000)가 주어진다. 문자열 S는 알파벳 소문자로만 이루어져&nbsp;있고, 길이는 100,000을 넘지 않는다.</p>

### 출력

<p>S의 K번째 부분 문자열을 출력한다. 만약, 부분 문자열의 개수가 K보다 작으면 -1을 출력한다.</p>
