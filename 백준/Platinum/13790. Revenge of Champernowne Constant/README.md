# [Platinum II] Revenge of Champernowne Constant - 13790

[문제 링크](https://www.acmicpc.net/problem/13790)

### 성능 요약

시간 제한: 8 초 (추가 시간 없음), 메모리 제한: 512 MB

### 통계

제출: 6, 정답: 4, 맞힌 사람: 4, 정답 비율: 66.667%

### 분류

임의 정밀도 / 큰 수 연산, 많은 조건 분기, 수학

### 문제 설명

<p>Champernowne constant is an irrational number. Its decimal representation starts with &ldquo;0.&rdquo;, followed by concatenation of all positive integers in the increasing order.</p>

<p>You will be given a sequence S which consists of decimal digits. Your task is to write a program which computes the position of the first occurrence of S in Champernowne constant after the decimal point.</p>

### 입력

<p>The input has multiple test cases. Each line of the input has one digit sequence. The input is terminated by a line consisting only of #.</p>

<p>It is guaranteed that each sequence has at least one digit and its length is less than or equal to 100.</p>

### 출력

<p>For each sequence, output one decimal integer described above. You can assume each output value is less than 10<sup>16</sup> .</p>
