### 문제 설명

<p>\(M\)개의 자연수 \(N\)과 정수 \(K\)가 주어졌을 때 이항 계수 \(\binom{N}{K}\)를 1,000,000,007로 나눈 나머지를 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 \(M\)이 주어진다. (1 &le; \(M\) &le; 100,000)</p>

<p>둘째 줄부터 \(M\)개의 줄에&nbsp;\(N\)과 \(K\)가 주어진다. (1 &le; \(N\) &le; 4,000,000, 0 &le; \(K\) &le; \(N\))</p>

### 출력

<p>총 \(M\)개의 줄에&nbsp;\(\binom{N}{K}\)를 1,000,000,007로 나눈 나머지를&nbsp;출력한다.</p>
