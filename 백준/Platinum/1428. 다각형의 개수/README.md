# [Platinum II] 다각형의 개수 - 1428

[문제 링크](https://www.acmicpc.net/problem/1428)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 128 MB

### 통계

제출: 84, 정답: 10, 맞힌 사람: 10, 정답 비율: 20.408%

### 분류

구현, 브루트포스 알고리즘, 기하학, 오일러 지표 (χ=V-E+F)

### 문제 설명

<p>다솜이는 엄청나게 큰 종이를 가지고 있다. 다솜이는 종이를 XY좌표로 생각한다. 그리고, 그 종이에 선을 N개 그었다. 선을 N개를 그리고 나서 종이를 본 후에 다솜이는 지금 내부에 선분이 없는 볼록 다각형이 총 몇 개가 있는지 궁금해졌다.</p>

<p>다솜이가 그은 선의 시작점과 끝점의 좌표가 주어졌을 때, 다각형이 총 몇 개 있는지 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 선의 개수 N이 주어진다. N은 50보다 작거나 같은 자연수이다. 둘째 줄부터 N개의 줄에 선분의 정보가 시작점의 X, Y좌표, 끝점의 X, Y좌표가 순서대로 주어진다. 모든 좌표는 -10,000보다 크거나 같고, 10,000보다 작거나 같은 정수이다. 모든 선분은 중복되지 않는다. 또, 시작점과 끝점이 같은 경우는 없다.</p>

### 출력

<p>첫째 줄에 다각형의 개수를 출력한다.</p>
