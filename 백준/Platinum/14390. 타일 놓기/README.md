# [Platinum I] 타일 놓기 - 14390

[문제 링크](https://www.acmicpc.net/problem/14390)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 512 MB

### 통계

제출: 191, 정답: 116, 맞힌 사람: 98, 정답 비율: 67.123%

### 분류

비트마스킹, 다이나믹 프로그래밍, 비트필드를 이용한 다이나믹 프로그래밍

### 문제 설명

<p>영선이는 방 바닥에 타일을 놓으려고 한다. 바닥은 직사각형 모양이며 1&times;1 크기의 정사각형으로 나누어져 있다. 타일의 크기는 1&times;k 크기이며, k는 양의 정수이다. 타일은 바닥의 변과 평행하게 놓아야 하며, 칸의 경계와 변이 일치해야 한다. 또, 타일은 겹쳐서 놓을 수 없다.</p>

<p>영선이의 방에는 기둥이 놓여져 있는데, 기둥의 크기는 1&times;1 이다. 또, 타일은 기둥이 놓여진 칸에 놓여질 수 없다.</p>

<p>방 바닥에 놓을 수 있는 타일의 최소 개수를 구하는 프로그램을 작성하시오. 이때, 바닥에 빈 칸이 있으면 안 된다.</p>

### 입력

<p>첫째 줄에 방의 세로 크기 N과 가로 크기 M이 주어진다. (1 &le; N, M &le; 10)</p>

<p>둘째 줄부터 방 바닥의 모양이 주어진다. &#39;.&#39;은 빈 칸, &#39;#&#39;은 기둥이 놓여진 칸이다.</p>

### 출력

<p>첫째 줄에 방 바닥을 모두 채우는데 필요한 타일의 최소 개수를 출력한다.</p>
