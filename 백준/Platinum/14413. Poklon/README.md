### 문제 설명

<p>Little Mirko is a very simple man. Mirko&rsquo;s friend Darko has given him an array of N natural integers and asked him Q queries about the array that Mirko must answer.</p>

<p>Each query consists of two integers, the positions of the left and right end of an interval in the array. The answer to the query is the number of different values that appear exactly twice in the given interval.</p>

### 입력

<p>The first line of input contains the integers N and Q (1 &le; N, Q &le; 500 000).</p>

<p>The second line of input contains N natural integers less than 1 000 000 000, the elements of the array.</p>

<p>Each of the following Q lines contains two integers, L and R (1 &le; L &le; R &le; N), from the task.</p>

### 출력

<p>The output must consist of Q lines, each line containing the answer to a query, respectively.</p>

### 힌트

<p>In the interval from the first to the third element, there is only one number (number 1) that appears exactly twice.</p>
