# [Platinum II] 직사각형 색칠 - 14886

[문제 링크](https://www.acmicpc.net/problem/14886)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 512 MB

### 통계

제출: 34, 정답: 11, 맞힌 사람: 11, 정답 비율: 40.741%

### 분류

수학, 다이나믹 프로그래밍, 비트마스킹, 비트필드를 이용한 다이나믹 프로그래밍, 분할 정복을 이용한 거듭제곱

### 문제 설명

<p>1&times;1크기의 칸으로 나누어져 있는 N&times;M크기의 그리드가 있다. 각각의 칸을 검정색이나 흰색으로 색칠하려고 한다. 이때, X&times;Y 크기의 직사각형이 모두 같은 칸으로 색칠되지 않아야 한다.</p>

<p>그리드를 색칠하는 방법의 수를 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 N, M, X, Y가 주어진다. (1 &le; X &le; 3, 2 &le;&nbsp;Y &le;&nbsp;M)</p>

<p>N과 M제한은 X값에 따라서 아래와 같이 달라진다.</p>

<ul>
	<li>X = 1인 경우: 2&nbsp;&le; N &le; 1,000,000, 2&nbsp;&le; M &le; 1,000</li>
	<li>X = 2인 경우: 2&nbsp;&le; N &le; 1,000,000, 2&nbsp;&le;&nbsp;M &le; 7</li>
	<li>X = 3인 경우: 3&nbsp;&le; N &le; 8, 2&nbsp;&le;&nbsp;M &le;&nbsp;5</li>
</ul>

### 출력

<p>첫째 줄에 경우의 수를&nbsp;1000000007로 나눈 나머지를 출력한다.</p>
