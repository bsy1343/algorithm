### 문제 설명

<p>임의의 자연수 n에 대하여, 00∙∙∙00 (0이 n개)에서 99∙∙∙99(9가 n개)까지 10<sup>n</sup>개의 수를 생각하자. 이 중에서 원래 수를 제곱한 값의 뒷부분 n자리가 원래 수와 똑같은 경우 그 수를 &lsquo;변치 않는 수&rsquo;라고 하자. 자릿수 n이 주어졌을 때 &lsquo;변치 않는 수&rsquo;는 언제나 4가지가 존재한다. 그중 두 개는 00∙∙∙00과 00∙∙∙01이고, 나머지 두 개 중 하나는 일의 자리 숫자가 5이고 다른 하나는 6이다. 예를 들어 n=4일 때에는 다음 수들이 &lsquo;변치 않는 수&rsquo;이다.</p>

<p style="text-align: center;">0000<sup>2</sup> = 0000, 0001<sup>2</sup> = 0001, 0625<sup>2</sup> = 390625, 9376<sup>2</sup> = 87909376</p>

<p>자릿수 n이 주어졌을 때, 일의 자리 숫자가 5인 &lsquo;변치 않는 수&rsquo;와 일의 자리 숫자가 6인 &lsquo;변치 않는 수&rsquo; 중에서 어느 쪽이 더 큰지를 출력하는 프로그램을 작성하는 것이 문제이다.</p>

<p>&nbsp;</p>

### 입력

<p>첫째 줄에 자릿수 n이 주어진다. (1 &le;&nbsp;n &le; 10,000)</p>

### 출력

<p>일의 자리 숫자가 5인 &lsquo;변치 않는 수&rsquo;가 더 크다면 5를, 일의 자리 숫자가 6인 &lsquo;변치 않는 수&rsquo;가 더 크다면 6을 첫째 줄에 출력한다.</p>
