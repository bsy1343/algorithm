### 문제 설명

<p>정수로 이루어진 크기가 N인 배열 arr과 C가 있다. 배열 C는 다음과 같이 구성되어 있다.</p>

<p>C[0] = arr[0], C[i] = gcd(C[i-1], arr[i]) (단, gcd(x, y)는 x와 y의 최대 공약수)</p>

<p>예를 들어서 배열 arr = [16,16,8,16,2]라고 하면, C = [16,16,8,8,2]가 된다. 배열 C가 주어졌을 때, 가능한 arr배열의 가짓수를 1000000007 (10<sup>9</sup>+7)로 나눈 나머지를 출력하시오.</p>

<p>단, arr에 들어갈 수 있는 수는 1이상 num이하이다.</p>

### 입력

<p>첫째 줄에는 정수 N(1 &le; N &le; 10<sup>5</sup>)과 num(1 &le; num &le; 10<sup>9</sup>)이 공백으로 구분되어 주어진다.</p>

<p>둘째 줄에는 배열 C에 들어있는 수 C[i] (i = 0, 1, &hellip;, N-1)가 공백으로 구분되어 주어진다. (1 &le; C[i] &le; num)</p>

### 출력

<p>문제의 정답을 출력하시오.</p>
