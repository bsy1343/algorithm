### 문제 설명

<p>새로운 언어를 만들었다. 이 언어에서&nbsp;&#39;단어&#39;는 아래 조건을 모두 만족하는 비어있지 않은 문자열이다.</p>

<ol>
	<li>영어 소문자 (&#39;a&#39;-&#39;z&#39;), 마침표 (&#39;.&#39;), 붙임표 (&#39;-&#39;), 그리고 별표 (&#39;*&#39;) 로만 구성되어 있는 비어있지 않은 문자열</li>
	<li>문자열에 포함된 연속한 세 글자가 모두 특수문자인 경우가 없는 문자열 (특수문자는 &#39;.&#39;, &#39;-&#39;, &#39;*&#39;)</li>
	<li>문자열에 포함된 연속한 세 글자가 모두 모음인 경우가 없는 문자열 (모음은 &#39;a&#39;, &#39;e&#39;, &#39;i&#39;, &#39;o&#39;, &#39;u&#39;)</li>
	<li>문자열에 포함된 연속한 세 글자가 자음/모음/특수문자 관계없이 모두 동일한 경우가 없는 문자열</li>
	<li>첫 글자가&nbsp;특수문자가 아니고 마지막 글자도 특수문자가 아닌 문자열</li>
</ol>

<p>예를 들어, 길이가 1인 단어는 총 26가지가 있다. 영어 소문자 한 글자는 각각이 단어이다. 추가로, &quot;boj&quot;, &quot;i.am.coder&quot;, and &quot;hello*world&quot; 모두 단어이다. 하지만, &quot;zzz&quot;, &quot;.boj&quot;, &quot;mioum&quot;, and &quot;hello-.-world&quot; 네 문자열은 단어가 아니다.&nbsp;</p>

<p>양수 a, b가 주어졌을 때, 길이가 a이상 b이하인 단어의 개수를 구하는 프로그램을 작성하시오.&nbsp;단, 수가 매우 클 수 있으므로 이 수를 (10<sup>9</sup> + 7)로 나눈 나머지를 출력한다.</p>

### 입력

<p>첫 줄에 두 정수 a와 b (1 &le; a &le; b &le; 10<sup>18</sup>)가 주어진다.</p>

### 출력

<p>첫 줄에 길이가 a이상 b이하인 단어의 개수를 (10<sup>9</sup> + 7)로 나눈 나머지를 출력한다.</p>
