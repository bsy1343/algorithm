# [Platinum III] 반복 부분문자열 - 1605

[문제 링크](https://www.acmicpc.net/problem/1605)

### 문제 설명

<p>알파벳 소문자로 이루어진 길이 L인 문자열이 있다. 이 문자열의 부분문자열 중, 적어도 한 번은 반복되는 (다시 말해서, 전체 문자열에서 두 번 이상 나타나는) 부분문자열을 &#39;반복 부분문자열&#39;이라고 부르자.</p>

<p>문자열이 주어지면, 가장 긴 &#39;반복 부분문자열&#39;의 길이를 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 문자열의 길이 L(1 &le; L &le; 200,000)이 주어진다. 둘째 줄에는 문자열을 이루는 L개의 알파벳 소문자들이 띄어쓰기 없이 주어진다.</p>

### 출력

<p>첫째 줄에 가장 긴 &#39;반복 부분문자열&#39;의 길이를 출력한다. 만일 &#39;반복 부분문자열&#39;이 하나도 존재하지 않는다면 0을 출력한다.</p>
