# [Platinum III] 비트와 가희 - 16153

[문제 링크](https://www.acmicpc.net/problem/16153)

### 성능 요약

시간 제한: 0.5 초 (추가 시간 없음), 메모리 제한: 256 MB

### 통계

제출: 528, 정답: 85, 맞힌 사람: 62, 정답 비율: 17.765%

### 분류

다이나믹 프로그래밍, 많은 조건 분기, 중간에서 만나기

### 문제 설명

<p>다음과 같은 조건이 <em>N</em>개 주어진다.</p>

<p><meta charset="utf-8" /></p>

<ul dir="ltr">
	<li><em>i</em>번째 bit가 1이어야 한다. 즉, 어떠한 수 <em>X</em>가 있다면 <em>X</em> &amp; (1 &lt;&lt; <em>i</em>)의 값이 0이 아니어야 한다.</li>
</ul>

<p>1 이상 <em>B</em>&nbsp;이하의 정수 중 이러한 조건을 만족하는&nbsp;<em>A</em>의 배수는 몇 개인지 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 <em>N</em>(0 &le; <em>N</em> &lt;&nbsp;32), <em>A</em>, <em>B</em>(0 &lt; <em>A</em>, <em>B</em> &lt;&nbsp;2<sup>31</sup>)가&nbsp;공백으로 구분되어 주어진다.</p>

<p>둘째 줄부터 <em>N</em>개의 각 줄에는 켜져야 하는 bit가 <em>N</em>개 주어진다. 이 값들은 30이하인 음이 아닌 정수이며, 켜져야 하는 bit는 중복으로 주어지지 않는 것이 보장된다.</p>

### 출력

<p>1이상 <em>B</em>이하의 정수 중&nbsp;주어진&nbsp;<em>N</em>개의 조건들을 만족하는&nbsp;<em>A</em>의 배수의 개수를 출력하라.</p>
