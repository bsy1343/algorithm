### 문제 설명

<p>동물원에서 막 탈출한 원숭이 한 마리가 세상구경을 하고 있다. 그는 수학을 매우 좋아하는 원숭이였다. 그래서 그는 N개의 직선을 가지고 기발한 문제를 생각해냈다.</p>

<p>문제는 이렇다. 좌표평면상에 N개의 직선과 상수 K가 주어진다. 직선은 모두 f<sub>i</sub>(x) = a<sub>i</sub> * x + b<sub>i</sub>의 꼴로 나타낼 수 있다. 그랬을 때 함수 g(x)를 {f<sub>1</sub>(x), f<sub>2</sub>(x), ..., f<sub>N</sub>(x)}의 중간값으로 정의하면, g(x) = K가 되는 x를 모두 찾는 문제이다.</p>

### 입력

<p>첫째 줄에 직선의 개수 N과 상수 K가 주어진다. N은 3이상 1,000,000이하인 홀수이고, K는 -100,000이상 100,000이하인 정수이다. 그 다음 줄부터 각 줄마다 두 개의 정수 a<sub>i</sub>, b<sub>i</sub> (0 &le; a<sub>i</sub> &le; 1,000,000, -1,000,000 &le; b<sub>i</sub> &le; 1,000,000) 가 주어진다. 중복되는 직선은 없다.</p>

### 출력

<p>첫째 줄에 g(x) = K 인 x의 구간의 시작과 끝을 출력한다. 각 수는 소수점 넷째자리에서 반올림하여 출력한다. 만약 x가 무한대로 발산하는 경우, 양의 무한대는 +inf, 음의 무한대는 -inf 로 출력하면 된다. g(x) = K를 만족하는 x가 존재하지 않는다면 &quot;impossible&quot;을 출력하면 된다. 답이 되는 구간이 1개 이하인 경우만 입력으로 들어온다.</p>

### 힌트

<p>-0.000 을 출력하지 않도록 주의한다. -0.0005 초과, 0.0005 미만의 수는 0.000 으로 출력하게 하면 된다.</p>
