# [Platinum I] Bob's Rummikub - 16708

[문제 링크](https://www.acmicpc.net/problem/16708)

### 분류

다이나믹 프로그래밍, 브루트포스 알고리즘, 비트마스킹, 백트래킹, 비트필드를 이용한 다이나믹 프로그래밍

### 문제 설명

<p>밥은 루미큐브 게임을 즐겨한다. 밥이 가진 카드와 테이블에 이미 내어져 있는 카드들을 모두 알고 있을 때, 밥이 낼 수 있는 카드 개수의 최댓값을 출력하시오.</p>

<p>루미큐브의 게임 규칙은 다음과 같다.&nbsp;</p>

<ol>
	<li>루미큐브는 1부터 7까지의 숫자가 적힌 4가지 색깔 (blue, red, yellow, black)의 카드로 하는 게임으로, 총 28개의 카드로 게임을 한다.</li>
	<li>특정 숫자와 특정 색깔을 가진 카드는 정확히 하나 존재한다.</li>
	<li>밥이 카드를 낼 때, 테이블에 내어진 카드들이 루미큐브세트들로 구성되도록 해야 한다.</li>
	<li>루미큐브세트에는 그룹과 연속이 있다. 그룹은 색이 다르지만 숫자는 같은 카드 3개나 4개를 말하고, 연속은 색이 같고 숫자가 연속되는 3개 이상의 카드를 말한다.&nbsp;</li>
	<li>테이블에 내어진 카드들이 루미큐브세트로 구성되었다는 것은, 카드들을 적당한 묶음으로 분할해, 모든 묶음이 루미큐브세트를 이루도록 할 수 있다는 것이다.</li>
	<li>밥이 카드를 낼 때, 루미큐브세트를 구성하기 위해서 테이블 위에 놓인 카드들을 활용할 수 있다. 테이블 위에 놓인 카드들은 여전히 루미큐브세트들로 구성되어야 한다.&nbsp;&nbsp;</li>
</ol>

### 입력

<p>첫 줄에는 밥이 가지고 있는 카드의 개수 <em>n</em>이 주어진다. (1&nbsp;&le; <em>n</em> &le; 28)</p>

<p>둘째 줄에는 밥이 가지고 있는 카드에 대한 정보가 사이에 공백을 두고 주어진다.</p>

<p>셋째 줄에는 테이블에 놓인 카드의 개수&nbsp;<em>m</em>이 주어진다. (0&nbsp;&le; <em>m</em> &le; 28-<em>n</em>)</p>

<p>넷째 줄에는 테이블에 놓인 카드에 대한 정보가 사이에 공백을 두고 주어진다.&nbsp;</p>

<p>카드들에 대한 정보는 색(char형)숫자(int형)으로 주어진다. 예시 입출력을 참고하여라.</p>

<p>단, 테이블에 놓인 카드들은 루미큐브세트들로 구성되어 있음이 보장되며, 동일한 카드는 두 번 이상 입력되지 않는다.</p>

### 출력

<p>밥이 낼 수 있는 카드 개수의 최댓값을 출력한다.</p>
