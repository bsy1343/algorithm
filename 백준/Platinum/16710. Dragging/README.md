# [Platinum I] Dragging - 16710

### 문제 설명

<p>태영이와 승한이는 연말을 맞아 해외 여행을 떠났다. 여행지에서 열심히 돌아다니던 둘은 점심 시간이 얼마 남지 않았다는 것을 깨달았다.</p>

<p>정확히 <em>K</em>시간 뒤부터 점심시간이고, 둘은 최대한 괜찮은 음식점에 찾아가서 식사를 하고 싶다. 하지만 문제는 태영이와 승한이의 음식 취향이 정반대라는 것이다.</p>

<p>태영이는 음식이 짤수록 좋아하고 승한이는 음식이 싱거울수록 좋아한다.</p>

<p>태영이와 승한이가 있는 마을에는 <em>N</em>개의 음식점이 있고, 이들은 <em>M</em>개의 길로 연결되어 있다.&nbsp;모든 길은 지나다니는데 정확히 10분의 시간이 걸린다.</p>

<p>두 음식점을 연결하는 길이 여러 개 존재할 수도 있고, 한 음식점에서 출발해서 같은 음식점으로 돌아오는 길이 존재할 수도 있다.&nbsp;</p>

<p>각 음식점은 정확히 하나의 음식만 판매하고, <em>i</em>번째 음식점에서 판매하는 음식의 &#39;짠 정도&#39;는 정수 <em>C<sub>i</sub></em>로 나타난다.</p>

<p>태영이와 승한이는 상대방에게 양보할 생각이 조금도 없고, 둘 사이의 힘의 균형이 팽팽하게 유지되기 때문에 남은 <em>K</em>시간동안 다음 규칙을 따라 이동하게 된다.</p>

<ol>
	<li>첫 20분 동안 태영이가 승한이를 질질 끌고 간다.</li>
	<li>다음 10분 동안 승한이가 태영이를 질질 끌고 간다.</li>
	<li>다음 10분 동안 태영이가 승한이를 질질 끌고 간다.</li>
	<li>다음 20분 동안 승한이가 태영이를 질질 끌고 간다.</li>
	<li>1~4번 과정이 총 <em>K</em>시간 동안 반복된다.</li>
</ol>

<p>예를 들어 2시간 즉, 120분이 남은 경우에는 1~4번 과정을 2번 반복하면 (20+10+10+20)+(20+10+10+20)분으로 총 120분이 지나가게 된다.</p>

<p>이 때 자신이 다른 사람을 질질 끌고 가는 시간에 움직이지 않고 멈춰 있을 수도 있지만, 태영이와 승한이의 각 차례가 끝날 때는 항상 어떤 음식점에 위치해 있어야 한다.&nbsp;</p>

<p>즉, 길 중간에 멈춰설 수는 없다.</p>

<p>태영이와 승한이는 마을 지도를 갖고 있고, 위 규칙대로 움직이되 자신에게 최대한 이득이 되는 방향으로 움직인다.</p>

<p>즉, 태영이는 최대한 짠 정도가 높은 음식을 먹을 수 있는 방향으로, 승한이는 최대한 짠 정도가 낮은 음식을 먹을 수 있는 방향으로 움직인다.</p>

<p>둘은 현재 1번 음식점에 있다. 이 때 태영이와 승한이가 먹게 될 음식의 &#39;짠 정도&#39;를 구하여라.</p>

### 입력

<p>첫 줄에 <em>N, M, K</em>가 사이에 공백을 두고 순서대로 주어진다. (1&nbsp;&le; <em>N</em> &le; 2&nbsp;&times; 10<sup>5</sup>, 0&nbsp;&le; <em>M</em> &le; 5 &times; 10<sup>5</sup>, 0&nbsp;&le; <em>K</em> &le; 5 &times; 10<sup>5</sup>)</p>

<p>두 번째 줄에는 <em>C<sub>1</sub>, C<sub>2</sub>, ... C<sub>N</sub></em>&nbsp;이 사이에 공백을 두고 순서대로 주어진다. (0&nbsp;&le;&nbsp;<em>C<sub>i</sub>&nbsp;</em>&le; 10<sup>9</sup>)</p>

<p>세 번째 줄부터 <em>M+2</em>번째 줄에서는 각 줄에 두 자연수 <em>S<sub>i</sub></em>와 <em>E<sub>i</sub></em>가 주어지는데, 이는 <em>S<sub>i</sub></em>번 음식점과 <em>E<sub>i</sub></em>번 음식점이 도로로 연결되어 있음을 의미한다. (1&nbsp;&le; <em>S<sub>i</sub>,</em>&nbsp;<em>E<sub>i</sub>&nbsp;</em>&le; <em>N</em>)</p>

### 출력

<p>태영이와 승한이가 먹게 될 음식의 &lsquo;짠 정도&rsquo;를 첫 줄에 출력한다.</p>
