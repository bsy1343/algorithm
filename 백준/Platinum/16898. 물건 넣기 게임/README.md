# [Platinum III] 물건 넣기 게임 - 16898

[문제 링크](https://www.acmicpc.net/problem/16898)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 512 MB

### 통계

제출: 148, 정답: 28, 맞힌 사람: 24, 정답 비율: 32.000%

### 분류

다이나믹 프로그래밍, 게임 이론

### 문제 설명

<p>물건 넣기 게임은 두 사람이 턴을 번갈아 가지면서 하는 게임이다.</p>

<p>가장 처음에는 서로 다른 박스&nbsp;A개와 서로 다른 물건 B개가 필요하다. 턴이 돌아올 때마다, 각 사람은 새로운 박스를 추가할 것인지, 새로운 물건을 추가할 것인지 정해야 한다. 새로운 박스나 물건을 추가한 후에, 물건을 박스에 넣는 방법의 개수가 N보다 크거나 같아지는 사람이 게임을 진다.</p>

<p>구사과와 큐브러버가 최적의 방법으로 물건 넣기 게임을 했을 때, 게임의 결과를 구하는 프로그램을 작성하시오. 게임은 구사과가 먼저 시작한다.</p>

### 입력

<p>첫째 줄에 A, B, N이 주어진다. (1 &le; A &le; 10,000, 1 &le; B &le; 30, 2 &le; N &le; 10<sup>9</sup>) 가장 처음에 물건을 박스에 넣는 방법의 개수는 N보다 작다.</p>

### 출력

<p>구사과가 지는 경우에는 &quot;koosaga&quot;, 큐브러버가 지는 경우에는 &quot;cubelover&quot;, 무승부인 경우에는 &quot;jh05013&quot;을 출력한다.</p>
