### 문제 설명

<p>길이가 N인 수열 A<sub>1</sub>, A<sub>2</sub>, ..., A<sub>N</sub>이 주어진다. 이때, 다음 쿼리를 수행하는 프로그램을 작성하시오.</p>

<ul>
	<li><code>s e</code>: s &le; i &lt; j &le; e 이면서 A<sub>i</sub> &gt; A<sub>j</sub>인 (i, j) 쌍의 개수를 출력한다. (1 &le; s &le; e &le; N)</li>
</ul>

### 입력

<p>첫째 줄에 수열의 크기 N (1 &le; N &le; 100,000), 쿼리의 개수 M (1 &le; M &le; 100,000)이 주어진다.</p>

<p>둘째 줄에는 A<sub>1</sub>, A<sub>2</sub>, ..., A<sub>N</sub>이 주어진다. (1 &le; A<sub>i</sub>&nbsp;&le; 1,000,000,000)</p>

<p>셋째 줄부터 M개의 줄에는 쿼리가 한 줄에 하나씩 주어진다.</p>

### 출력

<p>모든 쿼리마다 답을 출력한다.</p>
