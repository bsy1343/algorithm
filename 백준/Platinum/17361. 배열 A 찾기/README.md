### 통계

그래프 이론, 이분 탐색, 매개 변수 탐색, 방향 비순환 그래프, 위상 정렬

### 문제 설명

<p>다음 조건을 만족하는 배열 A를 찾아보자.</p>

<ul>
	<li>크기는 N이고, 2&times;10<sup>9</sup>보다 작거나 같은 자연수로 이루어져 있다.</li>
	<li>크기가 N인 배열 B보다 사전 순으로 뒤에 온다.</li>
	<li>M개의 (i, j)쌍은 A[i] &lt; A[j]를 만족한다.</li>
</ul>

### 입력

<p>첫째 줄에 배열의 크기 N과 쌍의 개수 M이 주어진다. 둘째 줄에는&nbsp;B[1], B[2], ..., B[N]이 주어진다. 셋째 줄부터 M개의 줄에는 A[i] &lt; A[j]를 의미하는 i, j가 주어진다.</p>

### 출력

<p>문제의 조건을 만족하는 배열 A 중에서 사전 순으로 가장 앞서는 것을 출력한다. 만약, 조건을 만족하는 배열이 없는 경우에는 -1을 출력한다.</p>

### 제한

<ul>
	<li>1 &le; N &le; 100,000</li>
	<li>0 &le; M &le; 100,000</li>
	<li>1 &le; B[i] &le; 10<sup>9</sup></li>
	<li>1 &le; i, j &le; N, i &ne; j</li>
</ul>
