### 문제 설명

<p>양의 정수 N과, N보다 큰 소수 P가 주어질 때, N!을 P로 나눈 나머지를 구하여라.</p>

### 입력

<p>첫째 줄에 N과 P가 공백으로 구분되어 주어진다.</p>

### 출력

<p>N!을 P로 나눈 나머지를 구하여라.</p>

### 제한

<ul>
	<li>1 <span class="st">&le;</span> N <span class="st">&lt;</span> P <span class="st">&le;</span> 10<sup>9</sup></li>
	<li>P는 소수</li>
</ul>
