# [Platinum V] 분수 - 18117

### 통계

### 문제 설명

<p>임의의 정수 <em>i</em>가 주어졌을 때, 어떤 수의 소수점 위 또는 아래 <em>i</em>째 자리&nbsp;숫자를 계산하는 알고리즘이 존재한다면, 그러한 수를 &quot;계산 가능한 수&quot; (computable number)라고 한다.</p>

<p>반면에, 그러한 알고리즘이 존재하지 않는다면, 그러한 수를 &quot;계산 불가능한 수&quot; (uncomputable number)라고 한다. 예를 들어, 랜덤으로 생성된&nbsp;프로그램이 무한 루프를 돌지 않을 확률을 나타내는 카이틴 상수 (Chaitin constant)는 계산 불가능한 수이다. 즉, 어떠한 알고리즘으로도 이 수를&nbsp;임의의 정확도로&nbsp;계산할 수 없다.</p>

<p>준석이는 분모와 분자가 자연수인 분수는 모두&nbsp;계산 가능한 수인지 궁금해졌다. 준석이를 도와&nbsp;<em>a</em> / <em>b</em>의 소수점 아래 <em>i</em>째 자리 숫자를 계산하는 알고리즘을 작성하여 모든 유리수가 계산 가능한 수임을 증명하자.</p>

### 입력

<p>첫째 줄에 테스트 케이스의 수 <em>T</em>가 주어진다. (1 &le;&nbsp;<em>T</em> &le;&nbsp;100)</p>

<p>각 테스트 케이스의 첫째 줄에 정수 <em>a</em>와 <em>b</em>가 공백으로 구분되어 주어진다. (1 &le;&nbsp;<em>a</em>, <em>b</em> &le;&nbsp;10<sup>18</sup>)</p>

<p>각 테스트 케이스의 둘째 줄에 정수 <em>i</em>와 <em>n</em>이 공백으로 구분되어 주어진다. (1 &le; <em>i</em> &le; 10<sup>18</sup>, 1 &le;&nbsp;<em>n</em> &le;&nbsp;100)</p>

### 출력

<p>각 테스트 케이스마다, <em>a</em> / <em>b</em>를 소수로 표현했을 때, 소수점 아래 <em>i</em>째 자리부터 (<em>i&nbsp;</em>+ <em>n&nbsp;</em>- 1)째 자리까지 <em>n</em>개의 숫자를 출력하라.</p>

<p>소수를 표현하는 방법이 여럿 있는 경우에는, 소숫점 아래에 0이 무한히 많은 표현 방법을 선택한다.</p>
