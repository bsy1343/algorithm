### 문제 설명

<p>집합 $S$에 대하여, &quot;$a,b \in S \Longrightarrow (a | b) \in S$&quot;가 성립한다면 (또한 그러할 때에만) &quot;$S$가 좋은 집합이다&quot;라고 정의하자. 여기서 &#39;$|$&#39;는 Bitwise-or를 의미한다. 예를 들어, {1, 3}은 좋은 집합이다. (1|1) = 1, (1|3) = 3, (3|3) = 3은 모두 {1, 3}의 원소기 때문이다. 다만, (1|2) = 3이므로 {1, 2}는 좋은 집합이 아니다.</p>

<p>전체 집합 $U = \{ 0, 1, \cdots, 2^N - 1 \}$의 부분집합 $A$에 대하여, &quot;$A$의 원소의 개수가 $K$고, $A$와 $(U \setminus A)$가 모두 좋은 집합&quot;이면 (또한 그러할 때에만) $A$를 &quot;$N, K$-분할&quot;이라고 한다.</p>

<p>$N$과 $K$가 주어질 때, &quot;$N, K$-분할&quot;이 존재하는지 밝히고, 있다면 그러한 예를 출력하는 프로그램을 작성하시오.</p>

### 입력

<p>첫 번째 줄에 자연수 $N$과 $K$가 사이에 공백을 두고 주어진다.</p>

### 출력

<p>만약 &quot;$N, K$-분할&quot;이 존재하지 않다면 첫 번째 줄에 &quot;<code>NO</code>&quot;를 출력한다.</p>

<p>만약 &quot;$N, K$-분할&quot;이 존재한다면 첫 번째 줄에 &quot;<code>YES</code>&quot;를 출력한다. 또한 두번째 줄에 &quot;$N, K$-분할&quot;인 집합의 원소를 오름차순으로 출력한다.</p>

<p>&quot;$N, K$-분할&quot;인 집합이 여러 개 존재한다면, 아무거나 출력해도 정답으로 인정된다.</p>

### 제한

<p>모든 입력 데이터는 다음 조건을 만족한다.</p>

<ul>
	<li>$1 \le N \le 12$</li>
	<li>$1 \le K \le 2^N$</li>
</ul>
