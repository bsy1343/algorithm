### 문제 설명

<p>Consider the infinite binary string $I$ formed by concatenating the binary representations of all the strictly positive integers in increasing order: $I = $ &quot;11011100...&quot;.</p>

<p>You are given a binary string $A$. Your task is to find the smallest integer $L$ such that $A$ is a suffix of $I[1 \ldots L]$.</p>

### 입력

<p>The only line of input contains the binary string $A$, $1 \le \left\lvert A \right\rvert \le 55$.</p>

### 출력

<p>Print a single line with a single integer: the number $L$.</p>
