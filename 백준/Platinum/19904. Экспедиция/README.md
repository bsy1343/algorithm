### 문제 설명

<p>Планируется отправить экспедицию к соседней звёздной системе. Были отобраны n кандидатов, пронумерованных от 1 до n, среди которых необходимо выбрать участников экспедиции. Организаторы хотят отправить в экспедицию как можно больше кандидатов.</p>

<p>Среди кандидатов был проведён опрос, в процессе которого каждый мог указать не более, чем одного из остальных кандидатов, с которым он не готов отправиться в экспедицию. Результатом опроса для i-го кандидата является целое число ai , которое равно номеру кандидата, с которым i-й кандидат не готов отправиться в экспедицию, либо &minus;1, если i-й кандидат готов отправиться в экспедицию в любом составе.</p>

<p>Теперь организаторы должны выбрать, кто из кандидатов отправится в экспедицию. Решено было выбрать участников экспедиции так, что если туда входит некоторый кандидат i, и a<sub>i</sub> &ne; &minus;1, то туда не входит кандидат a<sub>i</sub>. Организаторы хотят выбрать максимальное количество участников экспедиции.</p>

<p>Требуется написать программу, которая по заданным результатам опроса кандидатов определяет максимальное количество кандидатов, которых можно отправить в экспедицию.</p>

### 입력

<p>В первой строке входных данных находится целое число n &mdash; количество кандидатов (1 ⩽ n ⩽ 300 000).</p>

<p>В следующих n строках даны результаты опроса, i-я из этих строк содержит результат опроса i-го кандидата, целое число a<sub>i</sub> (a<sub>i</sub> = &minus;1 или 1 ⩽ a<sub>i</sub> ⩽ n, a<sub>i</sub> &ne; i).</p>

### 출력

<p>В единственной строке выведите одно целое число &mdash; максимальное количество кандидатов, которых можно отправить в экспедицию.</p>
