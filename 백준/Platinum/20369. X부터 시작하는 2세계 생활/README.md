# [Platinum I] X부터 시작하는 2세계 생활 - 20369

[문제 링크](https://www.acmicpc.net/problem/20369)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 1024 MB

### 통계

제출: 34, 정답: 22, 맞힌 사람: 18, 정답 비율: 72.000%

### 분류

많은 조건 분기, 다이나믹 프로그래밍

### 문제 설명

<p>제 이름은 베가입니다. 2세계라는&nbsp;이름은&nbsp;해당 세계가 2 x&nbsp;<em>N</em>&nbsp;형태의 격자 모양인 데서 유래됐다고 합니다. 종종 지구의 사람들은 2세계의 어딘가로 소환되는데, 그 사람이 2세계에 있는&nbsp;보물을&nbsp;전부 모으면 지구로 다시 돌아갈 수 있다고 전해집니다. 하지만, 2세계는 너무 광활하여 최대한 빠르게 보물을&nbsp;모으지 않으면 도중에 굶어 죽는다고 합니다. 얼마 전, 소환된&nbsp;어느&nbsp;야바위꾼과&nbsp;농부도&nbsp;그렇게&nbsp;목숨을 잃었지요. 저는 2세계에서 목숨을 잃는 지구인들을 더 이상 두고 볼 수 없습니다. 오랜 정보 수집을 바탕으로 저는&nbsp;마침내, 보물의 위치가 전부 표시된&nbsp;2 x&nbsp;<em>N</em> 형태의 지도를 완성시켰습니다.</p>

<p>여러분들께 부탁이 있습니다. 1 &le; <em>i</em> &le; 2, 1 &le; <i>j</i>&nbsp;&le; <em>N</em>을 만족하는 모든 (<em>i</em>, <em>j</em>)쌍에 대해, (<em>i</em>, <i>j</i>)에 소환된 지구인이 모든 보물을&nbsp;모을 때까지의 가장 빠른 시간을 알려주셨으면 합니다.&nbsp;지구인은 상하좌우로 인접한 칸으로 이동하는 데에&nbsp;1초가 걸린다고 합니다. 그리고 보물이&nbsp;있는 칸에 도착하면, 그 보물은&nbsp;0초 만에 즉시 회수된다고 합니다.</p>

<p>지구의 사람들을&nbsp;구할 수 있게 도와주세요!</p>

### 입력

<p>첫째 줄에 정수 <em>N&nbsp;</em>(1 &le; <em>N</em> &le; 200,000)이 주어진다.</p>

<p>둘째 줄부터 셋째 줄까지 베가가 완성한 2 x&nbsp;<em>N</em> 형태의 지도가 주어진다.</p>

<p>&#39;<code>#</code>&#39;은 보물이&nbsp;있는 칸, &#39;<code>.</code>&#39;은 비어있는 칸을 의미한다. <em>i+1</em>번째 줄의 <em>j</em>번째 문자는 좌표 (<em>i</em>, <em>j</em>)에 대응된다.</p>

### 출력

<p>1 &le; <em>i</em> &le; 2, 1 &le; <i>j</i>&nbsp;&le; <em>N</em>을 만족하는 모든 (<em>i</em>, <em>j</em>)쌍에 대해, (<em>i</em>, <em>j</em>)에 소환된 지구인이 모든 보물을&nbsp;모을 때까지의 가장 빠른 시간을 출력하라.</p>

<p><em>i</em>번째 줄의 <em>j</em>번째 정수는 좌표 (<em>i</em>, <em>j</em>)에서 소환된 경우에&nbsp;대응된다.</p>
