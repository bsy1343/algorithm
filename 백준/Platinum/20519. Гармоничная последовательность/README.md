### 문제 설명

<p>Цикл лекций в университете Флатландии посвящен изучению последовательностей.</p>

<p>Профессор называет последовательность целых чисел a<sub>1</sub>, a<sub>2</sub>, &hellip;, a<sub>n</sub> гармоничной, если каждое число, кроме a1 и an, равно сумме соседних: a<sub>2</sub> = a<sub>1</sub> + a<sub>3</sub>, a<sub>3</sub> = a<sub>2</sub> + a<sub>4</sub>, &hellip;, a<sub>n-1</sub> = a<sub>n-2</sub> + a<sub>n</sub>. Например, последовательность [1, 2, 1, &ndash;1] является гармоничной, поскольку 2 = 1 + 1, и 1 = 2 + (&ndash;1).</p>

<p>Рассмотрим последовательности равной длины: A = [a<sub>1</sub>, a<sub>2</sub>, &hellip;, a<sub>n</sub>] и B = [b<sub>1</sub>, b<sub>2</sub>, &hellip;, b<sub>n</sub>]. Расстоянием между этими последовательностями будем называть величину d(A, B) = |a<sub>1</sub> &ndash; b<sub>1</sub>| + |a<sub>2</sub> &ndash; b<sub>2</sub>| + &hellip; + |a<sub>n</sub> &ndash; b<sub>n</sub>|. Например, d([1, 2 ,1, &ndash;1], [1, 2, 0, 0]) = |1 &ndash; 1| + |2 &ndash; 2| + |1 &ndash; 0| + |&ndash;1 &ndash; 0| = 0 + 0 + 1 + 1 = 2.</p>

<p>В конце лекции профессор написал на доске последовательность из n целых чисел B = [b<sub>1</sub>, b<sub>2</sub>, &hellip;, b<sub>n</sub>] и попросил студентов в качестве домашнего задания найти гармоничную последовательность A = [a<sub>1</sub>, a<sub>2</sub>, &hellip;, a<sub>n</sub>], такую, что d(A, B) минимально. Чтобы облегчить себе проверку, профессор просит написать в качестве ответа только искомое минимальное расстояние d(A, B).</p>

<p>Требуется написать программу, которая по заданной последовательности B определяет, на каком минимальном расстоянии от последовательности B найдется гармоничная последовательность A.</p>

### 입력

<p>Первая строка входного файла содержит целое число n &ndash; количество элементов в последовательности (3 &le; n &le; 300 000).</p>

<p>Вторая строка содержит n целых чисел b<sub>1</sub>, b<sub>2</sub>, &hellip;, b<sub>n</sub> (&ndash;10<sup>9</sup> &le; b<sub>i</sub> &le; 10<sup>9</sup>).</p>

### 출력

<p>Выходной файл должна содержать одно целое число: минимальное возможное расстояние от последовательности во входном файле до гармоничной последовательности.</p>

### 힌트

<p>В приведенном примере оптимальной является, например, гармоничная последовательность [1, 2, 1, &ndash;1].</p>
