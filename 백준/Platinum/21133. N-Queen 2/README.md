### 문제 설명

<p>N-Queen 문제는&nbsp;크기가 N &times; N인 체스판 위에 퀸 N개를&nbsp;서로 공격할 수 없게 놓는&nbsp;문제이다.</p>

<p>N이 주어졌을 때, 퀸을 놓는 방법 한 가지를 출력하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 N이 주어진다. (4&nbsp;&le; N &le; 10,000)</p>

### 출력

<p>N개의 줄을 출력해야 한다. i번째 줄에는 하나의 정수를 출력해야 하고, 이 정수는 i번째 행에 있는 퀸이 있는 열의 번호이다.</p>
