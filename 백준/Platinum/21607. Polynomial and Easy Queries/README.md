### 문제 설명

<p>길이 $N$의 수열 $A$가 주어집니다. 이 수열에 아래 세 가지 종류의 쿼리를 처리하는 프로그램을 만들어 봅시다.</p>

<ul>
	<li>$1$ $l$ $r$: $l \le i \le r$인 모든 $A_i$를 $f(A_i)$로 바꿉니다.</li>
	<li>$2$ $l$ $r$: $l \le i \le r$인 모든 $A_i$를 $g(A_i)$로 바꿉니다.</li>
	<li>$3$ $x$: $A_x$를 출력합니다. 답이 매우 커질 수 있으니, $100 003$으로 나눈 나머지를 출력합니다.</li>
</ul>

<p>단, $f(x)=2x^2 -1$, $g(x) = 4x^3 - 3x$입니다.</p>

### 입력

<p>첫 줄에는 수열의 길이 $N$과 쿼리의 수 $Q$가 주어집니다.</p>

<p>둘째 줄에는 수열 $A$의 초기 상태 $A_1, A_2, \cdots, A_N$이 주어집니다.</p>

<p>셋째 줄부터 $Q$개의 줄에는 각 쿼리에 대한 정보가 위의 형태 ($t$ $l$ $r$ 또는 $t$ $x$)로 순서대로 주어집니다.</p>

### 출력

<p>모든 3번 쿼리에 대해, 그 답을 한 줄에 하나씩 출력합니다.</p>

### 제한

<ul>
	<li>$1 \le N \le 5 \times 10^5$</li>
	<li>$1 \le Q \le 5 \times 10^5$</li>
	<li>$1 \le A_i \le 10^5$</li>
	<li>모든 쿼리에 대해, $1 \le t \le 3$, $1 \le l \le r \le N$, $1 \le x \le N$</li>
	<li>3번 쿼리가 적어도 하나는 주어집니다.</li>
</ul>
