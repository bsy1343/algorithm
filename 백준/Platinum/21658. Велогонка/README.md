### 문제 설명

<p>Велосипедисты, участвующие в шоссейной гонке, в некоторый момент времени, который называется начальным, оказались в точках, удалённых от места старта на x<sub>1</sub>, x<sub>2</sub>, &hellip;, x<sub>n</sub> метров (n &ndash; общее количество велосипедистов). Каждый велосипедист двигается со своей постоянной скоростью v<sub>1</sub>, v<sub>2</sub>, &hellip;, v<sub>n</sub> метров в секунду. Все велосипедисты двигаются в одну и ту же сторону.</p>

<p>Репортёр, освещающий ход соревнований, хочет определить момент времени, в который расстояние между лидирующим &nbsp;в гонке велосипедистом и замыкающим гонку велосипедистом станет минимальным, чтобы с вертолёта сфотографировать сразу всех участников велогонки.</p>

<p>Требуется написать программу, которая по заданному количеству велосипедистов n, заданным начальным положениям велосипедистов x<sub>1</sub>, x<sub>2</sub>, &hellip;, x<sub>n</sub> и их скоростям v<sub>1</sub>, v<sub>2</sub>, &hellip;, v<sub>n</sub>, вычислит момент времени t, в который расстояние l между лидирующим и замыкающим велосипедистом будет минимальным.</p>

### 입력

<p>Первая строка входного файла содержит целое число n &nbsp;&ndash; количество велосипедистов.</p>

<p>В последующих n строках указаны по два целых числа: x<sub>i</sub> &ndash; расстояние от старта до i-го велосипедиста в начальный момент времени (0 &le; x<sub>i</sub> &le; 10<sup>7</sup>) и &nbsp;v<sub>i</sub> &ndash; его скорость (0 &le; v<sub>i</sub> &le; 10<sup>7</sup>).</p>

### 출력

<p>В выходной файл необходимо вывести два вещественных числа: t &ndash; время в секундах, прошедшее от начального момента времени до момента, когда расстояние в метрах между лидером и замыкающим будет минимальным, &nbsp;l &ndash; искомое расстояние.&nbsp;</p>

<p>Числа t и l должны иметь абсолютную или относительную погрешность не более 10<sup>&ndash;6</sup>, что означает следующее. Пусть выведенное число равно x, а в правильном ответе оно равно y. Ответ будет считаться правильным, если значение выражения |x &ndash; y| / max{1, |y|} не превышает 10<sup>&ndash;6</sup>.</p>

### 제한

<p>2 &le; n &le; 10<sup>5</sup>.</p>
