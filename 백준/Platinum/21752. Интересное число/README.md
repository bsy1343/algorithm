### 문제 설명

<p>Для заданного числа $n$ найдите наименьшее положительное целое число с суммой цифр $n$, которое делится на $n$.&nbsp;</p>

### 입력

<p>Во входном файле содержатся целое число $n$ ($1 \le n \le 1000$).</p>

### 출력

<p>Выходной файл должен содержать искомое число. Ведущие нули выводить не разрешается.</p>
