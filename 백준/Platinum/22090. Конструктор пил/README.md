### 문제 설명

<p>Иван &mdash; конструктор пил. Целью конструктора пил является формирование профиля пилы &mdash; длин зубьев. Ведь пила с различным размером зубьев пилит гораздо лучше, чем пила с одинаковыми размерами зубьев.</p>

<p>Однажды Ивану попалась одна интересная статья. В ней британские учёные доказали, что идеальной пилой будет пила, удовлетворяющая следующим условиям:</p>

<ul>
	<li>размеры всех зубьев пилы различны,</li>
	<li>второй зуб пилы больше, чем первый и третий,</li>
	<li>четвертый зуб пилы больше, чем третий и пятый,</li>
	<li>...</li>
</ul>

<p>То есть фактически размеры зубьев должны образовывать перестановку, причем зубья, которые стоят на чётных позициях, должны быть больше своих соседей.</p>

<p>Иван очень заинтересовала статья и он решил узнать сколько различных пил он может составить, если он имеет набор из&nbsp;<i>n</i>&nbsp;различных зубьев. Пилы, получающиеся друг из друга разворотом последовательности зубьев в обратном порядке считаются различными. Ответ может быть довольно большим, поэтому Иван решил найти его по модулю 10<sup>9</sup>&nbsp;+&nbsp;7.</p>

<p>Например, из четырех зубьев Ивана может составить 5 различных пил: (1, 3, 2, 4), (1, 4, 2, 3), (2, 3, 1, 4), (2, 4, 1, 3) и (3, 4, 1, 2).</p>

### 입력

<p>В первой и единсивенной строке содержится целое число&nbsp;<i>n</i>&nbsp;&mdash; количество зубьев в пиле, которую хочет сделать Иван (1&nbsp;&le;&nbsp;<i>n</i>&nbsp;&le;&nbsp;5&nbsp;000).</p>

### 출력

<p>Выведите искомое количество пил по модулю 10<sup>9</sup>&nbsp;+&nbsp;7.</p>
