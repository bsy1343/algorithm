### 문제 설명

<p>Андрей Сергеевич работает воспитателем в детском саду. Недавно он придумал новую веселую игру, для участия в которой все дети разбились на&nbsp;<i>n</i>&nbsp;команд. В качестве призов для участников игры Андрей Сергеевич подготовил&nbsp;<i>d</i>&nbsp;конфет.</p>

<p>По итогам игры оказалось, что в команде, занявшей&nbsp;<i>i</i>-е место, ровно&nbsp;<i>x<sub>i</sub></i>&nbsp;участников.</p>

<p>Теперь перед Андреем Сергеевичем стоит непростая задача: надо распределить конфеты. При этом должны выполняться следующие условия:</p>

<ul>
	<li>все участники одной команды должны получить поровну конфет: пусть участники команды, занявшей&nbsp;<i>i</i>-е место получат по&nbsp;<i>a<sub>i</sub></i>&nbsp;конфет;</li>
	<li>каждый ребенок должен получить хотя бы одну конфету:&nbsp;<i>a<sub>i</sub></i>&nbsp;&gt;&nbsp;0;</li>
	<li>участники команды, занявшей более высокое место, должны получить больше конфет: если&nbsp;<i>i</i>&nbsp;&lt;&nbsp;<i>j</i>, то&nbsp;<i>a<sub>i</sub></i>&nbsp;&gt;&nbsp;<i>a<sub>j</sub></i></li>
	<li>все конфеты должны быть распределены:&nbsp;<i>a</i><sub>1</sub><i>x</i><sub>1</sub>&nbsp;+&nbsp;<i>a</i><sub>2</sub><i>x</i><sub>2</sub>&nbsp;+ ...&nbsp;<i>a</i><sub><i>n</i></sub><i>x</i><sub><i>n</i></sub>&nbsp;=&nbsp;<i>d</i></li>
</ul>

<p>Помогите Андрею Сергеевичу раздать детям конфеты, или сообщите,что это невозможно.</p>

### 입력

<p>Первая строка содержит два целых числа&nbsp;<i>n</i>&nbsp;и&nbsp;<i>d</i>&nbsp;(1&nbsp;&le;&nbsp;<i>n</i>&nbsp;&le;&nbsp;100, 1&nbsp;&le;&nbsp;<i>d</i>&nbsp;&le;&nbsp;10<sup>6</sup>). Вторая строка содержит&nbsp;<i>n</i>&nbsp;целых чисел&nbsp;<i>x</i><sub>1</sub>, ...,&nbsp;<i>x<sub>n</sub></i>&nbsp;(1&nbsp;&le;&nbsp;<i>x<sub>i</sub></i>&nbsp;&le;&nbsp;100 для всех&nbsp;<i>i</i>&nbsp;от 1 до&nbsp;<i>n</i>).</p>

### 출력

<p>Если разделить конфеты возможно, то в первой строке выведите&nbsp;<code>YES</code>, а во второй выведите&nbsp;<i>n</i>&nbsp;целых чисел:&nbsp;<i>a</i><sub>1</sub>,&nbsp;<i>a</i><sub>2</sub>, ...,&nbsp;<i>a</i><sub><i>n</i></sub>. Если разделить конфеты указанным способом нельзя, выведите&nbsp;<code>NO</code>.</p>
