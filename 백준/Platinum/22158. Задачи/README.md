### 문제 설명

<p>При подготовке соревнований по программированию одна из проблем, встающих перед организаторами &mdash; выбор задач, которые будут в этом соревновании. Чтобы соревнование было интересным, необходимо, чтобы задачи в нем были разной сложности. Также, необходимо, чтобы задачи были на разные темы. Ведь кому интересно решать одинаковые задачи? К счастью, у организаторов есть множество задач, и им есть, из чего выбирать.</p>

<p>У вас есть несколько задач. Для каждой задачи известна ее уровень сложости &mdash; целое число, и то, какие темы она затрагивает. Вам нужно выбрать такое подмножество задач, чтобы выполнялись следующие требования:</p>

<ul>
	<li>Выбрана ровно одна задача каждого уровня сложности</li>
	<li>Каждая тема представлена</li>
	<li>Никакие две задачи не пересекаются по темам</li>
</ul>

<p>Сформируйте из имеющихся у вас задач набор задач на соревнование, удовлетворяющий написанным выше требованиям.</p>

### 입력

<p>Первая строка содержит три целых числа&nbsp;<i>n</i>&nbsp;(1&nbsp;&le;&nbsp;<i>n</i>&nbsp;&le;&nbsp;300) &mdash; количество задач,&nbsp;<i>d</i>&nbsp;(1&nbsp;&le;&nbsp;<i>d</i>&nbsp;&le;&nbsp;50) &mdash; количество уровней сложности задач и&nbsp;<i>m</i>&nbsp;(1&nbsp;&le;&nbsp;<i>m</i>&nbsp;&le;&nbsp;18) &mdash; количество различных тем. Далее, в&nbsp;<i>n</i>&nbsp;строках задается описание задач.</p>

<p>Каждая задача задается в следующем формате:&nbsp;<i>c<sub>i</sub>&nbsp;k<sub>i</sub>&nbsp;t<sub>i1</sub>&nbsp;t<sub>i2</sub>&nbsp;... t<sub>ik<sub>i</sub></sub></i>, где&nbsp;<i>c<sub>i</sub></i>&nbsp;(1&nbsp;&le;&nbsp;<i>c<sub>i</sub></i>&nbsp;&le;&nbsp;<i>d</i>) &mdash; сложность&nbsp;<i>i</i>-й задачи,&nbsp;<i>k<sub>i</sub></i>&nbsp;(0&nbsp;&le;&nbsp;<i>k<sub>i</sub></i>&nbsp;&le;&nbsp;<i>m</i>) &mdash; количество тем, которые затрагивает данная задача,&nbsp;<i>t<sub>ij</sub></i>&nbsp;(1&nbsp;&le;&nbsp;<i>t<sub>ij</sub></i>&nbsp;&le;&nbsp;<i>m</i>) &mdash; темы, которые затрагивает&nbsp;<i>i</i>-я задача. Внутри одной задачи все&nbsp;<i>t<sub>ij</sub></i>&nbsp;различны.</p>

### 출력

<p>Выведите в первой строке &laquo;<code>OK</code>&raquo;, если требуемый набор задач найден, и &laquo;<code>Impossible</code>&raquo; в противном случае. Если ответ есть, выведите в следующей строке&nbsp;<i>d</i>&nbsp;целых чисел &mdash; номера задач, которые входят в требуемый набор.</p>
