### 문제 설명

<p>Маленький Иван недавно решил написать свой собственный текстовый редактор. Этот текстовый редактор он собирается использовать для написания сложных программ, поэтому ему очень важна функция выделения парных скобок.</p>

<p>На ранней стадии текстовый редактор должен поддерживать всего одну операцию &mdash; изменение символа на позиции&nbsp;<i>i</i>. При этом редактор не разрешает иметь текст длиной более&nbsp;<i>n</i>&nbsp;символов. Каждый раз, когда новый символ является открывающей или закрывающей круглой скобкой, редактор должен подсвечивать парную ей скобку, если таковая имеется.</p>

<p>Определим понятие парной скобки. Пусть на позиции&nbsp;<i>i</i>&nbsp;в тексте располагается открывающая скобка. Тогда парной к ней будет закрывающая скобка на позиции&nbsp;<i>j</i>, для которой верны следующие свойства:</p>

<ul>
	<li><i>i</i>&nbsp;&lt;&nbsp;<i>j</i>;</li>
	<li>если рассмотреть текст с позиции&nbsp;<i>i</i>&nbsp;до позиции&nbsp;<i>j</i>&nbsp;включительно и удалить из него все символы, не являющиеся круглыми скобками, то получится правильная скобочная последовательность;</li>
	<li><i>j</i>&nbsp;минимально.</li>
</ul>

<p>Аналогичным образом определяется парная скобка для закрывающей скобки.</p>

<p>К сожалению, Ивану не удалось реализовать желаемую функциональность. Ваша задача состоит в том, чтобы по заданным операциям, для каждой операции изменения символа на круглую скобку находить к ней парную.</p>

### 입력

<p>Первая строка содержит целые числа&nbsp;<i>n</i>&nbsp;(1&nbsp;&le;&nbsp;<i>n</i>&nbsp;&le;&nbsp;100000) &mdash; максимальная длина текста, и&nbsp;<i>m</i>&nbsp;(1&nbsp;&le;&nbsp;<i>m</i>&nbsp;&le;&nbsp;100000) &mdash; число операций модификации символа.</p>

<p>Далее, в&nbsp;<i>m</i>&nbsp;строках описаны изменения операции изменения символа в виде целого числа&nbsp;<i>i</i>&nbsp;&mdash; позиции, в которой изменяется символ, и нового символа&nbsp;<i>c</i>&nbsp;(1&nbsp;&le;&nbsp;<i>i</i>&nbsp;&le;&nbsp;<i>n</i>,&nbsp;<i>c</i>&nbsp;является строчной буквой латинского алфавита, открывающей или закрывающей круглой скобкой). Изначально считается, что текст состоит из&nbsp;<i>n</i>&nbsp;строчных латинских букв &ldquo;a&rdquo;.</p>

### 출력

<p>Для каждой операции изменения символа на скобку, необходимо на отдельной строке вывести позицию парной ей скобки. Если парной скобки не существует, в соответствующей строке выведите число -1.</p>
