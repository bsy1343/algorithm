### 문제 설명

<p>С древности у каждого джавая был набор из трех видов оружия: лазерный меч, лазерная сабля и лазерный ножик для намазывания масла на хлеб (вдруг джавай проголодается).</p>

<p>Но так как это оружие джавайское, а не простое, на длины входящих в набор предметов накладывались следующие ограничения:</p>

<ul>
	<li>Длина ножика &mdash;&nbsp;<i>d</i><sub>1</sub>, длина сабли &mdash;&nbsp;<i>d</i><sub>2</sub>, длина меча &mdash;&nbsp;<i>d</i><sub>3</sub>&nbsp;должны быть простыми числами;</li>
	<li><i>d</i><sub>1</sub>&nbsp;&le;&nbsp;<i>d</i><sub>2</sub>&nbsp;&le;&nbsp;<i>d</i><sub>3</sub>;</li>
	<li>(<i>d</i><sub>1</sub>&nbsp;+&nbsp;<i>d</i><sub>2</sub>)<sup>2</sup>&nbsp;&minus; 1 делится на&nbsp;<i>d</i><sub>3</sub>;</li>
	<li>(<i>d</i><sub>2</sub>&nbsp;+&nbsp;<i>d</i><sub>3</sub>)<sup>2</sup>&nbsp;&minus; 1 делится на&nbsp;<i>d</i><sub>1</sub>;</li>
	<li>(<i>d</i><sub>3</sub>&nbsp;+&nbsp;<i>d</i><sub>1</sub>)<sup>2</sup>&nbsp;&minus; 1 делится на&nbsp;<i>d</i><sub>2</sub>.</li>
</ul>

<p>Компания &ldquo;Dart Generics Industries&rdquo; продает любой набор джавайского оружия по его номеру в&nbsp;<i>лексикографическом</i>&nbsp;порядке. А именно, упорядочим все джавайские наборы по неубыванию&nbsp;<i>d</i><sub>1</sub>, при равном&nbsp;<i>d</i><sub>1</sub>&nbsp;по неубыванию&nbsp;<i>d</i><sub>2</sub>, а при равных&nbsp;<i>d</i><sub>1</sub>&nbsp;и&nbsp;<i>d</i><sub>2</sub>&nbsp;по возрастанию&nbsp;<i>d</i><sub>3</sub>&nbsp;и пронумеруем их в этом порядке от 1 до бесконечности. Тогда по заданному&nbsp;<i>k</i>&nbsp;можно купить&nbsp;<i>k</i>-й набор в этом порядке.</p>

<p>Джавай&nbsp;<i>Anykey</i>&nbsp;хочет купить&nbsp;<i>k</i>-й набор. Подскажите ему размеры его оружия.</p>

### 입력

<p>Первая строка содержит одно целое число&nbsp;<i>k</i>&nbsp;(1&nbsp;&le;&nbsp;<i>k</i>&nbsp;&le;&nbsp;60000).</p>

### 출력

<p>Выведите три упорядоченных по неубыванию размера оружий, входящих в&nbsp;<i>k</i>-й комлект.</p>
