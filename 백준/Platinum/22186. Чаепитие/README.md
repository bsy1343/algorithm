### 문제 설명

<p>В одном из отделов крупной организации работает&nbsp;<i>n</i>&nbsp;человек. Как практически все сотрудники этой организации, они любят пить чай в перерывах между работой. При этом они достаточно дисциплинированы и делают в день ровно один перерыв, во время которого пьют чай. Для того, чтобы этот перерыв был максимально приятным, каждый из сотрудников этого отдела обязательно пьет чай одного из своих любимых сортов. В разные дни сотрудник может пить чай разных сортов. Для удобства пронумеруем сорта чая числами от 1 до&nbsp;<i>m</i>.</p>

<p>Недавно сотрудники отдела купили себе большой набор чайных пакетиков, который содержит&nbsp;<i>a</i><sub>1</sub>&nbsp;пакетиков чая сорта номер 1,&nbsp;<i>a</i><sub>2</sub>&nbsp;пакетиков чая сорта номер 2, ...,&nbsp;<i>a<sub>m</sub></i>&nbsp;пакетиков чая сорта номер&nbsp;<i>m</i>. Теперь они хотят знать, на какое максимальное число дней им может хватить купленного набора так, чтобы в каждый из дней каждому из сотрудников доставался пакетик чая одного из его любимых сортов.</p>

<p>Каждый сотрудник отдела пьет в день ровно одну чашку чая, которую заваривает из одного пакетика. При этом пакетики чая не завариваются повторно.</p>

### 입력

<p>Первая строка содержит два целых числа&nbsp;<i>n</i>&nbsp;и&nbsp;<i>m</i>&nbsp;(1&nbsp;&le;&nbsp;<i>n</i>,&nbsp;<i>m</i>&nbsp;&le;&nbsp;50). Вторая строка содержит&nbsp;<i>m</i>&nbsp;целых чисел&nbsp;<i>a</i><sub>1</sub>, ...,&nbsp;<i>a</i><sub>m</sub>&nbsp;(1&nbsp;&le;&nbsp;<i>a<sub>i</sub></i>&nbsp;&le;&nbsp;10<sup>6</sup>&nbsp;для всех&nbsp;<i>i</i>&nbsp;от 1 до&nbsp;<i>m</i>).</p>

<p>Далее следуют&nbsp;<i>n</i>&nbsp;строк &mdash;&nbsp;<i>i</i>-я из этих строк описывает любимые сорта&nbsp;<i>i</i>-го сотрудника отдела и имеет следующий формат: сначала следует положительное число&nbsp;<i>k<sub>i</sub></i>&nbsp;&mdash; количество любимых сортов чая этого сотрудника, а затем идут&nbsp;<i>k<sub>i</sub></i>&nbsp;различных чисел от 1 до&nbsp;<i>m</i>&nbsp;&mdash; номера этих сортов.</p>

### 출력

<p>Выведите одно целое число &mdash; искомое максимальное количество дней.</p>
