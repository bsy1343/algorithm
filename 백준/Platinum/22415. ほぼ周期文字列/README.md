# [Platinum I] ほぼ周期文字列 - 22415

[문제 링크](https://www.acmicpc.net/problem/22415)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 512 MB

### 통계

제출: 3, 정답: 2, 맞힌 사람: 2, 정답 비율: 66.667%

### 분류

문자열, 이분 탐색, 해싱

### 문제 설명

<p>文字列 $S$ が与えられる。この文字列 $S$ に対し、$Q$ 個のクエリに答えよ。 $i$ 番目のクエリでは、$S[l_i,\ r_i]$ から1文字まで変えてよいとき、$S[l_i,\ r_i]$ を周期 $t_i$ の文字列にできるかどうかを判定せよ。$S[l,\ r]$ は文字列 $S$ の $l$ 文字目から $r$ 文字目までの部分文字列を表す。</p>

<p>文字列 $W$ が周期 $t$ の文字列であるとは、 $i\ =\ 1,\ 2,\ ... ,\ |W|-t$ に対し、 $ W_{i} = W_{i+t}$ となることとする。</p>

### 입력

<p>入力は以下の形式で標準入力から与えられる。</p>

<blockquote>
<p>$S$</p>

<p>$Q$</p>

<p>$l_1$ $r_1$ $t_1$</p>

<p>$...$</p>

<p>$l_Q$ $r_Q$ $t_Q$</p>
</blockquote>

### 출력

<p>$Q$ 行にわたって出力せよ。 $i$ 行目には、$i$ 番目のクエリの答えを <code>Yes</code> または <code>No</code> で出力せよ。</p>

### 제한

<ul>
	<li>$1 \leq |S| \leq 10^5$</li>
	<li>$1 \leq Q \leq 10^5$</li>
	<li>$1 \leq l_i \leq r_i \leq |S|$</li>
	<li>$1 \leq t_i \leq r_i-l_i+1 $</li>
	<li>$S$はアルファベットの小文字のみからなる</li>
</ul>
