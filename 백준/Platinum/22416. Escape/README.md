# [Platinum III] Escape - 22416

[문제 링크](https://www.acmicpc.net/problem/22416)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 512 MB

### 통계

제출: 11, 정답: 2, 맞힌 사람: 2, 정답 비율: 100.000%

### 분류

그래프 이론, 그래프 탐색, 애드 혹

### 문제 설명

<p>頂点に正の値を持つ無向グラフが与えられる。 頂点には 1 から $N$ の番号がついており、$i$ 番目の頂点は $w_i$ の値を持っている。 1 番目の頂点からスタートし、直前に通った辺を通ることができないという制約のもとでグラフ上を移動することができる。 各頂点では，初めて訪れた時に限りその頂点が持つ値の点数を得られる。</p>

<p>取得できる点数の総和の最大値を求めよ。</p>

### 입력

<p>入力は以下の形式で標準入力から与えられる。</p>

<blockquote>
<p>$N$ $M$</p>

<p>$w_1$ $w_2$ $...$ $w_N$</p>

<p>$u_1$ $v_1$</p>

<p>$u_2$ $v_2$</p>

<p>$...$</p>

<p>$u_M$ $v_M$</p>
</blockquote>

<p>$1$ 行目には<b>グラフ</b>の頂点数 $N$ と辺の数を表す整数 $M$ が入力される。$2$ 行目には各頂点が持つ値 $w_i$ が入力される。さらに続けて $M$ 行に、各辺により繋がれる $2$ 頂点の番号が入力される。</p>

### 출력

<p>答えを1行に出力せよ。</p>

### 제한

<ul>
	<li>$1 \leq N \leq 100000$</li>
	<li>$N-1 \leq M \leq 100000$</li>
	<li>$1 \leq w_i \leq 1000$</li>
	<li>$1 \leq u_i, v_i \leq N$</li>
	<li>多重辺・自己辺は存在しない</li>
	<li>グラフは連結である</li>
</ul>
