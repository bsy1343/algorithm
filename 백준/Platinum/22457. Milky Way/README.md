### 문제 설명

<p>天の川交通公社は星間旅行ツアーの企画・運営を行う旅行会社である。 天の川交通公社では「天の川横断織姫彦星体験ツアー」という企画を計画中である。 このツアーは琴座のベガを出発し、 星々を巡って鷲座のアルタイルに向かうというものである。 あなたは天の川交通公社に社員であり、 ツアーの経路の選択を任されている。</p>

<p>簡単のため天の川は2次元座標上にあるものとし、 星は五芒星で表現されるものとする。 ツアーに使用する宇宙船には特殊なエンジンが搭載されており、 五芒星の線分上はエネルギー無しで移動することができる。 一方、五芒星の間を移動する際には、その距離に比例したエネルギーが必要となる。</p>

<p>近年、天の川交通公社の売上は低迷しており、 宇宙船のエネルギー代を始めとした各種必要経費の節約を迫られている。 あなたの仕事は、ベガからアルタイルに移動する際、 星間移動距離の総和が最小になるような経路を求め、 その総和を出力するプログラムを書くことである。</p>

<p>ある五芒星からその内部に含まれる別の五芒星に移動する場合、 五芒星同士が接していない場合は星間の移動として扱われることに注意せよ。</p>

### 입력

<p>入力は1つ以上のデータセットからなる。1つのデータセットは次の形式をしている。</p>

<pre>
<var>N</var>&nbsp;<var>M</var>&nbsp;<var>L</var>
<var>x<sub>1</sub></var>&nbsp;<var>y<sub>1</sub></var>&nbsp;<var>a<sub>1</sub></var>&nbsp;<var>r<sub>1</sub></var>
<var>x<sub>2</sub></var>&nbsp;<var>y<sub>2</sub></var>&nbsp;<var>a<sub>2</sub></var>&nbsp;<var>r<sub>2</sub></var>
<var>...</var>
<var>x<sub>N</sub></var>&nbsp;<var>y<sub>N</sub></var>&nbsp;<var>a<sub>N</sub></var>&nbsp;<var>r<sub>N</sub></var>
</pre>

<p>各テストケースの初めの1行は整数N、M、Lからなり、 Nは星の数(1 &le; N &le; 100)、 Mはベガの番号(1 &le; M &le; N)、 Lはアルタイルの番号(1 &le; L &le; N)を表す。 続くN行には各星の情報が与えられる。 各行はx<sub>i</sub>、y<sub>i</sub>、a<sub>i</sub>、r<sub>i</sub>の4つの整数からなり、 x<sub>i</sub>はi番目の星の中心のx座標(0 &le; x<sub>i</sub>&nbsp;&le; 1,000)、 y<sub>i</sub>はi番目の星の中心のy座標(0 &le; y<sub>i</sub>&nbsp;&le; 1,000)、 a<sub>i</sub>はi番目の星の中心座標と星の先端を結んだ直線がy軸となす角度(0 &le; a<sub>i</sub>&nbsp;&lt; 72)、 r<sub>i</sub>はi番目の星の中心座標から星の先端までの長さ(1 &le; r<sub>i</sub>&nbsp;&le; 1,000)である。 入力の終わりは3つのゼロを含む行で表される。</p>

<p>図 D-2の左の星はx=5, y=10, a=0, r=5の星を表しており、右の星はx=15, y=10, a=30, r=5の星を表している。</p>

### 출력

<p>各入力ごとに、ベガからアルタイルに移動する際に必要な星間移動距離の総和の最小値を1行に出力せよ。 出力には、0.000001を超える誤差があってはならない。</p>
