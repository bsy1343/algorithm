### 문제 설명

<p>1 &sim; NまでのN個の数字の順列A<sub>1</sub>, A<sub>2</sub>, &hellip;&hellip;, A<sub>N</sub>が与えられる。 この順列に対して区間[i, j](1 &le; i &le; j &le; N)の数字の順序を逆順にする操作reverse(i, j)を行うことが出来る。 例えば、[1, 2, 3, 4, 5]に対してreverse(2, 4)を適用すると[1, 4, 3, 2, 5]となる。 最小で何回の操作を行えば順列を昇順にソートされた状態にすることが出来るか計算せよ。</p>

### 입력

<p>入力は以下の形式で与えられる。</p>

<pre>
N
A<sub>1</sub> A<sub>2</sub> &hellip;&hellip; A<sub>N</sub>


</pre>

### 출력

<p>問題の解を1行に出力せよ。</p>

### 제한

<ul>
	<li>
	<p>Nは整数である</p>
	</li>
	<li>
	<p>2 &le; N &le; 10</p>
	</li>
</ul>
