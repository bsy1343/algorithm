# [Platinum II] 다리 건너기 게임 - 22852

[문제 링크](https://www.acmicpc.net/problem/22852)

시간 제한: 1 초, 메모리 제한: 1024 MB

### 통계

그래프 이론, 그래프 탐색, 너비 우선 탐색, 게임 이론

### 문제 설명

<p>제이와 케이는 다리 건너기 게임을 하려고 한다. 다리 건너기 게임은 시작점에 놓여 있는 말을 도착점으로 옮기는 게임으로, <em>N</em>개의 섬과 몇 개의 일방통행 다리로 이루어져 있다.&nbsp;섬에는 1부터 <em>N</em>까지의 번호가 붙어있다. 제이부터 시작하여 차례가 번갈아 가면서 진행되는데, 자신의 차례가 되면 아무것도 안 하고 건너뛰거나, 자신이 설치한 일방통행 다리를 따라 말을 옮길 수 있다. 자신의 턴에 말을 도착점인&nbsp;<em>N</em>번 섬으로 이동시키는 사람이 게임의 승자이다.</p>

<p>둘은 같은 게임판으로 너무 많은 게임을 해서 직접 게임판을 제작하기로 하였다. 제이는 자신만이 이용할 수 있는&nbsp;<em>J</em>개의 일방통행 다리를 설치했고, 케이 또한 자신만이 이용할 수 있는&nbsp;<em>K</em>개의 일방통행 다리를 설치했다.</p>

<p>제이와 케이는 자신의 승리를 목표로, 그럴 수 없다면 적어도 상대의 승리를 막는 것을 목표로 플레이하고, 매번 최선의 선택을 한다. 게임이 영원히 끝나지 않아 아무도 승리하지 않을 수도 있음에 유의하라.</p>

<p>둘은 자신들이 만든 게임판에 애정이 생겼고, 그래서 열심히 분석을 하기로 하였다. 제이와 케이는 도착점을 제외한&nbsp;<em>N-1</em>개의 섬이 출발점이 되었을 때의 결과를 각각 구해보려고 한다. 하지만, 너무나도 많은 분석량에 둘은 지쳐버렸다. 제이와 케이를 위해서 <em>N-1</em>가지의 상황에 대한 결과를 구해주자.</p>

### 입력

<p>입력의 첫 줄에 섬의 개수 <em>N</em>, 각각 설치한&nbsp;다리 개수 <em>J</em>와&nbsp;<em>K</em>가 주어진다.</p>

<p>그 다음 J개의 줄에는 각각 제이가 설치한 다리를 나타내는 두 정수&nbsp;<em>x, y</em>가 주어진다. 이는 <em>x</em>번 섬에서 <em>y</em>번 섬으로 가는 다리를 나타낸다.</p>

<p>그 다음 K개의 줄에는&nbsp;각각 케이가 설치한 다리를 나타내는 두 정수&nbsp;<em>x, y</em>가 같은 방식으로 주어진다.</p>

### 출력

<p><i>N-1</i>개의 문자를 한 줄에 공백 구분 없이 출력한다.&nbsp;<em>i</em>번째 문자는&nbsp;<em>i</em>번째 섬이 출발점일 때의 결과로, 제이가 이길 수 있다면 J, 케이가 이길 수 있다면 K, 아무도&nbsp;이길 수 없으면 D이다.&nbsp;</p>

### 제한

<ul>
	<li>2 &le; <em>N</em> &le; 100,000</li>
	<li>0&nbsp;&le; <em>J, K</em> &le; 100,000</li>
	<li>1 &le; <em>x, y</em>&nbsp;&le; N이고, <em>x</em>와 <em>y</em>는 서로 다르다.</li>
</ul>
