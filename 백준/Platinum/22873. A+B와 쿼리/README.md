### 문제 설명

<p>음이 아닌 $N$자리 10진수 정수 $A$와 $B$가 있다. 단, 각 수의 앞부분에 불필요한 0이 있을 수도 있다. 이때 $N+1$자리 정수 $C = A + B$를 생각하자. 마찬가지로 $A+B$의 값이 $N+1$자리가 안 될 경우 앞에 불필요한 0을 채워 $N+1$자리로 만든다.</p>

<p>$Q$개의 쿼리를 순서대로 처리해야 한다. 각 쿼리는 다음 중 하나이다.</p>

<ul>
	<li><code>A i d</code>: $A$의 오른쪽에서부터 $i$번째 자리 숫자를 $d$로 바꾼다.</li>
	<li><code>B i d</code>: $B$의 오른쪽에서부터 $i$번째 자리 숫자를 $d$로 바꾼다.</li>
</ul>

<p>각 쿼리를 수행할 때마다 $C = A+B$를 다시 계산하고, $C$의 각 자리 숫자 중 몇 개의 숫자가 바뀌었는지 출력하시오.</p>

### 입력

<p>첫째 줄에는 $N$과 $Q$가 주어진다. ($1 \leq N, Q \leq 300\ 000$)</p>

<p>둘째 줄에는 $A$, 셋째 줄에는 $B$가 주어진다.</p>

<p>다음 $Q$줄에는 한 줄에 하나씩 쿼리가 주어진다. 모든 쿼리에서 $1 \leq i \leq N$, $0 \leq d \leq 9$이다.&nbsp;</p>

### 출력

<p>각 쿼리를 수행할 때마다 $C = A+B$를 다시 계산하고, $C$의 각 자리 숫자 중 몇 개의 숫자가 바뀌었는지 출력한다.</p>

### 힌트

<p>쿼리를 실행하기 전과 각 쿼리를 실행한 후의 계산식은 차례로 다음과 같다.</p>

<ul>
	<li>$09905 + 80000 = 089905$</li>
	<li>$09905 + 80005 = 089910$</li>
	<li>$09995 + 80005 = 090000$</li>
	<li>$09995 + 90005 = 100000$</li>
</ul>
