### 문제 설명

<p>길이가 $N$인 두 수열 $(p_1, p_2, \ldots, p_N)$, $(q_1, q_2, \dots, q_N)$ 이 주어진다.</p>

<p>이때 다음 값을 구하여라.</p>

<p>$$\sum_{i=1}^{N} {\sum_{j=1}^{N} {\min(|p_i - p_j|, |q_i - q_j|)} }$$</p>

### 입력

<p>첫째 줄에 수열의 길이 $N$($ 1 \leq N \leq 1\ 000\ 000$) 이 주어진다.</p>

<p>둘째 줄에는 정수 $p_1, p_2, \ldots, p_N$ 이 공백으로 구분되어 주어진다. ($1 \leq p_i \leq 1\ 000\ 000$)</p>

<p>셋째 줄에는 정수 $q_1, q_2, \ldots, q_N$ 이 공백으로 구분되어 주어진다. ($1 \leq q_i \leq 1\ 000\ 000$)</p>

### 출력

<p>첫째 줄에 문제의 답을 출력한다.</p>
