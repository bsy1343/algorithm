### 문제 설명

<p>제제가 아끼는 라임 오렌지 나무인 밍기뉴는 정점의 개수가 $N$, 간선의 개수가 $N-1$인 연결 그래프이다. 밍기뉴의 $i$번 간선에는 $a_i$개의 라임 오렌지가 달려 있다. 제제와 포르투가는 밍기뉴 위에서 게임을 하려고 한다.&nbsp;첫 번째 턴에는 제제가 밍기뉴의 뿌리인&nbsp;$r$에 말을 놓고, 각 턴마다 두 사람은 다음 과정을 반복한다.</p>

<ul>
	<li>말이 놓인 정점과 인접한&nbsp;간선을 아무거나 선택한다.</li>
	<li>선택한 간선에 있는 라임 오렌지를 1개 이상 딴다.</li>
	<li>말을 선택한&nbsp;간선의 반대편 정점으로 이동한다.</li>
</ul>

<p>더 이상 턴을 진행할 수 없을 때, 즉 말이 놓인 정점과 인접한 어느 간선에도 딸&nbsp;라임 오렌지가 없을 때&nbsp;그 턴의 플레이어는 게임에서 패배한다. $r = 1, 2, \cdots, N$일 때, 제제가 이긴다면 &quot;Zeze&quot;, 포르투가가 이긴다면 &quot;Portuga&quot;를 따옴표 없이 출력하는 프로그램을 작성하자.</p>

### 입력

<p>첫 번째&nbsp;줄에는 밍기뉴의 정점의 개수 $N$이 주어진다.</p>

<p>두 번째 줄부터 $N$번째 줄까지 $N-1$개의 줄에 간선과 그 간선에 달린 라임 오렌지의 개수가 $x_i \ y_i \ a_i$의 형태로 주어진다.</p>

### 출력

<p>$r = 1, 2, \cdots, N$일 때 제제가 이긴다면 &quot;Zeze&quot;, 포르투가가 이긴다면 &quot;Portuga&quot;를 따옴표 없이 각각 $N$ 개의 줄에 걸쳐 출력한다.</p>

### 제한

<ul>
	<li>$1 \le N \le 3 \times 10^5$</li>
	<li>$1 \le a_i \le 10^9$</li>
</ul>
