# [Platinum V] 짝수싫어수 - 22957

[문제 링크](https://www.acmicpc.net/problem/22957)

### 성능 요약

시간 제한: 1 초 (추가 시간 없음), 메모리 제한: 1024 MB (추가 메모리 없음)

### 통계

제출: 115, 정답: 80, 맞힌 사람: 70, 정답 비율: 81.395%

### 분류

수학, 다이나믹 프로그래밍, 자릿수를 이용한 다이나믹 프로그래밍

### 문제 설명

<p>도도의 친구는 짝수를 싫어한다. 어느 정도냐면 짝수만 봐도 불을 뿜으며 빡! 하고 화를 낼 정도이다.&nbsp;</p>

<p>친구를 위해 도도는 짝수싫어수를 고안했다.</p>

<p>짝수싫어수는 $3$, $5$, $7$로 이루어져 있으며&nbsp;$3$의 개수, $5$의 개수, $7$의 개수가 <u><strong>동시에</strong></u> 짝수가 아닌 수이다. 숫자가 $0$번 사용되었으면 짝수로 생각한다.</p>

<p>예를 들어 $75$는 $7$의 개수가 $1$개, $5$의 개수가 $1$개, $3$의 개수가 $0$개로 모든 숫자의 개수가 동시에 짝수가 아니므로 짝수싫어수이다.</p>

<p>$10^N$보다 작은 수 중&nbsp;$K$번째로 큰 짝수싫어수를 출력하는 프로그램을 만들어보자.</p>

<p>답이 존재하지 않는 입력이 주어지는 경우는 없다.</p>

### 입력

<p>첫째 줄에 정수 $N$ ($1 \le&nbsp;N \le 30$) 과 정수 $K$ ($1 \le K&nbsp;\le 2^{63}-1$)가 주어진다.</p>

### 출력

<p>첫째 줄에&nbsp;$10^N$보다 작은 수 중&nbsp;$K$번째로 큰 짝수싫어수를 출력한다.</p>

### 힌트

<p>$2^{63}-1$ 은 8byte 자료형으로 표현할 수 있다.</p>

<p>하지만 예제 2 출력값인 $357335535573573353577573355337$ 은 8byte 자료형으로 표현할 수 없다.</p>
