# [Platinum V] 직각 이등변 삼각형 - 2297

[문제 링크](https://www.acmicpc.net/problem/2297)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 128 MB

### 통계

제출: 65, 정답: 22, 맞힌 사람: 19, 정답 비율: 55.882%

### 분류

정렬, 스위핑

### 문제 설명

<p>평면상에 n개의 직각이등변삼각형들이 있을 때, 이 삼각형들이 덮는 면적을 구하는 프로그램을 작성하시오.</p>

<p>직각이등변삼각형은 세 정수 x, y, m(m&gt;0)으로 표현된다. 이는 삼각형의 세 꼭짓점이 (x, y), (x+m, y), (x, y+m)임을 의미한다.</p>

### 입력

<p>첫째 줄에 정수 n(1 &le; n &le; 2,000)이 주어진다. 다음 n개의 줄에는 각 삼각형을 나타내는 x, y, m이 주어진다. x, y는 절댓값이 10,000,000을 넘지 않으며, m은 1,000이하이다.</p>

### 출력

<p>첫째 줄에 면적을 출력한다. 면적을 출력할 때는 소숫점 아래 한 자리를 반드시 출력한다.</p>
