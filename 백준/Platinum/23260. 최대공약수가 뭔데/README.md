### 문제 설명

<p>채완이는 1학년 후배들이 유클리드 호제법을 배웠다는 소식을 듣고 지문에 <code>&quot;최대공약수&quot;</code>가 들어간 문제를 만들기로 했다.</p>

<p>오름차순으로 정렬된 길이가 $K$인&nbsp;수열&nbsp;$\{A_1, A_2 \dots A_K\} $의&nbsp;&nbsp;최대공약수가 1일 때 이를 K-채완 수열이라 한다.</p>

<p>${A_1, A_2 \dots A_K} $의 최대공약수는 모든 $A_i (1 \leq i \leq K) $의 공통된 약수인 자연수 중 가장 큰 수를 의미한다.</p>

<p>서로 다른 $N$개의&nbsp;자연수가 주어질 때 서로 다른 $K$개를&nbsp;선택하여 K-채완 수열을 만드는 경우의 수를 구해보자.</p>

### 입력

<p>첫째 줄에 $N$, $K$가&nbsp;주어진다.</p>

<p>둘째 줄에 $1\,000\,000$이하인 자연수 $N$개가 공백으로 구분되어&nbsp;주어진다.&nbsp;</p>

### 출력

<p>K-채완 수열을 만드는 경우의 수를 $1\,000\,000\,007(=10^9+7)$로 나눈 나머지를&nbsp;출력하라.&nbsp;</p>

### 제한

<ul>
	<li>$1 \leq&nbsp;N&nbsp;\leq 1\,000\,000$</li>
	<li>$1&nbsp;\leq K \leq N$</li>
</ul>
