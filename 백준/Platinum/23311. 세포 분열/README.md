### 문제 설명

<p>대학원생 시철이는 대학원에서도 복전을 한다. 이번엔 무려 생명공학이다!</p>

<p>과제에 짓눌리는 걸 좋아하는 시철이답게, 오늘도 어김없이 세포 배양&nbsp;과제를 하고 있다.</p>

<p>시철이가 배양하는 세포는 상당히 특이한 성질을&nbsp;가지고 있다.</p>

<ol>
	<li>모든 세포는 분열할 때, 좌우로 갈라진다.&nbsp;세포 분열이 끝나고 나면, 같은&nbsp;종류의 세포가 하나 더 만들어지는 것이다.</li>
	<li>때때로 세포 분열 중 돌연변이가 발생해, 다른 종류의 세포가 하나 만들어질 수 있다. 돌연변이가 일어나면, 이전과 같은&nbsp;세포 $1$개와 이전과 다른 세포 $1$개가 만들어지고 이 둘의 순서는 무작위로 정해진다.</li>
	<li>세포 종류에 따라서 돌연변이가 아예 발생하지 않을 수도 있다. 만약 발생한다면, 오직 다른 한 종류로만&nbsp;가능하다.</li>
	<li>분열하는 세포는 무작위로 정해지고, 동시에 여러&nbsp;세포가 분열할 수 있다.</li>
</ol>

<p>불타는 금요일, 시철이는 배양지에 세포 하나를 올려두었다. 다음 주 월요일에 시철이가 연구실로 돌아왔을 땐 모든 세포 분열이 멈춰있는 상태였다. 곧바로 시철이는 이 배양지에 있는 세포들을 관찰했지만, 돌연변이가 너무 많이 생기는 바람에&nbsp;이 배양지가 본인의 배양지인지 동기의 배양지인지 헷갈리기 시작했다.</p>

<p>과연 배양지에 있는 세포들은 시철이가 올려둔 초기 세포에서 분열이 된 것일까?</p>

### 입력

<p>첫 번째 줄에 세포&nbsp;종류의 개수 $N$가 주어진다.&nbsp;($1 \leq N \leq 26$)</p>

<p>두 번째 줄에&nbsp;돌연변이를 나타내는 알파벳 소문자 $a_1, a_2, \cdots, a_N$가 공백 단위로&nbsp;주어진다. 이것은 $i$번째 알파벳에 해당하는 세포의 분열 중 돌연변이가 발생해 $a_i$번째 알파벳에 해당하는 세포가 생길&nbsp;수 있음을 뜻한다.&nbsp;$a_i$가 $i$번째 알파벳일 경우, 해당 종류의 세포는 돌연변이가 발생하지 않는 것을 의미한다.&nbsp;$a_i$는 $a$부터 $N$번째 알파벳&nbsp;중 하나다.</p>

<p>세 번째&nbsp;줄에 초기 세포의 종류를 나타내는 알파벳 소문자&nbsp;$K$가 주어진다. $K$는 $a$부터 $N$번째 알파벳 중 하나다.</p>

<p>네 번째 줄에 시철이가 관찰한&nbsp;세포 배열을 나타내는 $S$가 주어진다. $S$의 모든 원소는&nbsp;$a$부터 $N$번째 알파벳 소문자 중 하나고, 공백 없이 한 줄에 주어진다.&nbsp;($1 \leq |S|&nbsp;\leq 500$)</p>

### 출력

<p>관찰한 세포 배열이 초기 세포로부터 생겨날 수 있다면 <span style="color:#e74c3c;"><code><strong>YES</strong></code></span>를 출력한다.</p>

<p>그렇지 않으면 <span style="color:#e74c3c;"><code><b>NO</b></code></span>를 출력한다.</p>
