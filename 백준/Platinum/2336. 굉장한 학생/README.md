### 문제 설명

<p>N명의 학생이 참여하여 세 번의 시험을 치렀다. N명의 학생들은 세 번의 시험에 모두 응시하였다. 조교는 각각의 시험에서 같은 등수의 학생이 한 명도 없도록 성적을 매겼다.</p>

<p>A라는 학생이 B라는 학생보다 세 번의 시험에서 모두 성적이 좋다면, A가 B보다 &#39;대단하다&#39;고 한다. 또, C라는 학생보다 &#39;대단한&#39; 학생이 한 명도 없으면, C를 &#39;굉장하다&#39;고 한다.</p>

<p>세 번의 시험에서 각 학생의 성적이 주어졌을 때, &#39;굉장한&#39; 학생의 수를 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 N(1 &le; N &le; 500,000)이 주어진다. 다음 세 개의 줄에는 각 시험에서 1등인 학생부터 N등인 학생이 순서대로 주어진다. 학생의 번호는 1부터 N까지 매겨져 있다.</p>

### 출력

<p>첫째 줄에 &#39;굉장한&#39; 학생의 수를 출력한다.</p>
