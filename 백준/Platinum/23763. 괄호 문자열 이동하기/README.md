### 문제 설명

<p>올바른 괄호 문자열(VPS)은 다음과 같이 정의한다.</p>

<ol>
	<li>빈 문자열은 VPS이다.</li>
	<li><code>S</code>가 VPS라면 <code>(S)</code>도 VPS이다.</li>
	<li><code>S</code>와 <code>T</code>가 VPS라면 <code>ST</code>도 VPS이다.</li>
	<li>모든 VPS는 위 세 가지 규칙으로만 만들 수 있다.</li>
</ol>

<p>예를 들어&nbsp;&quot;<code>(())()</code>&quot;, &quot;<code>()()()</code>&quot;, &quot;<code>(()())</code>&rdquo; 등은 VPS이고,&nbsp;&quot;<code>())(()</code>&quot;, &quot;<code>(</code>&quot;, &quot;<code>(()()()</code>&quot; 등은 VPS가 아니다.</p>

<p>VPS를 좋아하는 태수는 길이가 $N$인 VPS $S$를 선물 받았다. 하지만 태수의 위시리스트에 담겨있던 VPS는 $T$였기에, $S$의 두 문자를 바꾸는&nbsp;작업을 최대 $N$번 해서&nbsp;$T$로 만들고 싶었다. 하지만 VPS를 좋아하는 태수는 다음과 같은 조건들을 만족할 때만 두 원소를 바꾸기로 했다.</p>

<ul>
	<li>$1 \leq L &lt; R \leq N$을 만족하는 $L$, $R$에 대해&nbsp;$S[L]$과&nbsp;$S[R]$을 바꾼다고 할 때, 두 문자 사이에 있는 문자열도&nbsp;VPS여야한다. 즉, $R-L=1$을 만족하거나&nbsp;$S[L+1 ... R-1]$이 VPS여야한다. 또한,&nbsp;두 문자를 바꾼 뒤에도 $S$는 VPS여야 한다.</li>
</ul>

<p>태수를 위해 $S$를 $T$로 바꾸는 방법을 알려주자. 답이 반드시 존재함을 증명할 수 있다.</p>

### 입력

<p>첫째 줄에 테스트케이스의 개수 $C$가 주어진다. $(1 \leq C \leq 5 \times 10^5)$</p>

<p>각 테스트케이스 별로 입력은 다음과 같이 주어진다.</p>

<p>첫째 줄에 정수 $N$이 주어진다.&nbsp;$(1 \leq N \leq 10^6)$</p>

<p>둘째 줄에 길이가 $N$인 VPS $S$가 주어진다.</p>

<p>셋째 줄에 길이가 $N$인 VPS $T$가 주어진다.</p>

<p>모든 테스트케이스의 $N$의 합은 $10^6$을 넘지 않는다.</p>

### 출력

<p>각 테스트케이스 별로 다음과 같이 출력한다.</p>

<p>첫째 줄에 작업 횟수 $X$ $(0 \leq X \leq N)$를 출력한다.</p>

<p>둘째 줄부터 $X$줄에 걸쳐, 바꾸는 두 원소의 인덱스 $L$ $R$ 을 작업 순서에 맞춰 출력한다.</p>

<p><strong>반드시 작업 횟수가 최소가 될 필요는 없다.</strong></p>
