### 문제 설명

<p>함수 $f : \{1, 2, \cdots, N\} &rarr; \{1, 2, \cdots,&nbsp;N\}$의 각각의 함숫값 $f(1), f(2), \cdots, f(N)$이 주어진다. 이 때, 다음 쿼리를 수행하는 프로그램을 작성하시오.</p>

<ul>
	<li><code>1 x</code> : $f(1)$의 값을 $x$로 변경한다.</li>
	<li><code>2 m x</code> : $f^m (x)$의 값을 출력한다.</li>
</ul>

<p>여기서 $f^m$은 $f$를 $m$번 합성한 함수로, $f^1 (x)&nbsp;= f(x)$이고 $m \ge&nbsp;2$인 모든 자연수 $m$에 대해 $f^m (x) = (f \circ f^{m-1})(x) = f( f^{m-1} (x) ) $를 만족한다.</p>

### 입력

<p>첫째 줄에 자연수 $N$이 주어진다.</p>

<p>둘째 줄에는&nbsp;초기 상태의 $f(1), f(2), \cdots, f(N)$이 차례대로 주어진다.</p>

<p>셋째 줄에는&nbsp;쿼리의 개수 $Q$가 주어진다.</p>

<p>넷째 줄부터 $Q$개의 줄에는 쿼리가 한 줄에 하나씩 주어진다.</p>

### 출력

<p>2번 쿼리의 결과를 한 줄에 하나씩 출력한다.</p>

### 제한

<ul>
	<li>$1 \le N \le&nbsp;200\,000$</li>
	<li>$1 \le f(i) \le N $&nbsp;</li>
	<li>$ 1 \le&nbsp;Q \le&nbsp;500\,000 $</li>
	<li>1번 쿼리의 경우 $ 1 \le x \le N $&nbsp;</li>
	<li>2번 쿼리의 경우 $ 1 \le&nbsp;m \le 10^9$, $ 1 \le x \le N $&nbsp;</li>
</ul>
