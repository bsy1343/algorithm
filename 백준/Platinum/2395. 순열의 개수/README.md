# [Platinum V] 순열의 개수 - 2395

[문제 링크](https://www.acmicpc.net/problem/2395)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 128 MB

### 통계

제출: 170, 정답: 32, 맞힌 사람: 28, 정답 비율: 26.667%

다이나믹 프로그래밍, 누적 합

### 문제 설명

<p>A = (a<sub>1</sub>, a<sub>2</sub>, &hellip;, a<sub>n</sub>)을 n개의 원소를 가진 집합 X = {1,2,3,&hellip;,n}의 순열이라고 하자. 즉, i &ne; j이면, a<sub>i</sub> &ne;&nbsp;a<sub>j</sub>이고 (1 &le; i, j &le; n), 모든 i (1 &le; i &le; n)에 대해서 a<sub>i</sub>&isin;X이다. 순열 A에 대해서 수열 P(A) = (p<sub>1</sub>, p<sub>2</sub>, &hellip;, P<sub>n-1</sub>)을 다음과 같이 정의한다.</p>

<p>만약 a<sub>i</sub>&nbsp;&gt; a<sub>i+1</sub>이면, p<sub>i</sub>&nbsp;= 0이고, 그렇지 않으면 p<sub>i</sub> = 1이다. (1 &le; i &le; n-1). 문제는 n과 순열 A가 주어질 때, P(A) = P(B)를 만족하는 순열의 수를 구하여 출력하는 것이다.</p>

<p>예를 들어 A=(1,3,2,4)이라면, P(A) = (1,0,1)이다. 아래와 같은 순열 B에 대해서 P(A)=P(B)가 성립한다.</p>

<p>(1,3,2,4), (1,4,2,3), (2,3,1,4), (2,4,1,3), (3,4,1,2)</p>

### 입력

<p>첫째 줄에 집합 X의 크기 n이 주어진다. n은 5,000이하인 양의 정수이다. 둘째 줄에 X의 순열 하나가 주어진다. 순열은 n개의 정수로 나타내고, 이들 사이에 공백이 있다.</p>

### 출력

<p>첫째 줄에 주어진 순열 A에 대하여 P(A) = P(B)를 만족하는 순열의 수를 출력한다. 이때 &nbsp;A 자신도 순열의 수를 헤아릴 때 포함시킨다. 답이 매우 클 수 있으니 1,000,000,000으로 나눈 나머지만 출력한다.</p>
