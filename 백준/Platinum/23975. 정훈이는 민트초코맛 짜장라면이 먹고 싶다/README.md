### 문제 설명

<p>정훈이는 민트초코맛 짜장라면을 좋아한다.</p>

<p>그러나 민트초코맛 짜장라면을 찾는 사람은 정훈이밖에 없기 때문에&nbsp;친절한 정훈이는 편의점 사장님들을 위해&nbsp;자신이 언제, 어떤 편의점을 방문할지 미리 알려주려고 한다.</p>

<p>앞으로 <em>K</em>일 동안, 정훈이는 집에&nbsp;가는 길에 있는 편의점 중에서 재고가 있는 첫&nbsp;편의점에 방문해&nbsp;민트초코맛 짜장라면을 1개씩 구매할 예정이다.&nbsp;민트초코맛을&nbsp;좋아하는 정훈이도&nbsp;하루에 2개 이상 먹는 것은 힘들기 때문에 하루에&nbsp;구매하는 라면은&nbsp;한 개를 넘지 않는다.</p>

<p>편의점은 1번부터 <em>N</em>번까지 규모 순으로 번호가 매겨져 있으며, 정훈이네 집은 규모가 가장 작은 1번 편의점이 위치한 곳에 있다. 정훈이는 편의점을 잇는 <em>M</em>개의 양방향 길로만 다니면서 항상 최단 경로로 집에 간다.&nbsp;단, 최단 경로가 여러 개라면 바로 다음에 보이는&nbsp;편의점의&nbsp;규모가 더 큰 쪽을 선택한다.</p>

<p><em>K</em>일 동안 정훈이가 위치한 곳의 편의점 번호가 주어질 때, 어떤 편의점에서 민트초코맛 짜장라면을 구매할지 구하여라.</p>

### 입력

<p>첫 줄에&nbsp;<em>N</em>,&nbsp;<em>M</em>, <em>K</em>가 공백으로 구분되어 주어진다. (1 &le; <em>N,&nbsp;M,&nbsp;K</em> &le; 1,000,000)</p>

<p>둘째 줄에 각&nbsp;편의점에 남아있는 민트초코맛 짜장라면의 개수 <em>A<sub>i</sub></em>가 주어진다. (0 &le; <em>A</em><sub><em>i</em>&nbsp;</sub>&le; 100)</p>

<p>셋째 줄부터 <em>M</em>+2번째 줄까지 두&nbsp;편의점의 번호와 두 편의점을 잇는 길의&nbsp;거리인 정수&nbsp;<em>C</em>가&nbsp;주어진다. (1 &le; <em>C</em> &le; 1,000)&nbsp;</p>

<p><em>M</em>+3번째 줄부터 <em>M</em>+2+<em>K</em>번째 줄까지&nbsp;정훈이가 있는 곳의 편의점 번호가 주어진다.</p>

### 출력

<p><em>K</em>일 동안&nbsp;정훈이가 민트초코맛 짜장라면을 구매할 편의점 번호를 출력한다.</p>

<p>집으로 가는 길이 없거나 라면을 구매할 수 없다면 -1을 출력한다.</p>
