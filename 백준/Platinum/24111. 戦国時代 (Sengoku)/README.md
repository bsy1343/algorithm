### 문제 설명

<p>時は戦国時代のまっただ中．JOI 国を率いる JOI 武将は，来たるべき戦いに備え，自らの領地に見張りを 配備することになった．</p>

<p>JOI 国の領地は，東西幅、南北幅が L の正方形の形をしている．領地は 1 &times; 1 の正方形の区画に分かれて おり，各区画は 0 5 x &lt; L, 0 5 y &lt; L を満たす整数 x, y によって (x, y) と表現される．区画 (0, 0) は北西隅の 区画であり，区画 (x, y) は，区画 (0, 0) から東に x，南に y 進んだ地点に位置する区画である．</p>

<p>それぞれの見張りは領地内のある区画にとどまり，次のような範囲を見張る．(x, y) の位置にいる見張り は，条件 |x &minus; i| = |y &minus; j| をみたす領地内の区画 (i, j) 全体を見張る (この範囲は，他の見張りの位置によって 変化することはない)．なお，同一の区画に 2 人以上の見張りがいるということはない．</p>

<p>JOI 武将が配備した N 人の見張りの場所が与えられると，少なくとも 1 人以上の見張りに見張られてい る領地内の区画の個数を求めるプログラムを作成せよ．</p>

### 입력

<p>標準入力から以下の入力を読み込め．</p>

<ul>
	<li>1 行目には整数 L と N が空白を区切りとして書かれている．</li>
	<li>続く N 行は，1 行につき 1 人の見張りについて記述している．これらの行のうちの i 行目は i 番目の 見張りについて記述しており，整数 x<sub>i</sub> と y<sub>i</sub> (0 &le; x<sub>i</sub> &lt; L, 0 &le; y<sub>i</sub> &lt; L) が空白を区切りとして書かれて いる．</li>
</ul>

### 출력

<p>標準出力に以下のデータを出力せよ．</p>

<ul>
	<li>1 行目には，1 人以上の見張りによって見張られている区画の個数を表す，1 つの整数が含まれてい なければならない．</li>
</ul>

### 제한

<ul>
	<li>1 &le; L &le; 100, 000, 000 領地の 1 辺の長さ</li>
	<li>1 &le; N &le; 100, 000 見張りの数</li>
</ul>
