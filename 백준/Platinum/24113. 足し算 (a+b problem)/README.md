### 문제 설명

<p>2 つの整数の和を出力するプログラムを作成せよ．</p>

### 입력

<p>標準入力から以下の入力を読み込め．</p>

<p>入力では足し合わせる 2 つの整数が以下の形式で順に与えられる．</p>

<ul>
	<li>1 つ目の整数を表すデータの 1 行目には整数 m (1 &le; m &le; 20, 000) が書かれており, 続く m 行のうちの i 行目には，整数 a<sub>i</sub> (0 &le; a<sub>i</sub> &le; 9) と l<sub>i</sub> (1 &le; l<sub>i</sub> &le; 100, 000, 000) が空白を区切りとして書かれている．こ れらは，1 つ目の整数が十進表記で上の桁から a<sub>1</sub> が l<sub>1</sub> 個, a<sub>2</sub> が l<sub>2</sub> 個, &hellip;, a<sub>m</sub> が l<sub>m</sub> 個並んだ整数である ことを意味する．a<sub>1</sub> &ne; 0, a<sub>i</sub> &ne; a<sub>i+1</sub> (1 &le; i &le; m &minus; 1) である．</li>
	<li>1 つ目の整数を表すデータの直後には，2 つ目の整数を表すデータが同様の形式で続いている．</li>
</ul>

<p>L<sub>i</sub> たちの値は，必ずしも 32 bit に収まらない．long long などの 64-bit データ型を使用すべきである． 詳細については，技術情報を参照のこと．</p>

### 출력

<p>標準出力に以下のデータを出力せよ．</p>

<p>2 つの整数の和が，十進表記で上の桁から A<sub>1</sub> が L<sub>1</sub> 個，A<sub>2</sub> が L<sub>2</sub> 個，&hellip;，A<sub>M</sub> が L<sub>M</sub> 個 (0 &le; A<sub>i</sub> &le; 9, A<sub>1</sub> &ne; 0, A<sub>i</sub> &ne; A<sub>i+1</sub>, L<sub>i</sub> = 1) 並んだ整数であるとき，出力は M + 1 行からなり，出力の 1 行目には整数 M を，i + 1 行 目 (1 &le; i &le; M) には整数 A<sub>i</sub> と L<sub>i</sub> をこの順に空白区切りで出力せよ．</p>

### 힌트

<p>この場合，入力で与えられる2つの整数は111222233333と999988877である．これらの和は112222222210 なので，出力は上のようになる．</p>
