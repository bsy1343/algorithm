### 문제 설명

<p>1차원 세계에는 1차원 체스라는 게임이 존재한다. 1차원 세계에 사는 체스 마스터 무지는 다음 중요한 경기를 위해 기보 분석에 빠져있다.</p>

<p>1차원 체스의 기보는 1부터 10,000까지의 수를 가진 수열 $A$로 나타난다.</p>

<p>또한 두 개의 기보를 비교할 때 처음으로 값이 달라지는 위치가 존재한다면 그&nbsp;이전 위치 수를 기점이라고 한다. 즉 기보쌍 {$A$, $B$}에 대해 기점은 다음과 같다.</p>

<p style="text-align: center;"><strong>$ 2 \le K$ $\le$ $min$($|A|$,$|B|$) 인 $K$에 대해 $A_{i} = B_{i} $ $(1 \le i &lt; K)$이고 $A_{K}$ $\neq$ $B_{K}$ 이면 두 기보의 기점은 $A_{K-1}$로 정의된다.</strong></p>

<p>기보쌍의&nbsp;기점이 존재하지 않을 수도 있다.</p>

<p>무지는 $N$ 개의 기보를 가지고 있고 가능한 모든 기보쌍에 대해&nbsp;특정 수들이 몇 번이나 기점이 되는지 궁금하다. 무지를 도와주자.</p>

### 입력

<p>입력의 첫 줄에 기보의 개수 $N$과 무지가 궁금한 수의 개수 $Q$ 가 정수로&nbsp;주어진다. ($ 2 \le N \le 100,000 $, $ 1 \le Q \le 10,000 $)</p>

<p>두 번째 줄부터 $N+1$ 줄까지&nbsp;각 줄마다 기보의 크기를 나타내는 정수 $S$ 가 주어지고 $A_{1} \; A_{2} \, ... \,A_{S}$ 의 형태로 기보의 값이 정수로 주어진다. ($ 1 \,\le \,S$, $ 1 \le A_{i} \le 10,000 $)</p>

<p>$N+2$줄부터 $N+Q+1$ 줄까지 무지가 궁금한 수들이 각 줄마다 $Q_{i}$ 형태로 주어진다. ($ 1 \le Q_{i} \le 10,000 $, $Q_{i}$ 는 모두 서로 다르다)</p>

<p>전체 기보의 크기 합은 $1,000,000$ 을 넘지 않는다. ($\sum&nbsp;S \le 1,000,000$)</p>

### 출력

<p>무지가 궁금해하는 수들 $Q$개 각각에 대해, 모든 기보쌍을 비교했을 때 그 수가 몇 번이나 기점이 되는지 $Q$개의 줄에 나눠서 순서대로 출력하자.</p>
