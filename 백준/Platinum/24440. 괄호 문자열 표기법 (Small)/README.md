### 문제 설명

<p>괄호 문자열은 다음과 같은 3개의 조건만으로 만들 수 있는 문자열을 말한다.</p>

<ul>
	<li>빈 문자열은 괄호 문자열이다.</li>
	<li>괄호 문자열 <em>a</em>에 대해 <code>(</code><em>a</em><code>)</code>는 괄호 문자열이다.</li>
	<li>두 괄호 문자열 <em>a</em>, <em>b</em>에 대해 <em>a</em><em>b</em>는 괄호 문자열이다.</li>
</ul>

<p>괄호 문자열의 예시로는 <code>(())()</code>, <code>(((()())())())</code>, <code>((()())()()())(())</code> 등이 있다.</p>

<p>다음과 같은 방법으로 괄호 문자열이 나타내는 값을 정한다.</p>

<ul>
	<li>빈 문자열이 나타내는 값은 1이다.</li>
	<li>괄호 문자열 <em>a</em>가 나타내는 값이 <em>x</em>라 할 때, <code>(</code><em>a</em><code>)</code>가 나타내는 값은 <em>x</em> + 1이다.</li>
	<li>두 괄호 문자열 <em>a</em>, <em>b</em>가 나타내는 값이 각각 <em>x</em>, <em>y</em>라 할 때, <em>a</em><em>b</em>가 나타내는 값은 <em>x</em> &times; <em>y</em>이다.</li>
</ul>

<p>예를 들어 <code>(())()</code>는 6을 나타낸다.</p>

<ul>
	<li><code>()</code>는 1 + 1 = 2를 나타낸다.</li>
	<li><code>(())</code>는 2 + 1 = 3을 나타낸다.</li>
	<li><code>(())()</code>는 3 &times; 2 = 6을 나타낸다.</li>
</ul>

<p>자연수 <em>N</em>이 주어지면 <em>N</em>을 나타내는 가장 짧은 괄호 문자열을 구해보자. 단, 그러한 문자열이 여러 개 있으면 사전 순으로 가장 앞서는 것을 구한다.</p>

### 입력

<p>첫째 줄에 테스트 케이스의 개수 <em>T</em>가 주어진다. (1 &le; <em>T</em> &le; 20)</p>

<p>둘째 줄부터 각 테스트 케이스 별로 한 줄에 하나씩 자연수 <em>N</em>이 주어진다. (2 &le; N &le; 1,000,000 = 10<sup>6</sup>)</p>

### 출력

<p>첫째 줄부터 각 테스트 케이스 별로 한 줄에 하나씩 <em>N</em>을 나타내는 가장 짧은 괄호 문자열을 출력한다. 단, 그러한 문자열이 여러 개 있으면 사전 순으로 가장 앞서는 것을 출력한다.</p>
