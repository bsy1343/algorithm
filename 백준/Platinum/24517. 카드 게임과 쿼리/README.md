### 문제 설명

<p>swoon은 raararaara와 게임을 하고 있다. 게임의 내용은 다음과 같다. 정수 $A$와 $B$가 주어지며, $A$를 $B$ 이상으로 만드는 사람이 승리한다.</p>

<p>양 플레이어 모두가 사용 가능한 $K$개의 카드가 놓여있으며, 각 카드에는 $1$부터 $K$까지의 정수가 하나씩 적혀 있다. 플레이어는 자신의 차례에 카드를 한 장 선택해 해당 카드에 적혀있는 수만큼 $A$에 더해야 한다. 선택한 카드는 사라진다. 사용할 수 있는 카드가 없다면, 다시 $K$개의 카드를 놓는다.&nbsp;</p>

<p>게임은 항상 swoon부터 시작하며, 모든 플레이어는 최적의 방법으로 게임을 한다.</p>

<p>swoon은 이 게임을 무조건 이기고 싶기 때문에 $A$, $B$, $K$가 주어졌을 때, 승자가 누구인지 알고 싶다. swoon을 위해 승자를 알려주는 프로그램을 작성해보자.</p>

### 입력

<p>첫째 줄에 쿼리의 개수 $Q$ ($1 \leq Q \leq 100\,000$)가&nbsp;주어진다.</p>

<p>둘째 줄부터 $Q+1$번째 줄까지 각 줄마다&nbsp;$A$, $B$, $K$가 주어진다. ($1 \leq A \lt&nbsp;B \leq 10^9$, $1 \leq K \leq 10$)</p>

### 출력

<p>쿼리마다 한 줄에 swoon이 이기는 경우에는 <span style="color:#e74c3c;"><code>swoon</code></span>을, raararaara가 이기는 경우에는 <span style="color:#e74c3c;"><code>raararaara</code></span>를 출력한다.</p>
