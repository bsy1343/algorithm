### 문제 설명

<p>GGG 표기법은&nbsp;등차수열을&nbsp;포함한 특수한 형태의 수열을 간단히 표기하는 방법 중 하나다.&nbsp;GGG 표기법에 대한 정의는 다음과 같다.</p>

<ul>
	<li>GGG&lt;$a$&gt;는 모든 항의 값이 $a$인 수열을 의미한다.</li>
	<li>GGG&lt;$a$, $b$&gt;는 초항이 $a$, 공차가 $b$인 등차수열을 의미한다.</li>
	<li>GGG&lt;$a_0$, $a_1$, $\cdots$, $a_m$&gt;은 초항이 $a_0$, GGG&lt;$a_1$,&nbsp;$\cdots$, $a_m$&gt;을 계차수열로 가지는&nbsp;수열을 의미한다.</li>
</ul>

<p>주어진 $N$차 다항식 $f(x) = f_0 + f_1 x^1 + \cdots + f_N&nbsp;x^N$에 대해서 수열 $\{ a_n \}$을&nbsp;음이 아닌 정수 $i$에 대해서&nbsp;$a_i = f(i)$로 정의할 때,&nbsp;수열 $\{ a_n \}$을 간단하게 표기하기 위해서 GGG 표기법을 사용할 것이다. 예를 들어&nbsp;$f(x) = x^2 + 1$일 때, 수열 $\{ a_n \}$은&nbsp;초항이 $1$, 계차가 $2i&nbsp;+&nbsp;1$인 수열이며, 그에 따라서 GGG 표기법으로&nbsp;GGG&lt;$1$, $1$, $2$&gt;로 표기할 수 있다.</p>

<p>주어진 $N$차 다항식 $f(x)$에 대한 수열&nbsp;$\{ a_n \}$에 대한 GGG 표기법을 구해보자.</p>

### 입력

<p>첫 번째&nbsp;줄에 음이 아닌 정수 $N$이 주어진다.</p>

<p>두 번째 줄에 $f_0$, $\cdots$,&nbsp;$f_N$을&nbsp;나타내는 정수 $N+1$개가&nbsp;공백으로 구분되어 주어진다. ($0 \le f_i &lt; 10^9 + 7$, $0 \le i \le N$)</p>

<p>$N \ge 1$일 때, $f_N \ne 0$임을 보장한다.</p>

### 출력

<p>다항식 $f(x)$에 대한 수열 $\{ a_n \}$을 GGG 표기법으로 출력한다. 단, 각 입력값이 매우 커질 수 있으니 입력값을 $10^9 + 7$로 나눈 나머지로 출력하시오.&nbsp;답이 여러 개인 경우, GGG 표기법의 입력값의 개수가 가장 적은 것을 출력한다.</p>
