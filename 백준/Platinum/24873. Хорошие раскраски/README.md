### 문제 설명

<p>Назовем раскраску клеток таблицы $n \times m$ <em>хорошей</em>, если никакие четыре клетки, центры которых образуют вершины прямоугольника со сторонами, параллельными осям координат, не покрашены в один цвет.</p>

<p>Иначе говоря, для раскраски не должно быть четверки целых чисел $x_1, x_2, y_1, y_2$, что $1 \leq x_1 &lt; x_2 \leq n$, $1 \leq y_1 &lt; y_2 \leq m$, и клетки $(x_1, y_1)$, $(x_2, y_1)$, $(x_1, y_2)$ и $(x_2, y_2)$ покрашены в одинаковый цвет.</p>

<p>Требуется написать программу, которая по заданным целым числам $n$, $m$ и $c$ находит любую хорошую раскраску таблицы $n \times m$ в $c$ цветов.&nbsp;</p>

### 입력

<p>В первой строке записаны три целых числа $n, m, c$ ($2 \leq n, m \leq 10$, $2 \leq c \leq 3$).</p>

<p>Гарантируется, что для заданных во входных данных значений существует хотя бы одна хорошая раскраска.</p>

### 출력

<p>Выведите $n$ строк по $m$ чисел в каждой.&nbsp;</p>

<p>В качестве $j$-го числа $i$-й строки выведите $a_{i,j}$ --- цвет клетки $(i,j)$ ($1 \leq a_{i,j} \leq c$).</p>

<p>Если есть несколько хороших раскрасок, можно вывести любую из них.</p>
