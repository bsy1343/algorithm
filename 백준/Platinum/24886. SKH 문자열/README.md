### 문제 설명

<p>알파벳 대문자 &#39;S&#39;, &#39;K&#39;, &#39;H&#39;로 이루어진 길이 $N$의 문자열 $T$가 주어진다.</p>

<p>주어진 문자열 $T$의 임의의 위치에 &#39;S&#39;를 $p$개, &#39;K&#39;를 $q$개, &#39;H&#39;를 $r$개 삽입해서 만든 문자열에서 &quot;SKH&quot;의 최대 등장 횟수를 구하라.&nbsp;</p>

<p>&quot;SKH&quot;는 <strong>연결되어 있을 때</strong>&nbsp;한 번으로 센다.</p>

### 입력

<p>첫째 줄에 문자열의 길이 $N(1 \le N \le 100\,000)$, &#39;S&#39;를 삽입 가능한 횟수 $p$, &#39;K&#39;를 삽입 가능한 횟수 $q$, &#39;H&#39;를 삽입 가능한 횟수 $r$이 주어진다. $(0 \le p+q+r \le 100\,000)$</p>

<p>둘째 줄에 &#39;S&#39;, &#39;K&#39;, &#39;H&#39;로만 구성된&nbsp;문자열 $T$가 주어진다.</p>

### 출력

<p>&quot;SKH&quot;를 가장 많이 포함하도록 &#39;S&#39;를 $p$개, &#39;K&#39;를 $q$개, &#39;H&#39;를 $r$개 삽입해서 만든 문자열에서 &quot;SKH&quot;의 최대 등장 횟수를 출력하라.</p>
