### 문제 설명

<p>$N$차 다항식 $f(x) = a_Nx^N + \cdots + a_1x + a_0$와 소수($1$과 자기 자신으로만 나뉘어 떨어지는 수. $1$은 소수가 아니다.) $P$가 주어진다.</p>

<p>이 때, $f(0) \mod P$, $f(1) \mod P$, $\cdots$, $f(P-1) \mod P$를 모두 구하는 프로그램을 작성하라. 단, $u \mod v$는 $u$를 $v$로 나눈 나머지를 나타낸다.</p>

### 입력

<p>첫 번째 줄에 두 정수 $N$, $P$ ($0 &le; N &le; 10^6$, $1 &le; P &le; 10^3$, $P$는 소수)가 공백 하나로 구분되어 주어진다.</p>

<p>두 번째 줄에는 $N+1$개의 정수 $a_N$, $\cdots$, $a_1$, $a_0$ ($0 &le; a_i &le; 10^9$)가 공백 하나로 구분되어 주어진다.</p>

### 출력

<p>$P$개의 줄에 걸쳐 답을 출력한다. $i$번째 줄에는 $f(i-1) \mod P$를 나타내는 정수가 출력되어야 한다.</p>
