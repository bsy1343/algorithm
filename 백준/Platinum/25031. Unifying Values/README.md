### 문제 설명

<p>$N$개의 수로 이루어진 수열이 주어진다. 우리는 이 수열을 <strong>둘 이상</strong>의 연속한&nbsp;부분으로 나눈 다음 각 부분에 속하는 수들의 합을 구할 수 있다. 이 때, 합을&nbsp;구한 모든 수가 같아지도록 수열을 나누는 방법의 개수를 구하는 프로그램을&nbsp;작성하라.</p>

<p>예를 들어 수열이 $[4, -3, 1, 0, 1]$이라고 하자. $[4, -3]$, $[1]$, $[0, 1]$의 세 부분으로 나누거나, $[4, -3]$, $[1, 0]$, $[1]$의 세 부분으로 나누면 모든 부분의 합이 $1$이 된다. 이외의 방법은 없으므로, 이 입력에 대한 답은 $2$가 된다.</p>

### 입력

<p>첫 번째 줄에 하나의 정수 $N$($1 &le; N &le; 10^4$)이 주어진다.</p>

<p>두 번째 줄에는 수열에 포함된 $N$개의 정수가 공백 하나로 구분되어 순서대로&nbsp;주어진다. 각 정수는 $-10^{14}$이상 $10^{14}$이하의 정수다.</p>

### 출력

<p>수열을 나누는 방법의 개수를 출력한다. 이 수가 매우 클 수 있으므로, $1\,000\,000\,007$로 나눈 나머지를 출력하도록 한다.</p>
