### 문제 설명

<p>어떤 함수 \(f(x)\) 의 (\(x=0\)에서의) \(n\)차 테일러 다항식이란,</p>

<p>\[ T_n f(x) := \sum_{i=0}^{n} \frac{f^{(i)}(0)}{i!} x^i\]</p>

<p>를 말한다.</p>

<p>차수가 \(n\) 이하이고 상수항이 \(0\)인 다항식 \( P(x)\)가 \(a_0 + a_1x + \cdots + a_nx^n\)의 꼴로 주어진다. 두 함수 \(\ln(1+P(x)) \)와 \( e^{P(x)}-1 \)의 \(n\)차 테일러 다항식을 계산해 보자.</p>

### 입력

<p>첫째 줄에 정수 \(n\)이 주어진다. (\(1 &le; n &le; 512\))</p>

<p>둘째 줄에 \(n+1\)개의 정수 \(a_0, a_1, \cdots, a_n \)이 주어진다. (\( 0 &le; a_i &lt; 998244353, a_0 = 0 \) ) </p>

### 출력

<p>첫째 줄에 함수 \(\ln(1+P(x))\)의 \(n\)차 테일러 다항식의 계수를 의미하는 \(n+1\)개의 정수를 오름차순으로 출력한다.</p>

<p>둘째 줄에 함수 \(e^{P(x)}-1 \)의 \(n\)차 테일러 다항식의 계수를 의미하는 \(n+1\)개의 정수를 오름차순으로 출력한다.</p>

<p>구하는 계수는 모두 유리수이므로, 기약분수 \(\frac{a}{b}\)로 나타낼 수 있다. 이 때 정수 \(a \times b^{998244351}\mod 998244353\)를 출력하면 된다.</p>
