# [Platinum IV] 완전한 수열 리버스 - 26089

[문제 링크](https://www.acmicpc.net/problem/26089)

### 성능 요약

시간 제한: 1 초 (추가 시간 없음), 메모리 제한: 1024 MB (추가 메모리 없음)

### 통계

제출: 120, 정답: 58, 맞힌 사람: 50, 정답 비율: 52.632%

### 분류

해 구성하기, 많은 조건 분기

### 문제 설명

<p>다음 두 조건을 모두 만족하는 수열을 <strong>완전한 수열</strong>이라고 한다.</p>

<ul>
	<li>수열의 길이는 소수이다.</li>
	<li>수열의 합은 소수이다.</li>
</ul>

<p>어떤 수열의 연속 부분 수열 중, 완전한 수열의 개수를 수열의 <b>안정도</b>라고 한다.</p>

<p>다음 두 조건을 만족하는 길이 $N$의 수열 $a_1,a_2,\cdots,a_N$을 아무거나 하나 구해보자.</p>

<ul>
	<li>수열의 원소는 $2\,000$이하의 음이 아닌 정수이다.</li>
	<li>수열의 안정도는 $M$이다.</li>
</ul>

### 입력

<p>첫째 줄에 정수 $N$이 주어진다. $(4\leq N \leq 500)$</p>

<p>둘째 줄에 정수 $M$이 주어진다. $(0\leq M \leq N)$</p>

### 출력

<p>첫째 줄에 조건을 만족하는 수열 $a_1,a_2,\cdots,a_N$을 공백으로 구분하여 출력한다. </p>

<p>조건을 만족하는 수열은 항상 존재한다.</p>

### 힌트

<p>소수는 $2$ 이상의 양의 정수이면서 자기 자신과 $1$ 이외의 양의 정수로 나누어떨어지지 않는 수이다.</p>
