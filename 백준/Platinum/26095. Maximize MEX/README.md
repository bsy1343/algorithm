# [Platinum II] Maximize MEX - 26095

[문제 링크](https://www.acmicpc.net/problem/26095)

### 성능 요약

시간 제한: 1 초 (추가 시간 없음), 메모리 제한: 1024 MB (추가 메모리 없음)

### 통계

제출: 202, 정답: 72, 맞힌 사람: 62, 정답 비율: 35.429%

### 분류

애드 혹, 이분 탐색, 매개 변수 탐색

### 문제 설명

<p>정수를 원소로 갖는 중복을 허용하는 집합 $S$가 주어진다.</p>

<p>초기에 집합 $S$는 $a_1,a_2,\cdots,a_N$의 $N$개의 원소로 이루어져 있다.</p>

<p>이 집합에 다음과 같은 연산을 원하는 만큼 실행하여 집합 $S$에 원소를 단 하나 남기려고 한다.</p>

<ul>
	<li>집합 $S$의 공집합이 아닌 부분집합 $T$를 선택한다. $S$에서 $T$를 뺀 후, $T$에 존재하지 않는 가장 작은 음이 아닌 정수를 $S$에 넣는다. </li>
</ul>

<p>연산을 적절히 실행하여 $S$에 원소가 단 하나 있을 때, 남아있는 원소의 최댓값을 구해보자.</p>

### 입력

<p>첫째 줄에 정수 $N$이 주어진다. $(2\leq N \leq 300\,000)$</p>

<p>둘째 줄에 정수로 이루어진 수열 $a_1,a_2,\cdots,a_N$이 공백으로 구분되어 주어진다. $(0\leq a_i &lt; N)$</p>

### 출력

<p>첫째 줄에 연산을 적절히 실행하였을 때, $S$에 남아있는 원소의 최댓값을 출력한다.</p>
