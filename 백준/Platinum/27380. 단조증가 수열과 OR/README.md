### 문제 설명

<p>음이 아닌 정수로 이루어진 길이 $N-1$의 수열 $A_1, A_2, \cdots, A_{N-1}$이 주어진다.</p>

<p>다음 조건을 만족하는 음이 아닌 정수로 이루어진 길이 $N$의 수열 $B$가 존재하는지 판별하고, 존재한다면 아무 것이나 하나 출력하라.</p>

<ul>
	<li>$(B_i$ OR $B_{i+1}) = A_i$ $(1 \le i \le N-1)$</li>
	<li>$B_i \le B_{i+1}$ $(1 \le i \le N-1)$</li>
</ul>

<p>여기서 OR은 Bitwise OR 연산을 의미한다.</p>

### 입력

<p>첫 번째 줄에 수열 $B$의 길이 $N$이 주어진다.</p>

<p>그 다음 줄에 $N-1$개의 정수 $A_1, A_2, \cdots, A_{N-1}$이 공백으로 구분되어 주어진다.</p>

### 출력

<p>만약 조건을 만족하는 수열 $B$가 존재하지 않는다면 <span style="color:#e74c3c;"><code>No</code></span>를 출력한다.</p>

<p>조건을 만족하는 수열 $B$가 존재한다면 첫 번째 줄에 <span style="color:#e74c3c;"><code>Yes</code></span>를 출력하고 그 다음 줄에 수열 $B$의 각 원소를 순서대로 출력한다.</p>

### 제한

<ul>
	<li>$2 \le N \le 200\,000$</li>
	<li>$0 \le A_i &lt; 2^{60}$ $(1 \le i \le N-1)$</li>
</ul>
