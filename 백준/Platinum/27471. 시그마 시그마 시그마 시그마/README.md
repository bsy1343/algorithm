### 문제 설명

<p>이제 막 시그마를 배운 에릭은 시그마 여러 개를 이어붙이는 놀이에 빠져버렸다. 오늘도 놀이를 하던 에릭은 $2$와 $3$은 소수라서 불편하기 때문에 시그마를 $4$개 이어붙였다.</p>

<p>$N$개의 정수 $A_1, A_2, \cdots, A_N$이 주어질 때, 다음 값을 구하는 프로그램을 작성하시오.</p>

<p>$$\sum_{i=1}^{N-1} \sum_{j=i+1}^{N} \sum_{k=i}^{j-1} \sum_{l=k+1}^{j} \max(A_k, A_l)$$</p>

### 입력

<p>첫 번째 줄에 정수 $N$이 주어진다.</p>

<p>두 번째 줄에 $N$개의 정수 $A_1, A_2, \cdots, A_N$이 주어진다.</p>

### 출력

<p>문제의 정답을 $998\,244\,353$으로 나눈 나머지를 출력한다.</p>

### 제한

<ul>
	<li>$2 \le N \le 3 \times 10^5$</li>
	<li>$0 \le A_i \le 10^9$</li>
</ul>
