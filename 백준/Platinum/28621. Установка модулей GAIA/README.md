### 문제 설명

<p>Всего есть $n$ модулей системы <code>GAIA</code>, таких как <code>MINERVA</code>, <code>AETHER</code> и другие. Модули пронумерованы от $1$ до $n$, и для них есть ровно $n$ слотов для подключения их к <code>GAIA</code>. Изначально модуль номер $i$ подключен к слоту номер $i$.</p>

<p>Существуют только две операции, позволяющие оперировать назначением модулей <code>GAIA</code> по слотам. Эти операции могут быть описаны перестановками $p$ и $q$ длины $n$. В соответствии с операцией $p$, модуль, подключенный к слоту $p_i$, перемещается в слот $i$. Аналогично для $q$: при применении операции $q$, модуль, подключенный к слоту $q_i$, переподключается к слоту $i$.</p>

<p>Чтобы <code>GAIA</code> функционировала корректно, требуется назначить каждому модулю слот, используя <em>частичную композицию</em> операций $p$ и $q$. Это означает, что для каждого $i$ к слоту номер $i$ должен быть подключен</p>

<ul>
	<li>либо модуль, подключаемый к нему применением операции $p$;</li>
	<li>либо модуль, подключаемый к нему применением операции $p$, а затем операции $q$.</li>
</ul>

<p>Иными словами, к слоту номер $i$ может быть подключен либо модуль с номером $p_i$, либо модуль с номером $(q \circ p)_i = q_{p_i}$. Для каждого $i$ этот выбор можно сделать независимо от других.</p>

<p>Помимо этого, известны также $m$ системных ограничений вида &lt;&lt;модуль номер $a_i$ не может располагаться на соседнем слоте с модулем $b_i$&gt;&gt;.</p>

<p>Определите, существует ли частичная композиция перестановок $p$ и $q$, обеспечивающая корректное функционирование <code>GAIA</code>, то есть при которой</p>

<ul>
	<li>каждый модуль подключен к своему слоту, и каждый слот занят только одним модулем;</li>
	<li>и удовлетворены все ограничения на расположение модулей в соседних слотах.</li>
</ul>

### 입력

<p>В первой строке ввода через пробел даны два целых числа $n$ и $m$ --- количество модулей системы и количество ограничений ($1 \leqslant n \leqslant 2 \cdot 10^5$; $0 \leqslant m \leqslant 2 \cdot 10^5$).</p>

<p>Во второй и третьей строках через пробел перечислены элементы перестановок $p$ и $q$, описывающих операции ($1 \leqslant p_i, q_i \leqslant n$). Гарантируется, что каждое число от $1$ до $n$ встречается в описании каждой операции ровно один раз.</p>

<p>В следующих $m$ строках даны ограничения на расположение модулей: в $i$-й из них через пробел даны два целых числа $a_i$ и $b_i$ --- номера модулей, которые не должны располагаться в соседних слотах ($1 \leqslant a_i, b_i \leqslant n$; $a_i \neq b_i$).</p>

### 출력

<p>Если невозможно построить удовлетворяющее условию назначение, выведите &lt;&lt;<code>-1</code>&gt;&gt; (без кавычек).</p>

<p>Иначе выведите через пробел $n$ целых чисел, $i$-е из которых равно $1$, если для $i$-го слота выбрано назначение, соответствующее $p$, и $2$, если выбрано назначение, соответствующее $q \circ p$.</p>

<p>Если есть несколько подходящих вариантов назначений, выведите любой из них.</p>
