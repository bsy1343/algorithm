# [Platinum III] 검정 직사각형 - 2874

[문제 링크](https://www.acmicpc.net/problem/2874)

### 문제 설명

<p>상근이는 N행 N열로 이루어진 모눈종이를 구매했다. 모눈종이는 가장 처음에 모든 칸이 흰 색이다. 상근이는 이 칸 중 일부를 검정색으로 칠하려고 한다.</p>

<p>모눈종이의 경계선을 따라서 붙어있는 칸을 선택한다면, 직사각형을 만들 수 있다. 그리고 그 직사각형 내부의 색이 모두 검정색이고, 크기가 두 칸 이상이라면, 이 직사각형을 검정 직사각형이라고 한다.</p>

<p style="text-align: center;"><img alt="" src="%EB%B0%B1%EC%A4%80/Platinum/2874.%E2%80%85%EA%B2%80%EC%A0%95%E2%80%85%EC%A7%81%EC%82%AC%EA%B0%81%ED%98%95/168af41e.png" data-original-src="https://upload.acmicpc.net/eafafd37-75e2-48e9-a17a-f8868b7430f3/-/preview/" onerror="this.onerror=null;this.src=this.dataset.originalSrc" style="width: 397px; height: 162px;" /></p>

<p>왼쪽에서 선택한 직사각형은 모두 검정 직사각형이 아니다. 1번 직사각형은 내부의 색 중 한 칸이 흰색이고, 2번은 한 칸이기 때문에 검정 직사각형이 아니다. 오른쪽 그림에서 선택한 세 개 직사각형은 모두 검정 직사각형이다.</p>

<p>이렇게 상근이가 적절히 색칠한 모눈종이가 주어졌을 때, 겹치는 칸이 없게 검정 직사각형 두 개를 선택하는 방법의 수를 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 N이 주어진다. (2 &le; N &le; 1000) 다음 줄부터 N개 줄에는 각 모눈종이 칸의 색이 주어진다. C는 검정색, B는 흰색을 의미한다.</p>

### 출력

<p>첫째 줄에 겹치는 칸이 없게 두 검정 직사각형을 선택하는 방법의 수를 10007로 나눈 나머지 값을 출력한다.</p>
