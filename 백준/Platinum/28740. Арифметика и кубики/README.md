### 문제 설명

<p>У Авроры есть $n$ кубиков. У каждого кубика есть шесть сторон, на каждой из которых написана цифра от $0$ до $9$. Цифры на одном кубике могут повторяться.</p>

<p>Феи решили научить Аврору арифметике, и дали задание --- собирать из кубиков числа. Аврора может выбрать произвольный набор кубиков, повернуть каждый кубик из набора произвольной стороной вверх и расставить их в произвольном порядке, чтобы получить желаемое число. Конечно же, Аврора собирает число без ведущих нулей.</p>

<p>Теперь, чтобы Аврора научилась считать, феи хотят попросить её по-очереди складывать натуральные числа в порядке возрастания. Кубики, использованные для складывания одного числа, могут быть использованы и для складывания следующих чисел. Помогите феям определить минимальное натуральное число, которое Аврора не сможет сложить, используя данный набор кубиков.</p>

### 입력

<p>В первой строке дано целое число $n$ --- количество кубиков ($1 \le n \le 100\,000$).</p>

<p>В каждой из следующих $n$ строк дана строка из шести цифр $a_{i,1}, a_{i,2}, \ldots, a_{i,6}$ ($0 \le a_{i,j} \le 9$).</p>

### 출력

<p>Выведите наименьшее натуральное число, которое Аврора не сможет сложить.</p>
