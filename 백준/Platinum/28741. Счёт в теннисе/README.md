### 문제 설명

<p>Аврора и Нотграсс решили сыграть в теннис и попросили Флитл побыть судьёй. Изначально их счет равнялся $0 : 0$. Затем, несколько раз очки одного из игроков увеличивались на $1$. А закончилась игра со счётом $a : b$.</p>

<p>Фислвит было скучно, поэтому она считала сумму НОД-ов очков игроков после каждого изменения счёта. НОД --- наибольший общий делитель двух чисел. Например, игра могла проходить так:</p>

<ul>
	<li>$0 : 0$</li>
	<li>$1 : 0$, $\textrm{НОД}(1, 0) = 1$</li>
	<li>$2 : 0$, $\textrm{НОД}(2, 0) = 2$</li>
	<li>$2 : 1$, $\textrm{НОД}(2, 1) = 1$</li>
	<li>$2 : 2$, $\textrm{НОД}(2, 2) = 2$</li>
	<li>$2 : 3$, $\textrm{НОД}(2, 3) = 1$</li>
</ul>

<p>В таком случае, у Фислвит получилась бы сумма $1 + 2 + 1 + 2 + 1 = 7$.</p>

<p>После игры Фислвит стало интересно, какое наименьшее число могло у неё получиться. Помогите ей найти это значение.</p>

### 입력

<p>В единственной строке даны два целых числа $a$ и $b$ --- финальные очки Авроры и Нотграсс соответственно ($0 \le a, b \le 10^9$).</p>

### 출력

<p>Выведите единственное целое число --- минимальное значение, которое могло получиться у Фислвит.</p>
