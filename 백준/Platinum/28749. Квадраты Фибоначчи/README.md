### 문제 설명

<p>Сэму приходится много времени путешествовать пешком, и чтобы немного отвлечься от однообразного занятия, он решает в уме всякие задачки. Сегодня он размышлял над последовательностью чисел Фибоначчи. Она строится по следующему правилу:</p>

<ul>
	<li>$f_0 = f_1 = 1$</li>
	<li>$f_i = f_{i - 2} + f_{i - 1}$, для всех $i \ge 2$</li>
</ul>

<p>Он посчитал значение $\sum\limits_{i = 0}^{n} f_i^2$, и теперь просит вас сделать то же самое, чтобы сравнить ответ. Так как это число может быть большим, посчитайте его по модулю $998\,244\,353$.</p>

### 입력

<p>В единственной строке дано одно целое число $n$ ($0 \le n \le 10^{18}$).</p>

### 출력

<p>Выведите одно целое число --- ответ на задачу.</p>
