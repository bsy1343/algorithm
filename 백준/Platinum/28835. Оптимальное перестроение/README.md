### 문제 설명

<p>Сегодня Аквамен решил навести порядок в своем огромном отряде из рыб. Для этого он выстроил их перед собой и стал внимательно изучать.</p>

<p>Всего у Аквамена есть $n$ рыб, стоящих в ряд. Самой главной характеристикой рыбы является ее сила. Сила $i$-й рыбы от начала ряда равна $a_i$. Отряд рыб подобран идеально: сила каждой рыбы является целым числом от $1$ до $n$, а также ни у каких двух рыб силы не совпадают. Иными словами, силы рыб образуют перестановку.</p>

<p>Назовем <em>беспорядочностью</em> ряда из рыб количество пар рыб, в которых более сильная рыба стоит в ряду раньше, чем менее сильная. Иными словами, беспорядочность ряда --- это количество пар номеров рыб $i$ и $j$, для которых $i &lt; j$ и $a_i &gt; a_j$.</p>

<p>Аквамен может отдать одну команду перестроения отряду. Для этого он должен назвать целое число $x$ от $1$ до $n$. После этого рыбы образуют новый ряд, в котором сначала стоят рыбы с силой, меньшей $x$, потом стоит рыба с силой ровно $x$, а потом стоят рыбы с силой, большей $x$. При этом рыбы, сила которых меньше или больше, чем $x$, не меняют своего относительного порядка при перестроении. Например, если рыбы стояли в порядке $\{3, 6, 1, 4, 2, 5\}$, и Аквамен выбрал значение $x = 4$, то после перестроения рыбы будут стоять в порядке $\{3, 1, 2, 4, 6, 5\}$.</p>

<p>Аквамен хочет отдать одну команду перестроения, чтобы после выполнения этой команды беспорядочность ряда рыб была как можно меньше. Помогите ему определить, какое минимальное значение беспорядочности он может получить.</p>

### 입력

<p>Первая строка входных данных содержит единственное целое число $n$ --- количество рыб в ряду ($1 \le n \le 3\,000\,000$).</p>

<p>Вторая строка содержит $n$ различных целых чисел $a_1, a_2, \ldots, a_n$ --- значения сил рыб в ряду ($1 \le a_i \le n$, $a_i \neq a_j$ для всех $i \neq j$).</p>

### 출력

<p>Выведите единственное целое число --- минимальную беспорядочность, которую может иметь ряд из рыб после одной команды перестроения.</p>
