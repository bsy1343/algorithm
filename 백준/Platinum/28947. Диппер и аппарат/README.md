### 문제 설명

<p>Сегодня Диппер нашел на чердаке странный аппарат. Он содержит $n$ слотов, пронумерованных от $1$ до $n$ и расположенных подряд. В каждом слоте находится строка. Изначально во всех слотах находятся пустые строки. Аппарат может по заданным $l$, $r$ и $s$ добавить в конец всех строк, находящихся с $l$-го по $r$-й слот, строчку $s$.</p>

<p>Диппер поспорил с Мэйбл, что она не сможет смоделировать действия аппарата. Для этого он будет давать команды, а Мэйбл будет параллельно их повторять. Для проверки того, что Мэйбл безошибочно повторяет поведение аппарата, Диппер будет иногда спрашивать у Мэйбл следующий вопрос: чему равна подстрока с $x$ по $y$ в строке, находящейся в слоте $i$.</p>

<p>Мэйбл решила написать программу, которая будет моделировать этот процесс. Она просит вас помочь ей.</p>

### 입력

<p>В первой строке находятся два натуральных числа $n$, $m$ --- количество слотов в аппарате и количество команд Диппера ($1 \le n \le 2 \cdot 10^5$; $1 \le m \le 2 \cdot 10^5$). </p>

<p>В следующих $m$ строках находятся $m$ команд по одной в строке, каждая команда может быть одного из двух типов:</p>

<ul>
	<li>$1$ $l$ $r$ $s$ --- добавить в конец всех строк, находящихся с $l$-го по $r$-й слот включительно, непустую строку $s$ ($1 \le l \le r \le n$). Строка $s$ состоит из строчных латинских букв.</li>
	<li>$2$ $i$ $x$ $y$ --- узнать значение подстроки с $x$ по $y$ включительно у строки, находящейся в слоте $i$ ($1 \le i \le n$; $1 \le x \le y$). Гарантируется, что длина строки, находящейся в слоте под номером $i$, не меньше, чем $y$.</li>
</ul>

<p>Гарантируется, что сумма длин строк, входящих в команды первого типа, не превосходит $10^6$.</p>

<p>Гарантируется, что сумма длин подстрок по всем командам второго типа не превосходит $10^6$.</p>

### 출력

<p>На каждую команду второго типа нужно вывести ответ в отдельной строке.</p>
