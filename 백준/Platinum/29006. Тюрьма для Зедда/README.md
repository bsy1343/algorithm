### 문제 설명

<p>Черный рейнджер нашел в жерле заброшенного вулкана $n$ прямоугольных листов металла разного размера, $i$-й лист имеет размер $a_i$ на $b_i$ метров.</p>

<p>Зордон поручил построить Черному рейнджеру тюрьму для злобного генерала Зедда, и эта находка оказалась для него просто подарком судьбы. Тюрьма должна иметь вид прямоугольного параллелепипеда каждая грань которого представляет собой цельный лист металла. Так что рейнджер собирается взять 6 из найденных им листов металла, сложить из них параллелепипед, и показать его Зордону. Листы нельзя гнуть или разрезать, листы, из которых будет сложена тюрьма не должны выступать за её края.</p>

<p>Поскольку чем больше тюрьма, чем надежнее, параллелепипед должен иметь максимальный возможный объем. Помогите герою выбрать 6 листов прямоугольников так, чтобы собрать из них самую большую тюрьму.</p>

<p>Листы можно поворачивать, таким образом, например, листы 4 на 7 метров и 7 на 4 метра считаются одинаковыми.</p>

### 입력

<p>В первой строке дано одно число $n$ --- количество листов металла у Черного Рейнджера ($6 \le n \le 200\,000$).</p>

<p>В следующих $n$ строках даны пары чисел $a_i, b_i$ --- размеры $i$-го куска металла ($1 \le a_i, b_i \le 10^6$).</p>

### 출력

<p>Выведите одно целое число --- максимальный объем прямоугольного параллелепипеда, который можно собрать из этих кусков металла. </p>

<p>Если из данных листов невозможно собрать прямоугольный параллелепипед, выведите <code>-1</code>.</p>
