# [Platinum IV] マラソンを観よう - 29022

[문제 링크](https://www.acmicpc.net/problem/29022)

### 성능 요약

시간 제한: 8 초 (추가 시간 없음), 메모리 제한: 1024 MB

### 통계

제출: 16, 정답: 7, 맞힌 사람: 5, 정답 비율: 71.429%

### 분류

수학, 정수론, 확장 유클리드 호제법

### 문제 설명

<p>陸上競技ファンのあなたは，とあるマラソンの大会を現地で観戦することにした．この大会では，<i>XY</i> 平面上の原点と点 <i>(a, b)</i> を結ぶ線分上がマラソンコースとなる．また，大会中の交通規制のため，大会の観戦は，<i>X</i> 座標と <i>Y</i> 座標がともに整数である点のうち，マラソンコース上にない点のみで行うことができる．</p>

<p>あなたは，もちろん，できるだけマラソンコースに近い地点で大会を観戦したい．上述の条件を満たす点のうち，マラソンコースまでのユークリッド距離が最も近い点を求めよ．ただし，そのような点が複数ある場合には，その中で <i>X</i> 座標が最も小さい点を求めよ．さらにそのような点が複数ある場合には，その中で <i>Y</i> 座標が最も小さい点を求めよ．</p>

### 입력

<p>入力は <i>300,000</i> 個以下のデータセットからなる．各データセットは次の形式で表される．</p>

<blockquote><i>a</i> <i>b</i></blockquote>

<p><i>a</i>, <i>b</i> はともに <i>1</i> 以上 <i>10<sup>9</sup></i> 以下の整数であり，マラソンコースの一方の端点の座標を表す．</p>

<p>入力の終わりは 2 つのゼロからなる行で表される．</p>

### 출력

<p>各データセットに対し，あなたがこのマラソン大会を観戦するのに最適な地点の <i>X</i> 座標と <i>Y</i> 座標を，空白区切りで一行に出力せよ．</p>
