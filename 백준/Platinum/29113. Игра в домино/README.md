### 문제 설명

<p>Собравшись в очередной раз, чтобы поиграть в домино, Люди Икс осознали, что игра им порядком надоела. Они решили придумать что-нибудь новое. Как обычно, с гениальной идеей выступил Гамбит. Он предложил следующую незамысловатую игру: по данному набору доминошек надо уметь определять длину самой длинной цепочки.</p>

<p>Каждая доминошка представляет собой пару чисел $a$, $b$ --- количество точек на двух половинах доминошки. Цепочкой называется последовательность доминошек, которую можно выложить в линию так, что для любых двух соседних доминошек с номерами $i$, $i+1$ в этой линии верно следующее: $b_i=a_{i+1}$. Доминошки нельзя поворачивать и переворачивать.</p>

<p>Люди Икс пока не научились оптимально играть в эту игру, поэтому обратились к вам за помощью.</p>

### 입력

<p>В первой строке входного файла дано число $n$ ($1 \le n \le 100000$) --- количество доминошек. В следующих $n$ строках даны пары чисел $a_i, b_i$ ($0 \le b_i \le a_i \le 10^9$) --- описание доминошек.</p>

### 출력

<p>В единственной строке выходного файла выведите одно число --- максимальную длину цепочки из доминошек.</p>
