### 문제 설명

<p>Недавно на уроках математики мальчик Коля изучал тему &lt;&lt;делимость&gt;&gt; и, в частности, наибольшие общие делители. Напомним, что наибольшим общим делителем двух натуральных чисел $a$ и $b$ называется наибольшее натуральное число $g$ такое, что и $a$ делится на $g$, и $b$ делится на $g$. При этом используется обозначение: $g =  \mathrm{gcd}(a, b)$.</p>

<p>На дом учительница математики задала Коле несколько задач на эту тему и среди них была задача посчитать для некоторого числа $k$ следующую сумму: $\sum\limits_{i = 1}^k{\mathrm{gcd}(k, i)}$. Коля быстро решил эту задачу и его заинтересовало следующее обобщение этой суммы --- чему равна сумма таких сумм для всех делителей числа $n$. При этом он решил брать внутреннюю сумму не до делителя $k$, а до самого числа $n$, и получил следующую формулу: $\sum\limits_{d|n}{\sum\limits_{i = 1}^n{\mathrm{gcd}(d, i)}}$, где $d|n$ обозначает, что число $d$ является делителем числа $n$.</p>

<p>И тут Коля обнаружил, что подсчет значения такого выражения для больших $n$ может занять значительное время, и потому он попросил Вас написать программу, которая будет находить значение такого выражения для различных $n$.</p>

### 입력

<p>Входной файл содержит единственное натуральное число $n$ ($1 \le n \le 10^{12}$).</p>

### 출력

<p>В выходной файл выведите единственное число --- значение выражения для данного $n$.</p>
