# [Platinum III] 수 맞추기 게임 - 29633

[문제 링크](https://www.acmicpc.net/problem/29633)

### 분류

수학, 다이나믹 프로그래밍, 확률론, 임의 정밀도 / 큰 수 연산

### 문제 설명

<p><em>K</em>명의 학생이 선생님이 고른 수를 맞추는 게임을 한다. 게임이 진행되는 방식은 다음과 같다.</p>

<ol>
	<li>선생님은 1 이상 <em>N</em> 이하 범위의 랜덤한 정수 <em>X</em>를 고른다. 단, 학생들은 <em>N</em>의 값을 미리 알고 있다.</li>
	<li>1번 학생부터 한 명씩 차례대로 1 이상 <em>N </em>이하의 수 <em>Y</em>를 골라 질문한다. 만약 <em>X=Y</em> 이면 해당 학생의 승리이다. 그렇지 않은 경우, 선생님은 <em>Y </em>의 값 및<em> X</em>와<em> Y</em>의 대소 관계를 모든 학생들에게 공개한다.</li>
	<li><em>K</em>번 학생까지 질문을 완료한 경우, 다시 1번 학생의 차례로 돌아간다. <em>X</em>를 맞추는 학생이 나올 때 까지 게임을 진행한다.</li>
</ol>

<p>위 게임을 할 때, 각 학생들은 자신의 승리 확률을 가장 높이는 질문을 한다. 또한 그러한 질문의 선택지가 여러가지 있는 경우에는 그 중에서 uniform random하게 수를 골라 질문한다. 학생들이 이 전략을 사용할 것임을 이미 모든 학생들이 알고 있는 상태이다.</p>

<p>당신의 목표는 1번 학생이 첫 차례에서 질문할 수 있는 수의 후보가 무엇인지를 찾는 것이다.</p>

<p>양의 정수 <em>M</em>과 학생의 수 <em>K</em>가 주어졌을 때, <em>N</em> = 1, 2, ..., <em>M</em> 인 경우 각각에 대해 첫 차례에 1번 학생이 질문할 수의 후보들을 구하는 프로그램을 작성하라.</p>

### 입력

<p>첫 번째 줄에 양의 정수 <em>M</em> (2 &le; <em>M</em> &le; 200)이 주어진다.</p>

<p>두 번째 줄에는 학생의 수 <em>K</em> (2 &le; <em>K</em> &le; 50)이 주어진다.</p>

### 출력

<p><em>M</em>개의 줄에 걸쳐 정답을 출력한다.</p>

<p><em>i</em>번째 줄에는 <em>N</em> = <em>i</em>인 경우에 1번 학생의 첫 질문으로 가능한 답의 후보들을 오름차순으로 <em>P<sub>1</sub></em>, <em>P<sub>2</sub></em>, ..., <em>P<sub>k</sub></em>라 할 때, <em>k+1</em>개 수를 공백을 사이에 두고<em> </em> <em>k P<sub>1</sub></em> <em>P<sub>2</sub></em> ... <em>P<sub>k </sub></em>의 형식으로 출력한다.</p>
