### 문제 설명

<p>알파벳 대문자 &lsquo;<span style="color:#e74c3c;"><code>A</code></span>-<span style="color:#e74c3c;"><code>Z</code></span>&rsquo;, 알파벳 소문자 &lsquo;<span style="color:#e74c3c;"><code>a</code></span>-<span style="color:#e74c3c;"><code>z</code></span>&rsquo;, 숫자 &lsquo;<span style="color:#e74c3c;"><code>0</code></span>-<span style="color:#e74c3c;"><code>9</code></span>&rsquo;를 원소로 가지는 집합 $C$의 부분 집합 $P$를 정의역으로 하는 함수 $f(c)$가 있다. $f(c)$의 값은 $1$ 이상 $9$ 이하의 자연수이다.</p>

<p>$1$ 이상 $9$ 이하인 자연수 $i$에 대해 집합 $S_i$를 다음과 같이 정의하자. 즉, $S_i$는 집합 $P$의 원소 중, 함수 $f$에 대한 함숫값이 $i$가 되는 문자들의 집합이다.</p>

<p>$$S_i=\{c \in P | f(c)=i\}$$</p>

<p>$S_i$에 속하는 문자가 각각 주어질 때, 다음 함수 $g(x)$의 값이 $a$ 이상 $b$ 이하가 되도록 하는 서로 다른 $x$의 개수를 구하여 보자.</p>

<p>$$g(x)=\sum_{i=1}^{\lvert x \rvert}{f(x_i)}$$</p>

<p>단, $x$는 $P$에 속한 문자들을 이어 붙여 만든 길이가 $1$ 이상인 문자열이고, $x_i$는 $x$의 $i$번째 문자이다. 또, $\lvert x \rvert$는 문자열 $x$의 길이를 의미한다.</p>

### 입력

<p>첫째 줄에 정수 $a$와 $b$가 공백을 기준으로 주어진다. $(1 \leq a \leq b &le; 10^{18};$ $b - a \lt 5\,000\,000)$</p>

<p>둘째 줄부터 열 번째 줄까지 $S_i$의 모든 문자가 $(i + 1)$번째 줄에 공백 없이 주어진다.</p>

<p>집합 $S_i$의 원소의 개수는 $1$ 이상 $50$ 이하이다.</p>

<p>주어지는 모든 문자는 알파벳 대소문자, 숫자로 구성되어 있고 각 문자는 최대 한 번씩만 등장한다.</p>

### 출력

<p>서로 다른 $x$의 개수를 $1\,000\,000\,007(=10^9 + 7)$로 나눈 나머지를 출력한다. 이 수는 소수이다.</p>
