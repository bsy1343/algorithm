### 문제 설명

<p>Juku seisab l&otilde;pmatul ruudulisel paberil koordinaatide alguspunktis $(0;0)$. Osadele ruutudele on asetatud kastid, mis varjavad n&auml;htavust. Leida, kui suur osa horisondist on n&auml;htav.</p>

### 입력

<p>Tekstifaili esimesel real on kastide arv $N$ ($0 \le N \le 100\,000$) ja j&auml;rgmisel $N$ iga&uuml;hel kaks t&auml;isarvu $X$ ja $Y$ ($|X| \le 10^9$, $|Y| \le 10^9$): &uuml;he kasti vasaku alumise nurga koordinaadid. V&otilde;ib eeldada, et mingid kaks kasti pole samas ruudus.</p>

### 출력

<p>Tekstifaili ainsale reale v&auml;ljastada reaalarv $p$ ($0 \le p \le 1$), mis n&auml;itab horisondi Jukule n&auml;htava osa suhet kogu horisonti (t&auml;isringi). V&auml;ljastatud vastus ei tohi t&auml;psest v&auml;&auml;rtusest erineda rohkem kui $0{,}00001$ v&otilde;rra.</p>
