### 문제 설명

<p>상근이는 매일 친구들에게 이상한 문제를 낸다.</p>

<p>오늘은 &quot;각 자리의 합이 S가 되는 수가 구간 [A,B]에는 몇 개가 있을까? 또, 그러한 수 중에서 가장 작은 수는 뭘까?&quot;</p>

<p>창영이는 이 질문의 답을 찾느라 3일 연속으로 밤을 샜다. 창영이가 잠을 잘 수 있게 대신 상근이 질문의 정답을 찾는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 A, B, S가 주어진다. (1 &le; A &le; B &lt; 10<sup>15</sup>, 1 &le; S &le; 135)</p>

### 출력

<p>첫째 줄에는 구간 [A,B]에 속하는 수 중 각 자리의 합이 S가 되는 수의 개수를 출력한다. 이 값이 적어도 1인 구간과 S만 입력으로 주어진다.</p>

<p>둘째 줄에는 그러한 수 중에서 가장 작은 수를 출력한다.</p>
