### 문제 설명

<p>Ant lentos užra&scaron;ytas teigiamas sveikasis skaičius $N$. Reikia perstatyti jo skaitmenis tokia tvarka, kad gautas skaičius $M$ būtų dalus i&scaron; kuo didesnio dvejeto laipsnio.</p>

<p>Nei pradinis, nei naujas skaičius negali prasidėti nuliu.</p>

### 입력

<p>Pirmoje eilutėje pateikiamas skaičiaus $N$ skaitmenų kiekis $D$.</p>

<p>Antroje eilutėje pateikiamas pats skaičius $N$.</p>

### 출력

<p>Pirmoje eilutėje i&scaron;veskite skaičių $M$, sudarytą i&scaron; skaičiaus $N$ skaitmenų, sura&scaron;ytų tokia tvarka, kad $M$ dalintųsi i&scaron; kuo įmanoma didesnio dvejeto laipsnio.</p>

<p>Antroje eilutėje i&scaron;veskite didžiausią dvejeto laipsnį, kuris dalina $M$.</p>

<p>Jei yra keli teisingi sprendiniai, i&scaron;veskite bet kurį i&scaron; jų.</p>

### 제한

<ul>
	<li>1 &le; D &le; 16</li>
</ul>
