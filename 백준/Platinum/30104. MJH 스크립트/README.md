# [Platinum I] MJH 스크립트 - 30104

[문제 링크](https://www.acmicpc.net/problem/30104)

### 성능 요약

시간 제한: 1 초, 메모리 제한: 1024 MB

### 통계

제출: 46, 정답: 6, 맞힌 사람: 5, 정답 비율: 16.129%

### 분류

수학, 구현, 그래프 이론, 문자열, 시뮬레이션, 파싱, 분할 정복을 이용한 거듭제곱

### 문제 설명

<p>정후와 함께 MJH 스크립트를 배웁시다.</p>

<p style="text-align: center;">&lt;MJH 스크립트의 명세&gt;</p>

<ul>
	<li>프로그램은 <code>a</code>부터 <code>z</code>까지의 로마자 소문자 이름을 가지고 있는 26개의 변수를 이용합니다.</li>
	<li>각 변수의 초깃값은 0입니다.</li>
	<li>프로그램은 하나 이상의 줄로 이루어집니다.</li>
	<li>한 줄은 하나의 명령어로 이루어지며, 줄 바꿈으로 끝납니다.</li>
	<li>가장 위의 줄인 첫째 줄부터 실행하여, 마지막 줄의 다음 줄로 이동한 경우 프로그램을 종료합니다.</li>
	<li>명령어는 다음 중 하나입니다.
	<ul>
		<li><code>&lt;lvalue&gt; = &lt;rvalue_1&gt; + &lt;rvalue_2&gt; + ... + &lt;rvalue_<em>k</em>&gt;</code>: 변수 <code>&lt;lvalue&gt;</code>에 <code>&lt;rvalue_1&gt;</code>부터 <code>&lt;rvalue_k&gt;</code>까지의 값을 더한 값을 $1\,000\,000\,007$로 나눈 나머지를 대입한 후 다음 줄로 이동합니다. $k$는 1 이상의 정수입니다. <code>&lt;lvalue&gt;</code>는 로마자 소문자로, 변수의 이름입니다.</li>
		<li><code>GOTO</code> <code><em>x</em></code>: $x$째 줄로 이동합니다. $x$는 프로그램의 줄의 수보다 작거나 같은 양의 정수입니다. 다만, 전체 <code>GOTO</code> 명령어를 통틀어 <code>GOTO</code> 명령어에 $P$ 번째로 진입한 경우에는 무한 루프로 판단하여 프로그램을 즉시 종료합니다.</li>
		<li><code>PRINT &lt;rvalue_1&gt; + &lt;rvalue_2&gt; + ... + &lt;rvalue_<em>k</em>&gt;</code>: <code>&lt;rvalue_1&gt;</code>부터 <code>&lt;rvalue_<em>k</em>&gt;</code>까지의 값을 더한 값을 $1\,000\,000\,007$로 나눈 나머지를 출력한 후 다음 줄로 이동합니다. $k$는 1 이상의 정수입니다.</li>
	</ul>
	</li>
	<li><code>&lt;rvalue_<em>i</em>&gt;</code>는 다음 중 하나입니다.
	<ul>
		<li>$1\,000\,000\,007$ 미만의 음이 아닌 정수로, 그 값을 나타냅니다.</li>
		<li>로마자 소문자로, 그 이름을 가진 변수의 값을 나타냅니다.</li>
		<li>$1\,000\,000\,007$ 미만, 2 이상의 정수 하나와 로마자 소문자 하나를 연달아 쓴 것으로, 정수와 그 이름을 가진 변수의 값을 곱한 값을 나타냅니다.</li>
	</ul>
	</li>
</ul>

<p>MJH 스크립트를 이해했다는 것을 증명하기 위해, 주어진 프로그램을 실행해 봅시다.</p>

### 입력

<p>첫째 줄에 정수 $P$가 주어집니다.</p>

<p>그다음 줄부터 프로그램이 주어집니다. 불필요한 공백은 주어지지 않습니다.</p>

### 출력

<p><code>PRINT</code>의 값을 한 줄에 하나씩 출력합니다. 단, 그 횟수가 10회 이상일 경우 첫 5회와 마지막 5회만 차례로 출력합니다.</p>

<p>그다음 줄에 <code>a</code>부터 <code>z</code>까지의 변수의 값을 공백으로 구분하여 차례로 출력합니다.</p>

### 제한

<ul>
	<li>1 &le; P &le; 10<sup>12</sup></li>
	<li>주어진 프로그램은 유한한 시간 안에 종료됩니다.</li>
	<li>프로그램의 길이는 1,000,000을 넘지 않습니다. 프로그램의 길이에 줄 바꿈은 포함하지 않습니다.</li>
</ul>
