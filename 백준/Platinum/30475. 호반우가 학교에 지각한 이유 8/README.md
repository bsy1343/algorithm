# [Platinum II] 호반우가 학교에 지각한 이유 8 - 30475

[문제 링크](https://www.acmicpc.net/problem/30475)

### 문제 설명

<p>이제 호반우는 그리운 경북대학교로 돌아가려고 한다. 그러나 문제가 하나 있었으니, 지구에서 이세계로 오기는 쉬워도 이세계에서 지구로 돌아가는 건 어렵다는 것이다.</p>

<p>$2$차원 좌표평면으로 나타낼 수 있는 우주에는 이세계를 제외한 $N$개의 행성이 존재하며, $x$값이 작은 순서대로 $1$번 행성, $2$번 행성, ... , $N$번 행성이라고 부른다.</p>

<p>$1 &le; i &le; N$인 $i$에 대해 $i$번 행성은 $1$사분면 위에 위치한 정수 좌표인 $(x_{i},\,y_{i})$에 있고 양의 정수 쌍 $a_{i},\,b_{i}$를 가진다.</p>

<p>처음에 호반우는 이세계이자 $0$번 행성의 위치인 $(0,\,0)$에 있으며 여러 행성을 거쳐 지구로 돌아가는 차원문이 있는 $N$번 행성에 가려고 한다.</p>

<p>호반우는 한 번의 이동에 $M$개의 행성을 지나칠 수 있다. 정확히는, 호반우가 $s$번 행성에 있을 때 $e$번 행성으로 이동하려면 $s \le e \le s+M$을 만족해야 한다. 이때의 이동 시간은 $0$번부터 $e$번까지의 행성들을 모두 포함하는 <strong>가장 작은 볼록 다각형</strong>을 만들었을 때 $s$번 행성이 볼록 다각형 경계에 위치한다면 $a_{e}$, 그렇지 않다면 $b_{e}$가 소요된다. 직선도 볼록 다각형으로 생각한다.</p>

<p>이세계로부터 $N$번 행성까지 빠르게 도착해 호반우가 학교에 지각하지 않도록 도와주자!</p>

### 입력

<p>첫 번째 줄에 테스트 케이스의 개수 $T$가 주어진다. $(1 &le; T &le; 60\,000)$</p>

<p>각 테스트 케이스 별로 첫 번째 줄에 이세계를 제외한 행성의 개수인 $N$과 양의 정수 $M$이 공백을 두고 주어진다. $(1 &le; M &le; N &le; 300\,000)$</p>

<p>두 번째 줄부터 $N$개의 줄에 걸쳐 $x_{i},\,y_{i},\,a_{i},\,b_{i}$가 공백을 두고 주어진다. $(1 &le; x_{i},\,y_{i},\,a_{i},\,b_{i} &le; 10^{9})$</p>

<p>모든 테스트 케이스에 대하여 항상 $x_{1}&lt;x_{2}&lt;x_{3}\,&lt;\cdots&lt;\,x_{N-1}&lt;x_{N}$을 만족한다.</p>

<p>모든 테스트 케이스의 $N$의 합은 $300\,000$을 넘지 않는다.</p>

### 출력

<p>각 테스트 케이스 별로 호반우가 이세계에서 출발해 $N$번 행성에 도착하기까지 걸리는 시간의 최솟값을 출력한다.</p>
