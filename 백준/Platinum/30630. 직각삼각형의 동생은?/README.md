### 문제 설명

<blockquote>
<p>직각삼각형의 동생은 직각삼각이다!</p>
</blockquote>

<p>이 글에 깊은 감명을 받은 성우는 직각삼각형과 관련된 문제를 내기로 결심했다. 문제의 내용은 다음과 같다.</p>

<p>$2$차원 평면 위에 놓인 검은색 점이 $N$개 주어진다. 이 점들의 $x$좌표와 $y$좌표는 모두 양수이다. 이제 당신은 $M$개의 질문에 각각 답해야 한다.</p>

<ul>
	<li>각 질문마다 주어지는 점 $(a_i, b_i)$에 대해, 점 $(0, 0)$, 점 $(a_i, b_i)$, 점 $(a_i, 0)$을 각 꼭짓점으로 갖는 삼각형 내부에 존재하는 검은색 점의 개수가 몇 개인지 구해야 한다. 단, 삼각형의 경계에 속하는 점은 삼각형의 내부에 속하는 것으로 생각한다.</li>
</ul>

<p>하지만 성우는 풀이를 생각하기 너무 귀찮았고, 그만 잠들어 버렸다. 성우 대신 당신이 이 문제를 풀어주자!</p>

### 입력

<p>첫 번째 줄에 점의 개수 $N$이 주어진다. $(1 \leq N \leq 100\, 000)$</p>

<p>그 다음 $N$개의 줄에 각 검은색 점의 $x$좌표와 $y$좌표를 나타내는 두 정수 $x_i$, $y_i$가 공백을 사이에 두고 주어진다. 모든 검은색 점의 좌표는 서로 다르다. $(1 \leq x_i, y_i \leq 10^9)$</p>

<p>그 다음 줄에 질문의 개수 $M$이 주어진다. $(1 \leq M \leq 100\, 000)$</p>

<p>그 다음 $M$개의 줄에 두 정수 $a_i$, $b_i$가 공백을 사이에 두고 주어진다. $(1 \leq a_i, b_i \leq 10^9)$</p>

### 출력

<p>$M$줄에 걸쳐 각 질문에 대한 답을 출력한다. $i$번째 줄에는 $i$번째 질문에 대한 답을 출력한다.</p>
