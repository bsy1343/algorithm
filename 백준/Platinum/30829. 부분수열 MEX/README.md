### 문제 설명

<p>어떤 양의 정수 $n$에 대해 $n$에서 숫자 몇 개를 지워 만들 수 있는 모든 수들의 집합을 $A_n$이라고 하자. 예를 들어, 양의 정수 $1234$에서 $1$번째 숫자와 $3$번째 숫자를 지우면 $24$가 되므로 $24$는 $A_{1234}$에 포함된다. 단, $0$으로 시작하는 수는 없다고 가정한다.</p>

<p>양의 정수 $n$이 주어질 때 $A_n$에 포함되지 않은 양의 정수 중 가장 작은 수를 구해보자.</p>

### 입력

<p>정수 $n$이 주어진다. $(1 \leq n &lt; 10^{200000})$</p>

### 출력

<p>$A_n$에 포함되지 않은 양의 정수 중 가장 작은 수를 출력한다.</p>
