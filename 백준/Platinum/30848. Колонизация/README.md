### 문제 설명

<p>Давным-давно в далекой, далекой галактике на одной очень отдаленной планете стало слишком много людей, и они решили начать колонизацию. Не долго думая, они выбрали своей первой жертвой самый ближайший для них спутник &mdash; Луну.</p>

<p>На Луну на огромных шаттлах завезли N автономных модулей, чтобы поселить в них людей. Каждый модуль представляет собой купол из бионического стекла, имеющий форму круга радиуса R<sub>i</sub>. Но вдруг оказалось, что эти модули не совсем автономные: пришлось соединить их переходами, причем переходы можно построить только в случае, если купола стоят вплотную друг к другу (т.е. ограничивающие их окружности касаются внешним образом).</p>

<p>Итак, нашим космонавтам требуется построить лунную базу, то есть расставить модули так, чтобы между любыми двумя куполами был переход, и вычислить площадь получившейся базы. Но важно заметить, что площадью базы считается не только площадь под самими модулями, но и та область, в которую нельзя попасть, не проходя через купол.</p>

<p>Это заселение продолжалось многие годы, и затем, как и на старой планете, на Луне стало слишком мало места, поэтому люди решили достраивать базы. Они придумали в базу из трёх модулей добавлять четвёртый, если он достаточно мал, чтобы уместиться между тремя другими куполами. И естественно им необходимо построить новые переходы, для этого новый модуль должен стоять вплотную к трём другим. Но нужно знать радиус этого четвертого купола, чтобы потом построить его на специальной М-фабрике.</p>

<p>На рисунке окружностями схематично показаны модули, серым обозначена площадь базы, синими черточками &mdash; переходы между модулями, красным &mdash; искомый новый модуль для достройки.</p>

### 입력

<p>В первой строке входных данных содержится число N &mdash; количество куполов (1 ⩽ N ⩽ 1000). В следующей строке содержатся N целых чисел &mdash; радиусы куполов (каждый радиус является натуральным числом и не превосходит 1000). В случае N = 3 в последней строке может также содержаться слово &laquo;INSERT&raquo;.</p>

### 출력

<p>В случае, если построить базу требуемым образом невозможно, выведите &minus;1. В противном случае выведите искомую площадь. В случае, если N = 3 и в последней строке содержалось слово &laquo;INSERT&raquo;, то в следующей строке необходимо вывести искомый радиус четвертого купола. Ваш ответ будет считаться верным, если его абсолютная или относительная погрешность не будет превосходить 10<sup>&minus;6</sup>.</p>
