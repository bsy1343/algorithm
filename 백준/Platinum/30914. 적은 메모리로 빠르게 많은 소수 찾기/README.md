# [Platinum II] 적은 메모리로 빠르게 많은 소수 찾기 - 30914

### 문제 설명



### 입력

<p>첫 줄에 정수 $N$, $A$, $B$가 주어진다. ($2 \leq N \leq 5 \times 10^9$; $0 \leq B &lt; A$)</p>

<p>$N$ 이하의 소수가 $A$개 이상 $10^6 A$개 이하인 입력만 주어진다.</p>

### 출력

<p>$N$ 이하의 소수를 차례로 $p_0, p_1, p_2, \cdots$라고 할 때, 모든 $p_i$ 중 $i \equiv B\ (\text{mod}\ A)$인 것들을 차례로 한 줄에 하나씩 출력한다.</p>
