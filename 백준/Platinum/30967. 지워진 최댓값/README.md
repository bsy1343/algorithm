# [Platinum I] 지워진 최댓값 - 30967

[문제 링크](https://www.acmicpc.net/problem/30967)

### 성능 요약

시간 제한: 1 초, 메모리 제한: 512 MB

### 통계

제출: 38, 정답: 22, 맞힌 사람: 20, 정답 비율: 66.667%

### 분류

수학, 자료 구조, 세그먼트 트리, 집합과 맵, 조합론, 트리를 사용한 집합과 맵

### 문제 설명

<p>길이가 $N$인 순열 $A$가 주어진다. $1 \le i \le j &lt; k \le l \le N$인 모든 가능한 $(i, j, k, l)$에 대해, $A$에서 $[i, j]$ 구간과 $[k, l]$ 구간을 삭제했을 때의 최댓값의 합을 구하여라. 비어있는 배열의 최댓값은 $0$으로 가정한다.</p>

### 입력

<p>첫 번째 줄에 $N$이 주어진다. $(3 \le N \le 10^5)$</p>

<p>두 번째 줄에 순열 $A$를 이루는 정수 $N$개가 공백으로 구분되어 주어진다. $(1 \le A_i \le N;$ 모든 $A_i$는 서로 다르다.$)$</p>

### 출력

<p>답을 $10^9+7$로 나눈 나머지를 출력한다.</p>

### 힌트

<p>$A$의 $[s, e]$ 구간은 $A_s, A_{s+1}, \cdots, A_e$를 의미한다.</p>
