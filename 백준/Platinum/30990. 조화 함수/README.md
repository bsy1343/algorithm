### 문제 설명

<p>실수 전체를 정의역으로 갖고 모든 계수가 실수인 다항함수 $p(x)$와 $q(x)$에 대해, $p&#39;(q(x))=q&#39;(p(x))$일 경우 두 함수가 <strong>조화를 이룬다</strong>고 정의한다. 조건을 만족하지 않으면 두 함수는 조화를 이루지 않는다. 단, 다항함수 $u(x)$에 대하여 $u&#39;(x)$는 $u(x)$의 도함수를 의미한다.</p>

<p>실수 전체를 정의역으로 갖는 다항함수 $f(x)=a_{n}x^{n} + a_{n-1}x^{n-1} + \cdots +a_{0}$와 $g(x)=b_{m}x^{m} + b_{m-1}x^{m-1} + \cdots + b_{0}$가 주어진다. $f(x)$와 $g(x)$의 모든 계수는 정수이다.</p>

<p>한 번의 시행에서, $f(x)$와 조화를 이루고 모든 계수가 실수인 다항함수 $h(x)$를 선택하여, $f(x)$를 $h(x)$로 바꿀 수 있다. </p>

<p>유한 번의 시행을 통하여 $f(x)=g(x)$가 되게 할 수 있는지 판별하라.</p>

### 입력

<p>각각의 입력 데이터는 하나 이상의 테스트 케이스로 구성되어 있다.</p>

<p>첫 번째 줄에 테스트 케이스의 개수 $T$가 주어진다. 이후 $T$개의 테스트 케이스가 하나씩 주어진다.</p>

<p>각 테스트 케이스의 첫 번째 줄에 $f(x)$와 $g(x)$의 차수를 나타내는 두 정수 $n$과 $m$이 공백으로 구분되어 주어진다. 입력의 편의를 위해, 이 문제에서는 모든 상수함수의 차수를 $0$으로 간주한다.</p>

<p>각 테스트 케이스의 두 번째 줄에 $f(x)$의 각 항의 계수를 나타내는 $n+1$개의 정수 $a_{n},\,a_{n-1},\,\cdots,\,a_{0}$가 공백으로 구분되어 주어진다.</p>

<p>각 테스트 케이스의 세 번째 줄에 $g(x)$의 각 항의 계수를 나타내는 $m+1$개의 정수 $b_{m},\,b_{m-1},\,\cdots,\,b_{0}$가 공백으로 구분되어 주어진다.</p>

### 출력

<p>총 $T$개의 줄에 각 테스트 케이스의 정답을 출력한다.</p>

<p>각 테스트 케이스별로, 유한 번의 시행을 통하여 $f(x)=g(x)$가 되게 할 수 있으면 <span style="color:#e74c3c;"><code>YES</code></span>, 아니면 <span style="color:#e74c3c;"><code>NO</code></span>를 출력한다.</p>

### 제한

<ul>
	<li>$1 \le T \le 10^{4}$</li>
	<li>$0 \le n, m \le 4$</li>
	<li>$-10^{6} \le a_{i}, b_{i} \le 10^{6}$</li>
	<li>$n \neq 0$이면 $a_{n} \neq 0$</li>
	<li>$m \neq 0$이면 $b_{m} \neq 0$</li>
</ul>
