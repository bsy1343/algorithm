### 문제 설명

<p>동현이와 정후는 밤하늘을 보고 있다.</p>

<ul>
	<li>동현: 정후야, 저 밤하늘을 봐. 오리온자리야! 마치 길이가 $N$인 두 순열이 교차하는 것 같지 않니?</li>
	<li>정후: 뭐라고?</li>
	<li>동현: 길이가 $N$인 두 순열 $A_1, A_2, \cdots, A_N$와 $B_1, B_2, \cdots, B_N$을 생각해 보자. 수열 $A$에서 제일 앞 $i$개의 수를 고르고, $B$에서 제일 앞 $j$개의 수를 골라 일렬로 나열했을 때 길이 $i+j$의 순열이 되는 경우는 몇 가지일까? $i, j$의 범위는 $0$ 이상 $N$ 이하야.</li>
	<li>정후: 간단하지! 그건...</li>
</ul>

<p>...이라고 대답해 버렸다. 정후를 도와 동현이의 퀴즈를 풀어 주자. 단, 길이 $M$의 순열이란 $0$ 이상 $M$ 미만의 수가 정확히 한 번씩 등장하는 수열이다. 길이 $0$의 수열도 순열이다.</p>

### 입력

<p>첫 번째 줄에 수열의 길이 $N$이 주어진다. 두 번째 줄에 수열 $A$, 세 번째 줄에 수열 $B$가 주어진다. $0\leq A_i, B_i &lt; N$ 이며, $i\neq j$일 때, $A_i\neq A_j, B_i\neq B_j$ 이다.</p>

### 출력

<p>동현이의 퀴즈에 대한 답을 출력한다.</p>

### 제한

<ul>
	<li>$1\leq N\leq 5\times10^5$</li>
	<li>주어지는 모든 수는 정수이다.</li>
</ul>
