### 문제 설명

<p>설날을 맞아, 도훈이네 부대에서 축구 대회를 개최한다고 한다! 도훈이는 팀의 감독으로서, 팀에 소속될 $11$명의 선수를 선발하는 역할을 맡게 되었다.</p>

<p>팀에 선발할 선수는 각각 <strong>공격수</strong>, <strong>미드필더</strong>, <strong>수비수</strong>, <strong>골키퍼</strong>의 네 가지 포지션 중 하나로 등록되어야 하며, 각 포지션에는 반드시 최소 한 명씩은 선발해야 한다. 단, 골키퍼는 한 명만 선발해야 한다.</p>

<p>도훈이의 팀에 지원한 선수들은 지원서에 각각 공격수, 미드필더, 수비수, 골키퍼에 대한 선호도 $a_i$, $b_i$, $c_i$, $d_i$를 작성하여 제출했다. 팀의 만족도는 각 선수들이 선발된 포지션에 대한 선호도의 합이다. 도훈이는 해당 정보를 바탕으로, 팀 선발 조건에 맞게 팀의 만족도를 최대화하는 방식으로 선수들을 선발하고자 한다. 도훈이를 위해, 팀 선발 조건에 맞게 $11$명을 선발한 경우 팀의 만족도의 최댓값을 구해주자.</p>

### 입력

<p>첫 번째 줄에 부대 내 병사 수 $N$이 주어진다.</p>

<p>이후 $N$개의 줄에 걸쳐, $i+1$번째 줄에 $i$번째 병사의 공격수, 미드필더, 수비수, 골키퍼 포지션별 선호도 $a_i$, $b_i$, $c_i$, $d_i$가 공백으로 구분되어 주어진다.</p>

### 출력

<p>팀 선발 조건에 맞게 선수들을 선발했을 때 팀의 만족도의 최댓값을 출력한다.</p>

### 제한

<ul>
	<li>$11 \le N \le 200\,000$</li>
	<li>$1 \le a_i \le 10^8$</li>
	<li>$1 \le b_i \le 10^8$</li>
	<li>$1 \le c_i \le 10^8$</li>
	<li>$1 \le d_i \le 10^8$</li>
	<li>모든 입력은 정수이다.</li>
</ul>
