### 문제 설명

<p>KSA의 일부 학생들은 자신이 좋아하는 분야만 더 깊게 파려 하는 경향이 있다. 민우가 시험 기간 동안 공부해야 하는 과목 $N$개를 순서대로 $1, 2, \cdots, N$이라 번호를 붙이고 과목 $i$의 공부량을 음이 아닌 정수 $s_i$라 하자.</p>

<p>민우는 각 $i$에 대해, 과목 $i$를 적어도 과목 $A_i$만큼 공부하고 싶어 한다. 즉, $s_i \geq s_{A_i}$여야 한다. ($A_i$의 값이 $i$일 수 있다.)</p>

<p>민우는 총 학점이 $\sum_{i=1}^N B_i s_i$에 비례한다고 믿는다. 시험을 망치기도, 공부를 과하게 하기도 싫어하므로 $X \leq \sum_{i=1}^N B_i s_i \leq Y$가 성립하도록 하고 싶어 한다.</p>

<p>$N$, $X$, $Y$의 값과 $A_1, \cdots, A_N$, $B_1, \cdots, B_N$가 주어졌을 때, 위 조건을 만족하는 공부 계획의 수를 구하여라.</p>

<p>다시 말해서 다음 조건을 만족하는 수열 $s$의 개수를 구하여라.</p>

<ul>
	<li>수열의 길이는 $N$이며, 음이 아닌 정수로 구성된다.</li>
	<li>$s_i \geq s_{A_i}$이다.</li>
	<li>$X \leq \sum_{i=1}^N B_i s_i \leq Y$이다.</li>
</ul>

### 입력

<p>첫 번째 줄에 세 개의 정수 $N$, $X$, $Y$가 공백으로 구분되어 주어진다.</p>

<p>두 번째 줄에 $N$개의 정수 $A_1, A_2, \cdots, A_N$가 공백으로 구분되어 주어진다.</p>

<p>세 번째 줄에 $N$개의 정수 $B_1, B_2, \cdots, B_N$가 공백으로 구분되어 주어진다.</p>

### 출력

<p>문제의 정답을 $998\,244\,353$으로 나눈 나머지를 출력한다.</p>

### 제한

<ul>
	<li>$1 \leq N \leq 1000$</li>
	<li>$1 \leq X \leq Y \leq 10^5$</li>
	<li>$1 \leq A_i \leq N$</li>
	<li>$1 \leq B_i \leq 10^5$</li>
</ul>
