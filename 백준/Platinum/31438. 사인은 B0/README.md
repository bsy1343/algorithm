# [Platinum III] 사인은 B0 - 31438

시간 제한: 1 초, 메모리 제한: 1024 MB

### 통계

제출: 313, 정답: 70, 맞힌 사람: 49, 정답 비율: 28.488%

### 분류

수학, 그리디 알고리즘, 조합론

### 문제 설명

<p>수학2 기말을 망쳐 B0를 받은 한별이는 최근 개방된 KSA 예지관에서 &quot;사인은 B0&quot; 노래만 부르고 있다고 한다. </p>

<blockquote>
<p>&quot;아나타노 아이도루 사인와 B0!&quot;</p>
</blockquote>

<p>한별이는 노래의 박자에 맞춰 $(N+1)\times (N+1)$ 크기의 격자 위에서 댄스를 추기 시작한다.</p>

<p>정수 $A$, $B$, $C$, $N$이 주어졌을 때, $(Ax+By+C)^{N}$에서 $x^{i}y^{j}$의 계수를 $M_{i,j}$이라고 하자. 한별이는 처음에 $(0,0)$ 칸에서 시작하여 다음에 따라 이동할 수 있다. </p>

<ol>
	<li>만약 $M_{i,j+1}$이 $M_{i,j}$ 이상인 경우, 한별이는 $(i,j)$에서 $(i,j+1)$으로 이동할 수 있다. </li>
	<li>만약 $M_{i+1,j}$이 $M_{i,j}$ 이상인 경우, 한별이는 $(i,j)$에서 $(i+1,j)$으로 이동할 수 있다. </li>
</ol>

<p>이때 움직일 수 있는 최장 경로의 길이를 출력하자. 경로의 길이는 한별이가 움직인 횟수와 같다. </p>

### 입력

<p>입력은 하나 이상의 테스트 케이스로 이루어져 있다. 첫 번째 줄에 테스트 케이스의 개수 $T$가 주어진다. 각 테스트 케이스는 아래와 같이 주어진다. </p>

<p>각 테스트 케이스의 첫 번째 줄에 네 개의 정수 $A$, $B$, $C$, $N$이 공백으로 구분되어 주어진다.</p>

### 출력

<p>각 테스트 케이스에 대해 한별이가 격자 위에서 움직일 수 있는 최장 경로의 길이를 출력한다.</p>

### 제한

<ul>
	<li>$1 \leq T \leq 100$</li>
	<li>$1 \leq A, B, C, N \leq 10^{9}$</li>
</ul>
