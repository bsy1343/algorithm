### 문제 설명

<p>$N$개의 정점으로 이루어진 트리가 주어진다. 트리의 각 정점은 $1$번부터 $N$번까지 번호가 붙어 있고, 트리를 구성하는 간선 또한 주어지는 순서대로 $1$번부터 $N-1$번까지 번호가 붙어 있다. 각 간선에는 $W_i$의 가중치가 존재한다.</p>

<p>다음 두 가지 유형의 질의가 합해서 $Q$개 주어진다. 2번 쿼리가 주어질 때마다, 답을 출력하는 프로그램을 작성하라.</p>

<ul>
	<li><code>1 e v</code>: $e$번 간선의 가중치 $W_e$를 $W_e\ \And \ v$로 바꾼다. $\And$는 <a href="https://ko.wikipedia.org/wiki/%EB%B9%84%ED%8A%B8_%EC%97%B0%EC%82%B0#AND">Bitwise AND</a> 연산을 의미한다.</li>
	<li><code>2 n</code>: $\sum_{i=1}^{N}{dist(n, i)}$를 출력한다. $dist(a, b)$는 $a$번 정점과 $b$번 정점 사이를 연결하는 유일한 경로에 포함된 간선 가중치를 모두 <a href="https://ko.wikipedia.org/wiki/%EB%B9%84%ED%8A%B8_%EC%97%B0%EC%82%B0#OR">Bitwise OR</a> 연산한 값이다. $a = b$인 경우에는 $dist(a, b) = 0$ 이다.</li>
</ul>

### 입력

<p>첫째 줄에 $N$과 $Q$가 공백으로 구분되어 주어진다. $(2 \le N \le 300\ 000; 1 \le Q \le 300\ 000)$</p>

<p>다음 $N-1$개의 줄에는 트리의 간선 정보를 의미하는 $A_i, B_i, W_i$가 공백으로 구분되어 주어진다. 트리의 $i$번 간선이 연결하는 두 정점의 번호가 $A_i, B_i$이고, 간선의 초기 가중치는 $W_i$라는 의미이다. $(1 \le A_i &lt; B_i \le N; 0 \le W_i &lt; 2^{20})$</p>

<p>다음 $Q$개의 줄에는 아래와 같은 형식 중 하나로 쿼리에 대한 정보가 주어진다.</p>

<ul>
	<li><code>1 e v</code>: $(1 \le e &lt; N; 0 \le v &lt; 2^{20})$</li>
	<li><code>2 n</code>: $(1 \le n \le N)$</li>
</ul>

<p>입력에서 주어지는 수는 모두 정수이고, 2번 쿼리가 적어도 하나 이상 주어짐이 보장된다.</p>

### 출력

<p>2번 쿼리의 수행 결과를 쿼리가 주어진 순서대로 한 줄에 하나씩 출력한다.</p>
