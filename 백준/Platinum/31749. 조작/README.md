### 문제 설명

<p>지민이의 반에는 $N$명의 학생이 있습니다. 지민이의 반은 최근에 중간고사를 봤는데, 지민이는 우연히 각 학생들의 중간고사 점수를 입수하게 되었습니다. $i$번째 학생의 중간고사 성적은 $A_i$입니다.</p>

<p>지민이는 평등을 매우 중요시하기 때문에, 이 점수들을 <strong>조작</strong>하여 점수의 최댓값과 최솟값의 차이를 최소화하고자 합니다. 이때, 한 번의 <strong>조작</strong>은 다음과 같은 과정으로 이루어집니다.</p>

<ul>
	<li>임의의 정수 $k$와 $1\le i\le N-1$인 정수 $i$를 선택하여, $A_i$와 $A_{i+1}$에 $k$를 더한다.</li>
</ul>

<p>성적은 음수가 될 수도 있습니다. 지민이를 도와 학생들의 성적을 $10^6$회 이하로 <strong>조작</strong>하여 $\max A-\min A$를 최소화하는 프로그램을 작성하세요.</p>

### 입력

<p>첫째 줄에 학생의 수 $N$이 주어집니다.</p>

<p>둘째 줄에 학생들의 성적 $A_1$, $A_2$, $\cdots$, $A_N$이 띄어쓰기를 사이에 두고 주어집니다.</p>

### 출력

<p>첫째 줄에 <strong>조작</strong>을 통해 지민이가 달성할 수 있는 $\max A-\min A$의 최솟값을 출력합니다.</p>

<p>둘째 줄에 최솟값을 달성하기 위해 필요한 <strong>조작</strong>의 횟수 $M$을 출력합니다. <strong>조작의 횟수를 최소화할 필요는 없습니다.</strong></p>

<p>셋째 줄부터 $M+2$번째 줄까지 두 정수 $i$, $k$를 띄어쓰기를 사이에 두고 출력합니다. 이때 $x+2$번째 줄에 출력하는 $i$, $k$는 $x$번째 <strong>조작</strong>이 $A_i$와 $A_{i+1}$에 $k$를 더하는 시행이었음을 나타냅니다. $(1 \le x \le M)$</p>

### 제한

<p>입력은 다음 조건을 만족합니다.</p>

<ul>
	<li>$2 \le N \le 10^5$</li>
	<li>$0 \le A_i \le 10^9$ $(1 \le i \le N)$</li>
</ul>

<p>출력은 다음 조건을 만족해야 합니다.</p>

<ul>
	<li>$0 \le M \le 10^6$. $10^6$회 이하의 <strong>조작</strong>으로 $\max A-\min A$의 값을 최소화할 수 있음을 증명할 수 있습니다.</li>
	<li>모든 출력에서 $1 \le i \le N-1$.</li>
	<li>$M$번의 <strong>조작</strong>을 하는 도중, 그리고 모든 <strong>조작</strong>이 완료된 후 $1 \le j \le N$인 모든 정수 $j$에 대해 $-10^{18} \le A_j \le 10^{18}$을 만족해야 합니다.</li>
	<li>$M$번의 <strong>조작</strong>이 모두 끝난 후에는 $\max A-\min A$의 값이 첫째 줄에 출력한 값과 동일해야 합니다.</li>
</ul>

### 힌트

<p>$\max A$는 $A$의 원소들 중의 최댓값을, $\min A$는 $A$의 원소들 중의 최솟값을 뜻합니다.</p>
