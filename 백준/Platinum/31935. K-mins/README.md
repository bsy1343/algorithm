### 문제 설명

<p>길이 $N$의 수열 $A=[A_{1}, A_{2},\dots, A_{N}]$이 있다. $1\leq i &lt; j \leq N$을 만족하는 임의의 두 정수 $i$와 $j$에 대하여 $A_{i}\neq A_{j}$를 만족한다. $1\leq l \leq r \leq N$을 만족하는 두 정수 $l$과 $r$에 대하여 함수 $f(l,r)$을 다음과 같이 정의하자.</p>

<p style="text-align: center;">$f(l,r)=A_{l},A_{l+1},\dots,A_{r}$의 값 중에서 $K$번째로 작은 값</p>

<p>만약 구간의 길이를 나타내는 값 $r-l+1$이 $K$보다 작다면, $f(l,r)=0$으로 정의한다.</p>

<p>$\sum_{l=1}^{N}\sum_{r=l}^{N}f(l,r)$의 값을 구해보자.</p>

### 입력

<p>첫 번째 줄에 수열 $A$의 원소의 개수 $N$과 정수 $K$가 공백으로 구분되어 주어진다. $(1\leq N \leq 100\,000; 1\leq K\leq 10)$</p>

<p>두 번째 줄에 수열 $A$를 이루는 $N$개의 정수 $A_{1},A_{2},\dots,A_{N}$이 공백으로 구분되어 주어진다. $(-10^9\leq A_{i}\leq 10^{9})$</p>

### 출력

<p>$\sum_{l=1}^{N}\sum_{r=l}^{N}f(l,r)$의 값을 출력한다.</p>
