시간 제한: 5 초, 메모리 제한: 1024 MB

### 분류

그래프 이론, 자료 구조, 이분 탐색, 많은 조건 분기, 분리 집합, 매개 변수 탐색, 최소 스패닝 트리, 삼분 탐색

### 문제 설명

<p>정점이 $n$개이고 간선이 $m$개인 단순 연결 그래프가 주어집니다. 각 간선의 가중치 함수는 일차함수 $w_i(t)=a_it+b_i$ ($-10^3 \le a_i, b_i \le 10^3$)의 꼴로 구성되며, 가중치의 값은 이 함수에 매개변수 $t$를 대입한 값입니다. 따라서, 매개변수 $t$의 값에 따라 이 그래프의 최소 스패닝 트리가 달라질 수 있습니다. 어떤 스패닝 트리의 가중치는 매개변수 $t$에 어떤 값을 대입했을 때 그 스패닝 트리에 포함된 간선의 가중치의 합이며, 최소 스패닝 트리의 가중치는 모든 스패닝 트리의 가중치 중 최솟값입니다. 정수 $S$가 주어질 때, 그래프의 최소 스패닝 트리의 가중치 합이 $S$가 되도록 하는 매개변수 $t$가 존재하는지 판단하고, 존재한다면 그러한 매개변수 $t$의 값을 구하세요.</p>

### 입력

<p>첫 번째 줄에 정점의 개수 $n$과 간선의 개수 $m$이 공백으로 구분되어 주어집니다. ($2 \le n \le 10^5$, $n-1 \le m \le \min({n \choose 2},10^5)$)</p>

<p>그다음 줄부터 $m$개의 줄에 걸쳐 각각 간선을 나타내는 네 정수 $u_i$, $v_i$, $a_i$, $b_i$가 주어집니다. 이는 정점 $u_i$와 $v_i$를 연결하는 간선이 존재하며 가중치 함수는 $a_it+b_i$라는 의미입니다. ($1 \le u_i &lt; v_i \le n$, $-10^3 \le a_i, b_i \le 10^3$)</p>

<p>$m+2$번째 줄에는 최소 스패닝 트리의 가중치 합을 나타내는 $S$가 주어집니다. ($-10^{12} \le S \le 10^{12}$)</p>

<p>주어지는 그래프는 단순 연결 그래프임이 보장됩니다.</p>

### 출력

<p>최소 스패닝 트리의 가중치 합이 $S$가 되게 하는 매개변수 $t$가 존재한다면, 첫 번째 줄에 <span style="color:#e74c3c;"><code>YES</code></span>를 출력하고, 그러한 $t$를 두 번째 줄에 출력하세요. 각 간선의 가중치 함수에 매개변수 $t$를 대입하였을 때, 최소 스패닝 트리의 가중치 합과 $S$의 절대 오차 또는 상대 오차가 $10^{-5}$ 이하인 경우 정답으로 인정합니다.</p>

<p>그러한 $t$가 존재하지 않다면, 한 줄에 <span style="color:#e74c3c;"><code>NO</code></span>를 출력하세요.</p>

### 제한

<ul>
	<li>가중치 합이 $S$가 되게 하는 매개변수 $t$가 존재하지 않으면서 가중치 합과 $S$의 차가 $0.1$ 이하가 되게 하는 $t$가 존재하는 입력은 주어지지 않습니다.</li>
</ul>
