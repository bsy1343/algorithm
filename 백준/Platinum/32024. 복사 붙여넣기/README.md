# [Platinum I] 복사 붙여넣기 - 32024

[문제 링크](https://www.acmicpc.net/problem/32024)

### 성능 요약

시간 제한: 3 초 (추가 시간 없음), 메모리 제한: 1024 MB (추가 메모리 없음)

### 통계

제출: 30, 정답: 26, 맞힌 사람: 21, 정답 비율: 87.500%

### 분류

수학, 다이나믹 프로그래밍, 누적 합, 조합론

### 문제 설명

<p>할 일이 없어진 밍구는 복사 붙여넣기 놀이를 하고 있다! 복사 붙여넣기 놀이는 파일을 복사 붙여넣기하면서 생기는 이름을 살펴보는 놀이이다.</p>

<p>파일의 이름은 0 이상 2<sup><em>w</em></sup> 미만의 정수로 이루어진 길이 1 이상의 수열이다. 한 번의 복사 붙여넣기는 다음 절차로 이루어진다.</p>

<ul>
	<li>복사 과정: 폴더에 있는 모든 파일을 선택한다.</li>
	<li>붙여넣기 과정: 각 선택된 파일 <em>S</em>에 대해서 다음을 반복한다. 이때 새로 생성한 파일은 선택되지 않는다.
	<ul>
		<li><em>S</em>의 뒤에 0을 추가한 이름을 가진 파일을 생성한다. 단, 이미 같은 이름을 가진 파일이 있다면 생성하지 않는다.</li>
		<li>0 &le; <em>i</em> &lt; <em>w</em>인 모든 <em>i</em>에 대해, <em>S</em>의 마지막 원소에 2<sup><em>i</em></sup>를 XOR한 이름을 가진 파일을 생성한다. 단, 이미 같은 이름을 가진 파일이 있다면 생성하지 않는다.</li>
	</ul>
	</li>
	<li>붙여넣기 과정이 끝난 후, 모든 선택된 파일을 선택 해제한다.</li>
</ul>

<p>예를 들어, <em>w</em> = 2이고, 폴더에 파일이 [0] 하나만 있을 때 복사 붙여넣기를 반복하면 폴더의 내용물은 다음과 같이 변한다.</p>

<ul>
	<li>초기 상황: [0]</li>
	<li>1회: [0], [0, 0], [1], [2]</li>
	<li>2회: [0], [0, 0], [0, 0, 0], [0, 1], [0, 2], [1], [1, 0], [2], [2, 0], [3]</li>
</ul>

<p>복사 붙여넣기 고수 밍구는 여러분에게 다음과 같은 문제를 주었다. 폴더에 파일이 [0] 하나만 있는 상황에서 <em>K</em>번 복사 붙여넣기를 했을 때, 밍구가 주는 파일의 이름 <em>A</em>가 사전 순으로 몇 번째에 위치하는지 구하는 것이다!</p>

<p>밍구를 위해 <em>A</em>가 사전 순으로 위치하는 순서를 998&thinsp;244&thinsp;353으로 나눈 나머지를 구해주자!</p>

### 입력

<p>첫째 줄에 정수 <em>w</em>, <em>K</em>가 공백으로 구분되어 주어진다. (1 &le; <em>w</em> &le; 60; 1 &le; <em>K</em> &le; 100&thinsp;000)</p>

<p>둘째 줄에 찾고자 하는 파일의 이름 <em>A</em>의 길이 <em>N</em>이 주어진다. (1 &le; <em>N</em> &le; 100&thinsp;000)</p>

<p>셋째 줄에 <em>A</em>의 <em>i</em>번째 원소 <em>A</em><sub><em>i</em></sub>들이 공백으로 구분되어 주어진다. (1 &le; <em>i</em> &le; <em>N</em>; 0 &le; <em>A</em><sub><em>i </em></sub>&lt; 2<sup><em>w</em></sup>)</p>

<p>복사 붙여넣기를 <em>K</em>번 했을 때 <em>A</em>의 이름을 가지는 파일이 존재하는 경우만 입력으로 주어진다.</p>

### 출력

<p>사전순으로 가장 앞서는 파일의 순서를 1이라고 할 때, <em>A</em>가 사전순으로 위치하는 순서를 998&thinsp;244&thinsp;353으로 나눈 나머지를 출력한다.</p>

### 힌트

<p>수열 <em>a</em> = [<em>a</em><sub>1</sub>, <em>a</em><sub>2</sub>, ⋯, <em>a</em><sub><em>n</em></sub>]이 수열 <em>b</em> = [<em>b</em><sub>1</sub>, <em>b</em><sub>2</sub>, ⋯, <em>b</em><sub><em>m</em></sub>]보다 사전순으로 앞선다는 것은 다음을 모두 만족시키는 양의 정수 <em>i</em>가 존재한다는 뜻이다.</p>

<ul>
	<li><em>i</em>보다 작은 모든 양의 정수 <em>j</em>에 대해, <em>a</em><sub><em>j</em></sub>와 <em>b</em><sub><em>j</em></sub> 모두 존재하면서 <em>a</em><sub><em>j</em></sub> = <em>b</em><sub><em>j</em></sub></li>
	<li><em>b</em><sub><em>i</em></sub>가 존재함</li>
	<li><em>a</em><sub><em>i</em></sub>가 존재하지 않거나 <em>a</em><sub><em>i</em></sub> &lt; <em>b</em><sub><em>i</em></sub></li>
</ul>
