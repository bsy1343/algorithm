# [Platinum IV] 過去問の共有 - 32092

[문제 링크](https://www.acmicpc.net/problem/32092)

### 성능 요약

시간 제한: 8 초 (추가 시간 없음), 메모리 제한: 1024 MB

### 통계

제출: 12, 정답: 11, 맞힌 사람: 9, 정답 비율: 90.000%

### 분류

수학, 다이나믹 프로그래밍, 비트마스킹, 확률론, 비트필드를 이용한 다이나믹 프로그래밍, 기댓값의 선형성

### 문제 설명

<p>JAG 大学 ICPC 学科には <i>N</i> 人の学生が在籍しており，それぞれの学生には <i>1</i> から <i>N</i> までの番号がついている．また，学生の交友関係が <i>M</i> 個存在する．<i>i</i> 番目の交友関係は，学生 <i>a<sub>i</sub></i> と <i>b<sub>i</sub></i> が友達であり，互いに連絡できることを表す．</p>

<p>学科の期末試験では，<i>N</i> 教科の試験が行われる予定である．はじめ，学生 <i>i</i> は教科 <i>i</i> の過去問を持っている． あなたは学生 1 である．友達同士で過去問を共有することを繰り返したとき，あなたが過去問の情報を何教科得られるかが気になった．そこで，友達同士で行われる過去問の共有が <i>K</i> 回行われたときの期待値を調べることにした．</p>

<p>次のイベントが順に <i>K</i> 回独立に発生することを考える．</p>

<ul>
	<li><i>M</i> 個の交友関係のうち一つがランダムに選ばれ，その学生二人が連絡を取りあう．このとき，それぞれが持つ過去問情報をすべて共有しあう．

	<ul>
		<li>言い換えると，片方が持つ過去問の教科の集合を <i>X</i>，もう片方を <i>Y</i> とすると，そのイベントの後，持っている過去問の教科の集合は双方とも <i>X &cup; Y</i> になる．</li>
	</ul>
	</li>
</ul>

<p><i>K</i> 回のイベントの終了後，学生 1 であるあなたが持っている過去問の教科数の期待値を <i>mod 998244353</i> で求めよ．</p>

<p>なお，この問題において，求める期待値は必ず有理数になることが証明できる．また，この問題の制約のもとでは，その値を既約分数 <i>P/Q</i> で表したとき，<i>Q ≢ 0 (mod 998244353)</i> となることも証明できる．よって，<i>R &times; Q &equiv; P (mod 998244353), 0 &le; R &lt; 998244353</i> を満たす整数 <i>R</i> が一意に定まる．この <i>R</i> が，期待値 <i>mod 998244353</i> である．ここで，<i>998244353 = 2<sup>23</sup> &times; 7 &times; 17 + 1</i> は素数である．</p>

### 입력

<p>入力は複数のデータセットからなる．データセットの個数は 50 を超えない．各データセットは次の形式で表される．</p>

<blockquote>
<p><i>N</i> <i>M</i> <i>K</i></p>

<p><i>a<sub>1</sub></i> <i>b<sub>1</sub></i></p>

<p><i>⋮</i></p>

<p><i>a<sub>M</sub></i> <i>b<sub>M</sub></i></p>
</blockquote>

<p>1 行目には <i>N</i>，<i>M</i> および <i>K</i> が与えられる．<i>N</i> は学生の人数であり，<i>2</i> 以上 <i>10</i> 以下の整数である．<i>M</i> は交友関係の個数であり，<i>1</i> 以上 <i>N(N-1)/2</i> 以下の整数である．<i>K</i> はイベントが発生する回数であり，<i>1</i> 以上 <i>50</i> 以下の整数である．</p>

<p>続く <i>M</i> 行には，交友関係の情報が与えられる．このうち <i>i</i> 番目の行には <i>a<sub>i</sub></i> および <i>b<sub>i</sub></i> が与えられ，それぞれ <i>1</i> 以上 <i>N</i> 以下の整数である．自分自身に交友関係があるような入力は与えられない．すなわち，すべての <i>1 &le; i &le; M</i> について <i>a<sub>i</sub> &ne; b<sub>i</sub></i> を満たす．また，同じ交友関係が複数与えられることはない．すなわち，すべての <i>1 &le; i, j &le; M</i> について，<i>i &ne; j</i> ならば <i>{ a<sub>i</sub>, b<sub>i</sub> } &ne; { a<sub>j</sub>, b<sub>j</sub> }</i> を満たす．</p>

<p>入力の終わりは 3 つのゼロからなる行で表される．</p>

### 출력

<p>各データセットについて，答えを 1 行で出力せよ．</p>
