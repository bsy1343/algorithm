### 문제 설명

<p>은호는 강한 연결 요소에 대해서 공부하였다. 이를 본 성현이는 영감을 얻어 여러 점들을 많은 선분으로 이은 <strong><em>매우 강한 연결 요소</em></strong>라는 것을 만들었다.</p>

<p>평면상의 서로 다른 점 $N$개가 주어질 때, 각 점들이 정점, 해당 점들을 양 끝점으로 갖는 선분들이 간선이 되는 그래프를 만들 수 있다. 여기서 어떤 두 선분도 그 양 끝점을 제외하고 만나지 않는 그래프 중, 가장 많은 선분을 사용한 그래프를 주어진 점들에 대한 <em><strong>매우 강한 연결 요소</strong></em>라고 한다. <em><strong>매우 강한 연결 요소</strong></em>가 가지는 간선의 개수를 <em><strong>매우 강한 연결 요소</strong></em>의 세기라고 한다.</p>

<p>성현이를 보던 은호는 한 가지 궁금증이 생겼다. $N$개의 점의 <strong><em>매우 강한 연결 요소</em></strong>의 세기는 얼마일까? 은호는 이 문제의 답을 찾았지만, 한자 공부를 하기 위해 안 알려주고 떠나버렸다. 그러니 여러분이 대신 구해보자.</p>

<p>평면상의 서로 다른 $N$개의 점이 주어질 때, 이들의 <strong><em>매우 강한 연결 요소</em></strong>의 세기를 출력하여라.</p>

### 입력

<p>첫 번째 줄에 양의 정수 $N$이 주어진다.</p>

<p>두 번째 줄부터 $N$개의 줄 중 $i$번째 줄에 $i$번째 점의 좌표 $x_i$와 $y_i$가 공백으로 구분되어 주어진다.</p>

<p>모든 점의 좌푯값은 정수이며, 어느 두 점도 같은 좌표를 가지고 있지 않다.</p>

### 출력

<p>첫 번째 줄에 문제의 답을 출력하여라.</p>

### 제한

<ul>
	<li>$1 \leq N \leq 200\,000$</li>
	<li>$-10^9 \leq x_i, y_i \leq 10^9$ ($1 \leq i \leq N$)</li>
	<li>$i \neq j \Rightarrow (x_i,y_i) \neq (x_j,y_j)$ ($1 \leq i,j \leq N$)</li>
</ul>
