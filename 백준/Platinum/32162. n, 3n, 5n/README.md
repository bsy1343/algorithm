### 문제 설명

<p>서로 다른 양의 정수로 구성된 증가하는 무한 수열 $a_i$는 다음의 성질을 충족하는 사전순으로 최소인 수열이다.</p>

<ul>
	<li>임의의 양의 정수 $n$에 대해, $n$, $3n$, $5n$ 중 정확히 하나의 수가 수열에 등장한다.</li>
</ul>

<p>양의 정수 $i$가 주어질 때, $a_i$의 값을 구하시오.</p>

### 입력

<p>첫 번째 줄에는 테스트 케이스의 개수 $T$가 주어진다. ($1\le T\le 100\, 000$)</p>

<p>그다음 $T$줄에 걸쳐서, 각 줄에 $i$의 값이 하나씩 주어진다. ($1\le i\le 100\, 000$)</p>

### 출력

<p>$T$줄에 걸쳐서, 각각의 $i$에 대한 $a_i$의 값을 한 줄에 출력한다.</p>
