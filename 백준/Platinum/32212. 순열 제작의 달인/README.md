### 문제 설명

<p><a href="/problem/31419">배열 제작의 달인</a> Jiro는 길이 $N$인 수열 $A$가 주어졌을 때, 다음 조건을 만족하는 길이 $N$의 순열 $P$의 개수를 구하고자 한다. 길이 $N$의 순열 $P$는 $1$부터 $N$까지의 모든 정수를 한번만 사용하여 만든 수열을 의미한다.</p>

<ul>
	<li>수열 $B$는 $(A_{P_1},A_{P_2},...,A_{P_N})$로 정의한다.</li>
	<li>수열 $B$의 <strong>가증스러움</strong>은 $1 \le i \le N$인 정수 $i$에 대해 다음 조건을 만족하는 $i$의 개수로 정의한다.
	<ul>
		<li>$i=1$ 또는 $\max(B_1,...,B_{i-1}) &lt; B_i$</li>
	</ul>
	</li>
	<li>수열 $B$의 <strong>가증스러움</strong>은 $K$ 이하이다.</li>
</ul>

<p>Jiro가 조건을 만족하도록 구한 순열 $P$의 개수를 $1\,000\,000\,007$으로 나눈 나머지를 구하시오.</p>

### 입력

<p>첫째 줄에 수열 $A$의 길이 $N$와 조건에 명시된 $K$가 공백으로 구분되어 주어진다. $(1 \le N \le 5\,000$; $1 \le K \le N)$</p>

<p>둘째 줄에 수열 $A_1$, $A_2$, $\cdots$, $A_N$이 공백으로 구분되어 주어진다. $(1 \leq A_i \leq 10^9)$</p>

### 출력

<p>Jiro가 조건을 만족하도록 구한 순열 $P$의 개수를 $1\,000\,000\,007$으로 나눈 나머지를 출력한다.</p>
