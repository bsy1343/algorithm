### 문제 설명

<p><strong>카드 뒤집기 1, 2는 서로 다른 문제다.</strong></p>

<p>준원이는 카드 $N$장을 가지고 있다. 카드의 앞면에는 정수 $1,2,\cdots ,N$이 하나씩 적혀 있고, 카드의 뒷면에는 모두 정수 $0$이 적혀있다. 준원이는 카드의 순서를 마음대로 바꾼 뒤 $N$장의 카드를 윗면이 천장을 향하도록 나열했다. 나열된 카드에서 천장을 향하는 면에 적힌 정수는 $p_1,p_2,\cdots ,p_N$이 되었다.</p>

<p>준원이는 $i$를 $1$부터 $N$까지 $1$씩 증가시키며 다음과 같은 행동을 반복한다.</p>

<ul>
	<li>$i$번째 카드와 $j(i&lt;j\le N)$번째 카드를 비교한다. $i,j$번째 카드의 천장을 향하는 면에 적힌 정수를 $X,Y$라고 할 때, $X&gt;Y$를 만족하는 모든 $j$번째 카드를 뒤집는다.</li>
</ul>

<p>모든 행동을 끝낸 후, 각 카드의 천장을 향하는 면에 적힌 정수를 구해보자.</p>

### 입력

<p>첫째 줄에 정수 $N(1\le N\le 200\, 000)$이 주어진다.</p>

<p>둘째 줄에 정수 $p_1,p_2,\cdots ,p_N(1\le p_i\le N)$이 공백으로 구분되어 주어진다.</p>

### 출력

<p>모든 행동을 끝낸 후, 각 카드의 천장을 향하는 면에 적힌 정수를 공백으로 구분하여 출력한다.</p>
