### 문제 설명

<p>홀수팀과 짝수팀이 <strong>표식</strong> 많이 만들기 대결을 $Q$번 하려고 한다. 각 팀의 목표는 양의 정수 $W$와 $B$가 주어지면 다음 규칙에 맞게 서로 다른 <strong>표식</strong>을 최대한 많이 만드는 것이다.</p>

<ul>
	<li><strong>표식</strong>은 1개 이상의 흰 정사각형을 한 줄로 이어 붙인 것을 의미한다.</li>
	<li>각 흰 정사각형 안에는 $W$개의 작은 도형이 세로로 나열되어 있으며, 작은 도형은 각각 검은 정사각형 또는 검은 정삼각형이다.
	<ul>
		<li>각 흰 정사각형 안에는 적어도 검은 정사각형이 하나 이상 있어야 한다. 즉, 어떤 흰 정사각형 안에 $W$개의 검은 정삼각형이 있는 <strong>표식</strong>은 만들 수 없다.</li>
	</ul>
	</li>
	<li><strong>표식</strong>은 회전하거나 뒤집을 수 없다.</li>
	<li>한 <strong>표식</strong>에 있는 검은 정사각형 개수의 합은 $B$이어야 한다.</li>
	<li>홀수팀은 흰 정사각형의 개수가 홀수인 <strong>표식</strong>만 만들어야 하고, 짝수팀은 흰 정사각형의 개수가 짝수인 <strong>표식</strong>만 만들어야 한다.</li>
</ul>

<p>홀수팀이 규칙에 맞게 만들 수 있는 서로 다른 <strong>표식</strong>의 개수가 $S_{odd}$개, 짝수팀이 규칙에 맞게 만들 수 있는 서로 다른 <strong>표식</strong>의 개수가 $S_{even}$개라면 홀수팀의 점수는 $S_{odd}$점, 짝수팀의 점수는 $S_{even}$점이 된다. 두 팀 중 점수를 더 많이 얻은 팀이 승리한다. 즉, $S_{odd}\gt S_{even}$ 이면 홀수팀의 승리이고, $S_{odd}\lt S_{even}$ 이면 짝수팀의 승리이다. $S_{odd}=S_{even}$이면 무승부이다. 각 대결에서 사용할 양의 정수 $W$와 $B$가 주어졌을 때 어느 팀이 승리하는지 알아내고, 두 팀의 득점 차이의 절댓값($|S_{odd}-S_{even}|$)을 구해보자.</p>

### 입력

<p>첫째 줄에 대결 횟수인 양의 정수 $Q$가 주어진다.</p>

<p>둘째 줄부터 $Q$개의 줄 각각은 한 번의 대결을 나타낸다.</p>

<p>각 줄에는 문제에 언급된 두 양의 정수 $W$, $B$가 공백으로 구분되어 주어진다.</p>

### 출력

<p>$Q$개의 줄에 걸쳐, 한 줄에 하나씩 각 대결의 결과를 <code style="color: rgb(231, 76, 60);">[결과]</code> <code style="color: rgb(231, 76, 60);">[점수차]</code> 의 형식으로 출력한다.</p>

<p><code style="color:#e74c3c;">[결과]</code>는 홀수팀이 승리한다면 <code style="color:#e74c3c;">odd</code>, 짝수팀이 승리한다면 <code style="color:#e74c3c;">even</code>, 무승부라면 <code style="color:#e74c3c;">tie</code>이다.</p>

<p><code style="color:#e74c3c;">[점수차]</code>는 두 팀의 득점 차( $ = |S_{odd}-S_{even}|$)를 $1\, 234\, 543$으로 나눈 나머지이다.</p>

<p>$1\, 234\, 543$은 소수이다.</p>

### 제한

<ul>
	<li>$1\le Q\le 100 000$</li>
	<li>$1\le W\lt 10^{18}$</li>
	<li>$1\le B\lt 10^{18}$</li>
</ul>
