### 문제 설명

<p>$N \times N$ 크기의 정사각형 격자에 동전들이 놓여 있다. 각 칸에는 동전이 하나씩 놓여있으며, 각 동전은 앞면 또는 뒷면이 위를 보고 있다. 다음의 시행을 최대 $\left\lceil \cfrac {N+1}{2} \right\rceil ^2$번 하여 모든 동전의 앞면이 위를 보게 하자.</p>

<ul>
	<li>네 정수 $y_1$, $x_1$, $y_2$, $x_2$ $(1\leq y_1\leq y_2\leq N; 1\leq x_1 \leq x_2 \leq N)$을 고른 후 $y_1 \leq i \leq y_2$, $x_1 \leq j \leq x_2$를 만족하는 모든 순서쌍 $(i, j)$에 대해 $i$행 $j$열에 있는 동전을 뒤집는다.</li>
</ul>

<p>주어진 문제 조건에서 항상 가능함을 보일 수 있다. 단, 해당 시행의 횟수를 최소화할 필요는 없다.</p>

### 입력

<p>첫째 줄에 격자의 크기를 나타내는 정수 $N$이 주어진다. $(1\leq N \leq 250)$</p>

<p>둘째 줄부터 $N+1$번째 줄까지 동전의 상태를 나타내는 문자열이 주어진다. $i+1$번째 줄 $j$번째 문자는 $i$행 $j$열에 있는 동전의 상태를 나타내고 앞면이 위를 향하도록 놓인 경우 <span style="color:#e74c3c;"><code>0</code></span>, 뒷면이 위를 향하도록 놓인 경우 <span style="color:#e74c3c;"><code>1</code></span>로 표시된다.</p>

### 출력

<p>첫째 줄에 시행의 횟수 $K$를 출력한다.</p>

<p>둘째 줄부터 $K+1$번째 줄까지 시행을 나타내는 네 정수 $y_1$, $x_1$, $y_2$, $x_2$를 공백으로 구분해 출력한다.</p>
