<blockquote>

<p>재우는 네모 네모 게임이라는 게임을 즐겨 한다. </p>

<p>네모 네모 게임은 가로 $M$칸, 세로 $1$칸의 격자판 위에 $N$개의 블록을 놓는 게임이다.</p>

<p>격자의 각 칸은 비어 있거나 <code><span style="color:#e74c3c;">X</span></code> 표시가 되어 있다. <span style="color:#e74c3c;"><code>X</code></span> 표시가 되어 있는 칸 위에는 블록을 놓을 수 없다.</p>

<p>$i$번째 블록은 격자 위 가로로 $A_i$칸, 세로로 $1$칸의 공간을 차지한다.</p>

<p>또한, 블록은 순서대로 왼쪽부터 배치해야 한다. 즉, $i$번째 블록은 $i-1$번째 블록의 오른쪽에 놓여야 한다.</p>

<p>블록은 겹칠 수 없다. 즉, 한 칸의 격자 위에는 최대 한 개의 블록만이 놓일 수 있다.</p>

<p>이러한 조건을 만족하면서 $N$개의 블록을 모두 격자판 위에 올릴 경우 게임에서 승리한다.</p>

<p>이 게임에서 승리하도록 블록을 배치하는 경우의 수는 매우 많다. 재우는 이 게임을 좀 더 고능하게 하기 위해서 항상 블록이 놓이는 칸들을 구하려고 한다.</p>

<p>재우를 대신하여 항상 블록이 놓이는 칸의 개수를 구해주자.</p>

### 입력

<p>첫 번째 줄에 블록 개수 $N$, 격자판의 가로 길이 $M$, <span style="color:#e74c3c;"><code>X</code></span> 표시된 칸의 수 $K$가 공백으로 구분되어 주어진다.</p>

<p>두 번째 줄에 각 블록의 가로 길이를 나타내는 $N$개의 정수 $A_1, A_2, \cdots , A_N$가 공백으로 구분되어 주어진다.</p>

<p>세 번째 줄에 <span style="color:#e74c3c;"><code>X</code></span> 표시된 칸들의 번호를 나타내는 $K$개의 정수 $X_1, X_2, \cdots , X_K$ ($i &lt; j$ 이면 $X_i &lt; X_j$)가 공백으로 구분되어 주어진다.</p>

<p>항상 조건을 만족하도록 블록을 배치할 방법이 있는 입력만 주어진다.</p>

### 출력

<p>첫 번째 줄에 항상 블록이 놓이는 칸의 개수를 출력한다.</p>

### 제한

<ul>
	<li>$1 \le N, K \le 10^6$</li>
	<li>$2 \le M \le 10^9$</li>
	<li>$1 \le A_i \le 10^9$</li>
	<li>$1 \le X_i \le M$</li>
	<li>$(\sum_{i=1}^N A_i) + K \le M$</li>
</ul>
