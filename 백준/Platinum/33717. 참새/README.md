### 문제 설명

<p>전깃줄 위에 일렬로 $N$마리의 참새가 앉아 있다. 왼쪽에서부터 $1, 2, 3, \dots , N$번 참새이다. 각 참새는 오른쪽 또는 왼쪽을 바라보고 있다.</p>

<p>참새는 특정한 규칙에 따라 운다.</p>

<ul>
	<li> $i$번 참새가 왼쪽을 바라보고 있다면, 자신보다 왼쪽에 있으면서 오른쪽을 바라보는 참새의 수만큼 운다.</li>
	<li> $i$번 참새가 오른쪽을 바라보고 있다면, 자신보다 오른쪽에 있으면서 왼쪽을 바라보는 참새의 수만큼 운다.</li>
</ul>

<p>각 참새가 운 횟수가 주어졌을 때, 참새의 방향을 복원하라.</p>

### 입력

<p>첫째 줄에 참새의 수를 나타내는 정수 $N (1 \leq N \leq 10^6)$이 주어진다.</p>

<p>둘째 줄에 각 참새가 운 횟수 $A_1, A_2, \dots , A_N (1 \leq A_i \leq N)$이 공백을 사이에 두고 주어진다.</p>

### 출력

<p>가능한 참새의 배치가 존재한다면, 길이 $N$의 $0$과 $1$로 이루어진 배열을 출력한다.</p>

<p>$i$번째 수가 $0$이라면 왼쪽을, $1$이라면 오른쪽을 바라보고 있음을 뜻한다.</p>

<p>가능한 배치가 여러 개라면 아무거나 출력한다. 불가능하다면 $-1$을 출력한다.</p>
