### 문제 설명

<p>어떤 양의 정수 $N$에 대해, 다음의 조건을 모두 충족하는 길이 $N$의 배열을 <strong>귀여운 배열</strong>이라고 하자.</p>

<ul>
	<li>이 배열에는 $1$ 이상 $N$ 이하의 모든 정수가 정확히 한 번씩 등장한다.</li>
	<li>이 배열의 모든 길이 $3$ 이상의 부분 수열은 등차수열이 아니다. 부분 수열의 원소는 이웃하지 않을 수 있다.</li>
</ul>

<p>그리고, 어떤 $N\times N$ 크기의 행렬에 대해 모든 행과 열이 귀여운 배열이면 이 행렬을 <strong>귀여운 행렬</strong>이라고 하자.</p>

<p>$N$이 주어졌을 때 귀여운 행렬이 존재하는지 여부를 판별하고, 존재한다면 그러한 행렬을 아무거나 하나 찾아 출력하시오.</p>

### 입력

<p>첫 번째 줄에 정수 $N$의 값이 주어진다.</p>

### 출력

<p>귀여운 행렬이 존재한다면 첫 줄에 <span style="color:#e74c3c;"><code>1</code></span>을 출력하고, 그 다음 $N$줄에 걸쳐 귀여운 행렬의 각 행의 $N$개 값을 공백으로 구분하여 출력한다.</p>

<p>귀여운 행렬이 존재하지 않는다면 첫 줄에 <span style="color:#e74c3c;"><code>0</code></span>을 출력한다.</p>

### 제한

<ul>
	<li>$1\le N\le 300$</li>
</ul>
