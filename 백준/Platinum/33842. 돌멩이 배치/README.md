### 문제 설명

<p>$N$개의 칸이 원형으로 배치되어있고 시계 방향으로 $0$부터 $N-1$까지의 번호가 붙어있다. 각 칸에는 돌멩이를 최대 1개까지 놓을 수 있다. $0\leq i &lt; N$인 모든 정수 $i$에 대해 $s_i$를 $i \bmod N, (i+1) \bmod N, (i+2) \bmod N, \cdots , (i+K-1) \bmod N$번 칸에 있는 돌멩이들의 총 개수라고 정의하자.</p>

<p>$0 \leq i \leq N-1$인 모든 $i$에 대해 $L \leq s_i \leq R$을 만족하도록 돌멩이를 배치하는 방법을 찾아보자. 불가능하다면 <span style="color:#e74c3c;"><code>-1</code></span>을 출력한다.</p>

### 입력

<p>첫 번째 줄에 테스트 케이스의 개수 $T$가 주어진다. $(1 \leq T \leq 10\ 000)$</p>

<p>각 테스트 케이스마다 $N$, $K$, $L$, $R$이 공백으로 구분되어 주어진다. $(1 \leq K &lt; N \leq 300\ 000;$ $0 \leq L \leq R \leq K)$</p>

<p>모든 테스트 케이스에서 $N$의 합은 $300\ 000$을 넘지 않는다.</p>

<p>입력되는 모든 수는 정수이다.</p>

### 출력

<p>각 테스트 케이스마다 불가능하다면 한 줄에 <span style="color:#e74c3c;"><code>-1</code></span>을 출력한다.</p>

<p>가능하다면 돌멩이의 배치를 나타내는 $N$개의 수를 공백으로 구분하여 출력한다. 각 수는 $0$ 또는 $1$이여야 하며, $i$번째 수는 $i-1$번 칸에 배치된 돌멩이의 개수를 나타낸다.</p>
