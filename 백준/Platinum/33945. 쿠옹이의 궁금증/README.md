### 문제 설명

<p>쿠옹이는 어느 날 갑자기 이런 궁금증이 들었다. '연산 결과가 $N$이 되는 수식은 몇 개가 있을까?'</p>


<p>쿠옹이는 연산 결과가 $N$이 되는 수식의 뒤에 <code>+0</code>, <code>-0</code> 등을 이어 붙이면 여전히 연산 결과가 $N$이므로 이 시행을 반복하면 연산 결과가 $N$인 수식을 무한히 만들 수 있다는 슬픈 사실을 깨닫고 말았다.</p>

<p>그래서 쿠옹이는 수식의 길이가 $M$이어야 한다는 제약을 추가했지만 이번에는 답을 내지 못했다. 여러분이 대신 이 문제를 풀어 주자!</p>

<p>수식은 다음과 같이 정의된다.</p>

<ul>
	<li><strong>항</strong>은 <code>0</code>, <code>1</code>, <code>2</code>, <code>3</code>, <code>4</code>, <code>5</code>, <code>6</code>, <code>7</code>, <code>8</code>, <code>9</code>만으로 구성되어 있으며 <code>0</code>으로 시작하지 않는 길이 1 이상의 문자열이다. 단 <code>0</code>은 <code>0</code>으로 시작하지만 예외적으로 <strong>항</strong>이다.</li>
	<li><strong>수식</strong>은 1개 이상의 <strong>항</strong>을 포함하며 각 <strong>항</strong>이 <code>+</code> 또는 <code>-</code>로 구분되어 있는 문자열이다.</li>
</ul>

<p>다르게 설명하면 수식은 다음 정규식을 만족하는 문자열을 말한다.</p>

<ul>
	<li><code>(([1-9][0-9]*|'0')[+-]))*([1-9][0-9]*|'0')</code></li>
</ul>

### 입력

<p>첫째 줄에 정수 $N$과 $M$이 공백으로 구분되어 주어진다. $(0 \le N \le 10^5, 1 \le M \le 11)$</p>

### 출력

<p>길이 $M$의 연산 결과가 $N$이 되는 서로 다른 수식의 수를 출력하라. 이때 그러한 수식이 아주 많을 수 있으므로 $10^9+7$로 나눈 나머지를 대신 출력하라.</p>
