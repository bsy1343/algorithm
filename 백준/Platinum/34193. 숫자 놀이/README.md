### 문제 설명

<p>양의 정수 $x$와 $n$이 주어졌을 때, $2n+1$개의 정수 $x,x+1,\cdots, x+2n$을 길이 $n$인 두 수열 $A=[A_1,A_2,\cdots,A_n]$과 $B=[B_1,B_2,\cdots,B_n]$, 그리고 하나의 수 $c$로 나누어 다음 수식들이 모두 충족되도록 해야 한다.</p>

<p>$$ \begin{align*} A_1 \,+\, &amp;c = B_1 \\ A_2 \,+\, &amp;c = B_2 \\ &amp;\vdots \\ A_n \,+\, &amp;c = B_n \end{align*} $$</p>

<p>이때 가능한 서로 다른 $c$의 값의 개수를 구하시오.</p>

### 입력

<p>첫 번째 줄에 테스트 케이스의 개수 $T$가 주어진다. ($1 \le T \le 100$)</p>

<p>그다음 줄부터 각각의 테스트 케이스에 대해, 양의 정수 $x$와 $n$이 공백으로 구분되어 한 줄에 주어진다. ($1 \le x,n\le 10^9$)</p>

### 출력

<p>각각의 테스트 케이스에 대해, 문제의 정답을 한 줄에 출력한다.</p>
