### 문제 설명

<p>당신은 숫자 $k$를 싫어한다. 그래서 수를 쓸 때, 숫자 $k$는 쓰지 않으려 한다.</p>

<p>$f(x)$를, $x$의 10진법 표현에서 숫자 $k$를 모두 지운 수로 정의하자. 만일 $x$의 10진법 표현이 숫자 $k$로만 이루어져 있다면 $f(x)$를 $0$으로 정의한다.</p>

<p>예를 들어, $k = 3$일 때 $f(3135) = 15$, $f(2345) = 245$, $f(8185) = 8185$, $f(333333) = 0$이다.</p>

<p>숫자 $k$와 수 $N$이 주어질 때, $f(1) + f(2) + ... + f(N)$을 $1\,000\,000\,007$로 나눈 나머지를 구하여라.</p>

### 입력

<p>첫째 줄에 숫자 $k$가 주어진다.</p>

<p>둘째 줄에 수 $N$이 주어진다.</p>

### 출력

<p>첫째 줄에 $f(1) + f(2) + ... + f(N)$을 $1\,000\,000\,007$로 나눈 나머지를 출력하라.</p>

### 제한

<ul>
	<li>$1 \le k \le 9$</li>
	<li>$1 \le N &lt; 10^{100\,000}$</li>
	<li>입력으로 주어지는 수는 모두 정수다.</li>
</ul>
