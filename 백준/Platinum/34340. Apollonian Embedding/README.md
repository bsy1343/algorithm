# [Platinum III] Apollonian Embedding - 34340

[문제 링크](https://www.acmicpc.net/problem/34340)

시간 제한: 1 초 (추가 시간 없음), 메모리 제한: 1024 MB (추가 메모리 없음)

### 문제 설명

<p>볼록 $N$각형이 주어진다. 각 정점들의 번호는 시계방향으로 $1,2,\cdots ,N$이다. $i$번 정점과 $i+1$번 정점 사이에는 양방향 간선이 존재한다 ($N+1$번 정점은 $1$번 정점으로 간주한다). 여기에 서로 교차하지 않는 $N-3$개 간선이 새로 주어진다. Apollonian network는 삼각형 그래프에서 시작해 재귀적으로 하나의 삼각형을 작은 3개의 삼각형으로 분할하는 것을 반복해서 만들 수 있는 그래프이다. 자세히는 다음과 같이 생성된다. 처음에는 정점 3개, 간선 3개로 이루어진 그래프로 시작한다. 이때 삼각형 면은 시작 정점 3개로 이루어진 삼각형 1개 뿐이다. 다음 시행을 $0$회 이상 반복해서 만들 수 있는 그래프를 Apollonian network라고 부른다.</p>

<ul>
	<li>정점 $u,v,w$로 구성된 삼각형 면을 고른다.</li>
	<li>새로운 정점 $x$를 추가한다.</li>
	<li>새로운 간선 $xu$, $xv$, $xw$를 추가한다.</li>
	<li>삼각형 면 $uvw$를 제거하고 새로운 삼각형 면 3개 $xuv$, $xvw$, $xwu$를 추가한다.</li>
</ul>

<p>주어진 그래프를 부분 그래프로 가지며 정점이 $N$개인 Apollonian network를 만들어야 한다. 가능한 방법이 여러 가지라면 아무것이나 출력해도 된다.</p>

### 입력

<p>첫 번째 줄에는 $N$이 주어진다. ($3 \leq N \leq 3000$)</p>

<p>이어지는 $N-3$개의 줄에는 대각선을 이루는 정점을 나타내는 두 정수 $u$, $v$가 주어진다. ($1 \leq u &lt; v \leq N$)</p>

### 출력

<p>Apollonian network를 위의 생성 과정을 따라서 생성한다.</p>

<p>첫 번째 줄에 Apollonian network의 시작 삼각형을 이루는 세 정점을 공백을 사이에 두고 출력한다.</p>

<p>다음 $N-3$개의 줄에는 제거하는 삼각형 면을 이루는 정점 3개 $u$, $v$, $w$와 추가하는 정점 $x$를 공백을 사이에 두고 출력한다. 각 줄에서 $u, v, w$의 출력 순서는 상관이 없다. 정점이 $N$개인 Apollonian network를 만들기 위해서는 시행을 $N-3$번 해야 됨을 알 수 있다.</p>
