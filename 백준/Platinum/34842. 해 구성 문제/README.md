### 문제 설명

<p>양의 정수 $N$이 주어질 때 다음 조건을 만족하는 길이 $N$의 정수 수열 $A_1,A_2,\ldots,A_N$을 구하여라.</p>

<ul>
<li>수열의 모든 원소는 서로 다르다.</li>
<li>$\max(A) - \min(A) \le N+2$</li>
<li>$\displaystyle \sum_{i=1}^{N} i A_i = 0$</li>
</ul>

### 입력

<p>첫째 줄에 양의 정수 $N$이 주어진다. ($1 \le N \le 300\,000$) </p>

### 출력

<p>만약 조건을 만족하는 수열 $A$가 있다면 첫째 줄에 <samp>Yes</samp>를 출력하고, 둘째 줄에 수열의 원소 $A_1,A_2,\ldots,A_N$를 공백으로 구분하여 출력한다. 조건을 만족하는 수열 $A$가 여러 가지라면 그중 아무거나 출력한다.</p>

<p>만약 조건을 만족하는 수열 $A$가 없다면 첫째 줄에 <samp>No</samp>를 출력한다.</p>
