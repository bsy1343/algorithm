### 문제 설명

<p>길이 $N$의 <strong>순열</strong> $P = (P_1, \, P_2, \, \cdots, \, P_N)$가 주어질 때, 길이 $N$의 적당한 <strong>순열</strong> $A = (A_1, \, A_2, \, \cdots, \, A_N)$와 $B = (B_1, \, B_2, \, \cdots, \, B_N)$를 구성하여</p>

<p>$$\displaystyle \large \sum_{k=1}^{N-1} \left( \min \left( B_{P_{A_k}}, B_{P_{A_{k+1}}} \right) + \min \left( B_{A_k}, B_{A_{k+1}} \right) \right)$$</p>

<p>을 최대화하라.</p>

### 입력

<p>첫 번째 줄에 순열의 길이 $N$이 주어진다. ($2 \leq N \leq 200 \, 000$)</p>

<p>두 번째 줄에 순열 $P$의 원소 $P_1, \, P_2, \, \cdots, \, P_N$이 공백으로 구분되어 주어진다.</p>

### 출력

<p>첫 번째 줄에 주어진 식을 최대화하는 순열 $A$의 원소 $A_1, \, A_2,\, \cdots, \, A_N$을 공백으로 구분하여 출력하라.</p>

<p>두 번째 줄에 주어진 식을 최대화하는 순열 $B$의 원소 $B_1, \, B_2,\, \cdots, \, B_N$을 공백으로 구분하여 출력하라.</p>

<p>가능한 답이 여러 개라면 아무거나 출력해도 된다.</p>

### 힌트

<p>길이가 $N$인 <strong>순열</strong>이란 순열의 원소로 $1$부터 $N$까지의 정수가 모두 빠짐없이 단 한 번씩 나오는 수열을 의미한다. 즉, 순열 $A = \left(A_1, \, A_2 , \, \cdots, \, A_N\right)$는 아래 조건을 만족한다.</p>

<ul>
<li> $A_i$는 $1$ 이상 $N$ 이하의 정수</li>
<li> $i \neq j$이면 $A_i \neq A_j$ </li>
</ul>
