### 문제 설명

<p>준규는 새로운 언어인 &quot;조말어&quot;를 창조하려고 한다. 먼저, 준규는 조말어의 기초가 되는 단어를 모두&nbsp;적은 목록을 만들었다. 이 목록에 있는 단어를&nbsp;&quot;기초 어휘&quot;라고 한다.</p>

<p>이제 기초 어휘를&nbsp;가지고 새로운 단어를&nbsp;창조하여&nbsp;&quot;조말어대사전&quot;을 만들 것이다. &quot;조말어대사전&quot;에 수록될 단어들은 다음의 조건&nbsp;중 하나를&nbsp;만족해야 한다.</p>

<ul>
	<li>&quot;기초 어휘&quot;이다.</li>
	<li>단어를 두 부분으로 나누었을 때, 앞 부분이&nbsp;기초 어휘의 앞 부분(혹은 전체)이고, 뒷 부분이&nbsp;기초어휘의 뒷 부분(혹은 전체)이다. (앞 부분과 뒷 부분은 prefix와 suffix로 이해하면 된다)</li>
</ul>

<p>위의 규칙을 따른 조말어대사전에는 총 몇 개의 단어들이 수록될까?</p>

### 입력

<p>첫째 줄에는 기초 어휘의 개수 n(1 &le; n &le; 10 000)이 주어진다. 그 다음, n개의 줄에는 각 기초 어휘가 주어진다. 기초 어휘는 길이가 1이상 40이하 이며 소문자 알파벳으로 이루어져있다.&nbsp;</p>

<p>또한 준규는 매우 게을러서 기초 어휘 목록에 중복된 단어가 있는지 검사하지 않는다. 주의하자.</p>

### 출력

<p>조말어대사전에 수록될 단어의 개수를 출력한다.</p>
