### 문제 설명

<p>페인트 토끼의 토끼굴은 $1$번 노드를 루트로 하는 $N$개의 노드로 이루어진 트리로 표현할 수 있다<strong>(단, $N$은 짝수)</strong>. 각 노드는 토끼굴의 방을 의미한다. 게임에서는 꾸미는 것도 중요하기 때문에 페인트를 좋아하는 토끼는 토끼굴에서 정확히 $\frac{N}{2}$개의 방을 색칠하려고 한다. 토끼에게는 단 하나의 페인트 탄이 있으며, 특정 노드에 탄을 쏘면 다음과 같은 규칙으로 페인트가 번진다.</p>

<ul>
<li>처음 탄을 맞은 노드는 색칠된다.</li>
<li>색칠된 노드의 모든 자식 노드는 연쇄적으로 색칠된다.</li>
</ul>

<p>GM 77은 토끼를 돕기 위해 서로 다른 두 노드 $u$, $v$를 연결하는 포탈을 <strong>정확히 하나</strong> 설치하기로 했다. 이 때 두 노드 사이에는 페인트가 양방향으로 흐른다. 즉, $u$가 색칠되면 추가로 $v$와 그 자손들이 모두 색칠되고, 반대로 $v$가 색칠되면 추가로 $u$와 그 자손들이 모두 색칠된다.</p>

<p>토끼가 불쌍했던 GM 77의 목표는 포탈을 적절히 설치하여, <u>어느 한 노드에 페인트 탄을 쐈을 때 정확히 $N/2$개의 노드가 색칠되게 만드는 발사 지점의 개수</u>를 최대화하는 것이다. 포탈을 최적으로 설치했을 때, 이러한 발사 지점의 최대 개수를 구하여라.</p>

### 입력

<p>첫째 줄에 노드의 수를 나타내는 정수 $N$이 들어온다.</p>

<p>두 번째 줄에는 $N-1$개의 정수 $P_2$, $P_3$, $\cdots$, $P_N$이 공백으로 구분되어 주어진다. $P_i$는 트리에서 $i$번 노드의 부모 노드의 번호를 의미한다.</p>

### 출력

<p>포탈을 적절히 설치했을 때 발사 지점의 최대 개수를 출력한다.</p>

### 제한

<ul>
<li>주어지는 모든 수는 정수이다.</li>
<li>$4\le N\le 100\, 000$, $N$은 짝수</li>
<li>$1\le P_i&lt;i$ ($2\le i\le N$)</li>
</ul>
