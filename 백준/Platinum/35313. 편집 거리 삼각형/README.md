# [Platinum III] 편집 거리 삼각형 - 35313

[문제 링크](https://www.acmicpc.net/problem/35313)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 1024 MB

### 통계

제출: 37, 정답: 17, 맞힌 사람: 16, 정답 비율: 50.000%

### 분류

구현, 두 포인터, 많은 조건 분기, 문자열, 브루트포스 알고리즘, 중간에서 만나기, 해 구성하기, 해싱

### 문제 설명

<p>두 문자열 $S$와 $T$ 사이의 편집 거리는 다음의 연산들을 사용하여 $S$에서 $T$로 바꾸기 위한 최소 연산 횟수로 정의한다.</p>

<ul>
<li>하나의 문자를 임의의 위치에 추가한다.</li>
<li>하나의 문자를 제거한다.</li>
<li>하나의 문자를 다른 문자로 바꾼다.</li>
</ul>

<p>알파벳 대문자로만 이루어진 문자열 $S$와 $T$가 주어진다. 역시 알파벳 대문자로만 이루어진 길이 $1$ 이상의 문자열 $U$ 중에서 다음 조건을 충족하는 문자열이 존재한다면, 그러한 문자열을 아무거나 하나 찾아보자.</p>

<ul>
<li>$S$와 $U$ 사이의 편집 거리는 정확히 $1$이고, $T$와 $U$ 사이의 편집 거리도 정확히 $1$이다.</li>
</ul>

### 입력

<p>첫째 줄에 문자열 $S$의 길이 $n$과 문자열 $T$의 길이 $m$이 공백으로 구분되어 주어진다. ($1\le n,m\le 5\, 000$)</p>

<p>둘째 줄에 알파벳 대문자로 이루어진 문자열 $S$가 주어진다.</p>

<p>셋째 줄에 알파벳 대문자로 이루어진 문자열 $T$가 주어진다.</p>

### 출력

<p>문제의 조건을 충족하는 길이 $1$ 이상의 문자열 $U$가 존재한다면, 첫째 줄에 <span style="color:#e74c3c;"><code>1</code></span>을 출력하고, 둘째 줄에 $U$를 출력한다. 그러한 $U$가 여러 가지라면 아무거나 하나를 출력한다.</p>

<p>존재하지 않는다면 첫째 줄에 <span style="color:#e74c3c;"><code>0</code></span>을 출력한다.</p>
