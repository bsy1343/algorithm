### 문제 설명

<p>승원이는 입부 시험을 위해 다음과 같은 문제를 만들었다.</p>

<ul>
<li> $X$진법으로 쓰인 $N$개의 수가 주어진다. 이 수들을 모두 곱한 값을 $X$진법으로 출력하라.</li>
</ul>

<p>하지만 테스트 케이스를 만들던 중 깜빡 졸아버려, 주어지는 수들의 각 자릿수가 모두 같은 숫자로 이루어지게 되었다. 즉, $222, 33, 88888$과 같은 수들만 주어진다. </p>

<p>이러한 조건에서, 주어진 $N$개의 수를 모두 곱한 값을 $X$진법으로 출력하는 프로그램을 작성하라.</p>

### 입력

<p>첫 번째 줄에 양의 정수 $N$과 진법 $X$가 공백으로 구분되어 주어진다.</p>

<p>두 번째 줄부터 $N$개의 줄 중 $i$번째 줄에 양의 정수 $a_i, l_i$가 공백으로 구분되어 주어진다. 이는 $i$번째 수가 숫자 $a_i$가 $l_i$번 반복된 $X$진법 수임을 의미한다.</p>

### 출력

<p>첫 번째 줄에 $N$개의 수를 모두 곱한 값을 $X$진법으로 출력하라.</p>

### 제한

<ul>
<li>$1 \le N \le 10$</li>
<li>$2 \le X \le 10$</li>
<li>$1 \le a_i &lt; X$ $(1 \le i \le N)$</li>
<li>$1 \le l_i \le 10^6$ $(1 \le i \le N)$</li>
</ul>
