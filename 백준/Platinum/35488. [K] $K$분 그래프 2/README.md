### 문제 설명

<p>하이바이는 최근 $K$분 그래프에 대해 배웠다. $K$분 그래프가 무엇인지 모른다면, 아래 정의를 참고하자.</p>

<p>양의 정수 $K$에 대해 <strong>$K$분 그래프</strong>란 다음 조건을 만족하는 가중치 있는 방향 그래프 $G=(V,E)$이다.</p>

<ul>
<li>그래프의 모든 닫힌 보행 $P$에 대해, $P$의 간선 가중치 합이 항상 $K$의 배수이다. 여기서 닫힌 보행이란 시작점과 끝점이 같으며, 같은 정점이나 간선을 여러 번 방문할 수 있는 경로를 말한다.</li>
</ul>

<p>마침 출제할 만한 문제가 없었던 하이바이는 주어진 그래프가 $K$분 그래프인지 판별하는 문제를 내기로 했다.</p>

### 입력

<p>첫째 줄에는 그래프 $G$의 정점 개수 $N$, 간선 개수 $M$, 문제에서 설명한 양의 정수 $K$가 공백으로 구분되어 주어진다. $(1\le N\le 300\, 000;$ $0\le M\le 500\, 000;$ $1\le K\le 10^9)$</p>

<p>둘째 줄부터 $M$개의 줄에 걸쳐 $i+1$번째 줄에는 $3$개의 정수 $v_i$, $w_i$, $x_i$가 공백으로 구분되어 주어진다. 이는 $v_i$번 정점을 시작점으로 하고 $w_i$번 정점을 끝점으로 하는 가중치가 $x_i$인 간선을 의미한다. $(1\le v_i,w_i\le N;$ $0\le x_i\le 10^9)$</p>

### 출력

<p>첫째 줄에는 주어진 그래프가 $K$분 그래프라면 <span style="color:#e74c3c;"><code>Yes</code></span>를, 아니면 <span style="color:#e74c3c;"><code>No</code></span>를 출력한다.</p>
