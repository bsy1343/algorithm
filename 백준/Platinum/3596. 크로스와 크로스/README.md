### 문제 설명

<p>크로스와 크로스 게임은 1 &times; n칸으로 이루어진 보드에서 진행한다. 두 플레이어는 서로 번갈아가면서 게임을 한며, 각 칸의 크기는 1 &times; 1이다.</p>

<p>각 플레이어는 자기 턴일때, 빈 칸중 한 곳에 &#39;&times;&#39; 마크를 한다.</p>

<p>만약, 어떤 플레이어가 연속된 3개의 &#39;&times;&#39;를 만들 수 있다면 그 플레이어가 이긴다.</p>

<p>n이 주어졌을 때, 두 플레이어가 현재 턴에서 선택할 수 있는 가장 좋은 방법으로 게임을 했을 때, 이기는 사람을 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 n이 주어진다. (3 &le; n &le; 2000)</p>

### 출력

<p>첫 번째 플레이어가 이긴다면 1을, 두 번째 플레이어가 이긴다면 2를 출력한다.</p>
