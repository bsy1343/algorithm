# [Platinum V] 애매함 - 3698

[문제 링크](https://www.acmicpc.net/problem/3698)

### 문제 설명

<p>단어의 첫 글자와 마지막 글자를 제외한 나머지 글자의 순서를 섞어도 그 단어는 쉽게 읽을 수 있다. 예를 들어, 문장 &quot;tihs snetncee mkaes prfecet sesne&quot;는 많은 사람들이 쉽게 읽을&nbsp;수 있다.</p>

<p>문장의 공백을 모두 제거해도 그 문장은 쉽게 읽을 수 있다. 예를 들면 &quot;thissentencemakesperfectsense&quot; 이다.</p>

<p>하지만, 글자의 순서를 섞고, 공백을 제거하면 문장은 쉽게 읽을 수 없다. &quot;tihssnetnceemkaesprfecetsesne&quot;와 같은 문장은 쉽게 읽을 수 없다.</p>

<p>글자의 순서를 섞고, 단어 사이의 공백을 제거한 문장이 주어진다. 또, 올바른 단어의 목록이 주어진다. 이때, 원래 문장을 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 테스트 케이스의 개수가 주어진다. 테스트 케이스의 개수는 최대 100개이다.</p>

<p>각 테스트 케이스의 첫째 줄에는 글자의 순서를 뒤섞고, 공백을 제거한 문장이 주어진다. 이 문장은 알파벳 소문자로만 이루어져 있으며, 길이는 최대 1000글자이다.</p>

<p>둘째 줄에는 올바른 단어의 개수 n (1 &le; n &le; 10,000)이 주어진다. 다음 n개 줄에는 올바른 단어가 주어진다. 모든 단어는 중복되지 않으며, 알파벳 소문자로만 이루어져 있다. 또, 길이는 최대 100글자이다.</p>

### 출력

<p>각 테스트 케이스마다, 원래 문장을 유일하게 만들 수 있으면 그 문장을 출력한다. 문장을 만들 수 있는 방법이 여러 가지인 경우에는 &quot;ambiguous&quot;를, 불가능한 경우에는 &quot;impossible&quot;을 출력한다.</p>
