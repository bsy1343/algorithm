### 문제 설명

<p>
	양의 정수 n과 k가 주어졌을 때, n = p * q이고, p &le; q, |q-kp| &le;&nbsp;10<sup>5</sup>를 만족하는 소수 p와 q를 찾는 프로그램을 작성하시오.</p>

### 입력

<p>
	첫째 줄에 n과 k가 주어진다. (1 &lt; n &lt; 10<sup>120</sup>, 1 &lt; k &lt; 10<sup>8</sup>)</p>

### 출력

<p>
	첫째 줄에 문제의 조건을 만족하는 소수 p와 q를 &quot;p * q&quot;형태로 출력한다.</p>
