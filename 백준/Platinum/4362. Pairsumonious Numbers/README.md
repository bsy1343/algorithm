# [Platinum III] Pairsumonious Numbers - 4362

[문제 링크](https://www.acmicpc.net/problem/4362)

### 문제 설명

<p>For 10 &gt; N &gt; 2 numbers we form N*(N-1)/2 sums by adding every pair of the numbers. Your task is to find the N numbers given the sums.</p>

### 입력

<p>Each line of input contains N followed by N*(N-1)/2 integer numbers separated by a space.</p>

### 출력

<p>For each line of input, output one line containing N integers in non-descending order such that the input numbers are pairwise sums of the N numbers. If there is more than one solution, any one will do; if there is no solution, print &quot;Impossible&quot;.</p>
