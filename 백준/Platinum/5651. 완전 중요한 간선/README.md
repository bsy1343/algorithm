# [Platinum I] 완전 중요한 간선 - 5651

[문제 링크](https://www.acmicpc.net/problem/5651)

### 성능 요약

시간 제한: 1 초, 메모리 제한: 256 MB

### 통계

제출: 2764, 정답: 916, 맞힌 사람: 536, 정답 비율: 29.194%

### 분류

그래프 이론, 최대 유량

### 문제 설명

<p>어떤 플로우 그래프가 주어졌을때, 한 간선의 용량이 1줄면 최대 유량도 1이 줄어드는 경우 그 간선을 완전 중요한 간선이라고 부른다. 그래프가 주어진 경우 완전 중요한 간선의 개수를 세어보자!</p>

### 입력

<p>입력은 여러개의 테스트케이스로 이뤄진다. 첫째 줄에는 테스트케이스의 수 K (1&lt;=K&lt;=15)가 주어진다. &nbsp;각 테스트 케이스에는 N, M&nbsp;(2 &lt;= N &lt;= 300; 2 &lt;= M &lt;= 5,000)가 주어지고 각각 정점의 수와 간선의 수를 뜻한다. 1번 정점은 source를 N번 정점은 sink를 뜻한다.&nbsp;</p>

<p>다음 M줄에 걸쳐서는 세개의 정수 f, t, b가 주어지는데 f에서 t로 가는 간선의 용량이 b(&lt;1000)라는 뜻이다. 모든 유량의 합은 20,000을 넘지 않는다.</p>

### 출력

<p>각 테스트케이스마다 한줄씩 완전 중요한 간선의 수를 출력한다.</p>
