### 문제 설명

<p>The form of a natural number a in a number system with base p contains of n ones (digts &quot;1&quot;) in a row.</p>

<p>Your task is to write a program that computes the greatest values of exponents of numbers 2 and 3 such that the remainder of a divided by both these values the remainder is 0.</p>

### 입력

<p>Two values of natural numbers p (1 &lt; p &lt; 10<sup>9</sup>) and n (n &lt; 10<sup>9</sup>) are input from the keyboard.</p>

### 출력

<p>Two non-negative integer numbers must be output on the screen: the values of the greatest exponents of two and three.</p>
