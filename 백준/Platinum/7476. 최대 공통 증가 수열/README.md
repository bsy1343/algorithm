### 문제 설명

<p>정수로 이루어진 두 수열이 있다. 두 수열의 공통 증가 부분 수열 중 가장 긴 것의 길이를 구하는 프로그램을 작성하시오.</p>

<p>길이가 N인 수열 S<sub>1</sub>, S<sub>2</sub>, ..., S<sub>n</sub>이 길이가 M인 수열 A<sub>1</sub>, A<sub>2</sub>, ..., A<sub>M</sub>의 증가하는 부분 수열이 되려면, 모든 1 &le; j &le; N에 대해서, S<sub>j</sub> = A<sub>i<sub>j</sub></sub>인 1 &le; i<sub>1</sub> &lt; i<sub>2</sub> &lt; ... &lt; i<sub>N</sub> &le; M가 존재하고, 모든 1 &le; j &lt; N에 대해 S<sub>j</sub> &lt; S<sub>j+1</sub>을 만족해야 한다.</p>

### 입력

<p>두 수열의 정보는 각각 두 줄에 걸쳐서 주어지며, 첫째 줄에는 수열의 길이 M이 주어진다. (1 &le; M &le; 500) 다음 줄에는 M개의 정수 A<sub>i</sub> (-2<sup>31</sup> &le; A<sub>i</sub> &lt; 2<sup>31</sup>)가 주어진다.</p>

### 출력

<p>첫째 줄에 최대 공통 증가 부분 수열의 길이를 출력하고, 둘째 줄에 그 부분 수열을 출력한다. 답이 여러 가지인 경우에는 아무거나 출력한다.</p>
