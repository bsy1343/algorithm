# [Platinum II] B-행렬 - 7783

[문제 링크](https://www.acmicpc.net/problem/7783)

### 성능 요약

시간 제한: 1 초, 메모리 제한: 128 MB

### 통계

제출: 58, 정답: 27, 맞힌 사람: 25, 정답 비율: 49.020%

### 분류

다이나믹 프로그래밍, 누적 합, 두 포인터

### 문제 설명

<p>m행 n열로 이루어진 바이너리 행렬이 주어졌을 때, 0으로만 이루어진 두 직사각형으로 덮을 수 있는 가장 큰 면적을 구하는 프로그램을 작성하시오. 두 직사각형은 겹치면 안 된다.</p>

### 입력

<p>첫째 줄에 m과 n이 주어진다. (0 &le; n, m &le; 200) 다음 m개 줄에는 행렬의 원소인 n개의 수가 주어진다.</p>

### 출력

<p>첫째 줄에 0으로만 이루어진 두 직사각형으로 덮을 수 있는 가장 큰 면적을 출력한다.</p>
