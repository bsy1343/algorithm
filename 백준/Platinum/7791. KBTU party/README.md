### 문제 설명

<p>At the latest KBTU &rdquo;Commencement&rdquo; party Bakhytzhan noticed that there were n girls D<sub>1</sub>,D<sub>2</sub>,...,D<sub>n</sub> and 2n&minus;1 boys B<sub>1</sub>,B<sub>2</sub>,...,B<sub>2n&minus;1</sub>. And every girl D<sub>j</sub> knows exactly boys B<sub>1</sub>,B2,...,B2j&minus;1. Throughout the evening party Bakhytzhan tried to count the number of possible choices of exactly r acquainted boys and girls pairs for dancing. That&rsquo;s why he missed his chance for dancing. Please, write a program that would help Bakhytzhan don&rsquo;t miss a chance to dance at the next party.</p>

### 입력

<p>The input file contains two integer numbers n and r (1 &le; n,r &le; 10<sup>6</sup>) separated by a space.</p>

### 출력

<p>Output file should contain answer to Bakhytzhan&rsquo;s problem &ndash; residue modulo 2946859</p>
