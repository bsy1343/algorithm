### 문제 설명

<p>Two positive integers are said to be coprime if 1 is their only common divisor. Given a sequence of positive integers a<sub>1</sub>, a<sub>2</sub>, ..., a<sub>n</sub>, find the number of pairs of its terms which are coprime.</p>

### 입력

<p>The first line of input contains one integer n (1 &le; n &le; 1,000,000) denoting the length of the sequence. The second line contains n integers a<sub>i</sub> (1 &le; a<sub>i</sub> &le; 3,000,000).</p>

### 출력

<p>Your program should output one integer representing the number of pairs (i, j) such that 1 &le; i &lt; j &le; n and a<sub>i</sub> is coprime with a<sub>j</sub>.</p>
