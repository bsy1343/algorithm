### 문제 설명

<p>In this problem we are interested only in prime prime powers, i.e., numbers of the form a<sup>b</sup>, where a and b are prime numbers. For a given number n, we want to find the k-th smallest prime prime power greater than n.</p>

### 입력

<p>The first and only line of the input contains two integers n and k (1 &le; n &le; 10<sup>18</sup>, 1 &le; k &le; 100,000).</p>

### 출력

<p>The first and only line of output should contain one integer m, such that m is the k-th smallest prime prime power greater than n.</p>
