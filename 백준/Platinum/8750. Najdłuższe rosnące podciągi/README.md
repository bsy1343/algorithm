# [Platinum III] Najdłuższe rosnące podciągi - 8750

[문제 링크](https://www.acmicpc.net/problem/8750)

### 성능 요약

시간 제한: 1 초, 메모리 제한: 128 MB

### 통계

제출: 15, 정답: 14, 맞힌 사람: 10, 정답 비율: 90.909%

### 분류

자료 구조, 세그먼트 트리, 이분 탐색, 가장 긴 증가하는 부분 수열 문제

### 문제 설명

<p>Twoim zadaniem jest obliczenie liczby najdłuższych rosnących podciąg&oacute;w ciągu <em>A</em>&nbsp;modulo liczba <em>m</em>.</p>

### 입력

<p>W pierwszej linii wejścia znajdują się dwie liczby całkowite <em>n</em>&nbsp;i <em>m</em>&nbsp;(1 &le; <em>n</em> &le; 500 000, 1 &le; <em>m</em> &le; 10<sup>9</sup>), odpowiednio długość ciągu <em>A</em>&nbsp;i liczba <em>m</em>. W następnym wierszu <em>n</em>&nbsp;liczb całkowitych - kolejne elementy ciągu <em>A</em>&nbsp;(0 &le; <em>a<sub>i</sub></em> &le; 10<sup>9</sup>) .</p>

### 출력

<p>Pierwszy i jedyny wiersz wyjścia powinien zawierać jedną liczę całkowitą - liczba najdłuższych rosnących podciąg&oacute;w modulo <em>m</em>.</p>
