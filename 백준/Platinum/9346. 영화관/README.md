# [Platinum II] 영화관 - 9346

[문제 링크](https://www.acmicpc.net/problem/9346)

### 성능 요약

시간 제한: 2 초, 메모리 제한: 64 MB

### 통계

제출: 81, 정답: 13, 맞힌 사람: 10, 정답 비율: 27.778%

### 분류

다이나믹 프로그래밍, 비트마스킹, 백트래킹, 비트필드를 이용한 다이나믹 프로그래밍

### 문제 설명

<p>신해빈이 운영하는 영화관은 8개의 좌석이&nbsp;<strong>R</strong>줄 만큼 있다. 어떤 좌석은 이미 예약되어있으며, 영화관에 들어오는 손님은 총 세 종류이다.</p>

<ul>
	<li>예약하고 온 손님</li>
	<li>예약하지 않고 온&nbsp;솔로</li>
	<li>예약하지 않고 온 커플 (너무&nbsp;당연하지만,&nbsp;커플은 같이 앉고 싶어한다)</li>
</ul>

<p>예약하고 온 손님은 그냥 예약한 좌석에 앉으면 된다. 이제 신해빈은 솔로와 커플들을 적절한 좌석에 앉혀야 한다. 그런데 솔로와&nbsp;커플은&nbsp;모두&nbsp;다른 사람의 옆이나 뒤에 앉기를 싫어한다. 이를 최대한 고려하여 각 손님의 좌석을 결정해준다면, 신해빈의 영화관은 매출이 늘어나&nbsp;순살치킨을 더 많이 먹을 수 있다.</p>

<p>신해빈은 치킨을 먹느라 바빠서&nbsp;우리가 이 문제를 해결해야한다. 문제의 조건으로&nbsp;예약된 좌석들의 위치, 솔로의 수 <strong>S</strong>, &nbsp;커플 쌍의&nbsp;수 <strong>C</strong>가 주어진다.</p>

<p>이제 다음의 조건들 :</p>

<ul>
	<li>솔로는 비예약석에만 앉는다.</li>
	<li>솔로는 같은 줄에 있는 어떤 사람 옆에도 앉기 싫다.</li>
	<li>솔로는 자신이 앉은 좌석 바로 앞 좌석에 어떤 사람이 앉아 있는게 싫다.</li>
	<li>커플은 같은 줄에 같이, 비예약석에&nbsp;앉아야&nbsp;한다.</li>
	<li>커플은 파트너를 제외하고는 같은 줄에 있는 어떤 사람 옆에도 앉기 싫다.</li>
	<li>커플은&nbsp;자신들이 앉은 좌석 바로 앞 좌석에 어떤 사람이 앉아&nbsp;있는게 싫다.</li>
	<li>(바로 앞좌석이라 함은, 바로 앞 줄의 같은 열에 있는 좌석을 말한다.)</li>
</ul>

<p>위의 조건을 모두&nbsp;만족시키면서 솔로와 커플을 앉힐 수 있는지 판단해야 한다.</p>

<p><em><strong>유의 :</strong> </em>&quot;어떤 사람&quot;은 예약 손님, 솔로, 커플을 모두 포함한다.</p>

<p><strong><em>유의 : </em></strong>예약하고 온 손님은 관대해서 위의 조건들을 안 따진다.</p>

### 입력

<p>첫째 줄에 테스트 케이스의 수 T (T &le; 20)가 주어진다.</p>

<p>각 테스트 케이스마다 첫째 줄에&nbsp;세 정수 R, S, C(1 &le; R &le; 20, 0 &le; S &le; 30, 0 &le; C &le; 30)가 주어지고, 그 다음 R개의 줄은 영화관 가장 앞쪽에 있는 좌석줄부터 시작하여 가장 뒤쪽에 있는 좌석줄과 대응된다. 각 줄은 길이 8의 바이너리 문자열인데, 0은 비예약석을 1은 예약석을 의미한다.</p>

### 출력

<p>각 테스트 케이스마다 명시한 조건들을 모두 만족시킬 수 있으면 &quot;YES&quot; 아니면 &quot;NO&quot;를 출력한다.</p>
