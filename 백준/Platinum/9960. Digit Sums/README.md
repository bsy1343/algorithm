### 문제 설명

<p>Given 3 positive integers A, B and C, find how many positive integers less than or equal to A, when expressed in base B, have digits which sum to C.</p>

### 입력

<p>Input will consist of a series of lines, each containing three integers, A, B and C, 2 &le; B &le; 100, 1&nbsp;&le;&nbsp;A, C &le; 1,000,000,000. The numbers A, B and C are given in base 10 and are separated by one or more blanks. The input is terminated by a line containing three zeros.</p>

### 출력

<p>Output will be the number of numbers, for each input line (it must be given in base 10).</p>
