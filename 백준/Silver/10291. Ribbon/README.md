### 문제 설명

<p>Jim wants to make a lot of bows for her birthday party. She buys many rolls of ribbon in different colors. When she starts to make a bow, she finds out that the first and the last inch of every roll are dirty and cannot be used. To make bows, she has to cut the ribbon as long as possible. However, each piece of ribbon must have the same length.</p>

### 입력

<p>The input must be read from standard input. Each line starts with integer n, which is the number of rolls, 2 &lt;= n &lt;= 10. The next n numbers are the ribbon&#39;s length of each roll, k, where 5 &lt;= k &lt;= 500000. Each number is separated by a space.</p>

### 출력

<p>For each line shows two integers. The first number is the length of piece of ribbon after cut off. The last number is the total amount of bows that Jim can make. Each number is separated by a space.&nbsp;</p>
