### 문제 설명

<p>One of the oldest and best-known unsolved problems in Mathematics is Goldbach&#39;s conjecture. It states that every even integer greater than 2 can be expressed as the sum of two primes. The expression of a given even number as a sum of two primes is called a Goldbach partition of that number. The following are examples of Goldbach partitions for some even numbers:</p>

<ul>
	<li>4 = 2+2</li>
	<li>10 = 3+7 = 5+5</li>
	<li>100 = 3 + 97 = 11 + 89 = 17 + 83 = 29 + 71 = 41 + 59 = 47 + 53</li>
</ul>

<p>Your task is to write a program to check the number of Goldbach partitions for a given even number.</p>

### 입력

<p>An even integer n (4 &lt;= n &lt;= 10000000) will be given in each input line. The end of input is indicated by a number 0.</p>

### 출력

<p>For each line of input, print an integer in one line showing the number of Golbach partitions.</p>
