### 문제 설명

<p>준규가 가지고 있는 동전은 총 N종류이고, 각각의 동전을 매우 많이 가지고 있다.</p>

<p>동전을 적절히 사용해서 그 가치의 합을 K로 만들려고 한다. 이때 필요한 동전 개수의 최솟값을 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 N과 K가 주어진다. (1 &le; N &le; 10, 1 &le; K &le; 100,000,000)</p>

<p>둘째 줄부터 N개의 줄에 동전의 가치 A<sub>i</sub>가 오름차순으로 주어진다. (1 &le; A<sub>i</sub> &le; 1,000,000, A<sub>1</sub> = 1, i&nbsp;&ge; 2인 경우에&nbsp;A<sub>i</sub>는 A<sub>i-1</sub>의 배수)</p>

### 출력

<p>첫째 줄에 K원을 만드는데 필요한 동전 개수의 최솟값을 출력한다.</p>
