### 문제 설명

<p>N개의 수 A<sub>1</sub>, A<sub>2</sub>, ..., A<sub>N</sub>이 입력으로 주어진다. 총 M개의 구간 i, j가 주어졌을 때, i번째 수부터 j번째 수까지&nbsp;합을 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 수의 개수 N이 주어진다. (1 &le; N &le; 100,000) 둘째 줄에는 A<sub>1</sub>, A<sub>2</sub>, ..., A<sub>N</sub>이 주어진다. (-1,000 &le; A<sub>i</sub> &le; 1,000) 셋째 줄에는 구간의 개수 M이 주어진다. (1 &le; M &le; 100,000) 넷째 줄부터 M개의 줄에는 각 구간을 나타내는 i와 j가 주어진다. (1 &le; i &le; j &le; N)</p>

### 출력

<p>총 M개의 줄에 걸쳐 입력으로 주어진 구간의 합을 출력한다.</p>
