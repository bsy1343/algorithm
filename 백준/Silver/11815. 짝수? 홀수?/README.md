### 문제 설명

<p>B를&nbsp;A로 나누었을 때 나머지가 0 이라면 A는 B의 약수라고 할 수 있다. (A &gt; 0, B &gt; 0) 예를 들면&nbsp;15 의 약수는&nbsp;1, 3, 5, 15 이다.</p>

<p>주어진 수가 가지는 약수 개수가 홀수인지 짝수인지 판별해보자.</p>

### 입력

<p>첫 번째 줄에는 전체 테스트 개수 (N)&nbsp;가 주어진다.&nbsp;(1 &le; N &le; 100)</p>

<p>두 번째 줄에는 약수 개수를 판별할&nbsp;수 (X)&nbsp;가 주어진다&nbsp;(1 &le; X&nbsp;&le; 10<sup>18</sup>).</p>

### 출력

<p>주어진 수의 약수 개수가 홀수이면 1, 짝수이면 0 을 출력하시오.</p>
