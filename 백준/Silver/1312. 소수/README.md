### 문제 설명

<p>피제수(분자) A와 제수(분모) B가 있다. 두 수를 나누었을 때, 소숫점 아래 N번째 자리수를 구하려고 한다. 예를 들어, A=3, B=4, N=1이라면, A&divide;B=0.75 이므로 출력 값은 7이 된다.</p>

### 입력

<p>첫 번째 줄에 A와 B(1 &le; A, B &le; 100,000), N(1 &le; N &le; 1,000,000)이 공백을 경계로 주어진다.</p>

### 출력

<p>A&divide;B를 했을 때, 소숫점 아래 N번째 수를 출력한다.</p>
