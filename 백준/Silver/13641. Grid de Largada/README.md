### 문제 설명

<p>Na Nlog&ocirc;nia, vai ser realizada a sensacional final mundial da f&oacute;rmula 17. Os competidores se alinham na largada e disputam a corrida. Voc&ecirc; vai ter acesso aos grids de largada e de chegada. A quest&atilde;o &eacute; determinar o n&uacute;mero m&iacute;nimo de ultrapassagens que foram efetuadas durante a competi&ccedil;&atilde;o.</p>

### 입력

<p>Cada caso de teste utiliza tr&ecirc;s linhas. A primeira linha de um caso de teste cont&eacute;m um inteiro N indicando o n&uacute;mero de competidores. Cada competidor &eacute; identificado com um n&uacute;mero de 1 a N. A segunda linha de cada caso tem os N competidores, em ordem do grid de largada. A terceira linha de cada caso tem os mesmos competidores, por&eacute;m agora na ordem de chegada.</p>

<p>Restri&ccedil;&otilde;es</p>

<ul>
	<li>2 &le; N &le; 24</li>
</ul>

### 출력

<p>Para cada caso de teste imprima uma linha contendo um &uacute;nico n&uacute;mero inteiro, que indica o n&uacute;mero m&iacute;nimo de ultrapassagens necess&aacute;rias para se chegar do grid de largada ao grid de chegada.</p>
