### 문제 설명

<p>We build the infinite sequence of uppercase letters (A-Z) which starts with single-letter strings (A, B, &hellip;, Z), continues with two-letter strings (AA, AB, &hellip;, AZ, BA, BB, &hellip;, BZ, &hellip;, ZZ), then threeletter strings and so on. The same-length strings are ordered lexicographically. We are interested in finding which letter sits at a given index in the sequence.</p>

### 입력

<p>The letter index, ranging between 0 and 2*10<sup>9</sup> .</p>

### 출력

<p>The letter at the specified index.</p>
