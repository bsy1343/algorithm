### 문제 설명

<p>어떤 정수 A가 있으면 그 수를 A = a<sub>1</sub> * a<sub>2</sub> * a<sub>3</sub> * a<sub>4</sub> ... * a<sub>n</sub>으로 했을 때 A&#39; = a<sub>1</sub> + a<sub>2</sub> + a<sub>3</sub> ... + a<sub>n</sub>이 성립하면 &quot;A는 A&#39;으로 변할 수 있다&quot;라고 한다. (a<sub>i</sub>는 정수) 만약 A&#39;이 A&quot;으로 변할 수 있으면 &quot;A는 A&quot;으로 변할 수 있다&quot;라고 한다.</p>

<p>이때 A와 B가 주어지면 A는 B로 변할 수 있는지 판별하시오.</p>

### 입력

<p>첫째 줄에는 테스트 케이스의 개수 T(1 &le; T &le; 100)이 주어진다. 테스트 케이스마다 두 정수 A, B(-2<sup>31</sup> &le; A, B &le; 2<sup>31</sup>-1)가 주어진다.</p>

### 출력

<p>각각의 테스트 케이스마다 한 줄에 변할 수 있으면 yes, 아니면 no를 출력한다.</p>

### 힌트

<p>6 = 2*3, 2+3=5이므로 성립한다.</p>
