### 문제 설명

<p>$n$개의 자연수로 이루어진 수열이 주어질 때, 특정 구간 $[i,j]$ ($i&le;j)$의 합이 $k$보다 큰 모든 쌍 $(i, j)$의 개수를 출력하시오.</p>

### 입력

<p>첫째 줄에는 자연수의 개수 $n$이 주어진다. ($1&le;n&le;100\,000$)</p>

<p>다음 줄에는 자연수 $n$개가 주어진다. 자연수는 $100\,000$보다 크지 않다.</p>

<p>그 다음 줄에는 자연수 $k$가 주어진다. ($1&le;k&le;1\,000\,000\,000$)</p>

### 출력

<p>특정 구간 $[i,j]$의 합이 $k$보다 큰 모든 쌍 $(i,j)$의 개수를 출력하시오.</p>
