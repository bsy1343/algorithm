### 문제 설명

<p>피카츄는 &quot;pi&quot;, &quot;ka&quot;, &quot;chu&quot;를 발음할 수 있다. 따라서, 피카츄는 이 세 음절을 합친 단어만 발음할 수 있다. 예를 들면, &quot;pikapi&quot;와 &quot;pikachu&quot;가 있다.</p>

<p>문자열 S가 주어졌을 때, 피카츄가 발음할 수 있는 문자열인지 아닌지 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 문자열 S가 주어진다. 문자열은 알파벳 소문자로 이루어진 문자열이며, 길이는 5000을 넘지 않는다.</p>

### 출력

<p>문자열 S가 &quot;pi&quot;, &quot;ka&quot;, &quot;chu&quot;를 이어 붙여서 만들 수 있으면 &quot;YES&quot;를 아니면 &quot;NO&quot;를 출력한다.</p>
