### 문제 설명

<p>(String permutation) Write a recursive method to print all the permutations of a string. The user need to enter the string which consists of a set of characters.</p>

### 입력

<p>First line of the input contains <em>T</em> (1 &le;&nbsp;<em>T</em>&nbsp;&le;&nbsp;200), the number of test cases. For each test case, there will a string of characters, <em>L</em> (1 &le; <em>L</em> &le; 5).</p>

### 출력

<p>For each test case, output a line in the format &ldquo;Case # x:&rdquo; where x is the case number (starting with 1), follow by the set of string permutation.</p>
