<p>Em diversas competi&ccedil;&otilde;es acad&ecirc;micas, como a Olimp&iacute;ada Brasileira de Inform&aacute;tica (OBI), uma certa quantidade de competidores se classifica de uma fase para a fase seguinte, garantindo uma das vagas dispon&iacute;veis.
