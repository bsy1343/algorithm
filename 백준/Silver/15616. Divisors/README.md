### 문제 설명

<p>You are given a sequence of n&nbsp;integers a<sub>1</sub>, a<sub>2</sub>, ..., a<sub>n</sub>. You should determine the number of such ordered pairs (i, j), that i, j&nbsp;&isin;&nbsp;{1, ..., n}, i &ne; j&nbsp;and a<sub>i</sub>&nbsp;is a divisor of a<sub>j</sub>.</p>

### 입력

<p>The first line of input contains one integer n&nbsp;(1 &le; n &le; 2,000,000). The second line contains a sequence of n&nbsp;integers a<sub>1</sub>, a<sub>2</sub>, ..., a<sub>n</sub> (1 &le; a<sub>i</sub> &le; 2,000,000).</p>

### 출력

<p>In the first and only line of output should contain one integer, denoting the number of pairs sought.</p>

### 힌트

<p>There are 6&nbsp;pars with the specified properties: (1, 2), (1, 4), (1, 5), (4, 1), (4, 2), (4, 5).</p>
