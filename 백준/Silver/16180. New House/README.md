### 문제 설명

<p>&nbsp;</p>

<p>Johan wants to build a new house and he wants his house as large as it can. Given an&nbsp;<i>N</i>&nbsp;x&nbsp;<i>N</i>&nbsp;grid land, find the largest square size that fit in the free area.&nbsp;</p>

### 입력

<p>The first line of input contains an integer&nbsp;<i>T</i>, the number of test cases follow.</p>

<p>Each test case starts with an integer&nbsp;<i>N</i>&nbsp;(1 &lt;=&nbsp;<i>N</i>&nbsp;&lt;= 10) denoting the size of the grid land. The next&nbsp;<i>N</i>&nbsp;lines each contains&nbsp;<i>N</i>&nbsp;characters. Each character be either &#39;.&#39; (free area) or &#39;#&#39; (obstacle). Your house should be build on free area.</p>

### 출력

<p>For each test case, output in a line the size of the largest square on free area.</p>
