# [Silver V] 내접사각형 만들기 - 17177

[문제 링크](https://www.acmicpc.net/problem/17177)

### 성능 요약

시간 제한: 1 초, 메모리 제한: 512 MB

### 통계

제출: 911, 정답: 528, 맞힌 사람: 456, 정답 비율: 58.991%

### 분류

수학, 기하학, 피타고라스 정리

### 문제 설명

<p>어떤 사각형의 네 꼭짓점이 원과 접할 때 이 사각형이 원에 내접한다고 한다. 세 선분의 길이가 주어지고 그중 가장 긴 선분의 길이가 내접할 원의 지름과 같을 때,&nbsp;나머지 한 선분의 길이를 구하여라.</p>

### 입력

<p>첫 번째 줄에&nbsp;세 변의 길이를 나타내는 세 정수&nbsp;<em>a, b, c</em>가 주어진다.&nbsp;(1 &le;&nbsp;<em>a, b, c</em>&nbsp;&le; 100) 세 선분은 내림차순으로 주어진다.</p>

### 출력

<p>첫째 줄에 나머지 한 변의 길이를&nbsp;출력한다. 원에 내접하는 사각형을 만들 수 없다면 -1을 출력한다. 단, 정답은 정수로 나오는 케이스만 주어진다.&nbsp;</p>

### 힌트

<p>톨레미의 정리에 따르면 다음과 같이 원에 내접한 사각형은&nbsp;마주 보는 두 쌍의 변의 곱의 합이 두 대각선의 곱과 같다.</p>
