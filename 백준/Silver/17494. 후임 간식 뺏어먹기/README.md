# [Silver I] 후임 간식 뺏어먹기 - 17494

[문제 링크](https://www.acmicpc.net/problem/17494)

### 문제 설명

<p>승엽이는 전역을 일주일 앞둔 최고참 말년 병장이다. 부대 내에서 아무도 승엽이를 막을 수 없다.</p>

<p>승엽이의 취미는 후임들이 작업이나 근무 나갔을 때 후임들의 관물장을 열어 간식을 뺏어먹는 것이다.</p>

<p>승엽이가 배부름을 느낄 수 있는 포만감 정도인 &lsquo;충분 포만감&rsquo; <em>M</em> 은 정해져 있고, 간식을 먹어 <em>M</em> 만큼 포만감을 느끼는 것이 목적이다. 승엽이는 <em>M</em> 이상 포만감을 채우지 못하면 화가 난다.</p>

<p>승엽이는 누구 간식을 뺏어 먹을지 고민하다가, 가장 만만한 후임인 현철이에게 자신이 배부를 만큼 간식을 가져오라고 시켰다.</p>

<p>현철이는 후임들을 괴롭히는 승엽이가 괘씸해서, 최대한 승엽이가 덜 좋아하는 간식들만 골라서 가져갈 생각이다.</p>

<p>평소 승엽이는 PX에 판매하는 모든 간식에 대해 얼마나 맛있는지 만족도를 평가해 두었다.</p>

<p>고생하는 현철이를 도와 승엽이가 배를 채우며 얻을 수 있는 최소의 만족도를 계산해 보자!</p>

### 입력

<p>첫 줄에 후임의 수 <em>N</em> (1 &le; <em>N</em> &le; 100), 승엽이의 충분 포만감&nbsp;<em>M</em> (1 &le; <em>M</em> &le; 100,000) 이 공백을 두고 주어진다.</p>

<p>다음 <em>N&nbsp;</em>개의 줄에 각각 해당 후임의 간식을 뺏어 먹고 얻을 수 있는 포만감 <em>W</em> (1 &le; <em>W</em> &le; 1,000)와 만족도 <em>H</em> (1 &le; <em>H</em> &le; 1,000)가 공백을 두고 주어진다.</p>

### 출력

<p>첫 줄에 승엽이가 충분 포만감 만큼 포만감을 채우며 얻을 수 있는 최소의 만족도를 한 줄로 출력한다.</p>

<p>승엽이가 간식으로 충분 포만감을 채울 수 없는 경우 &ldquo;<code>죄송합니다 한승엽 병장님</code>&rdquo; 을 한 줄로 출력한다.</p>
