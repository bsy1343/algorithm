### 문제 설명

<p>79를 영어로 읽되 숫자 단위로 하나씩 읽는다면 &quot;seven nine&quot;이 된다. 80은 마찬가지로 &quot;eight zero&quot;라고 읽는다. 79는 80보다 작지만, 영어로 숫자 하나씩 읽는다면 &quot;eight zero&quot;가 &quot;seven nine&quot;보다 사전순으로 먼저 온다.</p>

<p>문제는 정수 M, N(1 &le; M &le;&nbsp;N &le; 99)이 주어지면 M 이상 N 이하의 정수를 숫자 하나씩 읽었을 때를 기준으로 사전순으로 정렬하여 출력하는 것이다.</p>

### 입력

<p>첫째 줄에 M과 N이 주어진다.</p>

### 출력

<p>M 이상 N 이하의 정수를 문제 조건에 맞게 정렬하여 한 줄에 10개씩 출력한다.</p>
