### 문제 설명

<p>0과 1로 이루어진 길이가 무한한 문자열 <em>X</em>가 있다. 이 문자열은 다음과 같은 과정으로 만들어진다.</p>

<ol>
	<li><em>X</em>는 맨 처음에 &quot;0&quot;으로 시작한다.&nbsp;</li>
	<li><em>X</em>에서 0을 1로, 1을 0으로&nbsp;뒤바꾼 문자열 <em>X&#39;</em>을 만든다.</li>
	<li><em>X</em>의 뒤에 <em>X&#39;</em>를 붙인 문자열을 <em>X</em>로 다시 정의한다.&nbsp;</li>
	<li>2~3의 과정을 무한히 반복한다.</li>
</ol>

<p>즉, <em>X</em>는 처음에 &quot;0&quot;으로 시작하여 &quot;01&quot;이 되고, &quot;0110&quot;이 되고, &quot;01101001&quot;이 되고, ⋯&nbsp;의 과정을 거쳐 다음과 같이 나타내어진다.</p>

<p>&nbsp;&nbsp; &nbsp;&quot;011010011001011010010110011010011001011001101001⋯⋯&quot;</p>

<p>자연수 <em>k</em>가 주어졌을 때 <em>X</em>의 <em>k</em>번째에는 무슨 문자가 오는지 구하여라.</p>

### 입력

<p>첫 번째 줄에 자연수 <em>k</em> (1 &le;&nbsp;<em>k</em> &le;&nbsp;10<sup>18</sup>) 가 주어진다.</p>

### 출력

<p>첫 번째 줄에 <em>k</em>번째에 오는 문자를 출력하라.</p>
