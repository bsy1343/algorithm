### 문제 설명

<p>Consider a N &times; N &times; N lattice. One corner is at (0,0,0) and the opposite one is at (N, N, N). How many lattice points are visible from corner at (0,0,0)? A point X is visible from point Y iff no other lattice point lies on the segment joining X and Y.</p>

### 입력

<p>The first line contains the number of test cases T. The next T lines contain an interger N.</p>

### 출력

<p>Output T lines, one corresponding to each test case.</p>

### 제한

<ul>
	<li>T &le; 100, 1 &le; N &le; 100</li>
</ul>
