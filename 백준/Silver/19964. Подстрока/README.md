### 문제 설명

<p>Недавно разведка перехватила зашифрованное сообщение  --- строку $s$. Все ресурсы аналитического центра, в котором вы работаете, были брошены на его декодирование.</p>

<p>Ваш отдел занимается шифрами нового поколения. На данный момент известно всего $n$ таких шифров. Для каждого из них есть три характерных параметра  --- целые числа $l$, $r$ и строка $t$.&nbsp; Пусть строка $g$ была получена в результате применения этого метода. Тогда строка $g_lg_{l+1} \ldots g_{r-1}g_r$ (здесь $g_i$  --- это $i$-й символ строки $g$) содержит $t$ как подстроку.</p>

<p>Вам поручено определить для каждого типа шифрования, могло ли сообщение $s$ быть получено в результате его применения.</p>

### 입력

<p>Первая строка входного файла содержит строку $s$ &nbsp;($1 \le |s| \le 100\,000$, где $|s|$  --- длина строки $s$).&nbsp;</p>

<p>Вторая строка входного файла содержит целое число $n$  --- количество типов шифрования ($1 \le n \le 100\,000$). Последующие $n$ строк содержат по два целых числа $l_i$, $r_i$ и строку $t_i$, разделенные пробелами  --- характерные параметры $i$-го метода шифрования ($1 \le l_i \le r_i \le |s|$).</p>

<p>Все строки состоят из строчных букв латинского алфавита. Суммарная длина всех $t_i$ не превосходит $100\,000$.</p>

### 출력

<p>Выведите одну строку  --- для каждого типа шифрования &lt;&lt;<code>+</code>&gt;&gt;, если сообщение $s$ могло быть получено в результате его применения, или &lt;&lt;<code>-</code>&gt;&gt; в противном случае.</p>
