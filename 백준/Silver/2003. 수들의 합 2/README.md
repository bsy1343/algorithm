### 문제 설명

<p>N개의 수로 된 수열 A[1], A[2], &hellip;, A[N] 이 있다. 이 수열의 i번째 수부터 j번째 수까지의 합 A[i] + A[i+1] + &hellip; + A[j-1] + A[j]가 M이 되는 경우의 수를 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 N(1 &le; N &le; 10,000), M(1 &le; M &le; 300,000,000)이 주어진다. 다음 줄에는 A[1], A[2], &hellip;, A[N]이 공백으로 분리되어 주어진다. 각각의 A[x]는 30,000을 넘지 않는 자연수이다.</p>

### 출력

<p>첫째 줄에 경우의 수를 출력한다.</p>
