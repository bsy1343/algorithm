### 문제 설명

<p>An integer x is said to be a right-coupled number, if you can find two integers, say 0 &lt; a &le; b &le; x such that a &times; b = x and a/b &ge; 0.5. In this problem, your task is to determine whether a given integer is a right-coupled number or not.</p>

### 입력

<p>The first line of the input is an integer N denoting the number of test cases. Each test case is in one line, which contains a single integer 0 &lt; x &lt; 2<sup>15</sup>.</p>

### 출력

<p>If the given integer x is a right-coupled number, output 1; otherwise, output 0. Each is in a single line.</p>

### 제한

<ul>
	<li>1 &le; N &le; 1000</li>
	<li>0 &lt; x &lt; 2<sup>15</sup></li>
</ul>
