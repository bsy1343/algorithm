### 문제 설명

<p>Робот Бендер решил открыть аттракцион &lt;&lt;Кручу-Верчу&gt;&gt; с целью своего обогащения. Аттракцион состоит в следующем: Бендер прячет шарик под одним из $k$ одинаковых стаканчиков, расположенных на позициях от 1 до $k$, затем $n$ раз быстро меняет местами какие-то пары стаканчиков, после чего предлагает отгадать под каким из стаканчиков сейчас шарик.</p>

<p>Бендер --- робот, поэтому действует он по определенной программе. Бендер строит последовательность целых чисел $x_i$, при этом $x_1 = c$, а $x_i = a \cdot x_{i-1} + b$ для $i &gt; 1$.</p>

<p>На $i$-ом шаге Бендер меняет местами стаканчики на позициях с номерами $(x_i \bmod k) + 1$ и $\left( (x_i + 1) \bmod k \right) + 1$.</p>

<p>В начале робот прячет шарик под стаканчик на позиции с номером $r$. Бендер хочет, чтобы после $n$ обменов шарик оказался под стаканчиком на позиции с номером $l$.</p>

<p>Найдите такие $a$, $b$ и $c$, чтобы стаканчик с шариком переместился с $r$-й позиции на~$l$-ю.</p>

### 입력

<p>В единственной строке входного файла четыре целых числа $n$, $k$, $r$ и $l$ ($1 \le n \le 10^5$; $2 \le k \le 10$; $1 \le r, l \le k$).</p>

### 출력

<p>Если таких чисел не существует, выведите в выходной файл единственное слово &lt;&lt;<code>Impossible</code>&gt;&gt;. Иначе выведите три целых неотрицательных числа $a$, $b$ и $c$. Числа не должны превосходить $1000$.</p>
