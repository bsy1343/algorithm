### 문제 설명

<p>Little Q likes solving math problems very much. Unluckily, however, he does not have a good spatial ability. Every time he meets a 3D geometry problem, he will struggle to draw a picture.</p>

<p>Now he meets a 3D geometry problem again. This time, he doesn&#39;t want to struggle anymore. As a result, he turns to you for help.</p>

<p>Given a rectangular parallelepiped with length $a$, width $b$, and height $c$, please write a program to display it.</p>

### 입력

<p>The first line contains an integer $T$ ($1 \leq T \leq 50$), the number of test cases. For each test case:</p>

<p>The only line contains three integers $a$, $b$, $c$ ($1 \leq a,b,c \leq 20$), denoting the dimensions of the rectangular parallelepiped.</p>

### 출력

<p>For each test case, print several lines to display the rectangular parallelepiped. See the sample output for details.</p>
