### 문제 설명

<p>Задано натуральное число $A$. Необходимо представить его в виде суммы двух неотрицательных целых чисел $B$ и $C$ так, чтобы сумма цифр десятичных представлений чисел $B$ и $C$ была как можно больше.</p>

### 입력

<p>Входной файл содержит целое число $A$ ($1 \le A \le 10^{18}$).</p>

### 출력

<p>В первой строке выходного файла выведите $s$ --- максимальную возможную сумму цифр чисел $B$ и $C$. Во второй строке выведите через пробел сами числа $B$ и $C$, сумма которых равна $A$, а сумма цифр которых равна $s$. Если оптимальных ответов несколько, то выведите любой из них.</p>
