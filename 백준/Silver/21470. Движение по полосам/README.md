### 문제 설명

<p><img alt="" src="%EB%B0%B1%EC%A4%80/Silver/21470.%E2%80%85%D0%94%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5%E2%80%85%D0%BF%D0%BE%E2%80%85%D0%BF%D0%BE%D0%BB%D0%BE%D1%81%D0%B0%D0%BC/53d93e37.png" data-original-src="%EB%B0%B1%EC%A4%80/Silver/21470.%E2%80%85%D0%94%D0%B2%D0%B8%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5%E2%80%85%D0%BF%D0%BE%E2%80%85%D0%BF%D0%BE%D0%BB%D0%BE%D1%81%D0%B0%D0%BC/53d93e37.png" data-original-src="https://upload.acmicpc.net/8ad22aeb-26ae-457a-9c9b-2e1e633d94e6/-/preview/" onerror="this.onerror=null;this.src=this.dataset.originalSrc" style="width: 161px; height: 87px; float: right;" />При организации движения по сложным перекресткам, для того, чтобы траектории водителей, выполняющих различные маневры не пересекались, вводят ограничения на возможные маневры водителей, в зависимости от того, по какой полосе движения водитель подъехал к перекрестку. Для этого используется знак &lt;&lt;движение по полосам&gt;&gt;, на рисунке справа приведен пример такого знака, установленного перед одним из перекрестков в Санкт-Петербурге.</p>

<p>Рассмотрим дорогу, подходящую к перекрестку, на котором сходится $m$ дорог. Водитель, подъезжающий к перекрестку по этой дороге, потенциально может продолжить свое движение в $m$ различных направлениях --- обратно по дороге, по которой он приехал, а также по одной из оставшихся $m - 1$ дорог. Пронумеруем возможные направления числами от 1 до $m$ слева направо с точки зрения подъезжающего водителя, номер 1 получит разворот и возврат по дороге, по которой водитель подъезжал к перекрестку, номер 2 --- поворот на самую левую из дорог, и т. д.</p>

<p>Пусть дорога содержит $n$ полос для движения. Пронумеруем полосы от 1 до $n$ слева направо, самая левая полоса получит номер 1, следующая номер 2, и т. д. Знак &lt;&lt;движение по полосам&gt;&gt; разрешает каждой из полос движение по некоторым из $m$ возможных направлений. При этом должны выполняться следующие условия:</p>

<ol>
	<li>если с $i$-й полосы разрешено движение в $a$-м направлении, а с $j$-й&nbsp; полосы --- в $b$-м направлении, причем $i &lt; j$, то $a \le b$;</li>
	<li>с каждой полосы разрешено движение хотя бы в одном направлении;</li>
	<li>в каждом направлении разрешено движение хотя бы с одной полосы.</li>
</ol>

<p>Инспекция по безопасности дорожного движения заинтересовалась, а сколько различных знаков &lt;&lt;движение по полосам&gt;&gt; можно установить перед таким перекрестком. Помогите им найти ответ на этот вопрос.</p>

### 입력

<p>Входной файл содержит два целых числа: $m$ и $n$ ($2 \le m \le 50$, $1 \le n \le 15$).</p>

### 출력

<p>В выходной файл выведите одно число --- количество возможных знаков &lt;&lt;движение по полосам&gt;&gt;, которые можно установить перед перекрестком.</p>
