### 문제 설명

<p>Циклическим сдвигом строки s<sub>0</sub>s<sub>1</sub>&hellip;s<sub>n-1</sub> на k позиций назовем строку s<sub>k</sub>s<sub>k+1</sub>&hellip;s<sub>n</sub>s<sub>1</sub>..s<sub>k-1</sub>. Например, циклическим сдвигом строки &laquo;abcde&raquo; на две позиции является строка &laquo;cdeab&raquo;. В этой задаче далее будут рассматриваться только строки, состоящие из десятичных цифр от 0 до 9. Произвольной такой строке, первый символ которой не является нулем, можно сопоставить число, десятичной записью которого она является. Строкам, которые начинаются с нуля, никакое число сопоставляться не будет. Например, строке 123 сопоставляется число сто двадцать три, а строке 0123 не сопоставляется никакое число.</p>

<p>Пусть заданы две строки: s и t. Обозначим как S набор всех циклических сдвигов строки s, а как T &ndash; набор всех циклических сдвигов строки t. Например, если s = &laquo;1234&raquo;, то S содержит строки &laquo;1234&raquo;, &laquo;2341&raquo;, &laquo;3412&raquo;, &laquo;4123&raquo;. Как NUM(A) обозначим набор чисел, соответствующих строкам из набора A.&nbsp;</p>

<p>Требуется написать программу, которая по строкам s и t определит, максимальное число, представимое в виде разности x &ndash; y, где x принадлежит NUM(S), а y принадлежит NUM(T).&nbsp;</p>

<p>Например, если s = &laquo;25&raquo;, t = &laquo;12&raquo;, то NUM(S) содержит числа 25 и 52, NUM(T) &ndash; числа 12 и 21, их попарные разности это 25 &ndash; 12 = 13, 25 &ndash; 21 &nbsp;= 4, 52 &ndash; 12 = 40, 52 &ndash; 21 = 31. Из этих разностей максимальной является 40.</p>

### 입력

<p>Первая строка входного файла содержит строку s, вторая строка входного файла &ndash; строку t. Обе строки непусты, содержат только цифры, из которых хотя бы одна не является нулем, и имеют длину не более 3000 символов.</p>

### 출력

<p>В выходной файл выведите искомое число без ведущих нулей.</p>
