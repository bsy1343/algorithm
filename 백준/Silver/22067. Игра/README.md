### 문제 설명

<p>Петя посоветовал Васе новую игру. Эта игра очень понравилась Васе, и он захотел выиграть у Пети.</p>

<p>Игра представляет собой систему уровней, комнат и односторонних коридоров между ними. На&nbsp;<i>i</i>-м уровне ровно&nbsp;<i>i</i>&nbsp;комнат, пронумерованных от 1 до&nbsp;<i>i</i>, из каждой комнаты выходит ровно два коридора. Коридоры из комнаты на&nbsp;<i>i</i>-м уровне, имеющей номер&nbsp;<i>j</i>, ведут на уровень с номером&nbsp;<i>i</i>&nbsp;+&nbsp;1 в комнаты с номерами&nbsp;<i>j</i>&nbsp;и&nbsp;<i>j</i>&nbsp;+&nbsp;1. Каждый коридор имеет свою длину. Цель игры заключается в том, чтобы, начав в единственной комнате первого уровня, дойти до последнего уровня, преодолев минимальное расстояние.</p>

<p>Вася выбрал следующую стратегию в игре. Находясь в комнате, он бежит по более короткому коридору, ведущему из неё. Если же длины коридоров одинаковые, то он случайным образом равновероятно выбирает, по какому коридору побежать.</p>

<p>Вася заметил, что на одной и той же карте длина пути от первого уровня до последнего у него может быть разной, в зависимости от его случайных выборов. Он хочет посчитать математическое ожидание длины его пути.</p>

<p>Напомним определение математического ожидания случайной величины. Пусть величина принимает различные значения, причем значение&nbsp;<i>x<sub>k</sub></i>&nbsp;принимается с вероятностью&nbsp;<i>p<sub>k</sub></i>. Тогда математическим ожиданием называется сумма&nbsp;<i>x</i><sub>1</sub><i>p</i><sub>1</sub>&nbsp;+&nbsp;<i>x</i><sub>2</sub><i>p</i><sub>2</sub>&nbsp;+ ... +&nbsp;<i>x<sub>k</sub></i><i>p<sub>k</sub></i>&nbsp;+ ... (сумма берется по всем возможным значениям).</p>

### 입력

<p>Первая строка содержит одно натуральное число&nbsp;<i>t</i>&nbsp;&mdash; число тестовых примеров во входных данных. Далее следуют описания тестов.</p>

<p>Описание каждого теста состоит из&nbsp;<i>n</i>&nbsp;+&nbsp;1 строки. Первая строка содержит одно целое число&nbsp;<i>n</i>&nbsp;(1&nbsp;&le;&nbsp;<i>n</i>&nbsp;&le;&nbsp;1000), где&nbsp;<i>n</i>&nbsp;+&nbsp;1 &mdash; число уровней на карте.</p>

<p>Далее следует описание уровней,&nbsp;<i>i</i>-я строка содержит 2<i>i</i>&nbsp;целых чисел. Числа идут по парам и описывают длины коридоров,&nbsp;<i>j</i>-я пара чисел задает длины коридоров в комнаты&nbsp;<i>j</i>&nbsp;и&nbsp;<i>j</i>&nbsp;+&nbsp;1 на следующем уровне, соответственно. Длины коридоров не превышают 10<sup>9</sup>.</p>

<p>Сумма&nbsp;<i>n</i>&nbsp;по всем тестам не превосходит 1000.</p>

### 출력

<p>Для каждого теста в отдельной строке выведете матожидание длины пути. Ответ должен иметь относительную или абсолютную погрешность не более 10<sup>-6</sup>.</p>
