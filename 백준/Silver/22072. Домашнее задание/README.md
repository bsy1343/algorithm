### 문제 설명

<p>Сегодня на уроке информатики Петя узнал про системы счисления с различными основаниями. В системе счисления с основанием&nbsp;<i>k</i>&nbsp;используются цифры с весом от 0 до&nbsp;<i>k</i>&nbsp;&minus;&nbsp;1, для обозначения первых 10 цифр используются обычные цифры, а далее задействуются также буквы латинского алфавита и другие знаки.</p>

<p>Теперь Петя пытается решить следующее домашнее задание. Даны числа&nbsp;<i>x</i>,&nbsp;<i>y</i>&nbsp;и&nbsp;<i>z</i>, записанные с использованием десятичных цифр. Требуется найти все такие основания систем счисления&nbsp;<i>k</i>, что все цифры, использованные в числах, существуют в этой системе счисления, и при этом если считать, что&nbsp;<i>x</i>,&nbsp;<i>y</i>&nbsp;и&nbsp;<i>z</i>&nbsp;&mdash; числа, записанные в системе счисления с основанием&nbsp;<i>k</i>, то произведение&nbsp;<i>x</i>&nbsp;и&nbsp;<i>y</i>&nbsp;равно&nbsp;<i>z</i>.</p>

<p>Петя начал решать эту задачу и обнаружил, что при некоторых условиях подходит бесконечно много различных&nbsp;<i>k</i>. Он решил для начала понять, имеет ли место этот случай. Помогите ему это выяснить.</p>

### 입력

<p>Первая строка содержит число&nbsp;<i>t</i>&nbsp;&mdash; количество тестов во входных данных (1&nbsp;&le;&nbsp;<i>t</i>&nbsp;&le;&nbsp;1000).</p>

<p>Далее следует описание&nbsp;<i>t</i>&nbsp;тестов, каждый тест описывается тремя строками, которые содержат числа&nbsp;<i>x</i>,&nbsp;<i>y</i>&nbsp;и&nbsp;<i>z</i>, соответственно. Каждое из этих чисел положительно, состоит только из десятичных цифр и не содержит ведущих нулей. Длина каждого из чисел&nbsp;<i>x</i>&nbsp;и&nbsp;<i>y</i>&nbsp;не превышает 100, длина числа&nbsp;<i>z</i>&nbsp;не превышает 200.</p>

### 출력

<p>На каждый тест в отдельной строке выведите ответ на него &mdash; &laquo;<code>Infinity</code>&raquo;, если существует бесконечное число таких&nbsp;<i>k</i>, что если считать, что числа во вводе записаны в системе счисления с основанием&nbsp;<i>k</i>, то произведение&nbsp;<i>x</i>&nbsp;и&nbsp;<i>y</i>&nbsp;равно&nbsp;<i>z</i>, либо &laquo;<code>Finite</code>&raquo;, если таких&nbsp;<i>k</i>&nbsp;конечное число.</p>
