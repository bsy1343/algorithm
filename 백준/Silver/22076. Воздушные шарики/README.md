### 문제 설명

<p>Сегодня у Грю день рождения и миньоны решили подарить ему набор разноцветных воздушных шаров.</p>

<p>Грю исполняется ровно&nbsp;<i>k</i>&nbsp;лет, так что было решено подарить ему&nbsp;<i>k</i>&nbsp;шаров. Так как много шаров одинакового цвета &mdash; это скучно, вам было поручено выбрать из имеющихся шаров ровно&nbsp;<i>k</i>&nbsp;таким образом, чтобы количество различных цветов среди выбранных шаров было максимально.</p>

### 입력

<p>Первая строка содержит два целых числа&nbsp;<i>n</i>&nbsp;и&nbsp;<i>k</i>&nbsp;(1 &le;&nbsp;<i>k</i>&nbsp;&le;&nbsp;<i>n</i>&nbsp;&le; 10<sup>5</sup>) &mdash; количество шаров, имеющихся у миньонов и количество шаров, которые было решено подарить Грю. В следующей строке дано&nbsp;<i>n</i>&nbsp;чисел&nbsp;<i>a<sub>i</sub></i>&nbsp;(1 &le;&nbsp;<i>a<sub>i</sub></i>&nbsp;&le; 10<sup>9</sup>) &mdash; цвета шаров.</p>

### 출력

<p>Выведите ровно&nbsp;<i>k</i>&nbsp;целых чисел через пробел &mdash; цвета шаров, которые нужно подарить Грю. Если правильных ответов несколько, выведите любой из них.</p>
