### 문제 설명

<p>Перед организаторами финального этапа одного соревнования по программированию стоит сложная задача: спланировать все мероприятия. Спонсоры настаивают, чтобы их выступления были как можно длиннее, поэтому необходимо, чтобы разбор задач занял как можно меньше времени.</p>

<p>Вместе с участниками, на соревнования приехало&nbsp;<i>n</i>&nbsp;членов жюри, про каждого из которых известно, какие из предложенных участникам&nbsp;<i>m</i>&nbsp;задач он хочет разобрать. Каждый член жюри тратит на разбор любой задачи&nbsp;<i>t</i>&nbsp;секунд. На то, чтобы один из членов жюри сменил во время разбора другого, тратится&nbsp;<i>c</i>&nbsp;секунд. Если же проводящий разбор член жюри продолжает разбирать и следующую задачу, то потери времени нет.</p>

<p>Определите, какое наименьшее время может идти разбор всех задач, если задачи должны разбираться по порядку от первой до&nbsp;<i>m</i>-й и каждую задачу должен разбирать кто-либо из членов жюри, кто хочет ее разбирать.</p>

### 입력

<p>В первой строке задано число&nbsp;<i>T</i>&nbsp;&mdash; число тестов. Далее идёт описание&nbsp;<i>T</i>&nbsp;тестов.</p>

<p>В первой строке теста заданы целые числа&nbsp;<i>n</i>,&nbsp;<i>m</i>,&nbsp;<i>t</i>&nbsp;и&nbsp;<i>c</i>&nbsp;(1&nbsp;&le;&nbsp;<i>n</i>,&nbsp;<i>m</i>,&nbsp;<i>t</i>,&nbsp;<i>c</i>&nbsp;&le;&nbsp;100). Далее, в&nbsp;<i>n</i>&nbsp;строках заданы строки из&nbsp;<i>m</i>&nbsp;символов&nbsp;<code>0</code>&nbsp;или&nbsp;<code>1</code>, если&nbsp;<i>j</i>-й символ&nbsp;<i>i</i>-й строки равен&nbsp;<code>1</code>, это значит, что&nbsp;<i>i</i>-й член жюри хочет разбирать&nbsp;<i>j</i>-ю задачу, а если он равен&nbsp;<code>0</code>, то не хочет. Гарантируется, что каждую задачу хотя бы кто-то хочет разбирать.</p>

<p>Размер входного файла не превышает 2 мегабайт.</p>

### 출력

<p>Для каждого теста выведите единственное число&nbsp;&mdash; ответ на задачу.</p>
