### 문제 설명

<p>Все люди любят круглые числа. У этого явления достаточно много причин, главной из которых является то, что число, которое делится, например, на 100, существенно проще запомнить. К сожалению, в нашей жизни круглых чисел существенно меньше, чем всех остальных. Первоклассник Петя очень переживает по этому поводу. Поэтому он придумал свое определение круглого числа.</p>

<p>По версии Пети, число является круглым только в том случае, если из него можно удалить ровно&nbsp;<i>k</i>&nbsp;цифр так, что после этого число не будет содержать ведущих нулей, но будет делиться на 100. Например, при&nbsp;<i>k</i>&nbsp;равным 2, число 10304 является круглым, а число 1000&nbsp;&mdash; нет.</p>

<p>Теперь Петя хочет узнать, какие числа при каких&nbsp;<i>k</i>&nbsp;являются круглыми, а какие&nbsp;&mdash; нет. Помогите ему.</p>

### 입력

<p>Первая строка содержит целое число&nbsp;<i>n</i>&nbsp;&mdash; количество пар чисел&nbsp;<i>k</i>&nbsp;и&nbsp;<i>x</i>, про которые Петя хочет что-то узнать. Следующие&nbsp;<i>n</i>&nbsp;строк содержат по два целых положительных числа&nbsp;<i>k</i>&nbsp;и&nbsp;<i>x</i>&nbsp;каждая. Гарантируется, что любое число&nbsp;<i>k</i>&nbsp;строго меньше количества цифр в соответствующем числе&nbsp;<i>x</i>. Гарантируется, что все числа&nbsp;<i>x</i>&nbsp;не содержат ведущих нулей. Гарантируется, что сумма количеств цифр во всех числах&nbsp;<i>x</i>&nbsp;не превышает 10<sup>5</sup>.</p>

### 출력

<p>Для каждой пары чисел&nbsp;<i>k</i>&nbsp;и&nbsp;<i>x</i>&nbsp;из входного файла выведите -1 в случае, если число&nbsp;<i>x</i>&nbsp;не является круглым при данном&nbsp;<i>k</i>. В случае же, если число&nbsp;<i>x</i>&nbsp;являетсы круглым при данном&nbsp;<i>k</i>, выведите число&nbsp;<i>x</i>, из которого вычеркнули ровно&nbsp;<i>k</i>&nbsp;цифр, после чего оно стало делиться на 100. Выведенное вами число не должно содержать ведущих нулей, но может быть равно нулю.</p>
