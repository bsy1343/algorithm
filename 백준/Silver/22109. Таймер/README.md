### 문제 설명

<p>За хулиганство в школе Петя отбывает наказание под домашним арестом. В его комнате есть таймер, который показывает, сколько Петя уже пробыл под наказанием. Каждую секунду число на таймере увеличивается на единицу. Петя, желая сократить время своего наказания, нашел в нем уязвимость. Пусть таймер в настоящий момент показывает&nbsp;<i>a</i>&nbsp;секунд, тогда, использовав найденную уязвимость, Петя может сделать так, чтобы он показывал&nbsp;<i>b</i>&nbsp;секунд, где&nbsp;<i>b</i>&nbsp;&mdash;&nbsp;наименьшее число, большее либо равное&nbsp;<i>a</i>, которое делится на заданное число&nbsp;<i>y</i>.</p>

<p>Петя хочет, чтобы таймер показывал как можно больше секунд. Однако, Петя также не хочет попасться, поэтому он не будет использовать найденную уязвимость больше, чем&nbsp;<i>k</i>&nbsp;раз. Какое максимальное число на таймере Петя может получить через&nbsp;<i>t</i>&nbsp;реальных секунд? В момент, когда Петя обнаружил уязвимость, на таймере было число&nbsp;<i>x</i>.</p>

<p>Например, пусть в момент обнаружения уязвимости на таймере было число&nbsp;<i>x</i>&nbsp;=&nbsp;9,&nbsp;<i>y</i>&nbsp;=&nbsp;10, Петя планирует применить уязвимость не более&nbsp;<i>k</i>&nbsp;=&nbsp;10 раз и&nbsp;<i>t</i>&nbsp;=&nbsp;2. Тогда максимальное число, которое Петя может получить на таймере равно 30, для этого ему необходимо действовать так. Сразу применив уязвимость, он получает на таймере значение 10. Через 1 секунду на таймере будет значение 11, применив уязвимость, Петя получает на нем значение 20. Еще через секунду на таймере будет 21, применив уязвимость, Петя получит значение 30.</p>

### 입력

<p>Первая строка содержит целое число&nbsp;<i>T</i>&nbsp;(1&nbsp;&le;&nbsp;<i>T</i>&nbsp;&le;&nbsp;10<sup>4</sup>) &mdash; количество тестовых примеров. В каждой из следующих&nbsp;<i>T</i>&nbsp;строк содержится по четыре целых числа:&nbsp;<i>x</i>,&nbsp;<i>y</i>,&nbsp;<i>k</i>&nbsp;и&nbsp;<i>t</i>. (все числа находятся в диапазоне от 1 до 10<sup>9</sup>).</p>

### 출력

<p>Для каждого из&nbsp;<i>T</i>&nbsp;тестовых примеров выведите единственное число &mdash; максимальное возможное число, которое может показывать таймер через&nbsp;<i>t</i>&nbsp;секунд,</p>
