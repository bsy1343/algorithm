### 문제 설명

<p>Назовем&nbsp;<i>епрестановкой</i>&nbsp;такую перестановку&nbsp;<i>z</i>&nbsp;чисел от 1 до&nbsp;<i>n</i>, которая меняет первые два элемента местами:&nbsp;<i>z</i>&nbsp;= [2, 1, 3, 4, ...,&nbsp;<i>n</i>].</p>

<p>Пусть у вас епрестановка&nbsp;<i>z</i>&nbsp;и перестановка&nbsp;<i>p</i>&nbsp;=&nbsp;[<i>p</i><sub>1</sub>,&nbsp;<i>p</i><sub>2</sub>, ...,&nbsp;<i>p</i><sub><i>n</i></sub>] чисел от 1 до&nbsp;<i>n</i>. Рассмотрим&nbsp;<i>n</i>&nbsp;различных предметов, расположенных на позициях, пронумерованных от 1 до&nbsp;<i>n</i>. Предметы можно переставлять в соответствии с перестановками: после применения перестановки&nbsp;<i>q</i>, предмет, который находится на позиции&nbsp;<i>j</i>, перемещается на позицию&nbsp;<i>q<sub>j</sub></i>&nbsp;для каждого&nbsp;<i>j</i>&nbsp;от 1 до&nbsp;<i>n</i>.</p>

<p>Задано&nbsp;<i>m</i>&nbsp;пар целых чисел&nbsp;<i>a<sub>i</sub></i>&nbsp;и&nbsp;<i>b<sub>i</sub></i>. Необходимо для каждого&nbsp;<i>i</i>&nbsp;выяснить, возможно ли предмет, который находился на позиции&nbsp;<i>a<sub>i</sub></i>, переставить на позицию&nbsp;<i>b<sub>i</sub></i>, используя только перестановку&nbsp;<i>p</i>&nbsp;и епрестановку&nbsp;<i>z</i>. Для достижения цели&nbsp;<i>p</i>&nbsp;и&nbsp;<i>z</i>&nbsp;можно применять в любом порядке и сколько угодно раз.</p>

<p>Например, если&nbsp;<i>n</i>&nbsp;=&nbsp;4, а&nbsp;<i>p</i>&nbsp;=&nbsp;[1, 4, 3, 2], то элемент с позиции 4 можно переставить на позицию 1 (для этого, например, можно применить&nbsp;<i>p</i>&nbsp;&mdash; после этого предмет с позиции 4 окажется на позиции 2, а затем применить&nbsp;<i>z</i>), а элемент с позиции 3 на позицию 4 переместить нельзя, поскольку и&nbsp;<i>p</i>&nbsp;и&nbsp;<i>z</i>&nbsp;оставляют его на месте.</p>

### 입력

<p>Первая строка содержит два целых числа&nbsp;<i>n</i>&nbsp;и&nbsp;<i>m</i>&nbsp;&mdash; количество элементов в перестановке и количество запросов (2&nbsp;&le;&nbsp;<i>n</i>&nbsp;&le;&nbsp;10<sup>5</sup>, 1&nbsp;&le;&nbsp;<i>m</i>&nbsp;&le;&nbsp;10<sup>5</sup>). Следующая строка содержит&nbsp;<i>n</i>&nbsp;целых чисел&nbsp;<i>p<sub>i</sub></i>&nbsp;&mdash; перестановку&nbsp;<i>p</i>&nbsp;. Каждая следующая строка содержит по два целых числа&nbsp;<i>a<sub>i</sub></i>&nbsp;и&nbsp;<i>b<sub>i</sub></i>&nbsp;(числа&nbsp;<i>a<sub>i</sub></i>&nbsp;и&nbsp;<i>b<sub>i</sub></i>&nbsp;лежат в диапазоне от 1 до&nbsp;<i>n</i>).</p>

### 출력

<p>Для каждого запроса выведите в отдельной строке &laquo;<code>Yes</code>&raquo;, если можно переставить предмет с позиции&nbsp;<i>a<sub>i</sub></i>&nbsp;на позицию&nbsp;<i>b<sub>i</sub></i>, иначе выведите &laquo;<code>No</code>&raquo;.</p>
