### 문제 설명

<p>Задача сортировки состоит в упорядочивании заданного массива чисел (или других объектов) по возрастанию или убыванию. У этой задачи существует достаточно многих вариантов, для многих из которых существуют весьма эффективные алгоритмы. Одним из важных параметров этих алгоритмов является число обменов элементов массива, необходимых для упорядочивания.</p>

<p>Далее будем рассматривать вариант сортировки, который мы будем называть&nbsp;<i>k</i>-сортировкой. В этом варианте разрешается за одну операцию (называемую&nbsp;<i>k</i>-обменом) поменять местами значения двух элементов массива с номерами, отличающимися ровно на&nbsp;<i>k</i>. Например, если исходный массив имеет вид [6, 10, 4, 1, 2], а&nbsp;<i>k</i>&nbsp;= 3, то этот массив можно упорядочить по возрастанию за две операции (после первого обмена массив будет иметь вид [1, 10, 4, 6, 2], а после второго &mdash; [1, 2, 4, 6, 10]).</p>

<p>Задан массив целых чисел&nbsp;<i>a</i><sub>1</sub>, ...,&nbsp;<i>a<sub>n</sub></i>. Ваша задача состоит в том, чтобы определить минимальное число&nbsp;<i>k</i>-обменов, необходимое для упорядочивания этого массива по&nbsp;<b>неубыванию</b>.</p>

### 입력

<p>Первая строка содержит целое число&nbsp;<i>n</i>&nbsp;(1&nbsp;&le;&nbsp;<i>n</i>&nbsp;&le;&nbsp;300). Вторая строка содержит&nbsp;<i>n</i>&nbsp;целых чисел&nbsp;<i>a</i><sub>1</sub>, ...,&nbsp;<i>a<sub>n</sub></i>&nbsp;(1&nbsp;&le;&nbsp;<i>a<sub>i</sub></i>&nbsp;&le;&nbsp;10<sup>9</sup>&nbsp;для всех&nbsp;<i>i</i>&nbsp;от 1 до&nbsp;<i>n</i>). Третья строка содержит целое число&nbsp;<i>k</i>&nbsp;(1&nbsp;&le;&nbsp;<i>k</i>&nbsp;&le;&nbsp;<i>n</i>&nbsp;- 1).</p>

### 출력

<p>Если заданный массив можно упорядочить по неубыванию с использованием операций описанного типа, то выведите минимальное число&nbsp;<i>k</i>-обменов, необходимое для сортировки. Иначе, выведите одно число -1.</p>
