### 문제 설명

<p>$N$개의 돌이 일렬로 나열 되어 있다. $N$개의 돌에는 왼쪽부터 차례대로 수 $A_{1} A_{2} ... A_{i} ... A_{N}$로 부여되어 있다. 가장 왼쪽에 있는 돌에서 출발하여 가장 오른쪽에 있는 돌로 건너가려고 한다.</p>

<ol>
	<li>항상 오른쪽으로만 이동할 수 있다.</li>
	<li>$i$번째 돌에서 $j(i &lt; j)$번째 돌로 이동할 때 $(j - i)$ &times; (1 + |$A_{i} - A_{j}$|) 만큼 힘을 쓴다.</li>
	<li>돌을 한번 건너갈 때마다 쓸 수 있는 힘은 최대 $K$이다.</li>
</ol>

<p>가장 왼쪽 돌에서 출발하여 가장 오른쪽에 있는 돌로 건너갈 수 있는 모든 경우 중 $K$의 최솟값을 구해보자.</p>

### 입력

<p>첫 번째 줄에 돌의 개수 $N$이 공백으로 구분되어 주어진다.</p>

<p>두 번째 줄에는 $N$개의 돌의 수 $A_i$가 공백으로 구분되어 주어진다.</p>

### 출력

<p>가장 왼쪽 돌에서 출발하여 가장 오른쪽에 있는 돌로 건너갈 수 있는 모든 경우 중 가능한 $K$의 최솟값을 출력한다.</p>

### 제한

<ul>
	<li>$2 \le N \le 5,000$</li>
	<li>$1 \le A_{i} \le 1,000,000$</li>
	<li>$A_i$는 정수</li>
</ul>
