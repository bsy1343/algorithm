### 문제 설명

<p>균형 삼진법은 밑이 \(3\)이고, 자릿수가 \(0\), \(1\), \(-1\)로 이루어진 기수법이다.</p>

<p>이를 이용해 별도의 부호를 사용하지 않고서도&nbsp;모든 정수를 유일한 방법으로 나타낼 수 있다.</p>

<p>십진수를 입력 받아 균형 삼진법으로 출력하는 프로그램을 작성하시오.</p>

### 입력

<p>십진법으로 나타낸 정수 \(N\)이 주어진다.</p>

### 출력

<p>문제의 정답을 출력한다. 자릿수가 \(-1\)이라면 대신 &#39;<code>T</code>&#39;를 출력한다.</p>

### 제한

<ul>
	<li>-10<sup>9</sup>&nbsp;&le; \(N\) &le; 10<sup>9</sup></li>
</ul>

### 힌트

<p>\(10 = 1\times 3^2+0\times 3^1 + 1\times 3^0\)</p>

<p>\(5 = 1\times 3^2+(-1)\times 3^1 + (-1)\times 3^0\)</p>

<p>\(-8 = (-1)\times 3^2+0\times 3^1 + 1\times 3^0\)</p>
