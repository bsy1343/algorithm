### 문제 설명

<p>You are given a number $N$ and a digit $d$. Consider all numbers which <strong>do not contain</strong> digit $d$ in their decimal notation. Find the minimum such number for which the sum of its digits is exactly $N$.</p>

### 입력

<p>The first line contains two integers $N$ and $d$ ($2 \le N \le 10^{6}$, $0 \le d \le 9$).</p>

### 출력

<p>Print one integer: the minimum number which does not contain digit $d$ and has the sum of its digits equal to $N$.</p>
