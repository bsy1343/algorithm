### 문제 설명

<p>Given a list of numbers, determine whether one of these numbers equals the product of all the other numbers.</p>

### 입력

<p>The first line of the input file contains a single integer $N$ --- the number of elements in the initial list &nbsp;($2 \leq N \leq 10^5$). The next line contains $N$ space-separated numbers constituting the list. Each of these numbers is an integer not greater than $10^9$ in absolute value.</p>

### 출력

<p>In the first line of the output file, print <code>Yes</code> or <code>No</code>, depending on whether some number from the list is equal to the product of all the other numbers --- or not. If the answer is <code>Yes</code>, in the second line, print such a number. If several numbers satisfy the conditions of the problem, print any of them.</p>
