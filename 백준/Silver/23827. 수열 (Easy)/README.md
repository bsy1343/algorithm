### 문제 설명

<p>모든 원소가 양의 정수이고, 길이가 $N$인 수열 $A_1, A_2, ..., A_N$이 주어진다. $1 \le i &lt; j \le&nbsp;N$을 만족하는 모든 정수쌍 $(i, j)$에 대해 $A_i \times A_j$의 합을 $1\, 000 \, 000 \, 007$로 나눈 나머지를 구하시오.</p>

### 입력

<p>첫째 줄에 수열 $A$의 길이 $N$이 주어진다.</p>

<p>둘째 줄에 수열 $A_1, A_2, \cdots, A_N$이 공백으로 구분되어 주어진다.</p>

### 출력

<p>$1 \le i &lt; j \le&nbsp;N$을 만족하는 모든 정수쌍 $(i, j)$에 대해 $A_i \times A_j$의 합을 $1\, 000 \, 000 \, 007$로 나눈 나머지를 출력하여라.</p>

### 제한

<ul>
	<li>$2 \le N \le 500\,000$</li>
	<li>$1 \le A_i \le 500\, 000$ ($1 \le i \le N$)</li>
</ul>
