### 문제 설명

<p>2062 대선일, 유권자들의 투표가 끝나고 결과를 확인하는 일만 남았다. 이번 대선에 출마한 경곽당 <em>N</em> +&nbsp;1 번 후보 정후는 초조하게 결과를 기다리고 있다. 이번 대선에는 정후 말고도 후보 <em>N</em> 명이 더 출마했으며 각각 1 번부터 <em>N</em> 번까지이다.&nbsp;자신이 낙선할까 불안해진 정후는 개표 도중 계속하여 질문을 한다. 개표는 다음과 같은 사건 <em>Q</em> 회로 이루어진다.</p>

<ul>
	<li>1 <em>x</em> <em>p</em>: 투표지 <em>x</em> 장이 <em>p</em> 번 후보의 표임이 집계된다. 즉, <em>p</em> 번 후보의 누적 표 수가 <em>x</em>만큼 증가한다.</li>
	<li>2 <em>x</em> <em>y</em>: 정후가 묻는다. &quot;지금까지 집계된 표 이후 저를 찍은 표가 <em>x</em> 장, 제가 아닌 후보를 찍은 표가 <em>y</em> 장 더 집계된다면 제가 당선될 가능성이 있나요?&quot;</li>
</ul>

<p>정후를 위해 정후의 질문마다 답해 주자. 단, 최다 득표한 두 후보가 서로 같은 수의 표를 얻었다면 결선 투표를 치러야 하기 때문에&nbsp;<strong>당선이 아니다.</strong></p>

### 입력

<p>첫째&nbsp;줄에 두 정수 <em>N</em>과 <em>Q</em>가 공백으로 구분되어 주어진다. 둘째 줄부터 <em>Q&nbsp;</em>+ 1째 줄까지 <em>Q</em>&nbsp;개의 줄에 걸쳐 사건을 나타내는 세&nbsp;정수가 공백으로 구분되어 주어진다.</p>

### 출력

<p>정후의 각 질문에 대한 답을 한 줄에 하나씩 출력한다. 정후가 당선될 가능성이 있다면 <code>YES</code>를, 없다면 <code>NO</code>를 출력한다.</p>

### 제한

<ul>
	<li>1 &le; <em>N</em>&nbsp;&le; 100,000</li>
	<li>1 &le; Q &le; 300,000</li>
	<li>0 &le; <i>x</i>,<i> y&nbsp;</i>&le; 10<sup>6</sup></li>
	<li>1&nbsp;&le; <i>p</i>&nbsp;&le; <em>N</em> + 1</li>
	<li>마지막 사건은 정후의 질문이다.</li>
	<li>주어지는 모든 수는 정수이다.</li>
</ul>
