### 문제 설명

<p>두람이의 MBTI는 INFP이다.&nbsp;두람이는 친구들과 옷이 겹치는 것을 정말 싫어한다.&nbsp;</p>

<p>어느 날 기분이 좋지 않은&nbsp;두람이는 친구들과 우정 여행을 가기로 했다. 여행을 가서 두람이는 친구들과 $N$ 종류의 옷을&nbsp;각자 골라 입고 다 같이 원형으로 서서&nbsp;사진을 찍기로 했다. 사진을 찍는 총인원수는&nbsp;옷의 총개수와 같다. 오지랖이 넓은 두람이는 사진을 찍는 모두가&nbsp;이웃한 두 사람과 옷이 겹치지 않으면 행복해지기로 했다.&nbsp;</p>

<p>과연 두람이는 행복할 수 있을까?</p>

### 입력

<p>첫째 줄에 두람이와 친구들이 입고 있는 옷의 종류의 수&nbsp;$N$ ($1 \le N \le 10^5$)이 주어진다.</p>

<p>둘째 줄에는 종류별&nbsp;옷&nbsp;개수&nbsp;$d_1,&nbsp; d_2,&nbsp; \cdots,&nbsp; d_N$ ($1 \le d_i&nbsp;\le 10^5$)이 공백으로 구분되어 주어진다.</p>

### 출력

<p>두람이가 행복할 수 있으면<strong>&nbsp;</strong><span style="color:#e74c3c;"><code>Happy</code></span>를 출력하고, 행복할 수 없다면&nbsp;<span style="color:#e74c3c;"><code>Unhappy</code></span>를&nbsp;출력한다.</p>

### 힌트

<p>옷의 총개수는 C++의 32비트 int 자료형의 최댓값인 2,147,483,647을 초과할 수도 있다.</p>
