### 문제 설명

<p>알파벳 소문자로 구성된 문자열 <em>X</em>, <em>Y</em>, Z가 주어진다. 각각의 문자열에는 중복된 문자가 존재하지 않는다. 문자열 <em>S</em>에 있는 문자 중 임의로 <em>k</em>개를 선택하여 문자열 <em>S</em>에서의 순서를 유지하여 만든 모든 부분 문자열을 모아 놓은 집합을 문자열 <em>S</em>에 대한 조합 C(S, k)라고 하자. 예를 들어, 문자열 S = &#39;abc&#39;에 대한 조합 C(<em>S</em>, 2) = {&#39;ab&#39;, &#39;ac&#39;, &#39;bc&#39;}이다. 입력으로 문자열 <em>X</em>, <em>Y</em>, <em>Z</em>와 정수 <em>k</em>가 주어질 때 C(X, k), C(Y, k), C(Z, k)에 두 번 이상 나타나는 부분 문자열을 오름차순으로 출력하자.</p>

### 입력

<p>첫 번째 줄에 문자열 <em>X</em>가 주어진다.</p>

<p>두 번째 줄에 문자열 <em>Y</em>가 주어진다.</p>

<p>세 번째 줄에 문자열 <em>Z</em>가 주어진다.</p>

<p>네 번째 줄에 정수 <em>k</em>가 주어진다.</p>

### 출력

<p>C(X, k), C(Y, k), C(Z, k)에 두 번 이상 나타나는 부분 문자열을 오름차순으로 출력한다. 한 줄에 하나의 부분 문자열을 출력한다. 두 번 이상 나타나는 부분 문자열이 없으면 -1을 출력한다.</p>

### 제한

<ul>
	<li>1 &le;&nbsp;문자열 <em>X</em>, <em>Y</em>, <em>Z</em>의 길이 &le; 17</li>
	<li>1 &le; <em>k</em> &le; 문자열 <em>X</em>, <em>Y</em>, <em>Z</em> 길이의 최솟값</li>
	<li>문자열 <em>X</em>에는 중복된 문자가 존재하지 않는다.</li>
	<li>문자열 <em>Y</em>에는 중복된 문자가 존재하지 않는다.</li>
	<li>문자열 <em>Z</em>에는 중복된 문자가 존재하지 않는다.</li>
</ul>
