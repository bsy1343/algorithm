### 문제 설명

<p>Two fractions are considered &ldquo;neighbors&rdquo; if the numerator of the reduced positive difference between the two fractions is 1. For example, the fractions 1/11 and 1/12 subtract to be 1/132, and are therefore &ldquo;neighbors&rdquo;. However, 2/5 and 4/5 differ by 2/5, and are not neighbors. Determine and output either the &ldquo;neighbor&rdquo; fraction value, or output &ldquo;NOT NEIGHBORS&rdquo;.</p>

### 입력

<p>The first line of input contains a single positive integer, n, indication the number of data sets. Each data set is comprised of the positive integer values for two fractions, all on one line, in the order numerator, denominator, numerator, denominator, separated by single spaces.</p>

### 출력

<p>The positive &ldquo;neighbor&rdquo; difference between the two fractions or the phrase &ldquo;NOT NEIGHBORS&rdquo;.</p>
