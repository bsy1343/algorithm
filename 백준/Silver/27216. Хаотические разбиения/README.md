### 문제 설명

<p>Рассмотрим все представления числа $n$ в виде суммы различных целых возрастающих слагаемых: $n = a_1 + a_2 + \ldots + a_k$, $a_1 &lt; a_2 &lt; \ldots &lt; a_k$.</p>

<p>Будем называть такое разбиение <em>хаотическим</em>, если для него выполнено следующее условие: для любых трех подряд идущих слагаемых среднее <em>не равно</em> среднему арифметическому крайних. Иначе говоря, для всех $i$ от 1 до $k - 2$ выполнено $a_{i+1} \ne (a_i + a_{i+2}) / 2$.</p>

<p>Задано число $n$. Выведите все его хаотические разбиения на слагаемые.</p>

### 입력

<p>На ввод подается целое число $n$ ($1 \le n \le 80$).</p>

### 출력

<p>Выведите все хаотические разбиения на слагаемые числа $n$. Разбиения можно выводить в любом порядке. Выводите слагаемые в каждом разбиении, разделяя их знаком &lt;&lt;<code>+</code>&gt;&gt; без пробелов.</p>
