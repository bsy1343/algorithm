### 문제 설명

<p>Целое число $x$ называется <em>свободным от квадратов</em>, если нет такого целого числа $y &gt; 1$, что $x$ делится на $y^2$, то есть $x = y^2z$ для некоторого целого $z$. </p>

<p>Даны числа $l$ и $r$. Требуется найти число пар целых чисел $(a, b)$, таких что $l \le a &lt; b \le r$, и числа $a$, $b$, а также их произведение $ab$ свободны от квадратов.</p>

### 입력

<p>На вход подается две строки, первая содержит целое число $l$, а вторая  --- целое число $r$ ($1 \le l &lt; r \le 10^9$, $r - l \le 1000$).</p>

### 출력

<p>Выведите одно целое число  --- искомое число пар.</p>

### 힌트

<p>В примере подходят пары $a = 3, b = 5$, $a = 5, b = 6$. Число $4$ не может входить в пару, так как $4 = 2^2\cdot 1$, а пара $a = 3, b = 6$ не подходит, так как $ab = 3\cdot 6 = 18 = 3^2\cdot 2$. </p>
