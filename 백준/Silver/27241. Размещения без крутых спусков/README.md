### 문제 설명

<p><em>Размещением</em> из $n$ по $k$ называется массив $a[1..k]$, содержащий $k$ различных натуральных чисел, каждое из которых находится в диапазоне от $1$ до $n$. </p>

<p>Пара подряд идущих элементов размещения $a[i], a[i + 1]$ называется <em>спуском</em>, если $a[i] &gt; a[i+1]$. Спуск называется <em>крутым</em>, если $a[i] &gt; a[i + 1] + 1$. </p>

<p>По заданным $n$ и $k$ требуется вывести все размещения из $n$ по $k$ без крутых спусков. Размещения необходимо упорядочить по первому числу, при равенстве первого  --- по второму, затем по третьему и так далее. </p>

### 입력

<p>Первая строка ввода содержит натуральное число $n$, вторая строка ввода содержит натуральное число $k$ ($1 \le k \le n \le 13$).</p>

### 출력

<p>Выведите все размещения из $n$ по $k$ без крутых списков, по одному на строке. Внутри размещения разделяйте числа пробелами.</p>
