### 문제 설명

<p>Числа Фибоначчи определяются следующим образом: $F_1 = 1$, $F_2 = 2$, а для $n &gt; 2$ выполнено $F_n = F_{n - 2} + F_{n - 1}$. Таким образом, начало последовательности чисел Фибоначчи выглядит так $1, 2, 3, 5, 8, 13, 21, \ldots$.</p>

<p>Вам заданы числа $n$ и $k$. Требуется найти все способы представить число $n$ в виде суммы неубывающих чисел Фибоначчи, причем кажое число разрешается использовать не более $k$ раз.</p>

### 입력

<p>Первая строка ввода содержит число $n$ ($1 \le n \le 100$).</p>

<p>Вторая строка ввода содержит число $k$ ($1 \le k \le 20$).</p>

### 출력

<p>Выведите все искомые представления, по одному на строке. Разделяйте числа знаком &lt;&lt;<code>+</code>&gt;&gt;, не используйте пробелы.</p>

<p>Разбиения следует упорядочить по первому слагаемому, при равном первом слагаемом --- по второму, при равных первых двух --- по третьему, и так далее.</p>
