# [Silver IV] Новое слово - 27256

[문제 링크](https://www.acmicpc.net/problem/27256)

### 성능 요약

시간 제한: 1 초, 메모리 제한: 1024 MB

### 통계

제출: 32, 정답: 21, 맞힌 사람: 11, 정답 비율: 61.111%

### 분류

수학, 문자열, 조합론

### 문제 설명

<p>В информатике иногда образуют новые слова, взяв начало одного слова и конец другого. Например, из слов &lt;&lt;<code>tree</code>&gt;&gt; и &lt;&lt;<code>heap</code>&gt;&gt; образовано слово &lt;&lt;<code>treap</code>&gt;&gt;.</p>

<p>Дано слово $s$ и слово $t$. Сколько различных слов можно образовать, добавив к непустому началу слова $s$ непустой конец слова $t$?</p>

### 입력

<p>Первая строка входных данных содержит слово $s$.</p>

<p>Вторая строка входных данных содержит слово $t$.</p>

<p>Каждое из слов непусто и состоит из строчных латинских букв. Длина каждого из слов не превышает $100\,000$.</p>

### 출력

<p>Выведите одно целое число --- количество различных слов, которые можно образовать, добавив к непустому началу слова $s$ непустой конец слова $t$.</p>
