### 문제 설명

<p>Недавно на кружке по математике Миша узнал про разбиения на слагаемые. Разбиением числа $n$ на слагаемые называется представление его в виде суммы неубывающего набора натуральных чисел. Например, $9=1+2+2+4$ является разбиением числа 9 на слагаемые.</p>

<p>Миша называет разбиение интересным, если никакие два слагаемых в наборе не равны и не отличаются ровно на 1. Так, например, разбиение, приведенное выше не является интересным, а разбиение $9=1+3+5$ --- является.</p>

<p>Помогите Мише вывести все интересные разбиения числа $n$ на слагаемые.</p>

### 입력

<p>На ввод подается одно целое число $n$ ($1 \le n \le 80$).</p>

### 출력

<p>Выведите все интересные разбиения числа $n$ на слагаемые. Разбиения можно выводить в любом порядке. Соблюдайте формат из примера.</p>
