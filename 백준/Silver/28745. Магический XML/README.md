### 문제 설명

<p>Развлекаясь с ранее неизведанными заклинаниями, Малефисента случайно получила свиток с посланием из будущего. На свитке было написано какое-то занимательное заклинание.</p>

<pre>
&lt;note&gt;
&lt;to&gt;&lt;/to&gt;
&lt;from&gt;&lt;/from&gt;
&lt;heading&gt;&lt;/heading&gt;
&lt;body&gt;&lt;/body&gt;
&lt;/note&gt;</pre>

<p>Малефисента сразу заметила несколько закономерностей. А именно: заклинание представляет из себя правильную скобочную последовательность, в которой открывающаяся скобка соответствует шаблону &lt;&lt;<code>&lt;S&gt;</code>&gt;&gt;, а парная ей закрывающаяся --- шаблону &lt;&lt;<code>&lt;/S&gt;</code>&gt;&gt;, где строка <code>S</code> --- непустая строка из строчных латинских букв, равная для парных скобок.</p>

<p>У Малефисенты как раз оказалось старое неработающее заклинание. Она решила проверить, можно ли в нем переставить символы так, чтобы получившееся заклинание удовлетворяло тем же свойствам, что заклинание на свитке из будущего. Помогите Малефисенте переставить символы в ее заклинании желаемым образом, либо сообщите, что это невозможно.</p>

### 입력

<p>В единственной строке дана строка $s$, состоящая из строчных латинских букв и символов &lt;&lt;<code>&lt;</code>&gt;&gt;, &lt;&lt;<code>&gt;</code>&gt;&gt; и &lt;&lt;<code>/</code>&gt;&gt; --- заклинание Малефисенты ($1 \le |s| \le 100\,000$).</p>

### 출력

<p>Если переставить символы желаемым образом невозможно, выведите &lt;&lt;<code>Impossible</code>&gt;&gt;.</p>

<p>Иначе, выведите строку, полученную из исходной перестановкой символов, которая удовлетворяет желаемым свойствам.</p>
