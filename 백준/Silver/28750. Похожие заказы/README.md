### 문제 설명

<p>Как известно, Сэм Портер является своего рода постапокалиптическим курьером. От этой работы он далеко не в восторге, поэтому для её облегчения, он тщательно изучает заказы.</p>

<p>Сегодня Сэму выдали на доставку заказ, состоящий из $n$ коробок, у каждой из которых есть есть артикул, заданный строчной буквой латинского алфавита. Сэм сложил все коробки в одну стопку в некотором порядке, а затем выписал буквы артикулов в строку от верхней коробки к нижней, получив строку $s$. Теперь ему стало интересно, можно ли преобразовать эту строку таким образом, чтобы она стала совпадать со строкой $t$, которая описывала предыдущий заказ, состоявший тоже из $n$ коробок.</p>

<p>Сэм может один раз применить к строке $s$ следующее преобразование:</p>

<ul>
	<li>Ко всем артикулам применяется шифр Цезаря. То есть все буквы сдвигаются циклически на фиксированный шаг $d$ назад. Например, при $d = 4$, буква &lt;&lt;<code>e</code>&gt;&gt; превращается в &lt;&lt;<code>a</code>&gt;&gt;, а &lt;&lt;<code>c</code>&gt;&gt; --- в &lt;&lt;<code>y</code>&gt;&gt;.</li>
	<li>Сэм берет $k$ ($0 \le k &lt; n$) верхних коробок из стопки, ставит их пол, и ставит на них оставшуюся часть стопки сверху. Это соответствует циклическому сдвигу строки $s$ на $k$ позиций.</li>
</ul>

<p>Помогите Сэму узнать есть ли такое преобразование, которое переведёт строку $s$ в строку $t$.</p>

### 입력

<p>В первой строке дано одно целое число $n$ --- количество коробок в заказе ($1 \le n \le 200\,000$).</p>

<p>Во второй строке дана строка $t$, состоящая из $n$ строчных латинских символов --- описание предыдущего заказа.</p>

<p>В третьей строке содержится строка $s$, состоящая из $n$ строчных латинских символов --- описание текущего заказа.</p>

### 출력

<p>Если не существует преобразования, которое переведёт строку $s$ в строку $t$, выведите &lt;&lt;<code>Impossible</code>&gt;&gt;.</p>

<p>Иначе, выведите &lt;&lt;<code>Success</code>&gt;&gt;, а во второй строке выведите два числа $k$ и $d$, описывающие преобразование переводящее строку $s$ в строку $t$ ($0 \le k &lt; n$, $-26 &lt; d &lt; 26$). Если различных подходящих ответов несколько, вы можете вывести любой из них.</p>
