### 문제 설명

<p>Роман Сайонис пишет письмо с угрозами для Харли Квин. Он использует простой текстовый редактор и обычную QWERTY-клавиатуру.</p>

<p style="text-align: center;"><img alt="" src="%EB%B0%B1%EC%A4%80/Silver/28760.%E2%80%85%D0%9D%D0%B0%D0%B1%D0%BE%D1%80%E2%80%85%D1%82%D0%B5%D0%BA%D1%81%D1%82%D0%B0/19a07d08.png" data-original-src="https://upload.acmicpc.net/11148633-8ee1-449a-80de-8ed4a6bc1a4f/-/preview/" onerror="this.onerror=null;this.src=this.dataset.originalSrc" style="width: 600px; height: 200px;" /></p>

<p>В тексте Роман использует строчные и заглавные буквы английского алфавита, цифры, символы &lt;&lt;,&gt;&gt;, &lt;&lt;.&gt;&gt;, &lt;&lt;!&gt;&gt;, &lt;&lt;?&gt;&gt;, &lt;&lt;\$&gt;&gt;, &lt;&lt;(&gt;&gt; и &lt;&lt;)&gt;&gt;, а также пробелы. Текстовый редактор Сайониса настолько прост, что позволяет только дописывать в конец текущего сообщения по одному символу, нажимая на соответствующие клавиши на клавиатуре. Даже если Сайонис хочет набрать несколько одинаковых символов подряд, ему придется нажимать на клавишу заново для каждого символа. Помимо клавиш, соответствующих набираемым символам, Роман использует только клавишу shift, когда это необходимо для набора очередного символа. Среди используемых им символов, shift должен быть нажат, чтобы набрать заглавные буквы и символы &lt;&lt;!&gt;&gt;, &lt;&lt;?&gt;&gt;, &lt;&lt;\$&gt;&gt;, &lt;&lt;(&gt;&gt; и &lt;&lt;)&gt;&gt;. Пробел может быть набран как с нажатой клавишей shift, так и без неё. А во время набирания всех остальных символов, клавиша shift должна быть не нажата.</p>

<p>Роман очень спешит, поэтому хочет набрать сообщение, нажав на клавиши минимальное количество раз. Для экономии времени, он может нажать клавишу shift, набрать несколько символов, и только затем отпустить её --- это будет считаться одним нажатием клавиши shift. Помогите Роману определить минимальное количество нажатий на клавиши, которое ему придется сделать.</p>

### 입력

<p>В первой строке дано одно целое число $n$ --- число символов в тексте Романа ($1 \le n \le 10\,000$).</p>

<p>Во второй строке дана строка длины $n$, состоящая из строчных и заглавных букв английского алфавита, цифр, символов &lt;&lt;,&gt;&gt;, &lt;&lt;.&gt;&gt;, &lt;&lt;!&gt;&gt;, &lt;&lt;?&gt;&gt;, &lt;&lt;\$&gt;&gt;, &lt;&lt;(&gt;&gt;, &lt;&lt;)&gt;&gt; и пробелов.</p>

### 출력

<p>Выведите одно число --- минимальное количество нажатий на клавиши, которое придется сделать Роману.</p>
