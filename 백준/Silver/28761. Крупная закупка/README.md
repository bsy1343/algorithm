### 문제 설명

<p>После окончания разборок с украденным бриллиантом, новообразованной команде &lt;&lt;Хищных птиц&gt;&gt; нужно заняться множеством организационных вопросов. Разумеется, одним из самых важных является покупка оружия, без которого у них вряд ли есть шансы на долгое существование.</p>

<p>В магазине есть $n$ различных видов оружия, $i$-й из которых обладает мощностью $a_i$. Дина Лэнс, отвечающая за оружейные запасы, делает выбор по следующим критериям:</p>

<ol>
	<li>Требуется купить ровно $m$ единиц оружия.</li>
	<li>Среди $m$ купленных единиц оружия должно быть не менее $k$ различных видов.</li>
	<li>Среди всех наборов, удовлетворяющих первым двум критериям, надо выбрать набор с максимальной суммарной мощностью.</li>
	<li>Среди всех таких наборов максимальной мощности требуется выбрать такой, в котором максимальное количество единиц оружия одного вида минимально.</li>
</ol>

<p>Помогите ей с выбором и найдите такой оптимальный набор. Если наборов с такими свойствами несколько, выведите любой подходящий.</p>

### 입력

<p>В первой строке даны три целых числа $n$, $m$ и $k$ --- количество видов оружия в магазине, необходимое число единиц оружия и минимальное число различных видов в наборе ($1 \le k \le n \le 500\,000$, $k \le m \le 500\,000$).</p>

<p>В следующей строке даны $n$ целых чисел $a_i$ --- мощность оружия $i$-го вида ($0 \le a_i \le 10^9$).</p>

### 출력

<p>Выведите $m$ целых чисел $b_i$ --- номера видов оружия, которые входят в искомый набор ($1 \le b_i \le n$). Для каждого вида, его номер должен встречаться столько раз, сколько он входит в набор.</p>
