### 문제 설명

<p>Марио решил заняться ландшафтным дизайном. Сейчас его сад можно представить как $n$ вертикальных столбиков, $i$-й из которых начинается на $y$ координате $a_i$, и уходит бесконечно вниз. За одну операцию Марио может изменить высоту любого столбика на $1$ (обратите внимание, что высоты могут становиться отрицательными). Марио хочет изменить высоты столбиков с $a_i$ на $b_i$ так, чтобы для любого $1 \le i \le n - 2$ выполнялось $b_i = b_{i + 2}$, а любые два соседних столбика отличались по высоте ровно на $k$.</p>

<p style="text-align: center;"><img alt="" src="%EB%B0%B1%EC%A4%80/Silver/28910.%E2%80%85%D0%9B%D0%B0%D0%BD%D0%B4%D1%88%D0%B0%D1%84%D1%82%D0%BD%D1%8B%D0%B9%E2%80%85%D0%B4%D0%B8%D0%B7%D0%B0%D0%B9%D0%BD/3b9ce090.png" data-original-src="https://upload.acmicpc.net/0ada320b-7f8f-4480-93a4-05715765e8be/-/preview/" onerror="this.onerror=null;this.src=this.dataset.originalSrc" style="width: 416px; height: 155px;" /></p>

<p>Помогите Марио выяснить, какое минимальное количество операций ему придется сделать.</p>

### 입력

<p>В первой строке даны два целых числа $n$ и $k$  --- количество столбиков, и высота на которую должны отличаться два соседних столбика ($2 \le n \le 10^5$, $0 \le k \le 10^9$). В следующей строке даны $n$ целых чисел $a_i$  --- исходные высоты столбиков ($-10^9 \le a_i \le 10^9$).</p>

### 출력

<p>В единственной строке выведите минимальное число операций, которое придется сделать Марио, чтобы получить желаемые высоты столбиков.</p>

### 힌트

<p>В первом тесте Марио может, например, получить следующую последовательность $b_i$: $2$, $1$, $2$, $1$, $2$.</p>
