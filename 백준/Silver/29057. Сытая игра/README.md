### 문제 설명

<p>В последнее время среди жителей Панема начала набирать популярность &lt;&lt;Сытая игра&gt;&gt;. Суть её заключается вот в чём: при старте игры объявляются четыре положительных числа $d_1$, $d_2$, $d_3$ и $d_4$. Затем игрокам называют ещё 4 положительных числа $r_1$, $r_2$, $r_3$, $r_4$. Победителем считается игрок, который первый назовёт такое минимальное число $x$, что для любого $1 \le i \le 4$ и $x \equiv r_i (\mod d_i)$. Напишите программу, которая считает $x$ как можно быстрее, чтобы победить в игре!</p>

### 입력

<p>В первой строке входного файла находятся четыре положительных числа $d_1$, $d_2$, $d_3$, $d_4$ ($1 \le d_i \le 500$).</p>

<p>Во второй строке входного файла находятся четыре положительных числа $r_1$, $r_2$, $r_3$, $r_4$ ($0 \le r_i &lt; d_i$).</p>

<p>Гарантируется, что ответ всегда существует.</p>

### 출력

<p>Выведите минимальное число $x$, которое удовлетворяет описанным требованиям.</p>
