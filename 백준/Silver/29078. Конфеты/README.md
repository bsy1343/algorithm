### 문제 설명

<p>Галантерейщик Бонасье очень любит конфеты. Однажды за ужином его жена Констанция выложила в ряд $n$ конфет. У каждой конфеты есть тип $p_i$. Констанция предложила мужу выбрать последовательность подряд идущих конфет с одним условием --- в этой последовательности должно быть ровно два различных типа конфет. Бонасье просит вас узнать, какое максимальное количество конфет он может взять, учитывая условие Констанции.</p>

### 입력

<p>В первой строке находится одно натуральное число $n$ ($n \le 10^6$) --- количество конфет. Во второй строке содержатся $n$ целых чисел $p_i$ ($1 \le p_i \le 10^9$), где $p_i$ --- тип $i$-й конфеты.</p>

### 출력

<p>Выведите максимальное количество конфет, которые может взять Бонасье.</p>

### 힌트

<p>В первом примере на столе лежит три типа конфет 1, 2 и 3. Бонасье может взять первые три конфеты с типами 3, 3, 1, а может взять последние четыре конфеты 1, 2, 2, 1. Значит, максимальное количество конфет, которое он может взять равно четырем.</p>
