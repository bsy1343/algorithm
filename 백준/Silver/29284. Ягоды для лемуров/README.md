### 문제 설명

<p>Лемуры --- милые существа, обитающие на острове Мадагаскар. Вожаком их племени является энергичный и обаятельный лемур, который сам себя называет Королем Джулианом. Одной из его королевских обязанностей является раздача ягод всем своим подданным.</p>

<p>Для того, чтобы раздать ягоды своим подданным, Король Джулиан выстраивает всех лемуров своего племени в ряд. Каждый лемур, помимо своего номера $i$ в этом ряду, характеризуется также целым неотрицательным числом $a_i$ --- степенью любви к Королю Джулиану. Количество ягод, которые получает каждый лемур, напрямую зависит от того, насколько он и его соседи по строю любят своего вожака.</p>

<p>Перед тем, как выдать ягоды лемуру номер $i$, Джулиан считает количество таких лемуров с номерами $j &lt; i$, что</p>

<p style="text-align: center;">$j + a_j &gt; i$</p>

<p>После этого Джулиан умножает получившееся число на $a_i$, и результат этого умножения является числом ягод, которые необходимо выдать лемуру с номером $i$.</p>

<p>Джулиан хочет знать, сколько всего ягод ему придется выдать своим подданным. Помогите ему вычислить это число.</p>

### 입력

<p>Первая строка входного файла содержит одно целое число $n$ --- количество поданных Короля Джулиана. Следующая строка содержит $n$ целых чисел $a_i$ ($0 \le a_i \le n$) --- степени любви лемуров к своему вожаку.</p>

### 출력

<p>В выходной файл выведите единственное число --- суммарное количество ягод, которые придется выдать Королю Джулиану.</p>

### 힌트

<p>Ответ на тест, приведенный в примере, получается по формуле</p>

<p style="text-align: center;">$0 \times 4 + 1 \times 2 + 2 \times 2 + 2 \times 4 = 14$</p>
