### 문제 설명

<p>В королевстве наступила новая эра. Уже давно ничего не менялось в армии королевства, поэтому король решил выбрать новых предводителей армии.</p>

<p>В подчинении короля есть $n$ воинов. Так сложилоcь, что у каждого из воинов есть ровно три друга, которые его хорошо понимают и поддерживают.</p>

<p>Король решил из всех своих подчиненных выбрать шесть воинов, которые образуют &lt;&lt;Великую шестерку&gt;&gt;. Воины $a_1$, $a_2$, $\ldots$, $a_6$ образуют великую шестерку если:</p>

<ul>
	<li>Можно выбрать три воина $a_i$, $a_j$ и $a_k$ ($i \neq j$, $j \neq k$, $i \neq k$), среди которых любые два дружат между собой. Мы назовем их &lt;&lt;Главная тройка&gt;&gt;</li>
	<li>Каждому из трех воинов &lt;&lt;Главной тройки&gt;&gt; можно выбрать помощника, причем у каждого из них свой личный помощник</li>
	<li>Помощник воина $a_i$ --- это такой воин $a_t$, который входит в &lt;&lt;Главную шестерку&gt;&gt;, но не входит в &lt;&lt;Главную тройку&gt;&gt; и является другом $a_i$</li>
</ul>

<p>Король задумался, сколькими способами он может выбрать &lt;&lt;Великую шестерку&gt;&gt;. Помогите ему сделать это.</p>

<p>Два способа считаются различными, если отличаются наборы людей в &lt;&lt;Великой шестерке&gt;&gt;.</p>

### 입력

<p>В первой строке входного файла задано целое число $n$ ($1 \le n \le 10^5$) --- количество воинов в королевстве.</p>

<p>В следующих $n$ строках заданы друзья каждого из воинов. В $i + 1$-й строке три различных числа: номера воинов, которые являются друзьями воина с номером $i$.</p>

<p>Гарантируется, что данные не противоречивы, и что среди друзей воина нет его самого.</p>

### 출력

<p>В выходной файл выведите одно целое число: количество способов выбрать &lt;&lt;Великую шестерку&gt;&gt;.</p>
