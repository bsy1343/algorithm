### 문제 설명

<p>В задаче &lt;&lt;Резать!&gt;&gt; базового уровня предыдущей интернет-олимпиады участникам было предложено разрезать разлинованный в квадратную клетку листок бумаги. При этом резать листок следовало только по линиям сетки, и каждый из получившихся кусочков бумаги должен был состоять ровно из $k$ квадратных клеток исходного листа.</p>

<p>В данной задаче вам предлагается решить задачу жюри, а именно определить, верно ли был разрезан листок командой-участником.</p>

### 입력

<p>Первая строка входного файла содержит два целых числа $n$ и $m$ ($1 \le n, m \le 100$) --- соответственно, высоту и ширину листа бумаги в клетках. Вторая строка входного файла содержит число $k$ ($1 \le k \le 1000$). Гарантируется, что число $n \cdot m$ делится на $k$.</p>

<p>Далее следует $n$ строк по $m$ целых чисел в каждой --- для каждой из клеток исходного листа бумаги задан номер кусочка, в который клетка была определена командой-участником. Кусочки занумерованы натуральными числами начиная с единицы.</p>

### 출력

<p>Если листок был успешно разрезан командой, выведите в выходной файл &lt;&lt;<code>OK</code>&gt;&gt;. В противном случае выведите &lt;&lt;<code>Wrong answer</code>&gt;&gt;.</p>
