### 문제 설명

<p>В теории чисел <em>простыми числами-близнецами</em> называют пару таких простых чисел $(p, q)$, что $q - p = 2$. Например, пары $(3, 5)$ и $(11, 13)$ являются парами простых чисел-близнецов. Назовем <em>обобщенными числами-близнецами</em> пару простых чисел $(p, q)$, где $q - p = k$, $k$ --- некоторое натуральное число. Например, для $k = 4$ пара $(3,7)$ является парой обобщенных чисел-близнецов.</p>

<p>Существует предположение, что пар простых чисел-близнецов бесконечно много, однако это не доказано. Безусловно, выяснить по заданному $k$, сколько пар обобщенных близнецов содержит множество всех натуральных чисел, не менее сложная задача, чем аналогичная о простых близнецах.</p>

<p>Ваша же задача несколько проще --- выяснить по заданному $k$, сколько пар обобщенных близнецов содержит множество натуральных чисел от $1$ до $n$.</p>

### 입력

<p>Во первой строке входного файла через пробел заданы два натуральных числа $n$ и $k$ ($1 \le n, k \le 10^4$).</p>

### 출력

<p>В выходной файл выведите число пар простых чисел $(p, q)$, таких, что $1 \le p &lt; q \le n$ и $q - p = k$.</p>
