# [Silver I] 세상에는 많은 유튜버가 있고, 그중에서 버츄얼 유튜버도 존재한다 - 29754

### 문제 설명

<p>세상에는 많은 유튜버가 있고, 그중에는 버츄얼 유튜버도 존재한다. 매주 $5$회, 매주 총 $60$시간 이상 라이브 방송을 하는 버츄얼 유튜버를 <strong>진짜 버츄얼 유튜버</strong>라고 한다. 형식이는 버츄얼 유튜버를 좋아하는데 그중 <strong>진짜 버츄얼 유튜버</strong>들을 찾고 싶어 한다. $N$개의 줄에 유튜버의 이름, 방송 날짜, 방송 시작 시각, 방송 종료 시각이 주어질 때 <strong>진짜 버츄얼 유튜버</strong>의 이름을 사전 순으로 출력하시오.</p>

<p>날짜는 $1$일부터 시작이고 $1$일은 일요일이다. 일주일은 일요일부터 토요일까지로 한다.</p>

### 입력

<p>첫 번째 줄에 방송의 수 $N$과 형식이가 방송을 마지막으로 본 날짜 $M$이 공백으로 구분되어 주어진다. $(1 \leq N \leq 36\,400;$ $7 \leq M \leq 364;$ $M$은 $7$의 배수$)$</p>

<p>두 번째 줄부터 $N$줄에 걸쳐 <span style="color:#e74c3c;"><code>name day hh:mm hh:mm</code></span> 형식으로 버츄얼 유튜버의 이름, 방송 날짜, 방송 시작 시각, 방송 종료 시각이 공백으로 구분되어 주어진다. $(1 \leq \text{day} \leq M)$ 시각 형식은 $24$시간제를 사용한다. $(0 \le \text{hh} \le 23;$ $0 \le \text{mm} \le 59)$</p>

<p>이름(<span style="color:#e74c3c;"><code>name</code></span>)은 길이가 최대 $20$자인 공백 없는 문자열이고, 영어 소문자로만 이루어져 있다. 버츄얼 유튜버의 이름은 서로 다르므로 이름으로 구분할 수 있으며, 최대 $100$명 존재한다.</p>

<p>방송은 한 번에 최대 $23$시간 $59$분간 진행할 수 있으며, 전날에 방송을 시작하여 자정을 넘겨 다음 날까지 진행하는 경우는 없다. 즉, 방송 시작 날짜와 방송 종료 날짜는 항상 같다. 또한, 방송은 하루에 최대 한 번 진행한다.</p>

### 출력

<p><strong>진짜 버츄얼 유튜버</strong>의 이름을 사전 순으로 한 줄에 하나씩 출력하시오. 만약 <strong>진짜 버츄얼 유튜버</strong>가 없다면 <span style="color:#e74c3c;"><code>-1</code></span>을 출력하시오.</p>

### 힌트

<p>00:01에 방송을 시작해서 01:20에 방송을 끝냈다면 1시간 19분 동안 방송을 한 것이다.</p>

<p>방송 시작 시각이 방송 종료 시각보다 항상 이르다.</p>

<p>입력 순서는 방송 날짜의 오름차순이 아닐 수 있다.</p>
