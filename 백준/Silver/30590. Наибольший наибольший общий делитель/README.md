### 문제 설명

<p><em>Все встречали задачи с ленивым условием. Эта задача не является исключением.</em></p>

<p>Найдите пару различных целых чисел с наибольшим наибольшим общим делителем среди всех пар различных $x,y$ таких, что $L \le x,y \le R$.</p>

### 입력

<p>Даны два целых числа $L$ и $R$ $(1 \le L &lt; R \le 10^{18}, R - L \le 10^6)$.</p>

### 출력

<p>Выведите пару различных целых чисел с наибольшим наибольшим общим делителем. Если таких пар несколько, выведите любую из них.</p>
