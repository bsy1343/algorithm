### 문제 설명

<p>Dado n, dizer se n pode ser obtido como a soma de 2 ou mais números inteiros positivos consecutivos.</p>

### 입력

<p>Cada linha da entrada consistirá de um único inteiro 1 &lt; n &le; 1.000.000.000. A entrada termina com n = 0, que não deve ser processado.</p>

### 출력

<p>Para cada da linha da entrada, uma linha de saída deve ser produzida, contendo a palavra &ldquo;SIM&rdquo; caso n possa ser obtido da forma descrita e &ldquo;NAO&rdquo; caso contrário.</p>
