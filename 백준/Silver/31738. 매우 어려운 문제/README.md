### 문제 설명

<p>제목과 다르게 이 문제는 아주 쉽다.</p>

<p>2 이상의 정수 $N$, $M$에 대하여, $N!$을 $M$으로 나눈 나머지를 구하여라.</p>

### 입력

<p>첫 번째 줄에 두 정수 $N$과 $M$이 공백으로 구분되어 주어진다.</p>

### 출력

<p>첫 번째 줄에 문제의 답에 해당하는 정수를 출력한다.</p>

### 제한

<ul>
	<li>$2 \leq N \leq 10^{18}$ ($N$ 제한에 주의하라)</li>
	<li>$2 \leq M \leq 10^7$</li>
	<li>$N$, $M$은 정수이다.</li>
</ul>

### 힌트

<ul>
	<li>$n!$은 $1$ 이상 $n$ 이하의 모든 정수의 곱을 의미하며 &quot;$n$ 팩토리얼&quot;이라고 읽는다. 즉, $n!=n\times(n-1)\times(n-2)\times\cdots\times2\times1$이다.</li>
	<li>이 문제의 일부 테스트 케이스는 답의 범위가 $2^{31}$을 넘어갈 수 있으므로 long long 자료형을 쓰도록 하자.</li>
</ul>
