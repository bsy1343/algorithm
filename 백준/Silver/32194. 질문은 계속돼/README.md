### 문제 설명

<p>닝닝이는 예, 아니오로 답할 수 있는 질문을 좋아한다. 닝닝이 한 첫 번째 질문의 답은 &quot;예&quot;였다.</p>

<p>이후, 2번째부터 닝닝이는 다음과 같은 형태의 질문을 계속해서 할 것이다.</p>

<ul>
	<li><code><code>1 x y</code></code> : $x$번째 질문부터 $y$번째 질문의 답이 모두 &quot;예&quot;였습니까?</li>
	<li><code><code>2 x y</code></code> : $x$번째 질문부터 $y$번째 질문의 답이 모두 &quot;아니오&quot;였습니까?</li>
</ul>

<p>닝닝이가 위 질문을 하는 시점에, 당신은 이미 닝닝이가 한 $x$번째 질문부터 $y$번째 질문에 답한 적이 있다. 닝닝이가 하는 질문에 모두 답하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에, 닝닝이가 한 첫 번째 질문을 제외한 질문의 개수 $N$이 주어진다. 즉, 닝닝이는 총 $N+1$개의 질문을 했다.</p>

<p>이후 $N$개의 줄에, 닝닝이가 한 각 질문이 차례대로 <code><code>1 x y</code></code> 또는 <code><code>2 x y</code></code>의 형태로 주어진다.</p>

### 출력

<p>$N$개의 줄에 걸쳐, 닝닝이가 한 질문의 답을 순서대로 출력한다. 각각의 답이 &quot;예&quot;라면 <code>Yes</code>를, &quot;아니오&quot;라면 <code>No</code>를 출력한다.</p>

### 제한

<ul>
	<li>주어지는 모든 수는 정수이다.</li>
	<li>$1 \le N \le 200\,000$</li>
	<li>$i$ ($2 \le i \le N+1$)번째 질문에 대해, $1 \le x \le y &lt; i$.</li>
</ul>
