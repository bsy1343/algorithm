# [Silver III] 2 Keys Keyboard - 34744

[문제 링크](https://www.acmicpc.net/problem/34744)

### 성능 요약

시간 제한: 1 초, 메모리 제한: 1024 MB

### 통계

제출: 60, 정답: 43, 맞힌 사람: 40, 정답 비율: 72.727%

### 분류

그리디 알고리즘, 소인수분해, 수학, 정수론

### 문제 설명

<p>Imagine a very simple text editor that supports exactly two operations:</p>

<ul>
<li><strong>Copy All </strong>: Copy the entire current screen content into the clipboard. (Partial copy is <strong>not </strong>allowed.)</li>
<li><strong>Paste </strong>: Paste the content from the clipboard onto the screen.</li>
</ul>

<p>Initially, the screen contains a single character A. Your goal is to display exactly $N$ characters A on the screen using the minimum number of operations possible. Find the minimum number of operations required when acting optimally.</p>

### 입력

<p>The first line contains a single integer $N$, representing the number of characters A to be displayed. $(1 \leq N \leq 1\,000\,000)$</p>

### 출력

<p>Print a single integer: the minimum number of operations required.</p>
