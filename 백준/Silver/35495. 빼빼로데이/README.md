### 문제 설명

<p>$11$월 $11$일은 빼빼로데이다.</p>

<p><code>not_strober</code>는 $11$월 $1$일도 <span style="color:#e74c3c;"><code>1</code></span>이 많이 포함되어 있으니 빼빼로데이라고 생각했다.</p>

<p>이에 따라 <code>not_strober</code>는 $2023$년 $1$월 $1$일부터 $A$년 $B$월 $C$일까지, 날짜를 YYYY-MM-DD로 표기했을 때 <span style="color:#e74c3c;"><code>1</code></span>의 개수가 $T$개 이상인 날마다 한 번씩 록금이에게 빼빼로 한 개를 요구하기로 했다.</p>

<p>$2023$년 $1$월 $1$일부터 $A$년 $B$월 $C$일까지 <code>not_strober</code>가 요구한 빼빼로의 개수를 구해보자.</p>

### 입력

<p>첫 번째 줄에 정수 $T$가 주어진다.</p>

<p>두 번째 줄에 정수 $A$, $B$, $C$가 공백으로 구분되어 주어진다.</p>

### 출력

<p>첫 번째 줄에 <code>not_strober</code>가 요구한 빼빼로의 개수를 출력한다.</p>

### 제한

<ul>
<li>$1\le T\le 5$</li>
<li>$2023\le A\le 3162$</li>
<li>$1\le B\le 12$</li>
<li>$1\le C\le 31$</li>
<li>입력으로 주어지는 날짜 $A$년 $B$월 $C$일은 달력상 존재하는 날짜임이 보장된다.</li>
<li>날짜는 <a href="https://ko.wikipedia.org/wiki/%EA%B7%B8%EB%A0%88%EA%B3%A0%EB%A6%AC%EB%A0%A5">그레고리력</a>을 따른다.</li>
</ul>

### 힌트

<p>1, 3, 5, 7, 8, 10, 12월은 31일까지, 4, 6, 9, 11월은 30일까지 있으며, 2월은 평년에는 28일, 윤년에는 29일까지 있다.</p>

<p>윤년이란, 연도가 100의 배수를 제외한 4의 배수이거나 400의 배수인 해를 의미한다.</p>
