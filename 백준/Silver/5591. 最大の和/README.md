### 문제 설명

<p>n 個の整数からなる数列 a<sub>1</sub>, a<sub>2</sub>, ..., a<sub>n</sub> と正整数 k (1 &le; k &le; n) が与えられる．この とき，連続して並ぶ k 個の整数の和　 S<sub>i</sub> = a<sub>i</sub> + a<sub>i+1</sub> + ... +a<sub>i+k&minus;1</sub> (1 &le; i &le; n&minus;k + 1) の最大値を出力するプログラムを作りなさい．</p>

### 입력

<p>1 行目には正整数 n (1 &le; n &le; 100000) と正整数 k (1 &le; k &le; n) がこの順に空白で 区切られて書かれている．2 行目以降の第 1 + i 行目 (1 &le; i &le; n) には，数列の i 番 目の項 a<sub>i</sub> (&minus;10000 &le; a<sub>i</sub> &le; 10000) が書かれている．</p>

### 출력

<p>1 行だけからなり，その 1 行は S<sub>i</sub> の最大値だけを含む．</p>
