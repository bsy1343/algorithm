### 문제 설명

<p>Mamy daną tablicę liczb całkowitych. Chcielibyśmy znaleźć w tej tablicy takie dwa elementy <em>a</em>&nbsp;i <em>b</em>, aby zmaksymalizować wartość <em>r</em> = <em>a</em> + <em>b</em> + <em>odl</em>, gdzie <em>odl</em>&nbsp;to odległość między tymi liczbami. Elementy nie muszą być r&oacute;żne, a odległość pomiędzy tym samym elementem wynosi 0.</p>

### 입력

<p>Pierwszy wiersz wejścia zawiera jedną liczbę całkowitą <em>n</em>&nbsp;(1 &le; <em>n</em> &le; 10<sup>6</sup>), oznaczającą liczbę element&oacute;w w tablicy. Drugi wiersz wejścia zawiera <em>n</em>&nbsp;liczb całkowitych <em>a</em><sub>1</sub>, <em>a</em><sub>2</sub>, ..., <em>a<sub>n</sub></em>&nbsp;(1 &le; <em>a<sub>i</sub></em> &le; 10<sup>9</sup>), gdzie&nbsp;<em>a<sub>i</sub></em>&nbsp;oznacza <em>i</em>-ty element w tablicy.</p>

### 출력

<p>Pierwszy i jedyny wiersz wyjścia powinien zawierać jedną liczbę całkowitą, r&oacute;wną maksymalnej wartości <em>r</em>.</p>
