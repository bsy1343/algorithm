### 문제 설명

<p>N이 주어졌을 때, N!/10<sup>M</sup>이 정수가 되는 M&nbsp;중 가장 큰 것을 출력하시오.</p>

### 입력

<p>각 줄에&nbsp;5 &le; N &le; 10<sup>6</sup>인 N이&nbsp;있다. 입력의 마지막에는&nbsp;0이 주어진다.</p>

### 출력

<p>각 줄에 Case #x: M의 형태로 (x는 1부터 시작한다) 가장 큰 M을 출력한다.</p>

### 힌트

<p>시파르(Sifar)는 말레이어로 0을 뜻한다고 한다.</p>
